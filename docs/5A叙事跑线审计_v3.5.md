# 5A 叙事跑线审计 v3.5

## 一句话判断

5A 当前结构是成立的，不需要重写主线。下一步要做的是先把它当成一条完整小说线跑一遍：确认每个事件为什么发生、暗线靠什么递进、玩家点下去的钩子在哪里、人物有没有走出弧光。这样再改 `playtest/5a-full-data.js`，就不会因为单句润色把线写漂。

## 核心命题

5A 不是“宿舍和好线”，而是“共同生活的规则第一次被认真提出，并在现实压力下接受考验”。

它的情绪不是一路变好，而是：

1. 先承认门后的关系已经冷掉。
2. 再把旧账从玩笑和沉默里翻出来。
3. 然后尝试用规则把四个人重新放回同一张桌子。
4. 接着让规则在饭桌、期末、深夜噪音里接受复发考验。
5. 最后得到低温修补、有限收束，或唯一外流。

## 事件力度审计

| 阶段 | 现有事件 | 问题 | 下一轮优化方向 |
|---|---|---|---|
| 混乱盘点 | 宿舍群、晚风、家里、活动消息同屏 | 事件足够，但“多线压迫感”可以更明确 | 用手机屏幕排序、未读红点、置顶动作体现“第五幕收束” |
| 门口选择 | 门缝低音、洗漱盆、长椅、晨光湖、洗衣房 | 事件丰富，钩子存在 | 强化“门把手”作为进/不进门的第一物件钩子 |
| 长椅同质回流 | 周屿买夜宵、关东煮、自动贩卖机 | 很适合生活化，但还可更有针刺感 | 让周屿的玩笑更像防御，不要太快温和 |
| 湖边异质软回流 | 晚风微信、抖音、夏知微照片 | 功能明确，但容易漂到晚风/影像线 | 每段结尾必须把玩家推回青枫居，而不是开启新线 |
| 洗衣房回流 | 陆沉洗漱盆、毛巾、水印 | 是 5A 最好的暗线物件之一 | 下一轮可让“盆的位置”反复出现，成为陆沉边界暗线 |
| 旧账浮出 | 热水卡、水费、充电器、外放、聊天记录 | 现实事件成立 | 可以加“谁先截屏、谁删除、谁装没看见”的即时动作 |
| 陆沉底线 | 私聊、水房、陌生人进宿舍、东西被挪 | 人物弧光清楚 | 下一轮避免替陆沉总结，让他用更短但更硬的话表达 |
| 规则会 | 十分钟计时、旧账点名、规则纸、摔门风险 | 主冲突成立 | 需要加一点更真实的争吵节奏：打断、冷笑、停笔、床帘响 |
| 外流线 | 许澈、便利店、陌生座位、轻松但无根 | 方向正确 | 强化“轻松是诱惑，不是奖励”，避免玩家误读成更优路线 |
| 饭桌试运行 | 西园餐厅空椅、买水、边缘人物软回流 | 世界厚度够 | 需要让四人饭桌本身更有动作，不要被边缘人物抢戏 |
| 期末复发 | 复习表、耳机漏音、规则纸、热水杯 | 第二波峰成立 | 强化“规则第一次接受现实压力”，这是 5A 的结尾钩子 |

## 暗线设计

| 暗线 | 当前承载物 | 应继续强化的表现 | 禁止写法 |
|---|---|---|---|
| 宿舍互信 | 门缝、公共桌、饭桌空椅、规则纸 | 谁先看谁、谁让位、谁没有补刀 | 直接说“互信增加/下降” |
| 回避值 | 下楼、湖边、便利店、走廊背书 | 手机锁屏、脚步转向、话打了又删 | 把回避写成纯心理独白 |
| 陆沉边界 | 洗漱盆、毛巾、水房、被子 | 少说话，但每次说话都更具体 | 林亦舟替他总结、替他控诉 |
| 周屿面子 | 玩笑、床帘、杯盖、夜宵 | 玩笑变短、手摸床帘、递东西不解释 | “周屿怕被定罪”这类直说 |
| 唐骁规则感 | 复习表、笔、便签纸、计时器 | 笔尖停住、纸空着、规则从命令变协商 | “唐骁怕规则变玩笑”这类直说 |
| 林亦舟和稀泥 | 圆场话、备忘录、未发消息 | 话写了又删，从总结别人改成承认自己 | “林亦舟意识到”式总结 |
| 晚风牵引 | 微信、抖音、游戏截图、夜聊语音 | 提供喘气口，但不替他解决宿舍 | 在 5A 里解锁晚风主线 |
| 记忆与缺席 | 夏知微照片、饭桌空椅、毕业影像 | 让缺席被看见，但不改宿舍走向 | 把夏知微写成新核心 |
| 外流诱惑 | 许澈、便利店玻璃、陌生座位 | 越轻松越空，越顺手越无根 | 把外流写成爽线或新主线 |

## 人物弧光审计

### 林亦舟

当前弧光：从想绕开冲突，到承认自己也是当事人，再到不替别人总结，最后用具体行动维护规则。

下一轮重点：
- 多写“删字、停手、压低杯声、坐回空椅子”等动作。
- 少写“他明白了、他发现了、他意识到”。
- 让他每次进步都付出一点尴尬，而不是马上成熟。

### 周屿

当前弧光：用玩笑和面子防御，强冲突时可能摸床帘、冷笑、想离开；低温修补时能用递夜宵、按紧耳机等小动作回应。

下一轮重点：
- 周屿不能太快温柔。
- 他的修补应该别扭、短句、带一点刺。
- 关键钩子可以是“他明明想走，却把床帘松开了”。

### 唐骁

当前弧光：从规则当武器，到规则变成四个人都能执行的东西。

下一轮重点：
- 他的紧绷要和期末、复习表、公共桌面绑定。
- 他不需要大段解释自己，笔尖、纸面、停顿就能说明。
- 强化“他不是不在乎关系，只是不知道不用规则还能靠什么”。

### 陆沉

当前弧光：从退让和沉默，到在水房、私聊、规则会里说出边界。

下一轮重点：
- 他的台词应短、低、具体。
- 洗漱盆、杯子、被子、门边位置要持续出现。
- 林亦舟不能代替他成为控诉者，否则人物弧光会被抢走。

| 位置 | 当前钩子 | 是否有效 | 下一轮可加强点 |
|---|---|---|---|
| 入口 | 宿舍群红点压过其他消息 | 有效 | 增加“玩家知道其他线还在，但必须先处理门后” |
| 门口 | 门把手、门缝低音、洗漱盆 | 有效 | 门把手可成为第一视觉钩子 |
| 长椅 | 周屿夜宵出现 | 有效 | 让周屿一句玩笑露出火药味 |
| 湖边 | 晚风/夏知微二选一 | 有效但易漂 | 每条湖边结尾必须出现回青枫居的物件或路 |
| 旧账 | 谁发照片、谁垫水费、谁说没事 | 有效 | 增加“记录不完整”的不安 |

| 规则会 | 十分钟计时、规则纸、摔门风险 | 强 | 可加打断和冷场，不加大事件 |

| 输出 | 熄灯前低温、有限修补 | 有效 | 输出页要像“夜里留下的痕迹”，不要像结算页 |

## 漂移风险清单

1. 晚风线漂移：晚风只能作为微信、抖音、游戏状态里的情绪镜像，不能在 5A 里变成现实化恋爱主线。
2. 影像线漂移：夏知微只能用照片照出缺席，不打开 5E 作品线。
3. 学业线漂移：沈嘉禾只映照合作沉默和复习压力，不打开 5D 老师/学业主线。
4. 资源线漂移：秦越只用活动名单映照缺位，不打开 5F 资源实践线。
5. 外流误读：许澈线不能写成奖励线，必须是“轻松但无根”的宿命外流。
6. 强冲突漂移：摔门和争吵可以发生，但只能来自宿舍生活习惯、旧账、规则、期末压力，不加入诈骗、感情被渣等其他线事件。

### P1：再改回流小小说

- 长椅周屿线：让周屿的别扭和面子防御更真实。
- 晨光湖晚风/夏知微线：保持微心态，不开新线。
- 洗衣房陆沉线：强化边界暗线。
- 食堂侧门边缘人物线：让边缘人物照见主角，不抢主线。

### P2：最后改输出和系统同步

- 输出页必须全是场景痕迹和人物动作。
- 总剧情稿、互动系统 JSON、验收表等同步应在 5A 试玩文本稳定后进行。

## 下一轮改稿原则

- 不新增大节点，除非现有节点无法承载必要事件。
- 不改变唯一外流点。
- 不减少玩家选择数量。
- 不把状态变量写进玩家可见文本。
- 每次改稿后必须跑禁词扫描和 `node playtest/5a-full-data.test.js`。
