# 5A Chinese-Novelist 小说化重排 v3.7

## 定位

5A 不是“宿舍矛盾说明会”，而是一章发生在青枫居、西园餐厅和期末前夜里的短篇小说。它的核心悬念不是“谁对谁错”，而是林亦舟能不能从门外、聊天框、旧记录和饭桌侧门绕了一圈以后，仍然把自己放回那间四人寝。

本轮遵守 `chinese-novelist` 三条规则：

1. 展示而非讲述：用门把、床帘、热水卡、生活费记录、耳机漏音、空椅子和未读消息表现心态。
2. 冲突驱动剧情：每一段都必须让关系位置发生变化，不能只是补设定。
3. 悬念承上启下：每个承重节点结尾都要把玩家推向下一次具体动作。

## 角色写法校准

| 人物 | 缺陷 | 说话/动作特征 | 5A 写法 |
|---|---|---|---|
| 林亦舟 | 怕冲突，习惯圆场，把拖延当体面 | 删输入框、扣手机、站在门把前停顿 | 不能再替所有人总结，只能先承认自己那一块 |
| 周屿 | 面子重，玩笑过界，越尴尬越想开玩笑 | 拧杯盖、拉床帘、嘴上顶一句又停住 | 让他有可爱处，也要让玩笑造成后果 |
| 唐骁 | 规则感强，容易把保护写成控制 | 按平纸角、停笔、把清单往前推 | 规则不能像审判，要慢慢变成共处工具 |
| 陆沉 | 长期忍耐，害怕麻烦别人 | 鞋尖朝外、把盆往里收、床帘拉到一半 | 他不能被代表发声，必须留出自己的沉默和一句话 |
| 晚风 | 能接住情绪，但不能替现实做决定 | 微信语音条、抖音分享、游戏截图 | 在 5A 里把林亦舟推回现实，不开 5C |
| 夏知微 | 记录敏锐，容易拍到别人不想承认的空位 | 扣镜头盖、删照片、压相机背带 | 让她照见缺席，而不是讲道理 |
| 沈嘉禾 | 稳定但锋利，对低效率不耐烦 | 压资料边角、递一页重点 | 用合作经验映照宿舍，不开学业线 |
| 秦越 | 资源顺手，边界含糊 | 翻活动名单、递空白挂牌 | 用“缺位能不能顶上”映照饭桌空位 |
| 许澈 | 新鲜、轻快、无根 | 活动消息、便利店玻璃、顺手邀请 | 只在 5X 出现，制造短期轻松和长期空感 |

## 节点改写要求

| 节点 | 需要补的小说信息 | 禁止 |
|---|---|---|
| `ACT5_FULL_00_AUDIT` | 写清父母生活费/红包、晚风微信和抖音、共享文档、活动证明、宿舍群同时亮起 | 不能像系统盘点 |
| `ACT5_FULL_01_ROUTER` | 让“先处理宿舍”不是正确答案，而是今晚最躲不开的门 | 不能说其它线不重要 |
| `ACT5_5A05_RECORD_SCROLL` | 补手机袋、水费、热水卡、外放、空调、陆沉说没事；让旧账具体 | 不能只写“证据” |
| `ACT5_5A15_DOOR_SLAM_RISK` | 增加同质收束入口；同一场冲突根据互信与回避走向不同 | 不能预告两种方向 |
| `ACT5_5A16_CANTEEN_SEAT` | 饭桌测试：筷子、排队、空椅子、手机弹窗、侧门绕路 | 不能把饭桌写成自动和好 |
| `ACT5_5A20_RELAPSE_CHOICE` | 写第六幕入口：普通稳定、修补回升、低完成风险都来自复发后的处理 | 不能直接写圆满 |
| `ACT5_5X01_STAIRWELL_MUTE` | 外流不是爽逃，是静音旧关系后短暂轻松 | 不能出现“剧情安排”这种元叙事 |
| `ACT5_5X99_OUTFLOW_OUTPUT` | 写第六幕与终章只走外流单线，许澈关系轻快但无根 | 不能回 5A-5F |

## 5A v3.7 结尾钩子

正常修补输出的钩子：规则纸被夹进毕业活动文件夹，不是为了提前写毕业，而是告诉玩家第六幕还会把这张纸翻出来。

低温收束输出的钩子：拖鞋摆正、门留缝，说明关系没有恢复原样，但生活暂时没有继续下滑。

5X 外流输出的钩子：许澈的消息留在未读里，宿舍群和晚风都在列表里变远；第六幕与终章不会把玩家送回旧线，只会让旧关系从过场里经过。
