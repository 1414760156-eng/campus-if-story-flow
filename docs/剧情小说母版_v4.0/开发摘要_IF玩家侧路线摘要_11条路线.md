# IF 玩家侧路线摘要（11 条路线）

版本：v1
日期：2026-05-23
用途：给后续 UI / 试玩器提供玩家可读的路线名、入口提示、玩法感受、路线代价和毕业温度。本文只描述剧情侧摘要，不处理 H5。

## 使用口径

- 11 条路线都已接成完整结构：入口 / Act4-5 桥接或第五幕锁池 / Act6 定形 / Act7 毕业结算 / Ending。
- 玩家侧不要把路线写成“谁对谁错”或“奖励结局”。每条线都是林亦舟把生活重心投向某个关系、场域或逃离方向后的代价结算。
- 第七幕只保留双选压力；第四幕和第五幕仍按母版规格保留更丰富的选择铺垫、人物对话、反馈和现实代价。
- 路线锁定后不再打开其它完整路线，其它人物只作为软回声、现实压力或未被选择的后果出现。

| 分组 | 路线 | 入口 |
|---|---|---|
| 早期外流 | A3-ACTIVITY-PUBLIC / R3-PERFECT | 第三幕 P0-A，经第四幕生活半径和第五幕桥接 |
| 生活半径 | R4-WORK | 第四幕 P0-B，经第五幕工作线桥接 |

| 高难硬外流 | R5X-HARD | 第五幕 P0-E 硬锁 |

## 路线摘要

### A3-ACTIVITY-PUBLIC：活动父池

玩家侧标题：把名字写到公开事务里
一句话：林亦舟把重心投向活动、公开材料和可被看见的信用，但必须承担宿舍缺席、署名边界和候选来源被核验的压力。

- 入口：第三幕 P0-A 社团 / 活动父池；之后继续经历 Act4 活动生活半径和 Act5 当前池桥接。
- 玩家在做什么：处理招新、归档、公开材料、活动授权，并在第四幕活动生活半径和第五幕桥接里处理宿舍、亲密、项目与候选对象来源。
- 主要代价：公开信用越强，私人解释和宿舍在场越容易变少；活动不是万能出口。
- 毕业温度：可能成为活动圈低频确认对象，也可能留下压力扩散、宿舍回归、低可见或完美线种子。
- 不要写成：单纯社团成功线、恋爱候选入口合集、或跳过宿舍旧账的捷径。

### R3-PERFECT：完美子外流

玩家侧标题：把所有事情处理清楚
一句话：林亦舟选择低风险、清楚、体面和公开可信，但“做得对”会把他推向公共工具化和私人延迟。

- 入口：第三幕 P0-A 活动父池中的完美子外流；之后经历 Act4 低风险生活半径和 Act5 工具化 / 私人延迟桥接。
- 玩家在做什么：控制公开范围、补齐材料、定义职责边界，并在第五幕继续面对工具化压力和停止默认兜底的问题。
- 主要代价：材料齐全不等于关系被说开；越能兜底，越容易被继续当成可用工具。
- 毕业温度：公开清楚、体面低温、迟来的私人话、或安静边界。
- 不要写成：正确答案线、奖状线、无代价最优路线。

### R4-WORK：工作线

玩家侧标题：先把生活费和排班处理完
一句话：林亦舟把生活半径压进兼职、工时、钱账、证明和留校安排里，关系温度被现实时间切碎。

- 入口：第四幕 P0-B 暑假 / 兼职入口；第五幕必须经过 ACT5-WORK 桥接后进入 Act6。
- 玩家在做什么：处理排班、工资、证明、请假、宿舍最低事实，并在第五幕写清排班停止条件和毕业第一周现实安排。
- 主要代价：独立和稳定增加，但错过照面、饭局、宿舍解释和亲密推进的成本上升。
- 毕业温度：独立离校、体面淡化、错过最后一顿饭、家庭压力部分说开、低频真实联系。
- 不要写成：打工赚钱爽线，或第四幕直接跳第六幕的捷径。

### R5-WANFENG：晚风线

玩家侧标题：把关系说成两个人的事
一句话：林亦舟选择靠近晚风，把关系从语音、见面、礼物和报平安里说清，但必须解释 4XX 缺席和公开边界。

- 入口：第五幕 P0-C 主轴唯一直连亲密方向。
- 玩家在做什么：安排见面、保留私人入口、处理晚风生活圈、解释两边优先级和联系频率。
- 主要代价：甜不是免费；晚风关系越稳，宿舍毕业温度越可能降低。
- 毕业温度：继续联系、私人入口、体面分别、差一点、甜但有缺席成本。
- 不要写成：无条件恋爱胜利线，或用晚风替代宿舍收束。

### R5-ROMANCE：专注感情线

玩家侧标题：选择一个现实候选对象
一句话：林亦舟从 A3 女生候选来源进入亲密方向，关系必须经得起来源、公开边界、对方任务和现实安排的核验。

- 入口：第五幕 P0-C；必须来自 A3 中有连续事件压力的候选对象来源。
- 玩家在做什么：确认候选对象、处理协作外壳、署名授权、相册 / 文档入口和第一周现实安排。
- 主要代价：候选关系保住不等于宿舍不受影响；含混、否认、失约都会进入毕业结算。
- 毕业温度：沈嘉禾继续联系、夏知微继续联系、私人入口、体面分别、错过、甜但宿舍冷。
- 不要写成：默认主轴可直接跳其它女主，或多候选摇摆线。

### DEFAULT-4XX：宿舍修补线

玩家侧标题：把问题带回公共桌
一句话：林亦舟不私聊某个人、不逃开，也不替谁下结论，只是把旧账重新带回 4XX 的公共桌。

- 入口：第五幕 P0-D 宿舍站队共用入口。
- 玩家在做什么：当面说、补最低事实、处理公共桌、饭局、C407 事项和毕业前最后一句。
- 主要代价：修补只增加处理机会，不抹平裂痕；旧账必须本人确认，关系可能不亲近但还在。
- 毕业温度：说清一点、人在但不亲近、礼貌距离、不和解、低频联系。
- 不要写成：完美和解线或宿舍大团圆线。

### R5-ZHOU：周屿线

玩家侧标题：接住热闹背后的面子债
一句话：林亦舟站到周屿一侧，帮他处理热闹、公开场面和活动人情，但要承担宿舍误读与面子债。

- 入口：第五幕 P0-D，选择站向周屿。
- 玩家在做什么：陪跑活动、判断何时替他收场、确认授权、解释唐骁误读和活动进度。
- 主要代价：热闹带来人情，也带来公开误读、宿舍缺席和未被选择者的低温回声。
- 毕业温度：低频真实联系、半句道歉、体面距离、活动成本、误读回响。
- 不要写成：单纯帮朋友圆场线。

### R5-TANG：唐骁线

玩家侧标题：站在流程和边界旁边
一句话：林亦舟靠近唐骁，得到稳定、清晰和项目推进，也要面对规则压迫、情绪延后和低频关系。

- 入口：第五幕 P0-D，选择站向唐骁。
- 玩家在做什么：处理流程、代填权限、项目截止、当面边界、非功能性私聊入口。
- 主要代价：流程越稳，私人话越容易被推迟；关系会被流程化，周屿误读和冷边界仍会存在。
- 毕业温度：低频协作、非功能性私聊、体面断开、流程压力、误读成本。
- 不要写成：唐骁永远正确或流程万能线。

### R5-LUCHEN：陆沉线

玩家侧标题：等他说，而不是替他说
一句话：林亦舟靠近陆沉，核心不是拯救或代言，而是给现实压力留位置，等本人开口。

- 入口：第五幕 P0-D，选择站向陆沉。
- 玩家在做什么：处理排班、旧书、现实任务、最低公开范围、追问边界和不替他说完。
- 主要代价：越想替他说清，越容易把关系推向误读；理解不能写成救赎。
- 毕业温度：低频联系、迟来的半句话、礼貌空白、代言风险、现实压力被看见。
- 不要写成：拯救陆沉线或替他表达线。

### R5-LIEFLAT：摆烂线

玩家侧标题：什么都不选也会变成一种选择
一句话：林亦舟选择静音、短逃和最低存在感，换来短期安静，也让许多入口过期。

- 入口：第五幕 P0-D，选择不明确站队 / 不回复 / 回避。
- 玩家在做什么：静音、旁观、短逃、只做最低任务、看见入口如何过期。
- 主要代价：没人再追问，也没人再等；低存在感和空白毕业会成为真实结果。
- 毕业温度：迟来的最低消息、空白毕业、低存在感、错过已成事实、5X 远影。
- 不要写成：轻松逃课线或恶搞线。

### R5X-HARD：唯一硬外流线

玩家侧标题：离开旧线，进入一张新的桌子
一句话：林亦舟跟随许澈进入新圈，旧关系不再能回到原来的多线结构，新圈也不保证留下。

- 入口：第五幕 P0-E 隐藏高难硬锁；不是单次跟随许澈，而是在最后门口连续选择外走、坐到新桌并接受旧线不可回。
- 玩家在做什么：进入新圈、看见薄关系、处理旧人同场短句、确认许澈边界、删除置顶或发送旧消息。
- 主要代价：旧人远了，新圈也薄；许澈不是答案，只是一个不用解释的临时桌面。
- 毕业温度：清醒离开、薄圈散场、许澈边界、旧消息回应、删除置顶、站口空白。
- 不要写成：隐藏真结局、许澈拯救线或更高级路线。

## 后续接 UI 建议

- 路线列表先按入口阶段排序，再允许 UI 用分组筛选。
- 玩家首屏显示：玩家侧标题、一句话、入口阶段、主要代价、毕业温度。
- 详情页再显示：玩家在做什么、Act6 定形关键词、结局温度候选、不要写成。
- 试玩器读取 JSON 时，不应把 `ending_samples` 当成可直接达成条件展示给玩家；可展示为“可能毕业温度”。
