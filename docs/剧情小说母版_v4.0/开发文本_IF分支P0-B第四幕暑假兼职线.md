# IF 分支 P0-B 第四幕暑假 / 兼职线开发稿

## 路线定位

路线编号：`R4-WORK`

路线名称：兼职线 / 生存线

入口时期：第四幕，暑假、生活费波动、家庭暗线、留校和勤工选择集中出现时。

路线核心：林亦舟更早把生活重心投向钱、排班、节省、工时和现实压力。他不是突然变成“穷学生标签”，而是在暑假这个校园喘气期里第一次意识到：有些问题不是聊清楚就能解决，得有人去排班、取件、打印、算钱、错过饭局。

核心代价：时间被切碎。宿舍当面机会减少，4XX 对他的缺席更容易产生恶化解释；晚风、活动圈和宿舍修补都可能被“我今晚有班”“我得先去取件”“我先把这个钱补上”挤开。

毕业倾向：林亦舟更能独自处理离校、材料、行李和现实手续，但很多关系会停在“不是不在意，只是那时候真的没空解释”。

## 主线锁点

- 第四幕暑假仍是学校剧情的喘气期，不取消家庭暗线、返校压力和宿舍关系降温。
- 生活费变化、父母转账 / 忘记转账 / 生活用品减少等暗线仍继续推进。
- 陆沉仍有勤工、家庭和现实压力，但玩家进入兼职线不等于自动进入陆沉线。
- 第五卷宿舍站队爆发仍发生，林亦舟只是更常因为排班、兼职或钱的问题不在第一现场。
- 第六卷关系定形和第七卷毕业流程仍发生，兼职线只改变林亦舟的生活半径和结算温度。

## 入口条件

建议变量门槛：

- `money_pressure >= 3`
- `work_shift >= 2`
- `family_signal >= 2`
- `dorm_absence >= 1`

推荐触发方式：

- 玩家在第四幕多次选择留校、勤工、节省、兼职、帮人取件或现实优先。
- 玩家在家庭生活费节点不主动追问父母，而是先用自己的方式补窟窿。
- 玩家因为排班或赚钱错过 4XX 聚餐、活动复盘或晚风见面。

互斥提示：

- 如果玩家把暑假主要用于回寝室、和室友线下修补、处理公共桌旧账，会回到 `DEFAULT-4XX`。
- 如果玩家把暑假主要投向晚风现实见面和亲密推进，会转入亲密外部方向前置。
- 如果玩家既不赚钱也不修补，只用游戏、外卖、刷手机拖过暑假，会进入摆烂倾向。

## 核心事件链

### 事件一：生活费没有准时到账

地点：青枫居 4XX、校园快递站、便利店、食堂二楼。

参与人物：林亦舟、母亲、父亲、陆沉、宿管阿姨。

现实触发：月初生活费没有像之前一样准时到账。母亲没有像开学时那样买生活用品，父亲红包延后，林亦舟手机账单里同时出现饭卡充值、快递代取、打印费和一次晚风约见路费。

可拆选择：

- 直接问父母是不是忘了转账。
  - 结果：`family_signal +1`，`money_pressure -1`，但家庭暗线提前浮出。
  - 回响：母亲会先解释“这两天忙”，林亦舟能听出语气不一样。
- 不问家里，先压缩支出和饭卡充值。
  - 结果：`money_pressure +1`，`self_control +1`，`dorm_absence +1`。
  - 回响：4XX 聚餐时他会说“你们去”，别人可能以为他冷。
- 找陆沉问勤工岗位怎么排。
  - 结果：`work_route +1`，`luchen_reality_link +1`。
  - 回响：陆沉不会安慰，只会告诉他几点去、要身份证复印件、别迟到。
- 接一个临时代取快递 / 打印跑腿。
  - 结果：`work_shift +1`，`money_pressure -1`，`time_debt +1`。
  - 回响：宿管阿姨或快递站工作人员开始认得他，但他错过宿舍当面聊天。

对话逻辑：

- 母亲不直接说家里出事，而是用“等两天”“先别乱花”这类话露出变化。
- 陆沉不把赚钱讲成励志，只讲流程和注意事项。
- 宿管阿姨会提醒“代取要本人同意”，把规则压力落到具体动作上。

### 事件二：快递站排班

地点：校园快递站、青枫居楼下外卖架、宿管窗口。

参与人物：林亦舟、陆沉、快递站工作人员、周屿、唐骁。

现实触发：暑假留校人少，快递站临时缺人。陆沉提到可以补半天班，但要站满四小时，不能中途走。当天 4XX 原本约好去东区商业街吃饭，周屿还在群里发了店名。

可拆选择：

- 接快递站半天班，放弃饭局。
  - 结果：`work_shift +2`，`money_pressure -1`，`dorm_absence +1`。
  - 回响：周屿会说“行，你忙”，但下次不一定再叫他。
- 只接前两个小时，饭局迟到。
  - 结果：`work_shift +1`，`dorm_warmth 0`，`time_debt +1`。
  - 回响：唐骁会指出排班不能随便切，周屿会觉得他来了也不在状态。
- 拒绝排班，去饭局。
  - 结果：`dorm_warmth +1`，`money_pressure +1`，兼职线入口变弱。
  - 回响：林亦舟回寝后仍要面对账单，现实压力没有消失。
- 让陆沉帮忙问能不能换晚班。
  - 结果：`luchen_reality_link +1`，`work_shift +1`，`favor_debt +1`。
  - 回响：陆沉替他问了，但不会替他解释为什么缺席饭局。

对话逻辑：

- 快递站工作人员只认排班，不关心他的饭局。
- 周屿嘴上轻松，真正的不舒服藏在下一次“算了没叫你”里。
- 唐骁会认为“接了就要做完”，这为第五卷唐骁线或规则误读埋钩子。

### 事件三：打印店和证明材料

地点：打印店、明德楼材料窗口、教学楼走廊。

参与人物：林亦舟、沈嘉禾、打印店老板、唐骁、许棠。

现实触发：勤工申请、家庭情况说明、课程补交材料和活动证明同时需要打印。打印店排队，老板问他是不是又要打身份证复印件和表格。沈嘉禾正好来打印普通话报名材料，唐骁在群里催课程材料扫描版。

可拆选择：

- 先打印勤工和家庭相关材料。
  - 结果：`work_route +1`，`family_signal +1`，`public_credit 0`。
  - 回响：林亦舟把现实压力藏进文件袋，宿舍只看到他又没回消息。
- 先帮唐骁扫课程材料。
  - 结果：`tang_trust +1`，`work_shift 0`，`time_debt +1`。
  - 回响：项目稳定，但勤工窗口可能错过上午号。
- 请沈嘉禾帮忙看格式，但不让她替填家庭说明。
  - 结果：`shen_jiahe_link +1`，`boundary +1`。
  - 回响：沈嘉禾会尊重边界，也能成为后续现实协作或专注感情候选的支撑。
- 顺手接许棠的活动材料打印跑腿。
  - 结果：`small_income +1`，`public_boundary_risk +1`。
  - 回响：许棠会记住他可靠，但公开材料线可能牵出授权问题。

对话逻辑：

- 打印店老板可以有熟人感，但不需要姓名，重点是“又打这个啊”的现实重复。
- 沈嘉禾只提醒“家庭说明别让别人替你写”，不打听细节。
- 唐骁会把扫描文件命名规范发来，语气像工作，不像安慰。

### 事件四：错过晚风或活动圈

地点：地铁站口、光谷附近、东区商业街小舞台、快递站后门。

参与人物：林亦舟、晚风、秦越、周屿、程小满。

现实触发：晚风原本约林亦舟线下吃一碗面或走一段路，秦越也临时问他能不能帮活动收尾。快递站晚班突然延长，林亦舟必须选择先处理哪边。

可拆选择：

- 先完成晚班，再给晚风解释。
  - 结果：`work_shift +1`，`wanfeng_misread +1`，`money_pressure -1`。
  - 回响：晚风不会立刻吵，但会问“你是不是每次都先把自己排到最后”。
- 请同事帮十分钟，赶去见晚风。
  - 结果：`intimacy_link +1`，`favor_debt +1`，`work_reliability -1`。
  - 回响：亲密线有甜，但兼职线信任下降。
- 去帮秦越活动收尾，放弃晚风见面。
  - 结果：`activity_link +1`，`wanfeng_distance +1`，`work_shift 0`。
  - 回响：完美 / 活动线回升，兼职线不锁定。
- 明说自己今晚哪边都去不了，只把钱和时间说清。
  - 结果：`boundary +1`，`money_pressure +1`，`relationship_truth +1`。
  - 回响：可能失去一次甜，但减少误读。

对话逻辑：

- 晚风不能只温柔接住，她可以不催，但要让林亦舟知道“我也会等累”。
- 秦越不该变成抢人角色，她只是按活动流程找能帮忙的人。
- 程小满可以从晚风身边补一句现实旁观：“他要是不来，你别替他圆太多。”

### 事件五：兼职线确认节点

地点：快递站、青枫居 4XX、食堂二楼、星辰书屋。

参与人物：林亦舟、陆沉、唐骁、周屿、宿管阿姨。

现实触发：暑假结束前，林亦舟需要确认是否继续做固定勤工 / 兼职。4XX 群里同时出现返校后课程安排、宿舍清理、班会通知和一次聚餐投票。

确认条件：

- `money_pressure >= 3`
- `work_shift >= 3`
- `dorm_absence >= 2`
- 玩家选择继续用现实行动解决压力，而不是回到宿舍主轴当面修补。

可拆选择：

- 继续固定排班。
  - 结果：锁定 `R4-WORK`。
  - 回响：第六卷承接更多勤工、工时、钱和错过；陆沉现实线出现频率上升。
- 暂停兼职，回 4XX 修补关系。
  - 结果：回到 `DEFAULT-4XX`。
  - 回响：生活费压力不会消失，但宿舍当面机会恢复。
- 保留临时兼职，不绑定固定班。
  - 结果：兼职线半锁，后续可转学业 / 感情 / 宿舍。
  - 回响：玩家仍能在第五卷前段转向其他生活重心。
- 把排班和饭局都交给别人安排，自己只低限度参与。
  - 结果：`avoidance +1`，`lieflat_score +1`，兼职线不稳定。
  - 回响：容易滑向摆烂线，钱和关系都没真正解决。

对话逻辑：

- 陆沉会说“固定班不是轻松，是少变数”，不是鼓励。
- 周屿会半开玩笑说“你现在比我还难约”，但后续真的少叫他。
- 唐骁会把他的缺席写进后续排期，不吵。
- 宿管阿姨只看他晚归登记和钥匙，不理解他们之间的旧账。

## 变量建议

| 变量 | 用途 | 增长来源 | 过高后果 |
|---|---|---|---|
| `money_pressure` | 生活费和现实压力 | 生活费延迟、家庭用品减少、饭卡余额低 | 推动兼职线锁定，也压缩亲密和宿舍时间 |
| `work_shift` | 勤工 / 兼职投入 | 快递站、打印跑腿、临时排班 | 宿舍缺席增加 |
| `family_signal` | 家庭暗线显性程度 | 父母转账异常、母亲不再买用品、家庭说明材料 | 后续家庭线更早浮出 |
| `dorm_absence` | 宿舍缺席度 | 错过饭局、错过当面说明、晚归 | 第五卷误读加重 |
| `luchen_reality_link` | 与陆沉现实压力共鸣 | 问勤工、一起排班、食堂二楼短谈 | 可转陆沉线，但不能自动拯救 |
| `time_debt` | 时间欠账 | 迟到、改班、延迟回复 | 晚风、活动、宿舍都会产生误读 |
| `work_reliability` | 兼职可信度 | 按时到岗、完成排班 | 过低会关闭稳定兼职资源 |

## 后续承接

### 第五幕

兼职线进入第五幕时，林亦舟不一定更冷，但更常不在场。宿舍冲突爆发时，他可能刚下班、刚取完快递、刚在打印店排完队。别人会把“不在”解释成“不想管”，而他也会因为累、缺钱、怕解释太长而选择沉默。

第五卷可重点变化：

- 4XX 事项群里他回复更短。
- 周屿更少默认叫他吃饭。
- 唐骁把他排进表格时会加一句“你有班就提前说”。
- 陆沉会更能理解他，但不替他说。

### 第六幕

第六卷承接为现实成本和生活半径：

- 林亦舟更熟快递站、打印店、食堂二楼、星辰书屋和宿管窗口。
- 项目协作中，他更重视排期和成本，不愿无偿兜所有人的临时变化。
- 晚风若靠近，会面对他“不是不想见，是排班真的切不开”的现实。
- 陆沉线出现频率增加，但仍需玩家后续选择才能进入陆沉深线。

### 第七幕

第七卷毕业结算偏向：

- 离校手续、行李、校园卡、档案和快递处理能力更强。
- 旧关系可能因为长期错过而淡化。
- 有些人会承认他那时不容易，有些人只记得他一直不在。
- 若 `dorm_absence` 和 `time_debt` 长期过高，结局容易出现“他终于能独自处理很多事，却很难再把人叫回同一张桌子”。

## 开发备注

兼职线不是苦情线，也不是励志线。它的可玩性来自现实挤压：玩家每次看似只是少吃一顿、接半天班、晚回一条消息，最后都会改变林亦舟在宿舍、亲密关系和毕业流程里的位置。
