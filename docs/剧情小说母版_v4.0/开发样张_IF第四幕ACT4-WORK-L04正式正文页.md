# IF 第四幕 ACT4-WORK-L04 正式正文页样张

更新时间：2026-05-24

## 用途

本文是 `R4-WORK` 第四幕 `R4-ACT4-L04 补考与打印店` 的正式正文页样张，用来验证“母版锚点扩写”规则。

本样张不是新的独立剧情线，也不写回正式 JSON。它用于确认后续 `ACT4-WORK` 正式页应该如何从表格细稿扩成有沉浸感的剧情页：母版公共事件不变，工作线镜头改变；允许基于母版地点、人物关系、现实流程和物件扩写小剧情，但不能改变母版因果或打开新路线。

## 母版锚点

- 母本章节：第四卷第四章《低空飞过》、第五章《不要卡点》。
- 母版事件：数据结构 59 分、补考安排、资料包风险、唐骁发资料、周屿误以为可以替人踩雷、陆沉图书馆 / C201 提醒、晚风强调白天处理、母亲成绩追问、打印店和 C201 路线。
- R4 镜头：补考不是单独学业事件，而是和饭卡余额、打印费、勤工窗口、父母钱压、晚风低频报平安共同挤压林亦舟的白天。
- 当前池：`POOL-R4-WORK`。
- 禁止：不得把补考写成逃离工作线，不得把晚风、唐骁、陆沉或周屿扩成新路线入口。

## 正式正文页样张（P2 修正版为准）

### P1 缩写修订口径

本样张的目标不是把母版整段搬进 IF 页，也不是压成事件摘要。正式写回 JSON 时，应按“剧情基础上缩写”处理：

- 保留关键剧情颗粒：成绩 59、父亲信封、资料包风险、唐骁给目录、周屿“替人踩雷”误区、陆沉给流程、晚风强调白天处理、打印店和 C201。
- 非关键对白改为第三人称转述：寒暄、玩笑来回、重复确认和流程问答不逐句铺开。
- 只保留会改变边界的对白：例如“不要替我”“别卡点”“白天能排队”这类会影响玩家选择和变量理解的句子。
- 页数不骤减：后续标准版仍保留选择前连续场景、选择窗和三方向反馈，只压单页表达密度，不压成问答式节点。

### P2 修正版标准

P2 修正版是后续写回 JSON 的有效样张。后面的 P0 长样张原稿只作为素材池保留，不作为正式页长和对白密度标准。

P2 标准：

- 选择前保留 4 个连续剧情页：成绩页、资料包、图书馆、打印店 / C201。
- 每页只抓一个现场重心：一个地点、一个动作压力、一个关系回声。
- 每页先交代前因后果：发生了什么、林亦舟做了什么、最关键的人怎么反应、这个反应把下一页推向哪里。
- 对话只保留关键边界句；其余用第三人称转述。
- 一页最多保留 1 个主反应人物；其它人物延后到对应主题页，或只用一句背景带过。
- 选择后每个方向保留 2 个反馈页：一个现场行动页，一个承接回声页。
- 元信息只写在“字段备注”里，不混入玩家正文。

## P2 标准修正版

### `ACT4-WORK-L04-P2-S01` 青枫居 4XX / 成绩页

凌晨前，教务系统终于刷进去。

成绩表一格一格跳出来，白底黑字，像把这个暑假前欠下的账摊到了屏幕上。林亦舟那一行卡在“数据结构入门”后面，灰色小圈转了几次，页面往下一沉。

数据结构入门：59。

旁边没有红字，没有弹窗，只有一行安静的“待补考安排”。公共桌上，父亲给的信封还压在课本下面，旁边放着陆沉夹好的食堂小票、两瓶常温水和一支快没水的黑笔。

陆沉问：“出了？”

林亦舟盯着屏幕：“五十九。”

陆沉没有说“差一分而已”。他只是把自己的手机转过来一点，上面写着：数据结构入门，64。

“也不好看。”他说，“但能过。”

母亲的消息在这时弹出来：成绩查了吗？
晚风也发来一张游戏截图，角色站在出生点没动。

林亦舟看着这些聊天框。补考忽然不再只是一门课，它会占掉白天，会要打印费，会让他解释为什么不回家、为什么不见面、为什么还要去问勤工窗口。

他给母亲回：有一门没过，数据结构入门，补考安排下周出。

很久以后，母亲只回了一句：先睡，明天再说。

陆沉把勤工表往自己那边收了收：“明早我去图书馆。你要来的话，二楼靠窗那排空。”

林亦舟点了点头。点完才发现，明天已经被切开了：成绩、母亲、晚风、唐骁的资料、陆沉的图书馆，还有不知道能不能接上的勤工窗口。

### `ACT4-WORK-L04-P2-S02` 青枫居楼下外卖架 / 资料包

第二天中午，青枫居楼下的外卖架比前几天满。林亦舟拎起自己的两瓶水，旁边有人对着手机念：“补考资料包二十九块九，真的假的？”

年级群里也跳出相似链接，标题写得很满：往年题、老师重点、押题卷。链接很快被撤回，辅导员补了一句：补考以学院通知和课程教师安排为准。

宿舍群跟着亮起来。周屿先说自己差点点进去，唐骁追问他哪门课需要。周屿把玩笑绕回林亦舟，说只是替他先踩个雷。

林亦舟手里的水瓶被捏出一点响。

他在群里回：不用替我。

群里安静了一下。周屿很快解释自己没真买，林亦舟没有顺着玩笑往下接。外卖架旁刚才念资料包的男生把手机揣回兜里，拎着盖饭走了。

唐骁随后私聊发来一个文件夹截图：课件目录、实验题编号、错题空表。

唐骁：资料包别买。先保存成绩页。

林亦舟回：发目录就行。

水瓶外面的冷雾已经化掉。他把两瓶水带上楼，公共桌上那只信封还在原位。资料包没有买，补考也没有少一点；只是“谁能替他先试一试”这件事，被他先按住了。

### `ACT4-WORK-L04-P2-S03` 图书馆二楼 / 借阅单

图书馆二楼靠窗的位置空着。陆沉把一张借阅单推到林亦舟书边，又把备用圆珠笔放过去：“列个表，从第一章开始。”

林亦舟翻开《数据结构入门》，在第一行写下“线性表”。唐骁的压缩包里没有“必过”和“押题”，只有课件目录、实验题编号和错题空表。电脑风扇转起来，盖过了空调声。

晚风在这时发来语音。她说室友处理票务骗局时也用了空表，聊天记录和付款记录按时间排开，白天排队虽然烦，但至少有柜台、复印机和人。

语音后面跟着一句文字：白天能排队。

林亦舟把手机扣到桌上。陆沉从登记台那边抬头：“窗口三点半前还有人。”

这句话把下午切成两段。前半段给数据结构，后半段给勤工窗口；中间还要打印、确认 C201，给母亲回车票，也给晚风一个不用等到深夜才知道的安排。

### `ACT4-WORK-L04-P2-S04` 打印店 / 教学楼 C201

打印店柜台旁贴着价目表，下面另有一张手写纸：不代买，不推荐任何包过资料。

林亦舟把唐骁给的第一、二章目录和实验题传过去，又单独打开勤工申请表。

老板听见旁边有人问补考资料包能不能打，只说：“我只打你发来的文件。包过两个字，听着就不便宜。”

打印机吐出第一页“线性表”。纸还热，边缘有一点弯。陆沉把订书机推过来，等林亦舟自己对齐纸角。

两份纸被夹进同一个文件袋：一边是补考前两章，一边是勤工申请表。它们都要学生证复印件、黑色笔和本人签字，谁也不能把谁完全挤掉。

C201 在教学楼二楼尽头。陆沉按了按门把，说从青枫居过来十五分钟，下雨二十。年级群又提醒补考不要卡点。

林亦舟拍下门牌。镜头里没有人，只有门牌、楼梯口和走廊尽头的饮水机。可他知道第一周那天会有人排队，会有人翻学生证，也会有人临到门口才发现黑笔没带。

文件袋边缘在走廊风里轻轻响。现在他得决定，下午剩下的时间先交给哪一边。

### `ACT4-WORK-L04-P2-A01` 先补学业材料 / 现场行动

林亦舟没有马上去勤工窗口。他在 C201 门口把文件袋打开，先抽出数据结构前两章，再把勤工申请表压到后面。纸张合上时响了一下，像他把下午排出先后，而不是把工作扔掉。

唐骁在群里问打印范围。林亦舟只拍第一页标题和 C201 门牌，没有拍成绩。唐骁回：先看链表和队列，栈不用今天硬啃。

他坐到阳光书屋靠窗的位置，把目录抄进错题空表。三点二十一，勤工群弹出明德楼材料窗口缺人的消息。他看了一眼时间，把手机扣在桌上，继续翻第一章。

### `ACT4-WORK-L04-P2-A02` 先补学业材料 / 承接回声

三点三十七，那条窗口消息被别人接走。林亦舟没有立刻解释，也没有把这件事说成“没关系”。他在错题空表背面写下：工作窗口错过一次，补考资料完成第一章。

陆沉私聊问窗口是不是没了。林亦舟回“嗯”。陆沉只说：那今天把第一章看完。

林亦舟把手机放到一边，继续往下抄第二章目录。窗外有人推着一车快递经过，胶带声贴着玻璃划过去。窗口少了一次，补考资料多了一页；这两个结果都留在下午，谁也没替谁消失。

### `ACT4-WORK-L04-P2-B01` 先接临时班 / 现场行动

林亦舟站在教学楼一楼大厅，先回勤工群：周四下午我可以。

消息发出去后，他才把数据结构资料塞进书包外层。第一页“线性表”露出一角，被拉链压住。它没有消失，只是被排到稍后。

陆沉看见他的动作，没有说先看书，只问学生证带没带。林亦舟把证件袋摸出来给他看。陆沉点头：“别卡点。”

勤工群很快回复：周四下午两点半，明德楼材料窗口，带学生证复印件。

### `ACT4-WORK-L04-P2-B02` 先接临时班 / 承接回声

林亦舟把时间记进备忘录。备忘录里已经有 C201、周二返校、母亲要的车票截图，现在又多了一行明德楼材料窗口。每一行都像细线，拉住一点就会牵动别处。

晚风问他是不是还要排光谷。他回：今天不排。晚风只回了一句：这句比“到时候看”好。

回青枫居的路上，书包外层的纸被风轻轻拍着他的手臂。那是数据结构前两章，已经被他放到稍后。周四下午的窗口能换来饭钱和打印费，可拉链压住的那一角也在提醒他，补考不会因为接了班就自动通过。

### `ACT4-WORK-L04-P2-C01` 把白天能处理的事说清 / 现场行动

林亦舟没有把同一句话群发给所有人。

他先给母亲发：补考暂定周三上午，C201。我今天打印前两章，周二返校。母亲回得慢，最后只要车票截图和学生证。

他又给晚风发：今天先打印和确认教室，不去光谷。晚风回：收到。这个比晚上十一点突然说“我去不了”好多了。

最后，他在 4XX 群里发了一张 C201 门牌和文件袋照片，没有成绩，也没有母亲的聊天框。

### `ACT4-WORK-L04-P2-C02` 把白天能处理的事说清 / 承接回声

群里这次没有冷成一块。周屿问明德楼窗口是什么新副本，唐骁纠正是勤工窗口；陆沉提醒周四上午别排别的；唐骁补一句第一章今天看完；周屿把玩笑收住，说自己周二到，可以带吃的，但不带资料包。

林亦舟站在教学楼大厅，把手机屏幕熄灭。补考、钱、勤工和家里都还在，但它们不再全部躲在一句“到时候看”后面。

大厅门外的风把暑期安全提醒吹得贴到玻璃上，纸角卷起。林亦舟把文件袋抱紧了一点，往外走。问题没有少，只是每个人终于知道自己要等的是什么。

### `ACT4-WORK-L04-S01` 青枫居 4XX / 教务系统

凌晨前，教务系统终于刷进去。

成绩表一格一格跳出来，白底黑字，像有人把这个暑假之前欠下的账摊到屏幕上。周屿先在群里喊了一声“我靠”，唐骁很快回“别只发语气词”，陆沉坐在阳台门边，没有探身来看林亦舟的屏幕。

林亦舟那一行卡在“数据结构入门”后面。灰色小圈转到第五次，页面往下一沉。

数据结构入门：59。

后面窄窄一个括号，写着“待补考安排”。没有红字，没有感叹号，甚至没有弹窗。它只安静地停在那里，把林亦舟的手指钉在触控板上。

公共桌上还放着父亲给的信封。信封压在《数据结构入门》下面，书脊被现金撑起一点，像这本书自己也知道现在不只是一本课本。旁边是陆沉夹好的食堂小票、两瓶常温水、空调遥控器和一支快没水的黑笔。每样东西都不大，却都像能开出一张单子：饭钱、打印费、补考资料、勤工窗口、回家的车票。

陆沉终于开口：“出了？”

“嗯。”

“多少？”

林亦舟盯着那行字：“五十九。”

陆沉手里的笔停在勤工表边上。他没有说“差一分而已”，也没有说“补考不难”。他把自己的手机转过来一点，只让林亦舟能看见那一行：数据结构入门，64。

“也不好看。”陆沉说，“但能过。”

这句话没有安慰人的尾音，反而比安慰更容易听。林亦舟把页面截图保存，手指刚离开屏幕，母亲的消息从上方弹出来。

母亲：成绩查了吗？

隔了几秒，又一条。

母亲：不用报喜，也不用瞒。

晚风那边也亮着。她发了一张游戏截图，角色站在出生点没动，旁边队友已经跑远。

晚风：要不要开麦骂教务网页？

林亦舟看着三个聊天框。母亲问成绩，晚风留着麦，唐骁在群里说“不要乱买资料包”，父亲的钱还夹在数据结构课本里。他第一次觉得补考不是一项单独任务。它会占用白天，会要打印费，会让他解释为什么不回家、为什么不见面、为什么还要去问勤工。

他给母亲回：有一门没过，数据结构入门，补考安排下周出。

消息发出去后，母亲那边一直显示正在输入。正在输入消失，又出现，最后发来的只有一句：先睡，明天再说。

晚风问：那你明天白天说？

林亦舟回：嗯。

陆沉把勤工表往自己那边收了收：“明早我去图书馆。你要来的话，二楼靠窗那排空。”

林亦舟点头。点完才发现，明天的白天已经被切开了：成绩、母亲、晚风、唐骁的资料、陆沉的图书馆，还有不知道能不能接上的勤工窗口。

### `ACT4-WORK-L04-S02` 青枫居楼下外卖架

第二天中午，青枫居楼下外卖架比前几天满。塑料袋一层层挂着，标签被热风吹得翻起来，有几袋汤汁把袋底烫出雾。一个留校男生站在架子前翻手机，嘴里念：“补考资料包二十九块九，真的假的？”

旁边的人说：“别买，去年有人买到一堆百度文库。”

林亦舟脚步慢了一点。手机刚好震动，年级群里有人发了一个链接，标题写得很满：计科数据结构补考必过资料，含往年题、老师重点、押题卷。

链接很快被撤回。

辅导员发：不要购买非官方来源资料，补考以学院通知和课程教师安排为准。

宿舍群紧跟着亮起来。周屿先说自己差点点进去，唐骁追问他哪门课需要。周屿把玩笑绕回林亦舟，说只是替他先踩个雷。

林亦舟拎着水，瓶身被他捏出一点响。周屿的玩笑不是恶意，但“替”这个字太轻，轻到像把 59 分、补考名单和打印费都当成可以随手挡掉的小雨。

这句必须截住。他在群里回：不用替我。

周屿很快解释自己没真买。林亦舟没有顺着玩笑往下接，只把刚才那句话留在群里，让它比玩笑更硬一点。

群里安静下来。外卖架旁刚才念资料包的男生把手机揣回兜里，拎起自己的盖饭走了。塑料袋摩擦声很轻，却把这几句话衬得更硬。

唐骁私聊发来一个文件夹截图，文件名排得整齐：课件、实验题、章节目录、错题空表。

唐骁：先截图保存成绩页。

唐骁：资料包别买。

唐骁：我下午把第一、二章整理出来。不是答案，只是目录和题型。

三句话像唐骁桌上的线夹，能把散乱的纸夹住，也会把纸边压出硬痕。林亦舟看了一会儿，回：发目录就行。

陆沉从图书馆方向回来，旧包贴在背上，短袖后面汗湿了一小块。他看见林亦舟站在外卖架旁，抬了抬下巴：“水买了？”

“嗯。”

“上去吧。”

两个人一起上楼。四楼走廊很热，尽头有人开门吹空调，冷气漏出来，很快被热风冲散。陆沉进门后把包放到桌边，没问资料包的事，只说：“窗口下午三点有人值班。勤工申请表要学生证复印件。”

林亦舟把两瓶水放到公共桌上：“补考也要学生证。”

“所以别卡点。”陆沉说。

他说的是 C201，也是勤工窗口。林亦舟看着公共桌，父亲的信封、补考截图、唐骁的文件夹、陆沉的窗口时间和晚风那句“白天说”都挤在同一张桌面上。补考资料不能被工作吞掉，工作窗口也不会因为他没过一门课就自动等他。

### `ACT4-WORK-L04-S03` 图书馆二楼 / 阳光书屋

图书馆二楼靠窗那排确实空着。

玻璃窗把树影叠到桌面上，有人趴着背四级词汇，嘴里含着一支没开的中性笔；有人把计算机二级题库摊成两页，一页选择题，一页错题截图。暑假里的学校本来该慢下来，偏偏这些纸和屏幕让时间又挤成开学前那种窄缝。

陆沉在登记台后面坐着。他把一张空白借阅单压到林亦舟书边，又把备用圆珠笔推过来。笔帽上缠着透明胶，像这支笔已经被很多人临时借过。

“列个表。”陆沉说，“从第一章开始。”

林亦舟翻开《数据结构入门》。书里的“线性表”三个字很熟，熟到像已经见过很多次，却从来没有坐下来把它看完。他在借阅单第一行写：第一章，线性表。

唐骁的压缩包下载得很快。文件夹里没有“必过”，没有“押题”，只有课件目录、实验题编号和一张空白错题表。林亦舟把第一、二章打开，电脑风扇转起来，声音盖过了空调。

晚风在这时发来语音。

“我室友那个票的事，辅导员让她把聊天记录和付款记录按时间截出来。她一边骂骗子一边给每张图命名，命名命到崩溃。你昨天说的空表，她用了。”

语音后面是一张照片：派出所外面的长椅、几张打印出来的聊天记录、一杯快化掉的柠檬茶。晚风没有拍人脸，只拍纸。

晚风：白天很烦，但白天能排队。

林亦舟看着那句话，把手机扣在桌上。白天能排队，也能打印，能去教学楼确认 C201，能问勤工窗口，能给母亲回成绩，能把晚风那边的骗局材料按时间排好。夜里所有事都像天塌了，白天至少有楼梯、柜台、复印机和人。

陆沉从登记台那边抬头：“你下午去打印？”

“先打第一、二章。”

“勤工表呢？”

林亦舟停了一下。父亲的现金还夹在书里，饭卡余额不高，打印费要花，补考要复习。每一项都是真的。

“一起打。”他说，“但先看补考资料。”

陆沉点头，没有替他判断。他只补一句：“窗口三点半前还有人。”

这句话把下午切成两段。前半段给数据结构，后半段给勤工窗口；中间还要经过打印店、C201、母亲的消息和晚风那句“白天能排队”。

### `ACT4-WORK-L04-S04` 打印店 / 教学楼 C201

打印店里空调开得低，门帘掀开时带出一股纸粉和墨的味道。柜台旁贴着价目表，下面另有一张手写纸：不打印、不代买、不推荐任何包过资料。

老板坐在电脑前，鼠标点得很快：“U 盘还是微信传？”

“微信。”

林亦舟把唐骁发来的第一、二章课件目录和实验题传过去，又单独点开勤工申请表。旁边一个女生正在打印四级准考证，手机屏幕上亮着报名页面；后面两个男生讨论计算机二级题库，一个说“押题靠谱不靠谱”，另一个说“先看免费题吧”。柜台最边上，一个男生拿着手机问：“老板，这个数据结构补考资料包能不能打？”

老板头也没抬：“我只打你发来的文件。包过两个字，听着就不便宜。”

店里笑了一声，又很快安静。打印机吐出林亦舟的资料，第一页标题是“线性表”。纸很热，刚拿起来时边缘有一点弯。陆沉把订书机推过来，等林亦舟自己对齐纸角。

“你不打全？”陆沉问。

“先打前两章。”

“够今天看。”

林亦舟把勤工申请表夹在资料后面。两份纸厚度不一样，却都要学生证复印件、黑色笔和本人签字。补考资料是为了不让学业继续漏下去；勤工申请是为了不再把所有缺口都压给父亲那几张钱。它们挤在同一个文件袋里，谁也不能把谁完全挤掉。

走出打印店，晚风问：查路线？

林亦舟站在树荫下按住语音键：“我现在去 C201。”

晚风回得很快：像提前踩 Boss 房。

过了几秒，她又发：光谷不急。你别把见面当成补考后的奖励，我也不想坐在商场里看你一直算时间。

林亦舟看着这句，没马上回。C201 在教学楼二楼尽头，暑假只开了几扇门，一楼大厅的灯没全亮。门牌下面有几道旧胶印，像之前贴过考场安排。陆沉伸手按了按门把，没有用力。

“从青枫居到这里，十五分钟。”陆沉说。

“下雨呢？”

“二十。”

“迟到呢？”

陆沉看了他一眼：“你周二就回来。”

林亦舟把 C201 的门牌、楼梯口和走廊尽头的饮水机一起拍下来。镜头里没有人，但他知道第一周那天会有人排队、有人翻学生证、有人临到门口才发现黑笔没带。

年级群又亮。

辅导员：补考同学务必提前返校，不要卡点。

林亦舟把手机放回口袋，手里的文件袋被汗压软了一点。里面一边是数据结构前两章，一边是勤工申请表。纸张贴在一起，边缘在走廊风里轻轻响。

### `ACT4-WORK-L04-CHOICE-01` 选择剧情点

位置：`ACT4-WORK-L04-S04` 后。

选择前状态：林亦舟已经拿到数据结构前两章资料、勤工申请表，确认 C201 路线；母亲知道有一门没过，晚风降低光谷见面压力，唐骁提供资料但不代整理，陆沉提供流程但不代判断，周屿已经被“不要替我”截住一次。

## 三方向选择后正文样张

### `ACT4-WORK-L04-A01` 先补学业材料：把工作表压到第二层

林亦舟最后没有马上去窗口。

他在 C201 门口把文件袋打开，先把数据结构前两章抽出来，又把勤工申请表往后压了一层。纸张重新合上时发出很轻的一声，像他把今天下午拆成了先后顺序，而不是把某一件事扔掉。

唐骁在群里问：打印了吗？

林亦舟拍了文件袋，只拍第一页标题和 C201 门牌，没有拍成绩。

唐骁：先看链表和队列。栈不用今天硬啃。

周屿：唐主任开始远程执法。

唐骁：这是最低建议。

林亦舟没有回玩笑。他坐到阳光书屋靠窗的位置，把第一章翻开。窗外有学生乐队试音，主唱反复唱同一句副歌，音准飘来飘去。旁边一桌有人在背普通话测试注意事项，“不要卡点”四个字又从别人嘴里跑出来，像整座学校都在提醒他不要把时间当成能无限退改的票。

他把唐骁的目录抄到错题空表第一栏。抄到第三行，勤工群弹出消息：明德楼材料窗口周四下午仍缺一人，有意者今天三点半前回复。

林亦舟看了一眼时间，三点二十一。

他把手机扣在桌上，又把第一章最后一页翻过去。不是不缺钱，也不是不想接班。只是如果这时候离开，今天的补考资料又会变成“晚上再说”。过去很多事就是这样被挪到晚上，最后挪成一句没说出口的“算了”。

三点三十七，勤工群那条消息被别人接走。

陆沉发来私聊：窗口没了？

林亦舟：嗯。

陆沉：那今天把第一章看完。

这句话很短，没有安慰，也没有责怪。林亦舟把错题空表翻到背面，在空白处写下：工作窗口错过一次，补考资料完成第一章。

第五幕读取：学业被留在工作线内部，而不是被工作挤掉；但现实窗口确实少了一次。

### `ACT4-WORK-L04-B01` 先接临时班：把收入窗口接住

林亦舟站在教学楼一楼大厅，先回了勤工群。

林亦舟：周四下午我可以。

消息发出去后，他才把数据结构资料塞进书包外层。第一页“线性表”露出一角，被拉链压住。它没有消失，只是被排到了稍后。

陆沉看见他的动作，没说“你先看书”。他只问：“C201 路线记住了？”

“记住了。”

“学生证？”

林亦舟从包里摸出来给他看。透明壳角上磨出白边，照片里的他眼神还没学会躲镜头。陆沉点头：“别卡点。”

这三个字比“好好复习”更像工作线能听懂的提醒。林亦舟把学生证塞回证件袋，又看了一眼饭卡余额。数字不高，但周四下午如果能排上，至少打印费和接下来几天的饭钱不必继续从父亲那只信封里抽。

勤工群很快回：收到，周四下午两点半，明德楼材料窗口，带学生证复印件。

他把时间记进备忘录。备忘录里已经有 C201、B204、周二返校、母亲要的车票截图。现在又多了一行明德楼材料窗口。每一行都像一根细线，稍微一拉就会牵动别的东西。

晚风发来：你现在在教学楼？

林亦舟：嗯。确认 C201，顺便接了一个周四窗口。

晚风隔了一会儿才回：那今天别硬排光谷。

林亦舟：不排。

晚风：这句比“到时候看”好。

回青枫居的路上，林亦舟没有打开数据结构资料。书包外层的纸被走路带起的风轻轻拍着他的手臂，像提醒，也像催促。工作线倾向在这一刻更清楚：他先接住能变成饭钱和打印费的窗口，但补考还在包外层，不会因为他接了班就自动通过。

第五幕读取：工作线投入增强，短期钱压下降；但 `time_debt` 增加，后续必须补一次正式复习反馈。

### `ACT4-WORK-L04-C01` 把白天能处理的事说清

林亦舟没有把同一句话群发给所有人。

他先给母亲发：补考暂定周三上午，C201。我今天打印前两章，周二返校。

母亲回得慢。正在输入出现又消失，最后只发：票买了截图给我。学生证放好。

他又给晚风发：今天先打印和确认教室，不去光谷。

晚风回：收到。这个比晚上十一点突然说“我去不了”好多了。

最后，他在 4XX 群里发了一张照片。照片里有 C201 门牌、楼梯口和文件袋一角，没有成绩，也没有母亲的聊天框。

林亦舟：补考教室确认了。资料只打前两章。周四可能接明德楼窗口。

周屿：明德楼窗口是什么新副本？

唐骁：勤工窗口。

周屿：那我不替你踩雷。

林亦舟看着这句，手指停了一下。

林亦舟：嗯，不用替。

陆沉：周四两点半的话，上午别排别的。

唐骁：数据结构第一章今天看完。

周屿：我周二到，能带吃的。别让我带资料包。

这次群里没有安静成一块硬石头。每个人都只接住自己能接的那一点。母亲要车票和学生证，晚风收掉光谷期待，唐骁盯第一章，陆沉盯窗口时间，周屿把玩笑改成了“不带资料包”。

林亦舟站在教学楼大厅，把手机屏幕熄灭。大厅门外的风把暑期安全提醒吹得贴到玻璃上，纸角卷起。问题没有少，补考、钱、勤工和家里都还在，但它们不再全部躲在一句“到时候看”后面。

第五幕读取：关系解释能力提高，后续迟到或缺席时优先检查是否曾经给过最低事实。
