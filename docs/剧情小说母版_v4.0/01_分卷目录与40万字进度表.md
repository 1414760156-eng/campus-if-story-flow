## 分卷节奏

### 第一卷：行李落在青枫居

目标：写出“住进来”的重量。大学不是抽象自由，而是从通知书、行李箱、父母饭桌、新生群、武生院站、青枫居床位和第一晚灯光开始。

必须出现：

- 录取通知书。
- 计算机科学与技术学院、计科专业、宿舍四人同专业的事实。
- 家庭饭桌。
- 新生群与宿舍四人第一次线上接触。
- 父母买东西、生活费、床品、药、收纳、洗衣液。
- 武生院站、校门、报到动线。
- 青枫居宿舍、床位、公共桌、门把手、第一晚灯光。
- 晚风作为游戏入口的早期影子。
- 宿舍四人的初始爱好差异：手游、PC 游戏、吉他、歌单、运动习惯或不运动。
- 图书馆、阳光书屋、星辰书屋作为新生地图里的读书空间预埋。
- 食堂阿姨、宿管阿姨、快递驿站、便利店和奶茶店作为新生第一次记住校园的生活触点。
- 个人偏好只做轻暗示，不急着成熟固定；但要留下未来能变成集体记忆的小物件、小票、外卖袋或饭桌玩笑。
- 夏知微、沈嘉禾、秦越的弱预埋，不抢宿舍主轴。

### 第二卷：把手机交出去

目标：让“规则”和“集体生活”第一次落到身体上。

必须出现：

- 图书馆仓库领军训服。
- 晨光体育场训练。
- 晨光两校、乐跑和体育选课作为军训后校园身体线的远钩子。
- 英语四级报名传闻、听力耳机、计算机二级和普通话等证书考试作为军训后时间压力的远钩子，不抢手机袋事件。
- 校医院护士、食堂窗口、便利店买水和运动场边补给点，让军训身体疲惫落地。
- 军训手机袋 / 物资区事件。
- 宿舍四人对规则、玩笑、沉默和帮忙的不同反应。
- 健源体育场汇演。
- 汇演不是爽文成功，而是把身体疲惫、集体秩序和小责任一起推到台前。

### 第三卷：热闹把人带走

目标：热闹不是丰富，而是开始分流。每个人被不同东西拉走。

必须出现：

- 第一个早八。
- 计科专业导论课和陈老师。
- 高等数学、程序设计基础、计算机导论等大一上课程露面。
- 计算机机房、实验截图、共享仓库或实验报告的早期压力。
- 英语四级报名、听力耳机、打印准考证、计算机二级和普通话报名信息开始进入群聊和饭桌闲聊。
- 大一第一次期末周：图书馆占座、星辰书屋夜自习、青枫居键盘声、外卖夜宵、借耳机、讲题和作息摩擦要进入日常，不必成为单独大事件。
- 数学竞赛、程序设计竞赛、数学建模、创新创业类比赛和校运动会只作为入口或传闻出现，先不抢主持招新和水房主事件。
- 体育选课、乐跑打卡、晨光体育馆预约和老师带队练武术的入口。
- 东区商业街和健源体育场的乐队表演、舞蹈表演、学生唱歌作为常态热闹。
- 奶茶店、打印店、小饭馆、快递驿站和食堂窗口作为支线人物与宿舍四人擦肩的生活场景。
- 图书馆、阳光书屋、星辰书屋分别承担查资料、小组讨论和夜间自习。
- 晨光大道招新。
- 小组作业和共享文档。
- 主持招新入口。
- 周屿表达欲、唐骁规则感、陆沉现实压力。
- 夏知微照片、沈嘉禾资料、秦越活动、晚风夜聊入口。

### 第四卷：夏天没有停下来

目标：暑假不是中场休息，而是所有线换方式继续。

必须出现：

- 宿舍变空。
- 晨光湖和空校园。
- 家庭生活费、父亲工作、母亲生活用品变化。
- 晚风聊天从游戏进入微信和抖音。
- 光谷活动、陆沉现实压力、秦越资源感。
- 夏知微记录、沈嘉禾资料、陈老师通知。
- 西区外卖、东区商业街、后街夜市作为暑假和返校前后的生活半径。
- 英语四级二战、六级准备、计算机二级、普通话、教师资格证、驾照或软考备考影响暑假时间分配和聊天背景。
- 后街水果摊、修自行车摊、校医院、便利店夜班、奶茶店和小饭馆承担暑假情绪出口。
- 成绩查询、补考预警、开学初借资料和新学期返校压力要作为暑假后半段暗流；补考不写成笑料，只写它如何改变聚餐、见面和自尊。
- 线性代数、离散数学、数据结构入门或程序设计实验带来的学业压力。
- 晨光体育馆、室外篮球场、室外羽毛球场、网球场和乐跑路线成为支线相遇地点。
- 寝室游戏、PC 端 3A 大作、吉他唱歌和楼层投诉作为宿舍日常延伸。
- 第四卷引导式外流点：后街、东区商业街、光谷、地铁口、便利店或校道边缘可以制造“先不回去”的冲动和无旧账的轻松，但每次必须回到宿舍、家庭、晚风、学业或留校状态。
- 许澈在第四卷只能作为远影、名字、便利店工牌、活动群一瞥或路人采访预感出现，不能提前成为林亦舟的新关系核心。
- 班会前的返校压力。

### 第五卷：所有路都窄下来

目标：第五幕是可玩性核心。开头多线混乱，中段拆成清晰单线，后段每条线内部完成冲突、选择和代价。

最新回改方向：第五卷不能只是材料流转和表格追查，也不能让四人一直和谐协作。它必须成为下册矛盾爆发卷：每个主要冲突都逼人物站队，在室友、规则、自己前途、兴趣活动、情感关系、家庭压力和外部人际之间做选择。站队后要留下寝室小动作和关系裂痕，作为第六卷戏剧收场的根。

补充硬规则：第五卷只负责把队伍站出来，不负责把所有关系修好。它要留下可继承后果：谁护过谁，谁没叫谁，谁把谁写进材料，谁私下吐槽过谁，谁开始绕开某个室友，谁在外面找到更轻松的关系。第六卷必须顺着这些后果写关系收束，而不是重新从表格、材料和礼貌沟通开始。

## 游戏之外剧情的纳入方式

游戏之外剧情分三种：

| 类型 | 作用 | 能否进入正式节点 |
|---|---|---|
| 生活插章 | 丰富世界和人物真实感 | 可变成低操作高沉浸段 |
| 支线前史 | 解释支线人物为什么这样行动 | 可变成条件化记忆或对话背景 |
| 现实事件库 | 提供被骗、争吵、老师误会、生活费变化等冲突素材 | 可变成支线事件或可选回流 |
| 考试与证书背景 | 提供四六级、计算机二级、普通话、教资、竞赛和运动会造成的时间压力 | 多数作为氛围和对话，少数可成为强剧情点 |
| 生活血肉碎片 | 提供食堂、奶茶店、校医院、后街夜市、修车摊、KTV、台球厅等情绪出口 | 可变成短场景、回流段或终章回响 |
| 角色偏好与集体记忆索引 | 管理谁喜欢什么、为什么去某处、个人偏好如何和宿舍/社团/班级形成共同记忆 | 可变成短场景、微选择、关系回响或终章物件 |
| 人物视觉锚点 | 管理外貌、发型、衣服、鞋包、配饰、换季和场合穿着怎样反映性格、经济状态、关系变化和时间流逝 | 可变成出场描写、微心态、关系温度、场景辨识和终章旧物回响 |
| 寝室座位与桌面索引 | 管理青枫居 4XX 四人座位、桌面、电脑手机外设、插排、公共桌、空调遥控器和旧物如何反映人物 | 可变成宿舍冲突、生活习惯差异、低操作沉浸段、5A 修补素材和终章清桌回响 |

新增内容越多，越要标清楚它属于哪一层。不能把参考层误写成玩家必经事实。
