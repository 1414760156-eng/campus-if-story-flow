# IF 第四幕 ACT4-WORK 工作线生活半径剧情页

更新时间：2026-05-24

## 用途

本文用于把 `R4-WORK` 第四幕从入口概要推进到可审读的正式剧情页样板。

当前版本覆盖 `R4-ACT4-L01` 至 `R4-ACT4-L06`，对应母本第四卷暑假生活半径的六个公共锁点：《空宿舍》《退改规则》《站外半小时》《低空飞过》《不要卡点》《两间教室》《座位表》《联系人》《签字》。本文件不是独立外传，也不替换小说母本主事件；它只把同一时期的母本事件改成工作线镜头：钱、排班、饭卡、交通、父母转账、勤工流程、补考材料、去向登记和本人签字变成林亦舟此时的主要生活半径。

## 硬规则

- 路线：`R4-WORK`
- 当前池：`POOL-R4-WORK`
- 当前样板范围：6 个母本锁点、24 个剧情页、6 个选择剧情点。
- 每个选择窗 3 项：1 个工作线主轴推进 + 2 个池内微内流点。
- 每个微内流点必须展开为可计数小剧情链：至少 4 拍，包含触发、对话 / 行动、小抉择、回声；普通微内流点建议 3-5 个剧情段。
- 每个方向选择后都必须有反馈和承接剧情，主轴推进方向也不例外；本文表格中的“微剧情链 / 结果”只是审读索引，正式 JSON / 试玩层不能只改变量后直接跳到下一个选择点。
- 正式剧情页必须吸收小说母版的场景、人物对话、物件细节和生活噪音；不能把丰富母版压缩成“事件摘要 + 选项问答”。当前表格稿只是结构细稿，后续 P1 需要把每页扩成可沉浸阅读的正文页。
- 若母版没有逐段写出某个小事件，可基于母版地点、人物关系、现实流程和物件扩写小剧情；扩写只增加沉浸与当前路线生活半径，不改变母版公共锁点因果，也不打开新路线。
- 其它人物只能作为母本事件中的软回声，不开启陆沉线、晚风线、宿舍线、感情线或 5X。
- 第四幕完整稿最终交给 `ACT5-WORK-B01`，不得直跳 Act6。
