# 开发长线_IF可玩支线R5-WANFENG晚风线

## 用途

本文用于把 `R5-WANFENG` 从第五幕亲密分歧骨架扩展成可拆剧情页、选项页、变量回调和毕业结算的 A 级主轴亲密外流线。

它不是“恋爱奖励线”，也不是把宿舍主轴换成单纯甜线。它规定的是：玩家在第五幕宿舍关系开始退位、事项群变短、4XX 不再自然同步时，选择继续靠近晚风，林亦舟的大学生活如何被语音、游戏、微信、抖音状态、线下见面、礼物、公开边界、交通时间和钱一起改写。

本线只承接主轴能直接进入的晚风方向。沈嘉禾 / 夏知微等其它感情线仍属于 `R5-ROMANCE`，必须由 A3 社团 / 活动父池里已经形成连续生活半径的女生候选派生，不能从本线中临时跳出。

## 路线定位

路线编号：`R5-WANFENG`

路线池编号：`POOL-R5-WANFENG`

路线等级：A 级主轴亲密外流线

共享范围：第一卷、第二卷共享；第三幕后晚风以消息、语音、游戏和状态持续存在；第五幕亲密分歧后进入独立命运池。

核心体验：林亦舟选择把原本可能回到 4XX 当面说的话，先说给晚风听。晚风能听见他、偏向他、指出他在逃避，也会有自己的生活圈、队友、室友、通勤和情绪边界。靠近她会带来甜，也会带来迟到、缺席、花钱、被看见、被误读和必须解释的代价。

本线的可玩重点不是“追到晚风”，而是：玩家是否愿意承担亲密关系从屏幕走到现实以后产生的时间成本、公开成本和旧关系退位后果。

## 不改变的母本锁点

- 第五幕宿舍强冲突仍然发生。晚风线只改变林亦舟有没有回去、什么时候回去、如何解释，不取消 4XX 的旧账。
- 晚风不是宿舍问题的解决者。她可以听、问、提醒、偏向林亦舟，也可以不接受他把她当逃跑理由。
- 周屿、唐骁、陆沉仍按母本生活半径推进：活动、项目、勤工、毕业流程都继续发生，只是林亦舟更常缺席。
- 第六卷 C407、课程演示、关系定形、公开边界和事项群回声仍发生。
- 第七卷预登记、毕业照、答辩、清寝、饭局和离校手续仍发生。
- 本线不能临时打开沈嘉禾 / 夏知微完整感情线；她们只能作为 A3 派生线或软照面出现。

## 核心变量

| 变量 | 含义 | 高值倾向 | 风险 |
|---|---|---|---|
| `wanfeng_link` | 和晚风的连续关系强度 | 更容易进入稳定亲密、继续联系 | 过高会挤压宿舍当面处理 |
| `wanfeng_boundary` | 晚风关系边界清晰度 | 能说清见面、公开、游戏和私聊边界 | 过低会导致误读、失约和受伤 |
| `romance_focus` | 亲密投入度 | 晚风成为主叙事半径 | 宿舍缺席与旧账延迟 |
| `dorm_absence` | 宿舍缺席 | 4XX 不再默认等他 | 误读加重，毕业同框降温 |
| `time_debt` | 时间债 | 迟到、晚回、错过群消息 | 第六卷事件窗口被压缩 |
| `money_pressure` | 交通、礼物、奶茶、见面花费 | 亲密现实感增强 | 钱压不能写成浪漫滤镜 |
| `public_boundary` | 朋友圈、抖音、合照和他人可见边界 | 关系更真实，也更能承受被问 | 过低会引发公开风险 |
| `wanfeng_misread` | 晚风误读或受伤 | 产生小冷、暂停、体面后退 | 高值触发错过结算 |
| `dorm_visibility` | 4XX 是否看见这段关系 | 提供起哄、建议和误读场 | 高值增加旧关系压力 |
| `future_talk` | 未来讨论 | 第七卷城市、实习、考研结算更具体 | 说得太满会产生承诺压力 |
| `intimacy_cost` | 亲密代价 | 甜和现实绑定 | 高值压缩学习、宿舍和打工 |
| `graduation_temperature` | 毕业结算温度 | 继续联系、体面分别、错过或私人入口 | 由前述变量综合决定 |

## 体量与密度

建议 8 章开发体量，对应约 60,000-90,000 字正文容量。本文只提供可开发骨架，不直接写足正文。

最低配置：

- 至少 32 次有效选择。
- 至少 6 个内部回流点。
- 每个内部回流点至少 2 个回流方向。
- 每个回流方向至少 3-4 个连续剧情拍点。
- 至少 3 次强后果回调：线下见面迟到、公开边界误读、第六卷事项群变短、第七卷毕业照 / 饭局 / 私聊结算。

## 章节开发详表

### 第一章：今晚还排吗

章节目标：确定玩家从主轴进入晚风线。林亦舟不是突然抛下宿舍，而是在 4XX 事项群和晚风语音同时亮起时，选择先接住晚风那边的一句话。

现实触发：
- 晚风问“今晚还排吗”，游戏队伍差一个人。
- 4XX 事项群同时在讨论站队、材料或饭局。
- 林亦舟知道自己无论先回哪边，另一边都会留下延迟。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `WANFENG-CH01-N01` | 读取亲密入口 | 手机 / 床上 / 楼道 | 林亦舟、晚风、4XX |
| `WANFENG-CH01-N02` | 游戏语音与群消息并压 | 游戏房间 / 事项群 | 林亦舟、晚风、周屿、唐骁 |
| `WANFENG-CH01-N03` | 第一次明确偏向 | 青枫居楼下 / 私聊 | 林亦舟、晚风 |

关键选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `WANFENG-CH01-C01` | 先进语音，告诉晚风自己这边很乱 | `wanfeng_link +2`、`romance_focus +1`、`dorm_absence +1` | 第六章晚风会追问“你是不是又先躲到我这边” |
| `WANFENG-CH01-C02` | 先回 4XX，再跟晚风说明 | `dorm_visibility +1`、`wanfeng_boundary +1`、`romance_focus +0.5` | 晚风线保留但进入更慢 |
| `WANFENG-CH01-C03` | 两边都用短句糊过去 | `wanfeng_misread +1`、`dorm_absence +1`、`avoidance +1` | 亲密入口不稳，后续更容易小冷 |
| `WANFENG-CH01-C04` | 明确说今晚想和晚风多聊一会儿 | `wanfeng_link +2`、`romance_focus +2`、`dorm_absence +1` | 锁入 `POOL-R5-WANFENG`，4XX 当晚少问一次 |

本章结尾钩子：游戏房间里只剩两个人没准备，晚风没有催，只问：“你是不是又把另一边静音了？”

### 第二章：地铁口没等到雨停

章节目标：让线下见面成为真实成本。约见不是浪漫切片，而是交通、天气、迟到、门禁、宿舍消息和是否解释一起发生。

现实触发：
- 晚风提出线下见一面，地点在地铁口、东区商业街或校园外奶茶店。
- 4XX 同时需要确认一个当面事项。
- 林亦舟如果赴约，就会错过公共桌上最适合解释的一段时间。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `WANFENG-CH02-N01` | 约见确认 | 微信 / 地铁口 | 林亦舟、晚风 |
| `WANFENG-CH02-N02` | 迟到与门禁 | 校门 / 地铁站外 | 林亦舟、晚风、生活人物 |
| `WANFENG-CH02-N03` | 见面后的旧账延迟 | 校园夜路 / 4XX 群 | 林亦舟、晚风、4XX 软回声 |

关键选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `WANFENG-CH02-C01` | 准时赴约，4XX 只留最低事实 | `wanfeng_link +2`、`time_debt +1`、`dorm_absence +1` | 4XX 事项群后续变短 |
| `WANFENG-CH02-C02` | 先处理宿舍，再迟到见晚风 | `wanfeng_boundary +1`、`wanfeng_misread +1`、`dorm_visibility +1` | 晚风不爆发，但记录“你又迟到了” |
| `WANFENG-CH02-C03` | 取消见面并说明真实原因 | `wanfeng_boundary +2`、`romance_focus -1/+1`、`dorm_warmth +1` | 关系降速但更清楚 |
| `WANFENG-CH02-C04` | 不说清原因，只说“有点事” | `wanfeng_misread +2`、`avoidance +1`、`time_debt +1` | 第五章晚风生活圈会反照他的含混 |

本章结尾钩子：晚风把伞收起来，说“下雨不是你迟到的理由，顶多是背景。”

### 第三章：礼物不贵

章节目标：处理亲密关系中的笨拙和钱压。礼物不能只写成甜，要写预算、快递、室友看见、买错规格和是否过度正式。

现实触发：
- 节日、生日、游戏纪念日或一次普通的“想带点东西”。
- 林亦舟在快递站、奶茶店、淘宝订单和 4XX 公共桌之间切换。
- 室友可能看到包装或订单。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `WANFENG-CH03-N01` | 礼物预算 | 淘宝 / 奶茶店 / 快递站 | 林亦舟 |
| `WANFENG-CH03-N02` | 室友建议 | 4XX 公共桌 | 林亦舟、周屿、唐骁、陆沉 |
| `WANFENG-CH03-N03` | 送出或退缩 | 校门口 / 微信 / 快递柜 | 林亦舟、晚风 |

关键选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `WANFENG-CH03-C01` | 送实用小礼物并说明“不贵” | `wanfeng_link +1`、`money_pressure +1`、`public_boundary +0.5` | 晚风会问“你为什么先解释价格” |
| `WANFENG-CH03-C02` | 不送礼物，只认真陪她打一局或走一段路 | `romance_focus +1`、`time_debt +1`、`wanfeng_boundary +1` | 甜更自然，但错过 4XX 当面节点 |
| `WANFENG-CH03-C03` | 问室友建议 | `dorm_visibility +1`、`teasing_risk +1`、`money_pressure +1` | 周屿起哄，唐骁给预算，陆沉问用不用得上 |
| `WANFENG-CH03-C04` | 临时退缩，不送也不解释 | `wanfeng_misread +1`、`missed_chance +1`、`romance_focus -1` | 第四章公开边界降温 |

本章结尾钩子：晚风把礼物照片发来，没发朋友圈，只回：“你下次别把心意写成报销单。”

### 第四章：朋友圈可见

章节目标：处理公开边界。亲密关系被看见后，问题不再是“甜不甜”，而是谁能看、谁会问、谁被排除在解释外。

现实触发：
- 晚风发朋友圈、抖音状态或游戏截图。
- 林亦舟出现在照片边缘，或评论区有人问。
- 4XX 有人刷到，反应各不相同。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `WANFENG-CH04-N01` | 状态可见 | 朋友圈 / 抖音 | 林亦舟、晚风 |
| `WANFENG-CH04-N02` | 宿舍刷到 | 4XX / 公共桌 | 周屿、唐骁、陆沉、林亦舟 |
| `WANFENG-CH04-N03` | 公开边界讨论 | 私聊 / 校园夜路 | 林亦舟、晚风 |

关键选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `WANFENG-CH04-C01` | 和晚风说清哪些能发、哪些不适合发 | `public_boundary +2`、`wanfeng_link +1`、`dorm_visibility +1` | 第七章公开边界可用 |
| `WANFENG-CH04-C02` | 默认可以发，不主动讨论 | `romance_focus +1`、`public_boundary -1`、`dorm_visibility +1` | 可能被 4XX 误读为“他已经不在这边” |
| `WANFENG-CH04-C03` | 被室友问起时否认或打哈哈 | `wanfeng_misread +2`、`dorm_visibility +1`、`public_boundary -1` | 晚风受伤，进入小冷 |
| `WANFENG-CH04-C04` | 主动承认但不展开私事 | `public_boundary +1`、`dorm_trust 0/+0.5`、`romance_focus +1` | 4XX 知道关系存在，但旧账仍未解决 |

本章结尾钩子：周屿把手机扣在桌上，笑了一下，说：“行啊，热闹都不在我们这边了。”

### 第五章：她也有队伍

章节目标：写出晚风自己的生活圈。她不是只在屏幕那头等林亦舟的人，她有室友程小满，有游戏队友闻峥、叶遥，有自己的排位、作业、通勤、社交和不想被敷衍的时候。

现实触发：
- 晚风的队伍缺人，但不是非林亦舟不可。
- 程小满或队友看出她在等消息。
- 林亦舟意识到晚风也会因为他迟到、含混、失约而调整自己的生活。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `WANFENG-CH05-N01` | 晚风生活圈亮相 | 游戏语音 / 宿舍背景音 | 晚风、程小满、闻峥、叶遥 |
| `WANFENG-CH05-N02` | 林亦舟被旁观 | 语音房 / 微信 | 林亦舟、晚风生活圈 |
| `WANFENG-CH05-N03` | 是否尊重她的节奏 | 游戏结算 / 私聊 | 林亦舟、晚风 |

关键选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `WANFENG-CH05-C01` | 承认自己总让她等 | `wanfeng_boundary +2`、`wanfeng_link +1`、`romance_focus +1` | 晚风愿意继续，但不再无条件等 |
| `WANFENG-CH05-C02` | 把玩笑接过去，躲开严肃话题 | `avoidance +1`、`wanfeng_misread +1`、`romance_focus 0` | 第六章被明确指出“挡箭牌” |
| `WANFENG-CH05-C03` | 尊重她和队友安排，不临时插队 | `public_boundary +1`、`future_talk +1`、`wanfeng_link +1` | 关系更成熟，但甜度不靠占有 |
| `WANFENG-CH05-C04` | 因为吃味要求她先陪自己 | `intimacy_cost +1`、`wanfeng_misread +2`、`public_boundary -1` | 触发公开边界压力 |

本章结尾钩子：晚风关掉队伍麦，声音低下来：“你别把我这边也当成你可以随时暂停的房间。”

### 第六章：别拿我当挡箭牌

章节目标：把宿舍强冲突反压到亲密关系。林亦舟如果总把晚风当喘气口，晚风会指出这件事；4XX 那边也会感到他不再在场。

现实触发：
- 第五幕站队爆发或 4XX 事项群进入关键窗口。

## 对话逻辑

### 晚风

- 她不是安慰奖励，也不是只会等人的屏幕头像。
- 她的温柔常常带一点清楚：会听林亦舟讲，也会问“那你打算怎么回那边”。
- 她可以偏向林亦舟，但不会替他把宿舍旧账判成“都是别人不懂你”。
- 她受伤时不需要大吵，可以表现为语音变短、队伍先开、状态不再给他看。

### 林亦舟

- 不能写成突然恋爱脑。他靠近晚风，是因为这段关系让他第一次愿意解释自己。
- 他的问题不是喜欢晚风，而是容易把晚风当成延迟面对 4XX 的理由。
- 每一次甜都要留下一个现实钩子：迟到、钱、公开、事项群变短、宿舍少问一句。

### 周屿

- 周屿会先起哄，后面慢慢意识到林亦舟真的不常在场。
- 不能把周屿写成恋爱阻碍。他的压力来自“热闹少了一个默认会来的人”。

### 唐骁

- 唐骁更关心流程是否被影响，也会把见面、迟到和项目稳定联系起来。
- 他不能被写成反对亲密关系，只是不接受林亦舟用“我有事”替代明确分工。

### 陆沉

- 陆沉不会评价晚风，只会在林亦舟缺席时更少默认他会回来。
- 他可以给出很短的实用判断，例如“你要去就提前说”，但不当恋爱导师。

### 程小满 / 闻峥 / 叶遥

- 他们是晚风生活圈，不是反派阻碍。
- 他们的功能是让玩家看见：晚风也有人际位置，也会被别人问，也不能无限等林亦舟。

## 漂移防线

- 本线不是“甜线奖励”。每次靠近晚风都必须同时带出时间、钱、公开边界或宿舍缺席成本。
- 本线不是“晚风拯救 4XX”。晚风可以提醒林亦舟，但不能替他解决宿舍关系。
- 本线不能把晚风写成只等林亦舟的人。她必须有自己的生活圈、游戏队伍、室友、计划和情绪边界。
- 本线不能开启沈嘉禾 / 夏知微完整路线；其它感情线必须等 `R5-ROMANCE` 按 A3 候选来源开发。
- 本线内部回流不能打开其它命运池。所有回流只改变晚风关系温度、公开边界、误读、宿舍缺席、资源代价和毕业结算。
- 本线不能取消母本公共锁点，只能改变林亦舟进入锁点的位置：更晚、更需要解释、更难两边都说清。

## 开发优先级

1. 先实现入口 `P0C_ACT5_WANFENG_BOUNDARY` 与 `P0C_WANFENG_STEP_FORWARD` 的路由池锁定。
