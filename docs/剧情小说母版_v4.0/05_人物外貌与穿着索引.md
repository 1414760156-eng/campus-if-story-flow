# 05 人物外貌与穿着索引

## 核心原则

人物不能只靠名字、台词和功能站住。读者需要在脑子里看见他们：谁的背包总是只挂一边肩，谁把鞋带系得很紧，谁的旧外套袖口洗得发白，谁在正式场合穿得过分用力，谁因为家里钱紧开始反复穿同一件外套。

外貌与穿着不是静态立绘，也不是“谁更好看”的排名。它们是人物性格、经济状态、场合判断、关系温度和时间流逝的可见痕迹。

人物的衣服、鞋包和身体动作需要与寝室物件互相支撑。四人座位、桌面、电脑手机外设、床帘、插排和公共桌的具体规则见 `06_青枫居四人寝室与桌面索引.md`。

写作时遵守四条：

1. 先写能被看见的东西，再让读者自己判断人。不要用“成熟”“清冷”“阳光”“高冷”“女神”“男神”替代细节。
2. 外貌描写必须服务现场。新生入校、军训服、早八、主持试训、期末周、约会、运动会、家长来校、毕业照，每个场合能看见的东西都不同。
3. 视觉锚点要能重复，也要能变化。重复让人物可识别，变化让人物真的过了大学四年。
4. 宿舍四人同场时优先写对比。四个人站在同一条走廊、同一个食堂窗口、同一个机房门口，穿着和动作要能看出他们不是同一种人。

## 描写插入时机

不要每章开头都给人物做外貌介绍。优先在以下场合写：

- 第一次正式出场。
- 换季、换年级、放假返校。
- 重要事件前后：军训、汇演、班会、主持试训、运动会、考试周、实习面试、毕业照。
- 关系变化时：约会、争吵后见面、道歉、宿舍修补、外流线里见新人。
- 经济和家庭压力改变时：旧鞋、重复穿的外套、没买新生活用品、红包备注变短以后仍然把衣服穿得整齐。
- 人物装大人和露怯时：穿得正式但袖口皱、准备好的衬衫不合身、外套拉链卡住、领口被汗洇湿。

## 宿舍四人视觉锚点

| 人物 | 初始视觉锚点 | 阶段变化 | 场景使用 | 禁忌 |
|---|---|---|---|---|
| 林亦舟 | 干净、低调，新生期衣服多是家里新买的基础款；短发常被枕头压出一点翘；背包拉链上挂着开学时留下的证件袋或小挂件 | 大一下以后开始有固定几件 T 恤和外套；家庭线变重后，鞋边磨损、手机壳旧角、反复买常温水可以承担压力；见晚风前会换更干净的衣服，但不一定熨得平 | 站在门口、便利店买水、地铁口见面、食堂排队、毕业照前整理衣领，都可以写他的迟疑和想装镇定 | 不写成“普通路人脸”；不要用外貌平庸替代性格回避 |
| 周屿 | 背包常单肩挂，外套敞着，头发不太服帖；T 恤、卫衣、球鞋或小配饰更容易有颜色和图案；走路比别人快半步 | 主持试训后开始尝试衬衫、外套、发型和舞台感，常过分用力；低落时帽檐压低、背包改成双肩，嘴上仍然开玩笑 | 小舞台、篮球场、后街夜市、宿舍开黑、被人送水或起哄告白时，衣服和动作能显示他想被看见又怕被说穿 | 不写成单纯“阳光开朗”；不要让花哨穿着变成浅薄标签 |
| 唐骁 | 衣领、袖口、鞋带、电脑包都收拾得规整；颜色偏稳，衣服不一定贵，但讲究合适、耐穿和功能；出门前会看时间和天气 | 运动、竞赛、面试、毕业手续都能看见他提前准备的衣物和备用物；亲密关系或强冲突时，最先乱的是纸角、袖口和领口 | 东北饺子馆、图书馆、机房、晨光体育馆、打印店、毕业手续现场，都可以用他的整齐照出他怕失控 | 不强行加眼镜、精英感或冷面人设；规则感要从动作里出来 |
| 陆沉 | 黑色旧背包，衣服少而干净，短袖和外套洗得发软；鞋不新但刷得干净；常把东西往里收，不挡别人 | 勤工、家庭压力和留校会让他的衣物重复率更高；冬天旧外套拉链拉到高处，夏天衣角被汗贴住；被照顾时不立刻换新，却会把别人送的东西收好 | 食堂二楼、图书馆、星辰书屋、勤工岗位、夜跑、青枫居水房，都可以用旧物和整洁显示他的自尊 | 不把节省写成可怜，不把沉默写成阴郁；不要让别人替他的衣物下结论 |

## 家庭、老师和无名生活人物

| 人物层级 | 视觉锚点 | 使用方式 |
|---|---|---|
| 父亲 | 旧背包、工作鞋、口袋里的硬币、短袖领口、车站外晒出的汗、转账后仍舍不得买贵水 | 不靠大段诉苦写家庭压力，而靠鞋底、背包、现金、站外小店和说话时看表来写 |
| 母亲 | 家居服、围裙、买生活用品的袋子、收纳袋、快递盒、手机语音里的厨房声 | 她的关心从亲手买东西变成少买、晚买、让林亦舟自己看着办，外观和物件要跟着变化 |
| 陈老师 | 普通衬衫、旧公文包、激光笔、讲义边角、办公室桌上的材料夹 | 老师不是反派，视觉上要像大学里真实会遇到的老师：疲惫、务实、按材料说话 |
| 食堂阿姨 / 店员 / 摊主 / 护士 / 宿管 | 围裙、袖套、口罩、手套、收银小票、体温计、钥匙串、雨后的修车工具 | 他们不需要姓名，但可以被认出。读者记住一双袖套、一句“少辣吗”、一串钥匙，就能记住这个校园被人生活过 |

## 年级与季节变化

大一：衣服和物品带着家庭准备的痕迹。新买床品、母亲塞的薄外套、父亲转的钱、开学文件袋、军训服和新鞋磨脚，都是新生感。

大二：人物开始有自己的固定穿法。谁常去哪个食堂窗口、谁常穿哪件外套、谁运动前绑护腕、谁约会前换衣服，都开始稳定。

大三：压力改变外观。实训、竞赛、证书、感情和家庭线挤在一起后，衣服会重复，头发会乱，鞋会脏，书包会沉，正式场合也可能露出赶时间的皱。

大四：视觉回响比新鲜更重要。旧准考证、证书照、磨旧的球鞋、毕业照衬衫、最后一次穿的外套、搬空宿舍时露出的旧物，负责把四年压回来。

季节也要进入人物。武汉夏天的汗、雨季的鞋、冬天宿舍走廊里的厚外套、春天运动会前的短袖和外套叠穿，都比抽象“天气很热/很冷”更能让场景落地。

## 集体场景写法

同一场景里，尽量让视觉细节形成关系：

- 早八：唐骁衣领平，周屿背包歪，陆沉旧外套拉链拉高，林亦舟头发被枕头压翘。
- 军训：同一套迷彩服抹平差异，但袖口、腰带、鞋带、帽檐和手机藏放位置会把差异重新显出来。
- 食堂：谁把外套搭在椅背，谁先擦桌子，谁把小票夹起来，谁衣袖沾了汤。
- 约会：林亦舟换了干净衣服但袖口皱，晚风鞋跟磨脚却嘴硬，甜蜜和笨拙同时出现。
- 宿舍争吵：不要写“大家都很紧张”，写周屿把帽檐压低，唐骁把袖口卷上去又放下，陆沉把旧外套往椅背里推，林亦舟盯着门口自己的鞋。
- 毕业照：衣服终于相对整齐，但鞋、袖口、旧包、没摘掉的胸牌和站位会暴露四年留下的差异。

## 禁令

- 不要用外貌标签替代人物行动。
- 不要把人物写成固定立绘，每次出场都同一套描写。
- 不要把经济压力写成廉价可怜；旧物也可以干净、有尊严。
- 不要为了区分人物给所有人强行加夸张配饰。
- 不要让支线人物的外貌抢走宿舍四人主轴。
- 不要在第四卷把许澈写得过于完整，破坏第五卷唯一硬外流的代价。
