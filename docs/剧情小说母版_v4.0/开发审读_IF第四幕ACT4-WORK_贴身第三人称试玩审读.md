## 分锁点审读

### L01 空宿舍与暑假分流

通过。新版开头能看见行李箱、公共桌、光谷截图、母亲电话和勤工表，事件从宿舍打包自然推到“三件事先处理哪一件”。比第二人称版本更像小说，也更容易记住“空宿舍不是空白时间”。

关注项：选择引导句“三件事都在等林亦舟先动一下”略偏抽象，但没有越界成选项说明。

### L02 生活费与退改规则

通过。车票、退改规则、便利店兼职表和三个聊天框的压力链清楚，选择页没有再解释功能。该锁点承担“钱和时间第一次被算成表”的作用成立。

### L03 站外半小时与家庭现金

通过。父亲信封、面馆硬币、图书馆登记、补考名单能形成连续事件，五百块不再只是变量。家庭压力仍留在工作线池内，没有打开家庭完整线。

### L04 补考与打印店

通过。C201、打印店、勤工临时班和 C407 扫描件之间的冲突清楚。选择页“手里只拿得稳一沓纸”是当前第四幕里较好的短句压力。

### L05 返校前两间教室与座位表

基本通过。座位表、文件袋、回执、明德楼窗口能支撑“被看见 / 低可见处理 / 先跑窗口”的三向选择。

关注项：该锁点贯穿物仍比 L01、L04 稍散，后续若第五幕承接宿舍误读，可优先从 L05 回填一两处“座位表照片 / 文件袋位置”的回声。

### L06 联系人与签字

通过。去向页面、联系人、本人签字、透明文件夹能直接接到 Act5 B01。它是第四幕收束点，功能明确。

关注项：L06 的“签下去前，先决定哪一栏不再空着”仍有轻微选择引导感，但比之前的说明句自然。

## 结论

第四幕当前已经可以作为新版口径样板进入试玩。它解决了上一版的两个主要问题：

- 不再用密集第二人称制造疲劳。
- 不再让玩家感觉每页都在解释选项功能。

当前不建议继续大改第四幕。后续只保留两个轻修方向：

- 如果试玩时仍觉得姓名偏密，可以做一次“林亦舟 -> 他 / 主语省略”的轻量节奏 pass。
- 如果 L05 承接第五幕时不够强，再补座位表和文件袋回声。

## 下一步建议

进入第五幕前，先把本轮第四幕口径定为后续标准：

1. 正文：贴身第三人称。
2. 选择页：短句压力，不解释功能。
3. 微心态：写玩家能真实发出 / 写下 / 做出的动作。
4. 第五幕扩写时优先补“具体事件”和“校园人物参与”，不要先堆选项。
