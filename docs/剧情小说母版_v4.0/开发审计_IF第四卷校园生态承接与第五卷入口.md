更新时间：2026-05-29

## 结论

第四卷不能只作为“暑假勤工线”从第三卷跳到第五卷。它必须承担中间桥梁：证明第三卷留下的社团人物、公开生活、老师 / 店员 / 老板 / 宿管 / 驿站人员等校园日常关系没有消失，只是被暑假、钱、排班、补考和本人签字压低了音量。

因此，第四卷 R4-WORK 的修订顺序应先于第五卷扩写。第五卷 12-15 个主选择窗口要有根，第四卷就必须先把这些根带到暑假末和返校前。

## 四层承接权限

1. 第三卷已选择方向的核心边缘人物承接
   - 如果玩家第三卷明确选过主持、融媒体、摄影、材料协作、幕后、志愿等方向，对应人物可以在第四卷成为真实义务、未读消息、时间冲突、材料请求或微选择权重。
   - 他们不重开路线，只证明工作线挤掉了什么。

2. 第一至三幕照面人物的微心态承接
   - 没有选过方向，但有过照面、短协作、群消息或公共场合互动的人物，只能进入微心态层。
   - 他们不能临时给关键建议，也不能突然变成候选。

3. 日常校园人物记忆承接
   - 老师、打印店老板、驿站工作人员、宿管阿姨、便利店店员、食堂窗口、书屋老板、活动场地人员可以记住、误解、退回、放行或多问一句。
   - 他们不抢戏，但要让玩家感觉林亦舟被校园系统记住。

4. 第五卷入口种子
   - 每个第四卷块都要留下可被第五卷读取的对象：透明文件夹、签字、打印小票、活动群未读、老师规则、驿站记忆、门禁记录、座位表等。

## 第四卷到第五卷过渡要求

`ACT4-WORK-L06-TO-ACT5-WORK-B01` 不能只写“暑假结束”。它要带着这些东西进入第五卷：

- 透明文件夹
- 本人签字边界
- 老师规则
- 打印店老板记忆
- 活动群未读
- 勤工排班种子
- 宿舍缺席种子

玩家侧写法必须用物件、消息、收据、签字、门禁和具体动作承接，不能写成系统说明。
