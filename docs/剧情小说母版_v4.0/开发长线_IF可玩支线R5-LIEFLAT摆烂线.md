# 开发长线_IF可玩支线R5-LIEFLAT摆烂线

## 用途

本文用于把 `R5-LIEFLAT` 从第五幕站队分歧骨架扩展成可拆剧情页、选项页、变量回调和毕业结算的 B 级生活方式外流线。

它不是小说正文，也不是“少写内容”的省略线。它规定的是：玩家在第五幕强冲突后既不站队、也不当面修补、也不进入 5X 硬外流时，林亦舟的后续大学生活如何进入“静音、晚回、绕开、低投入、小事不处理和入口过期”的命运池。母本公共事件照常发生，但镜头、可操作内容和关系后果都按摆烂线重新嫁接。

## 路线定位

路线编号：`R5-LIEFLAT`

路线池编号：`POOL-R5-LIEFLAT`

路线等级：B 级生活方式线

共享范围：第一卷、第二卷共享；第三幕以后只按关键选择进入对应命运池；本线入口在第五幕站队分歧。

核心体验：林亦舟没有选择回到 4XX 当面修补，也没有选择周屿、唐骁、陆沉任一侧，更没有进入 5X 唯一硬外流。他只是每次都晚一点、少回一点、先刷一下手机、先打一局游戏、先去后街绕一圈。短期看没有大爆炸，长期看每个人都慢慢少问他一次。

本线重点不是惩罚玩家，而是把“不处理”也写成真实的可玩路径：玩家可以选择如何回避、回避到什么程度、什么时候想补、补的时候是不是已经过了最佳时间。摆烂线的代价不是戏剧化决裂，而是入口过期、低温失联、毕业时低存在感，以及“好像也没发生什么，但很多事都没再等他”。

## 进入条件

建议满足以下条件进入本线：

| 条件 | 建议值 | 说明 |
|---|---:|---|
| `avoidance` | `>= 3` | 玩家多次选择不当面处理 |
| `lost_chance` | `>= 2` | 至少两个关系入口已经错过 |
| `lieflat_score` | `>= 2` | 玩家连续选择省力、晚回、静音或无效回复 |
| `strong_alignment` | `false` | 未达到周屿、唐骁、陆沉或修补线的关系门槛 |
| `hard_outflow` | `false` | 未进入 `R5X-HARD` 唯一硬外流 |

## 不改变的母本锁点

- 第五幕宿舍强冲突必然发生。玩家不站队，不代表冲突自动消失。
- 周屿、唐骁、陆沉仍按母本生活半径推进：他们会继续活动、项目、勤工、毕业流程，只是不再默认等林亦舟。
- 晚风、许澈和后街照面只能作为软回声。玩家可以短暂逃开，但不能借摆烂线临时打开晚风线或 5X 硬外流。
- 第六卷的 C407 项目、课程演示、材料归档、关系定形仍发生，只是林亦舟更多作为旁观者、迟到者、已读未回者或最低责任参与者出现。
- 第七卷的预登记、毕业活动材料、开题 / 答辩、毕业照、清寝、最后一顿饭、离校手续仍发生。
- 5X 仍是唯一硬外流。本线内部回流不能临时打开 5X，也不能跳回周屿线、唐骁线、陆沉线或修补线主分支。

## 核心变量

| 变量 | 含义 | 高值倾向 | 风险 |
|---|---|---|---|
| `lieflat_score` | 摆烂倾向累计 | 更容易进入低投入、晚回、静音路径 | 高值会锁定低温结算 |
| `avoidance` | 回避程度 | 冲突热度暂时降低 | 入口慢慢关闭 |
| `lost_chance` | 已错过的当面机会 | 不会立刻爆炸 | 第六卷别人少问他一次 |
| `missed_chance` | 过期入口数量 | 可生成遗憾结算 | 补救不再能转深线 |
| `low_conflict` | 弱冲突累积 | 表面稳定 | 问题不爆，但关系变薄 |
| `late_regret` | 迟来后悔 | 第七卷可触发一句补话 | 不能直接回到深修补线 |
| `dorm_trust` | 4XX 是否还把林亦舟算进同一张桌 | 高值保留体面同框 | 低值时饭局、清寝、毕业照都少问他 |
| `old_dorm_distance` | 和旧宿舍的距离 | 关系淡化、低频同场 | 高值会走空白毕业 |
| `time_debt` | 拖延造成的时间债 | 事件低温推迟 | 截止时间会替关系做决定 |
| `money_pressure` | 夜市、游戏、临时花费和现实成本 | 逃开也有成本 | 钱压不能变成 5X 入口 |
| `hard_outflow_shadow` | 5X 硬外流影子 | 许澈 / 后街可软照面 | 高值只提示风险，不在本线开启 5X |
| `graduation_temperature` | 毕业结算温度 | 体面淡化、低温同框、空白离校或轻微补话 | 由前述变量综合决定 |

## 体量与密度

建议 6 章开发体量，对应约 45,000-70,000 字正文容量。本文只提供可开发骨架，不直接写足正文。

最低配置：

- 至少 24 次有效选择。
- 至少 5 个内部回流点。
- 每个内部回流点至少 2 个回流方向。
- 每个回流方向至少 3-4 个连续剧情拍点。
- 至少 3 次强后果回调：第六卷事项群 / 过期表格 / 第七卷饭局或毕业照。

## 章节总表

| 章 | 标题 | 母本事件 | 核心任务 | 关系压力 |
|---:|---|---|---|---|
| 1 | 你们先说 | `ME-STAND-BURST` / `ME-4XX-GROUP` | 进入不站队路径，处理第一轮静音和无效回复 | 不选任何人也会被理解成态度 |
| 2 | 一局就下 | `ME-WANFENG-CIRCLE` / `ME-DAILY-CAMPUS` | 用游戏、短视频或消息延迟逃开冲突 | 短逃让当晚安静，也让入口变短 |
| 3 | 后街不算外流 | `ME-DAILY-CAMPUS` | 去夜市、台球厅或便利店绕开 4XX | 许澈可软照面，但不能硬外流 |
| 4 | 过期表格 | `ME-CLASS-MATERIAL` / `ME-C407-DEMO` | 表格、材料、截止时间替关系做决定 | 不处理不等于无后果 |
| 5 | 湖边没人追出来 | `ME-NOT-ENTER-DOOR` | 想补又不进门，发现没人再追出来 | 低温不是吵架，是不再追问 |
| 6 | 最后一栏空着 | `ME-GRADUATION` | 第七卷嫁接：毕业照、饭局、清寝、离校 | 低存在感、体面淡化或迟来的轻微补话 |

## 章节开发详表

### 第一章：你们先说

章节目标：确定玩家进入摆烂线后的第一种代价。林亦舟没有选边，也没有回公共桌。他只是让群消息多停了一会儿。这个选择不会立刻毁掉关系，但会让别人下一次少问他一次。

现实触发：
- 4XX 事项群连续弹出消息。
- 周屿、唐骁、陆沉分别留下不同风格的短句。
- 林亦舟看见消息，但不想现在处理。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `LIEFLAT-CH01-N01` | 读取不站队入口 | 事项群 / 床上 / 走廊 | 林亦舟、4XX |
| `LIEFLAT-CH01-N02` | 第一轮静音 | 手机界面 / 4XX 门口 | 林亦舟 |
| `LIEFLAT-CH01-N03` | 无效回复后果 | 事项群 / 公共桌 | 4XX |

关键选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `LIEFLAT-CH01-C01` | 只回“你们先说” | `avoidance +1`、`dorm_trust -1`、`low_conflict +1` | 第四章表格默认不等他 |
| `LIEFLAT-CH01-C02` | 发一个中立表情 | `lieflat_score +1`、`misread +1` | 周屿当玩笑，唐骁当无效回复 |
| `LIEFLAT-CH01-C03` | 直接静音去做别的 | `lost_chance +1`、`late_regret +1`、`dorm_trust -1` | 当晚没事，后面少一个入口 |
| `LIEFLAT-CH01-C04` | 打字又删掉 | `avoidance +1`、`repair_attempt +0.5`、`time_debt +1` | 第五章湖边空转更明显 |

本章结尾钩子：群消息终于停了，林亦舟松了一口气，过了十分钟才发现没人再问他“你怎么看”。

### 第二章：一局就下

章节目标：把短逃写成可玩选择。游戏、短视频、晚风低频消息都不是新主线入口，而是让玩家暂时离开冲突现场的方式；逃开的时间会进入 `time_debt`。

现实触发：
- 冲突后林亦舟想开一局游戏或刷一会儿手机。
- 晚风或旧朋友发来低频消息。
- 4XX 群里仍有未处理事项。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `LIEFLAT-CH02-N01` | 游戏短逃 | 手机 / 床上 | 林亦舟、晚风软回声 |
| `LIEFLAT-CH02-N02` | 消息延迟 | 私聊 / 事项群 | 林亦舟 |
| `LIEFLAT-CH02-N03` | 回到现实 | 青枫居 / 门禁时间 | 林亦舟、4XX 软回声 |

关键选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `LIEFLAT-CH02-C01` | 打完一局再说 | `lieflat_score +1`、`time_debt +1`、`lost_chance +1` | 4XX 已经自己处理一部分 |
| `LIEFLAT-CH02-C02` | 中途退出游戏回群 | `late_regret +1`、`dorm_trust +0.5`、`time_debt +1` | 保留体面，但不转深修补 |
| `LIEFLAT-CH02-C03` | 只回晚风，不回 4XX | `avoidance +1`、`dorm_trust -1`、`wanfeng_low_echo +1` | 晚风只作为低频朋友回声 |
| `LIEFLAT-CH02-C04` | 两边都不回，继续装没看见 | `lieflat_score +2`、`missed_chance +1`、`graduation_temperature -1` | 第六章低存在感加重 |

本章结尾钩子：游戏结算页弹出“再来一局”，4XX 群的新消息只剩一句“算了，先这样”。

### 第三章：后街不算外流

章节目标：处理 5X 影子。林亦舟可以去后街夜市、便利店或台球厅短暂逃开，也可以遇见许澈，但本线不能开启 5X。这里的重点是：新桌子看起来轻松，但林亦舟没有真正跨出去。

现实触发：
- 林亦舟不想回寝，去后街买夜宵。
- 许澈或新圈人物短暂擦肩。
- 旧群消息仍在屏幕上，但他没有点开。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `LIEFLAT-CH03-N01` | 后街夜市 | 后街 / 便利店 | 林亦舟、生活人物 |
| `LIEFLAT-CH03-N02` | 许澈软照面 | 台球厅门口 / 便利店 | 林亦舟、许澈 |
| `LIEFLAT-CH03-N03` | 夜归回寝 | 青枫居楼下 | 林亦舟、4XX 软回声 |

关键选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `LIEFLAT-CH03-C01` | 买夜宵拖到门禁前 | `money_pressure +1`、`time_debt +1`、`avoidance +1` | 回寝时公共桌已散 |
| `LIEFLAT-CH03-C02` | 和许澈聊两句但不跟走 | `hard_outflow_shadow +1`、`old_dorm_distance +1` | 5X 只是影子，不锁线 |
| `LIEFLAT-CH03-C03` | 想跟新圈走，最后又回寝 | `hard_outflow_shadow +2`、`late_regret +1`、`lieflat_score +1` | 第六卷仍在旧命运池内 |
| `LIEFLAT-CH03-C04` | 直接回寝，但不上公共桌 | `low_conflict +1`、`dorm_trust -1`、`old_dorm_distance +1` | 弱冲突继续累积 |

本章结尾钩子：林亦舟在楼下看见 4XX 灯还亮着，却绕到自动贩卖机前多站了一分钟。

### 第四章：过期表格

章节目标：让“拖延”变成硬后果。表格、材料、截止时间不会替玩家保留情面；唐骁不再催，周屿不再绕着叫人，陆沉不再补一句。

现实触发：
- C407 或班级材料还有一次补交机会。
- 唐骁在群里贴出截止时间，但没有单独提醒林亦舟。
- 林亦舟发现自己其实能补，但已经过了最佳时间。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `LIEFLAT-CH04-N01` | 表格补交 | C407 / 共享文档 | 林亦舟、唐骁 |
| `LIEFLAT-CH04-N02` | 最低责任选择 | 事项群 / 课堂 | 林亦舟、4XX |
| `LIEFLAT-CH04-N03` | 系统关闭 | 教务平台 / 手机 | 林亦舟 |

关键选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `LIEFLAT-CH04-C01` | 立刻补最低项 | `late_regret +1`、`dorm_trust +0.5`、`missed_chance 0` | 保住流程，关系不转热 |
| `LIEFLAT-CH04-C02` | 继续拖，等别人提醒 | `missed_chance +2`、`dorm_trust -1`、`lieflat_score +1` | 唐骁不再单独催 |
| `LIEFLAT-CH04-C03` | 找人问“现在还能补吗” | `repair_attempt +1`、`late_regret +1`、`old_dorm_distance +1` | 对方回复事实，不展开旧账 |
| `LIEFLAT-CH04-C04` | 装不知道，让系统关闭 | `lost_chance +2`、`graduation_temperature -1`、`low_conflict +1` | 第六章毕业材料低温 |

本章结尾钩子：系统提示“提交已截止”，唐骁没有再发第二遍链接。

### 第五章：湖边没人追出来

章节目标：写摆烂线的低温核心。林亦舟想补，但没有进门；他希望有人追出来，结果没人追出来。不是大家恶意冷淡，而是他们已经习惯不等他。

现实触发：
- 林亦舟绕到湖边，不进 4XX。
- 夏知微、无名生活人物或宿管阿姨可短暂软照面。
- 4XX 群里的话题已经结束。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `LIEFLAT-CH05-N01` | 楼下空转 | 青枫居楼下 / 湖边 | 林亦舟 |
| `LIEFLAT-CH05-N02` | 软照面提醒 | 湖边 / 栈道 | 林亦舟、夏知微或生活人物 |
| `LIEFLAT-CH05-N03` | 回寝或继续绕 | 青枫居 | 林亦舟、4XX 软回声 |

关键选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `LIEFLAT-CH05-C01` | 进门，但只说一句生活小事 | `repair_attempt +1`、`dorm_trust +0.5`、`old_dorm_distance 0` | 体面保留，旧账不降 |
| `LIEFLAT-CH05-C02` | 继续绕湖，等情绪过去 | `avoidance +1`、`late_regret +1`、`old_dorm_distance +1` | 第六章饭局少问他 |
| `LIEFLAT-CH05-C03` | 和软照面人物聊别的，不聊冲突 | `low_conflict +1`、`missed_chance +1`、`external_echo +1` | 外部人物只做镜子，不开新线 |
| `LIEFLAT-CH05-C04` | 打开群想补，发现话题已过 | `lost_chance +1`、`late_regret +2`、`graduation_temperature -1` | 入口过期成为明确回声 |

本章结尾钩子：林亦舟终于回到门口，门内有人关灯，没人问他刚才去哪儿。

### 第六章：最后一栏空着

章节目标：完成第七卷毕业结算入口。摆烂线的毕业不是强决裂，而是空栏、少问、没留座、合照边缘和离校前没说出口的一句话。

现实触发：
- 毕业照、饭局、清寝、离校手续把旧人重新拉回同一流程。
- 表格里某一栏空着，或饭局投票没人再等林亦舟。
- 林亦舟想补，但很多事已经被流程替他结算。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `LIEFLAT-CH06-N01` | 毕业照边缘 | 操场 / 毕业照队列 | 林亦舟、4XX |
| `LIEFLAT-CH06-N02` | 最后一顿饭投票 | 群聊 / 饭店 | 林亦舟、4XX |
| `LIEFLAT-CH06-N03` | 离校空栏 | 学院窗口 / 校门 | 林亦舟 |

关键选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `LIEFLAT-CH06-C01` | 主动补一个具体小事，但不重开旧账 | `late_regret +1`、`graduation_temperature +1`、`dorm_trust +0.5` | 触发体面淡化结算 |
| `LIEFLAT-CH06-C02` | 在饭局投票里继续不表态 | `old_dorm_distance +2`、`missed_chance +1`、`graduation_temperature -1` | 可能没有留座 |
| `LIEFLAT-CH06-C03` | 给其中一人发迟来的私聊 | `repair_attempt +1`、`late_regret +2`、`low_conflict -1` | 可触发轻微补话，但不转深线 |
| `LIEFLAT-CH06-C04` | 什么都不做，按流程离校 | `lieflat_score +1`、`old_dorm_distance +2`、`graduation_temperature -2` | 触发空白离校 |

本章结尾钩子：离校手续盖完章，林亦舟看见那一栏终于不用填了，却想不起是谁最后一次问过他。

## 内部回流点

本线设置 5 个内部回流点。它们都只能回到 `POOL-R5-LIEFLAT`，不能借周屿、唐骁、陆沉、晚风或许澈照面开启其它完整路线。

### `RET-LIEFLAT-01` 静音事项群

| 方向 | 第1拍：现实触发 | 第2拍：照面或打断 | 第3拍：玩家选择 | 第4拍：回流与结算 |
|---|---|---|---|---|
| A 静音回 | 事项群连续弹消息。 | 林亦舟把手机扣下，屏幕还在亮。 | 静音一小时 / 只回收到 / 晚点再看。 | 截止时间拉回；结算 `avoidance`、`missed_chance`。 |
| B 表情回 | 群里等态度，林亦舟发了一个表情。 | 周屿接成玩笑，唐骁当成没回复。 | 继续表情 / 补一句最低事实 / 撤回。 | 回弱冲突主线；结算 `lieflat_score`、`misread`。 |
| C 没人问回 | 等他再打开群，话题已经结束。 | 新消息只剩“那就这样”。 | 追问结果 / 当作结束 / 私聊一个人。 | 回主线时结算 `lost_chance`、`late_regret`。 |

### `RET-LIEFLAT-02` 游戏局短逃

| 方向 | 第1拍：现实触发 | 第2拍：照面或打断 | 第3拍：玩家选择 | 第4拍：回流与结算 |
|---|---|---|---|---|
| A 一局回 | 冲突后想开一局游戏。 | 4XX 群消息压在通知栏上。 | 打完一局 / 中途退 / 直接关机。 | 游戏结束回；结算 `time_debt`、`lieflat_score`。 |
| B 晚风回 | 晚风发来低频消息。 | 她不知道 4XX 正在出事。 | 回晚风 / 说今天不聊 / 不回任何人。 | 回摆烂主线；结算 `wanfeng_low_echo`、`dorm_trust`。 |
| C 门禁回 | 一局结束时门禁快到点。 | 楼道里已经没什么声音。 | 赶回去 / 再打一局 / 发群问还要不要说。 | 回寝时结算 `lost_chance`、`late_regret`。 |

### `RET-LIEFLAT-03` 后街夜市

| 方向 | 第1拍：现实触发 | 第2拍：照面或打断 | 第3拍：玩家选择 | 第4拍：回流与结算 |
|---|---|---|---|---|
| A 夜宵回 | 林亦舟不想回寝，去后街吃夜宵。 | 钱包余额和门禁时间一起提醒。 | 买烧烤 / 买便利店饭团 / 不买只走。 | 夜归回；结算 `money_pressure`、`avoidance`。 |

| C 旧群回 | 后街很热闹，旧群未读还在。 | 林亦舟点开又退出。 | 回一句 / 删除草稿 / 当没看见。 | 回低温主线；结算 `old_dorm_distance`、`late_regret`。 |

### `RET-LIEFLAT-04` 过期表格

| 方向 | 第1拍：现实触发 | 第2拍：照面或打断 | 第3拍：玩家选择 | 第4拍：回流与结算 |
|---|---|---|---|---|

| C 关闭回 | 系统提示提交已关闭。 | 屏幕上没有人骂他。 | 截图留着 / 关掉页面 / 发群道歉。 | 回第六章毕业材料；结算 `lost_chance`、`graduation_temperature`。 |

### `RET-LIEFLAT-05` 湖边空转

| 方向 | 第1拍：现实触发 | 第2拍：照面或打断 | 第3拍：玩家选择 | 第4拍：回流与结算 |

| B 软照面回 | 夏知微或生活人物短暂出现。 | 对方只聊天气、照片或路灯，不替他解围。 | 聊别的 / 说一点冲突 / 直接告别。 | 回主线；结算 `external_echo`、`missed_chance`。 |
| C 没人追回 | 林亦舟期待有人问他在哪。 | 手机没有新消息。 | 主动回去 / 当作没人需要 / 私聊一个人。 | 回毕业低温；结算 `old_dorm_distance`、`graduation_temperature`。 |

## 对话逻辑

### 林亦舟

- 不能写成突然变坏。他只是每次都想晚点再说、明天再补、先不打扰。
- 他的内心不是没感觉，而是不断把感觉推迟到“现在不适合说”。
- 摆烂线的选择必须有轻微诱惑：短期安静、少挨骂、少站队、少承担。
- 每个轻松选择后都要留下一个小后果：少问一次、少留一个座、少等一分钟、少发一条私聊。

### 周屿

- 周屿会先用玩笑帮他糊过去，后面慢慢不再叫他去热闹里。
- 不能把周屿写成主动抛弃林亦舟；他的退场是“算了，他可能不想来”。

### 唐骁

- 唐骁会从催促变成只发群公告。不是报复，而是默认林亦舟不会及时接。
- 他对摆烂线的压力是流程不再为林亦舟单独保留余地。

### 陆沉

- 陆沉不会追着问。林亦舟越不处理，陆沉越容易把这当成一种默认边界。
- 不能把陆沉的沉默复制成摆烂线的省事理由；陆沉线是边界，摆烂线是逃避。

### 晚风 / 许澈 / 软照面人物

- 晚风只能作为低频消息回声，不能把摆烂线改成晚风线。
- 许澈只能作为 5X 影子，不能在本线里锁硬外流。
- 夏知微、沈嘉禾、许棠或生活人物可以提醒“过期”和“授权”，但不能替玩家修补宿舍关系。

## 毕业结算

| 结算 ID | 条件倾向 | 结局温度 |
|---|---|---|
| `LIEFLAT-END-POLITE-FADE` | `late_regret >= 3`, `dorm_trust >= 2`, `missed_chance <= 3` | 毕业时仍有体面同框，但旧账不再展开，像一段低声淡化的关系 |
| `LIEFLAT-END-LOW-PRESENCE` | `lieflat_score >= 5`, `old_dorm_distance >= 4` | 林亦舟没有被讨厌，只是很多名单、饭局和合照都不再默认算他 |
| `LIEFLAT-END-LATE-MESSAGE` | `repair_attempt >= 2`, `late_regret >= 4` | 离校前发出一条迟来的私聊，对方回复礼貌，但不重开旧线 |
| `LIEFLAT-END-MISSED` | `missed_chance >= 5` 或 `lost_chance >= 5` | 不是大事毁掉关系，而是很多入口过期，毕业时像被生活轻轻推出去 |
| `LIEFLAT-END-5X-SHADOW` | `hard_outflow_shadow >= 3`, `hard_outflow = false` | 林亦舟看见过新圈入口，但没有真正进入；旧人远了，新人也没接住 |

## 漂移防线

- 本线不是“省内容线”。玩家每一次回避都必须产生变量、回声、时间债或过期入口。
- 本线不是“中立线”。不站队不会保住所有关系，只会让别人逐渐不再默认等他。
- 本线不是 5X。后街、许澈、新圈只能作为硬外流影子，不能在本线打开完整 5X。
- 本线不能把晚风写成自动安慰，也不能把外部人物写成替 4XX 关系兜底的人。
- 本线不能让内部回流打开其他命运池。所有回流只改变本线关系温度、误读、资源代价和毕业结算。
- 本线不能取消母本公共锁点，只能改变林亦舟进入锁点的位置：更迟、更边缘、更少被问。

## 开发优先级

1. 先实现入口 `P0D_STAND_NO_REPLY` 与路由池锁定。
2. 再实现 6 章主事件链。
3. 再实现 5 个内部回流点，每个回流点至少 2 个方向。
4. 再写第六卷公共锁点嫁接：事项群 / C407 / 表格过期 / 关系定形。
5. 最后写第七卷毕业结算：饭局 / 毕业照 / 清寝 / 离校空栏。
