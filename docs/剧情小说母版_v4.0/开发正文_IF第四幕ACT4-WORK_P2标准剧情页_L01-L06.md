## 用途

本文把 `R4-WORK` 第四幕 6 个生活半径锁点统一整理为 P2 正文稿。它不是正式 JSON，也不新增独立主线；后续应先做完整审读，再统一拆成页级 JSON 字段。

执行口径：

- 先写完整正文，再统一拆 JSON。
- 每页只抓一个现场重心，先交代前因后果。
- 非关键对白用第三人称转述，关键边界句保留。
- 每个选择窗 3 项：1 个工作线主轴推进 + 2 个池内微内流点。
- 每个方向都保留选择后反馈和承接页。
- 全部内容留在 `POOL-R4-WORK`，其它人物只做当前池软回声。

2026-05-26 试玩器确认口径：

- 本稿已用 `playtest/r4-work-act4-preview.html` 做 L01-L06 可操作样板确认；预览口径可作为后续 R4-WORK 第四幕正式 JSON 拆分的玩家侧文本标准。
- 玩家页不得出现开发说明、系统词或路线解释，例如“本窗抉择”“微内流点”“主轴推进”“当前池”“最低事实”“具体改法”“风险”“代价”“收入入口”“低频信任”等。
- 抉择引导文只承接当前剧情压力，必须像现场自然逼出的下一步，不能像在解释选项功能。
- 主选项写成玩家能理解的方向动作；微心态选项必须写成你此刻会真实发出、说出口、写下或做出的具体内容。
- 微心态卡片说明只写该动作落下后的即时现场反应，不能写“沟通策略”“策划标签”“最优路径暗示”或变量后果。
- 内流点仍可服务变量和 Act5 回声，但变量、回声、锁池说明只能留在写回字段或 JSON 字段，不进入玩家正文。

## R4-ACT4-L01 空宿舍与暑假分流

母本锁点：第四卷第一章《空宿舍》

### `ACT4-WORK-L01-P2-S01` 青枫居 4XX / 打包声

暑假去向通知弹出来时，林亦舟正蹲在 4XX 地上给行李箱合扣。

周屿的箱轮卡在门槛边，唐骁把证书材料塞进文件袋，陆沉坐在公共桌旁，手机屏幕还亮着。

班群通知里，留校、返家、实习、兼职并排在页面上，每一项后面都有说明栏。

晚风刚发来光谷截图，母亲的未接电话压在上面。林亦舟低头看着屏幕，手里的箱扣迟迟没有扣上。

周屿问他是不是也要回家。他看了一眼空床下面那只没收好的拖鞋，没有立刻答。

### `ACT4-WORK-L01-P2-S02` 4XX 公共桌 / 三个入口

公共桌很快被几样东西占满：回家车次、晚风截图、勤工群公告，还有唐骁摊开的证书材料。

林亦舟把手机放在桌上，屏幕自动暗下去，又被母亲的电话点亮。

她问票看了没有。父亲明天要走，能见就见一面。

陆沉把自己的手机推过来，群公告停在暑期勤工表上。图书馆、楼栋登记、明德楼窗口，一格一格排得很细。

林亦舟没有立刻接哪一边，只把桌上的纸往自己这边拢了一下。纸角互相压住，手机又震了一下。

### `ACT4-WORK-L01-P2-S03` 4XX 阳台 / 勤工群

阳台风把衣架吹得轻轻相碰。林亦舟站在门边，看陆沉把勤工表放大。

第一次报名要本人到场，身份证复印件和学生证都要带，窗口班不能迟到。

他把这几项记进备忘录。每打一行，晚风的截图就往通知栏下面沉一点，母亲电话还挂在未接记录里。

楼下传来箱轮声。周屿已经下楼，唐骁把文件袋夹在胳膊下，4XX 的门口一下空出来。

留校也要每天出门、签到、带复印件。林亦舟看着那张表，才发现空宿舍不等于空白时间。

### `ACT4-WORK-L01-P2-S04` 空宿舍 / 电话

4XX 空出一半后，铃声在屋里响得很清楚。

林亦舟接起母亲电话。她没有催，只说父亲明天要走，站外见一面也行。

晚风的光谷截图还在屏幕上，陆沉那张勤工表也摊在桌边。他一只手拿着手机，另一只手按住被风吹起的纸角。

电话那头安静了几秒。母亲问：“票能不能先看一下？”

三件事挤在同一块屏幕上。林亦舟的拇指停在最上面的输入框。

### `ACT4-WORK-L01-P2-CHOICE-01`

### `ACT4-WORK-L01-P2-A01` 接住家里电话

林亦舟先把母亲的电话接住。

她问票，他就把购票页重新打开；她说父亲明天要走，他把到站时间记进备忘录。

晚风截图还亮着，勤工表也还摊在桌边。林亦舟没有解释太多，只先回：我看票，明天去站外见他。

母亲那边正在输入很久，最后回：别买太晚的，路上注意。

这句话落下来后，明天的第一格已经被写上了。

### `ACT4-WORK-L01-P2-A02` 接住家里电话

晚上十点，林亦舟把明天行程排出来。

站外见父亲排在前面，光谷和勤工窗口都被挪到后面。

陆沉看见他打开车次页，只提醒别卡最后一班。

林亦舟点头，把手机放回桌上。通知栏还亮着，但家里这通电话没有再变成未接。

### `ACT4-WORK-L01-P2-B01` 问清勤工

林亦舟挂掉电话，先问陆沉勤工怎么排。

陆沉把群公告点开：本人报名，带身份证复印件和学生证，窗口班不能迟到。

林亦舟把几项抄进备忘录。母亲还等车次，晚风还等光谷时间，但勤工已经从一条消息变成能跑的流程。

陆沉没有劝他，只把常温水推过来。

### `ACT4-WORK-L01-P2-B02` 问清勤工

4XX 只剩桌前一盏灯。

林亦舟把报名要求写在一页，又把父亲、光谷、车票写在下面。

四行字互不相让。他盯着它们看了很久，最后先把勤工群置顶。

缺口没有少，但现在至少有一条能从明天开始跑。

### `ACT4-WORK-L01-P2-C01` 回应晚风

林亦舟下楼买水时，先点开晚风的截图。

输入框里原本想打“到时候看”，他删掉了。

他回她：明天先处理家里，光谷不能按原时间。我今晚告诉你能不能改短。

晚风过了一会儿回：光谷不会跑，你别从朋友圈才让我知道。

站在便利店门口，塑料袋勒着手指，林亦舟回了一个好。

### `ACT4-WORK-L01-P2-C02` 回应晚风

回到楼上，4XX 的门半开着。

晚风聊天框安静下来，但母亲的电话和勤工表还在。

林亦舟把水放到公共桌上，瓶底碰到桌面，声音很轻。

她没有把他拉走，只让他不能继续失踪。

## R4-ACT4-L02 生活费与退改规则

母本锁点：第四卷第二章《退改规则》

### `ACT4-WORK-L02-P2-S01` 4XX / 购票软件

林亦舟把购票软件打开，车次一排排挤在屏幕上。

清晨班便宜，但要凌晨出门；下午班稳，却撞上光谷；晚上那班只剩候补，下面还写着退改规则。

他点进规则页，又退出来。灰色小字没有骂人，只把钱和时间写得很硬。

母亲发来一句：票别太晚。晚风问他能不能给个准话。

林亦舟把手机扣在桌上，掌心还记着屏幕发热的位置。

### `ACT4-WORK-L02-P2-S02` 陆沉桌边 / 排班眼光

陆沉坐在桌边看自己的排班表。林亦舟把票价给他看，他只扫了一眼。

“改签别拖到最后。”陆沉说完，就把勤工群公告翻出来。

报名材料、窗口时间和复印件要求挤在一起，林亦舟才发现省下的那几十块不只是车票钱。

公共桌上没有人替他算账。车票、饭卡、打印费、去光谷的路费，各自占着一行。

他把它们写在同一张备忘录里，第一行很快就不够用了。

### `ACT4-WORK-L02-P2-S03` 东区商业街便利店 / 兼职表

东区商业街的便利店门口，冰柜嗡嗡响。

林亦舟站在灯下，把兼职表重新打开。老板娘在收银台后面扫了一眼，说暑假人少，能固定最好。

他没立刻应。手机里父亲的到站时间、晚风的光谷截图、母亲催票的消息还排在一起。

便利店玻璃门映出他的脸，也映出手机屏幕。他忽然不想再把每个人都回成“我看看”。

他把常温水放到柜台上，付款前又看了一眼余额。

### `ACT4-WORK-L02-P2-S04` 商业街灯下 / 三个聊天框

商业街灯亮起来时，三个聊天框都还在。
