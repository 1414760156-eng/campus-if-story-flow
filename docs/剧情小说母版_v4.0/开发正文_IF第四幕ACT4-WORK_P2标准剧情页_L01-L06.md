## R4-ACT4-L01 空宿舍与暑假分流

母本锁点：第四卷第一章《空宿舍》

### `ACT4-WORK-L01-P2-S01` 青枫居 4XX / 打包声

午后的蝉声被拉链声压住。

唐骁把暑假表压在公共桌上，表格按日期排得很窄，像每一天都已经提前被切开。周屿一边收耳机线，一边说回家路上还得补听施然发来的音频。陆沉没接玩笑，他把床单折进留校行李袋，动作比平时慢一点。

林亦舟拖箱子时带出一只灰袜子。

周屿笑了一声，唐骁抬眼，陆沉把袜子捡起来放到椅背上。笑声很短，很快又被新的拉链声盖住。

公共桌上只剩几样没被分走的东西：半包纸巾、空调遥控器、唐骁压着的暑假表、陆沉手机里亮着的留校登记截图。林亦舟站在桌边，突然意识到暑假不是所有人一起离开。

它更像一张表。

每个人都被填进不同的格子里。

外卖送到楼下时，唐骁把暑假表往旁边挪了挪，给饭盒腾出位置。林亦舟的手机就在这时亮了第一次。

### `ACT4-WORK-L01-P2-S02` 4XX 公共桌 / 三个入口

外卖盒摊开后，手机接连亮了三次。

晚风发来光谷歌会截图，页面底部写着退票和改期说明。
母亲发来一句：能不买就先别买。
父亲的生活费备注只有四个字：路上用。

林亦舟没有马上回。

饭卡余额停在屏幕上，数字不算难看，却经不起车票、打印费和两顿饭一起扣。周屿还在说回家的车，唐骁低头核对暑假要交的证书材料，陆沉把自己的饭盒盖上，提醒他常温水还剩一瓶。

林亦舟把余额页划掉。通知栏没有消失，三条消息还叠在那里，像三扇门同时开着。

晚风那边是光谷。
母亲那边是回家。
父亲的钱把“路上”两个字压得很重。

他抬头时，陆沉正看着阳台外面，没有问他怎么选。过了一会儿，陆沉的手机震了一下。

“勤工群发暑期表了。”他说。

### `ACT4-WORK-L01-P2-S03` 4XX 阳台 / 勤工群

阳台上的衣架被风吹得轻轻碰在一起。

陆沉把手机递过来。暑期值班表排得很细：图书馆、楼栋登记、明德楼窗口，时间被切成一格一格。林亦舟看了两眼，才发现留校不是“有空”。

留校只是换了一种更碎的排法。

“第一次去要带身份证复印件。”陆沉说，“别让别人代签。”

这话不像劝他留下，也不像帮他决定。它只是把一条现实路摆出来：能换饭钱，能减少向家里开口，也会占掉白天。

林亦舟看着那些空格。每一格都不大，却足够把光谷、回家、补考和宿舍剩下的时间挤到边上。

楼下传来车轮压过减速带的声音。周屿的箱子已经拖到门口，唐骁把证书材料塞进包里。

宿舍快空了。
可通知栏还满着。

### `ACT4-WORK-L01-P2-S04` 空宿舍 / 电话

周屿和唐骁走后，4XX 突然空出一半。

桌上的外卖盒被收走了，暑假表还留着。空床位比平时更亮，风从阳台进来，把陆沉那张勤工表吹得轻轻抬了一下。

晚风又发来一条：周二下午到光谷。

母亲电话紧跟着压进来。铃声在空床之间响得很清楚。

林亦舟接起电话。母亲先问他票看了没有，又说父亲明天要走，能见就见一面。她没有催，可每一句都像把时间往前推了一格。

陆沉坐在对面，没有抬头，只把勤工表往桌边推了推。

林亦舟看着光谷时间、空床位和那张暑期值班表。到这一步，选择已经不是去哪边比较开心。

而是哪一边可以被他说清楚。
哪一边只能继续欠着。

### `ACT4-WORK-L01-P2-A01` 接住家里电话 / 现场行动

林亦舟把母亲电话接完，没有马上去点晚风的截图，也没有问陆沉勤工表。

他先回母亲：我看票，明天去站外见他。

母亲那边正在输入了很久，最后只回：别买太晚的，路上注意。

这句话没有责备，却把明天的时间压得更实。林亦舟把父亲到站时间、车票、站外半小时写进备忘录，勤工表还摊在公共桌边，纸角被风吹起又落下。

晚风的光谷消息仍然亮着。他知道自己会欠一个答复，但至少家里这通电话没有被他拖成又一条未读。

### `ACT4-WORK-L01-P2-A02` 接住家里电话 / 承接回声

晚上十点，林亦舟终于把明天行程排出来。

先去站外见父亲，再看能不能赶回学校。光谷被他暂时挪到后面，勤工窗口也没有立刻问清。这个顺序不漂亮，却比什么都不说要稳一点。

陆沉看见他把车次页打开，只提醒：“别卡最后一班。”

林亦舟点头。公共桌上还留着暑假表和勤工表，通知栏里也还压着晚风的消息。家里的门被他先接住了，另外两扇门没有关，只是都往后推了一格。

### `ACT4-WORK-L01-P2-B01` 问清勤工 / 现场行动

林亦舟把母亲电话挂断后，没有立刻回晚风。

他问陆沉：“勤工怎么排？”

陆沉看了他一眼，把群公告点开。需要本人报名，第一次带身份证复印件和学生证，窗口班不能迟到，临时换班要提前说。陆沉讲得很慢，每一句都像在把“缺钱”从情绪里拆成可执行的流程。

晚风的消息还停在屏幕顶端，母亲也还在等他给票。林亦舟没有点开，只把勤工群保存进通讯录。

### `ACT4-WORK-L01-P2-B02` 问清勤工 / 承接回声

4XX 只剩陆沉桌前的台灯亮着。

林亦舟把报名要求抄进备忘录，旁边又写下光谷、父亲、回家票。四行字互不相让，谁也没有被真正解决。

陆沉把常温水推过来：“固定班不是轻松，是少变数。”

林亦舟点头。少变数也要付代价。晚风那条“周二下午到光谷”仍然没有回，母亲也还在等票。只是从这一刻起，他至少知道自己想从哪里开始补缺口。

### `ACT4-WORK-L01-P2-C01` 回应晚风 / 现场行动

林亦舟下楼买水时，才点开晚风的截图。

他没有回“到时候看”，只打字：我明天先处理家里的事，光谷不能按原时间。我今晚告诉你能不能改短。

晚风那边过了一会儿发来语音。她没有问他是不是不想来，只说：“光谷不会跑。你别从朋友圈让我知道你已经回去了。”

林亦舟站在便利店门口，手里的常温水被塑料袋勒出响声。他回：我提前说。

### `ACT4-WORK-L01-P2-C02` 回应晚风 / 承接回声

回到楼上时，周屿和唐骁已经拖着箱子下去了。

4XX 的门半开着，陆沉在里面叠被套。林亦舟把水放到公共桌上，晚风的聊天框终于安静下来，但安静不等于问题解决。

他仍然要看票，仍然要给母亲回车次，也仍然要决定要不要问勤工。晚风没有把他拉走，只是在这个暑假开始的位置，留下了一条要求：不能失踪。

## R4-ACT4-L02 生活费与退改规则

母本锁点：第四卷第二章《退改规则》

### `ACT4-WORK-L02-P2-S01` 4XX / 购票软件

林亦舟把回家车次一班班拉出来。

清晨班便宜一点，但要凌晨三点半起，打车去汉口站；下午班最稳，却正好撞上晚风的光谷时间；晚上那班只剩候补，退改费另算。

母亲最后打来电话。

## R4-ACT4-L04 补考与打印店

母本锁点：第四卷第四至第五章《低空飞过》《不要卡点》

### `ACT4-WORK-L04-P2-S01` 青枫居 4XX / 成绩页

凌晨前，教务系统终于刷进去。

成绩表一格一格跳出来，白底黑字，像把这个暑假前欠下的账摊到了屏幕上。林亦舟那一行卡在“数据结构入门”后面，灰色小圈转了几次，页面往下一沉。

数据结构入门：59。

旁边没有红字，没有弹窗，只有一行安静的“待补考安排”。公共桌上，父亲给的信封还压在课本下面，旁边放着陆沉夹好的食堂小票、两瓶常温水和一支快没水的黑笔。

陆沉问：“出了？”

林亦舟盯着屏幕：“五十九。”

陆沉没有说“差一分而已”。他只是把自己的手机转过来一点，上面写着：数据结构入门，64。

“也不好看。”他说，“但能过。”

母亲的消息在这时弹出来：成绩查了吗？
晚风也发来一张游戏截图，角色站在出生点没动。

林亦舟看着这些聊天框。补考忽然不再只是一门课，它会占掉白天，会要打印费，会让他解释为什么不回家、为什么不见面、为什么还要去问勤工窗口。

他给母亲回：有一门没过，数据结构入门，补考安排下周出。

很久以后，母亲只回了一句：先睡，明天再说。

陆沉把勤工表往自己那边收了收：“明早我去图书馆。你要来的话，二楼靠窗那排空。”

林亦舟点了点头。点完才发现，明天已经被切开了：成绩、母亲、晚风、唐骁的资料、陆沉的图书馆，还有不知道能不能接上的勤工窗口。

### `ACT4-WORK-L04-P2-S02` 青枫居楼下外卖架 / 资料包

第二天中午，青枫居楼下的外卖架比前几天满。林亦舟拎起自己的两瓶水，旁边有人对着手机念：“补考资料包二十九块九，真的假的？”

年级群里也跳出相似链接，标题写得很满：往年题、老师重点、押题卷。链接很快被撤回，辅导员补了一句：补考以学院通知和课程教师安排为准。

宿舍群跟着亮起来。周屿先说自己差点点进去，唐骁追问他哪门课需要。周屿把玩笑绕回林亦舟，说只是替他先踩个雷。

林亦舟手里的水瓶被捏出一点响。

他在群里回：不用替我。

群里安静了一下。周屿很快解释自己没真买，林亦舟没有顺着玩笑往下接。外卖架旁刚才念资料包的男生把手机揣回兜里，拎着盖饭走了。

唐骁随后私聊发来一个文件夹截图：课件目录、实验题编号、错题空表。

唐骁：资料包别买。先保存成绩页。

林亦舟回：发目录就行。

水瓶外面的冷雾已经化掉。他把两瓶水带上楼，公共桌上那只信封还在原位。资料包没有买，补考也没有少一点；只是“谁能替他先试一试”这件事，被他先按住了。

### `ACT4-WORK-L04-P2-S03` 图书馆二楼 / 借阅单

图书馆二楼靠窗的位置空着。陆沉把一张借阅单推到林亦舟书边，又把备用圆珠笔放过去：“列个表，从第一章开始。”

林亦舟翻开《数据结构入门》，在第一行写下“线性表”。唐骁的压缩包里没有“必过”和“押题”，只有课件目录、实验题编号和错题空表。电脑风扇转起来，盖过了空调声。

晚风在这时发来语音。她说室友处理票务骗局时也用了空表，聊天记录和付款记录按时间排开，白天排队虽然烦，但至少有柜台、复印机和人。

语音后面跟着一句文字：白天能排队。

林亦舟把手机扣到桌上。陆沉从登记台那边抬头：“窗口三点半前还有人。”

这句话把下午切成两段。前半段给数据结构，后半段给勤工窗口；中间还要打印、确认 C201，给母亲回车票，也给晚风一个不用等到深夜才知道的安排。

### `ACT4-WORK-L04-P2-S04` 打印店 / 教学楼 C201

打印店柜台旁贴着价目表，下面另有一张手写纸：不代买，不推荐任何包过资料。

林亦舟把唐骁给的第一、二章目录和实验题传过去，又单独打开勤工申请表。

老板听见旁边有人问补考资料包能不能打，只说：“我只打你发来的文件。包过两个字，听着就不便宜。”

C201 在教学楼二楼尽头。陆沉按了按门把，说从青枫居过来十五分钟，下雨二十。年级群又提醒补考不要卡点。

林亦舟拍下门牌。镜头里没有人，只有门牌、楼梯口和走廊尽头的饮水机。可他知道第一周那天会有人排队，会有人翻学生证，也会有人临到门口才发现黑笔没带。

文件袋边缘在走廊风里轻轻响。现在他得决定，下午剩下的时间先交给哪一边。

### `ACT4-WORK-L04-P2-A02` 先补学业材料 / 承接回声

三点三十七，那条窗口消息被别人接走。林亦舟没有立刻解释，也没有把这件事说成“没关系”。他在错题空表背面写下：工作窗口错过一次，补考资料完成第一章。

林亦舟把手机放到一边，继续往下抄第二章目录。窗外有人推着一车快递经过，胶带声贴着玻璃划过去。窗口少了一次，补考资料多了一页；这两个结果都留在下午，谁也没替谁消失。

林亦舟站在教学楼一楼大厅，先回勤工群：周四下午我可以。

消息发出去后，他才把数据结构资料塞进书包外层。第一页“线性表”露出一角，被拉链压住。它没有消失，只是被排到稍后。

陆沉看见他的动作，没有说先看书，只问学生证带没带。林亦舟把证件袋摸出来给他看。陆沉点头：“别卡点。”

晚风问他是不是还要排光谷。他回：今天不排。晚风只回了一句：这句比“到时候看”好。

回青枫居的路上，书包外层的纸被风轻轻拍着他的手臂。那是数据结构前两章，已经被他放到稍后。周四下午的窗口能换来饭钱和打印费，可拉链压住的那一角也在提醒他，补考不会因为接了班就自动通过。

### `ACT4-WORK-L04-P2-C02` 把白天能处理的事说清 / 承接回声

群里这次没有冷成一块。周屿问明德楼窗口是什么新副本，唐骁纠正是勤工窗口；陆沉提醒周四上午别排别的；唐骁补一句第一章今天看完；周屿把玩笑收住，说自己周二到，可以带吃的，但不带资料包。

林亦舟站在教学楼大厅，把手机屏幕熄灭。补考、钱、勤工和家里都还在，但它们不再全部躲在一句“到时候看”后面。

大厅门外的风把暑期安全提醒吹得贴到玻璃上，纸角卷起。林亦舟把文件袋抱紧了一点，往外走。问题没有少，只是每个人终于知道自己要等的是什么。

位置被看见了，补考也被看见了。但这一顿饭没有把失败做成笑话，也没有把关心做成审问。林亦舟把筷子拆开，终于能说一句具体的：“链表写完了。”

林亦舟把兼职和留校说明都填进去。

### `ACT4-WORK-L06-P2-B01` 只写留校 / 现场行动

林亦舟没有把兼职细节写满。

周屿说：“你至少写个人能找到。”

陆沉没有替他编理由，只把留校申请样式推过去。林亦舟把页面保存，知道这张表能过，但过得不干净。

### `ACT4-WORK-L06-P2-B02` 只写留校 / 承接回声

B204 签字时，辅导员看了备注一眼。

“后面窗口确定了，补一次说明。”

林亦舟点头，把这句话记进备忘录。签字栏落下去，纸面上没有红字，也没有当场退回，但那块空白已经跟着他回到 4XX。

晚上，周屿问他到底算留校还是兼职。林亦舟说：“现在算留校，后面补窗口。”

这句话能让表格通过，却不能让误读消失。第五幕如果进入工作主轴，第一件事就要补这块说明。

### `ACT4-WORK-L06-P2-C01` 先问父母 / 现场行动

林亦舟没有按默认顺序保存联系人。

他站到阳台，先给父亲发：学校要真实有效，忙的时候能接电话吗？

父亲隔了几分钟回：你妈方便就写她。

他又给母亲发：我把你放第一联系人。父亲放第二。

母亲只回：可以。别怕他多想。

林亦舟看着这句话，手指停了一会儿，才回到页面把两个号码重新排好。

### `ACT4-WORK-L06-P2-C02` 先问父母 / 承接回声

联系人保存成功后，系统弹出确认页。

母亲在第一栏，父亲在第二栏。这个顺序没有解决家里的钱，也没有让父亲的体面变轻，但它至少让学校真能找到人。

林亦舟把截图发到家庭群，没有解释太多，只说：按学校要求填好了。

父亲没有回，母亲回了一个“嗯”。阳台外面天已经暗下来，4XX 里有人催签字。林亦舟把手机收好，拿着黑笔往 B204 去。联系人不是系统默认，是他自己排出来的。
