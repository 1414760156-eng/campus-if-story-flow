# IF 可玩支线合理性审查（chinese-novelist）

审查时间：2026-05-19

审查对象：

- `开发索引_IF命运分歧开发矩阵.md`
- `开发规格_IF路线Schema与P0节点池.md`
- `开发索引_IF分支P0文本总索引.md`
- `开发规划_IF可玩支线长篇扩展总表.md`

本审查不按出版小说评估，而按互动剧情游戏母版评估。`chinese-novelist` 在这里的作用不是把开发稿写成正文，而是检查每条 IF 是否具备小说级的因果、人物张力、行动代价和后续钩子。

核心标准只有四条：

1. 玩家选择必须改变林亦舟的生活半径，而不是只改好感数字。
2. 每条路线必须有吸引力，也必须有现实代价。
3. 主线锁点不能被 IF 改写，只能改变林亦舟抵达这些锁点时的位置、关系和温度。
4. 每条线都要能在第七卷毕业节点结算出不同温度，而不是最后统一回同一个圆满结局。

## 总结论

当前支线开发方向是合理的，可以继续推进。

最重要的合理性在于：它已经从“给支线人物写外传”改成了“时期外流点 + IF 命运分歧”。这更适合游戏，也更符合母版小说的功能。玩家不是在每个事件里把世界切成无数平行宇宙，而是在第三幕、第四幕、第五幕几个关键时期选择林亦舟靠近哪种生活。

这会带来更好的可玩性：

- 选择宿舍修补，就获得更多公共桌、饭局、旧账、清寝和毕业照压力。
- 选择活动 / 完美，就获得更多公开表达、流程、材料和被工具化的代价。
- 选择兼职，就获得更多钱、排班、生活费、错过和现实压力。
- 选择亲密关系，就获得更多陪伴、礼物、公开边界、误读和宿舍缺席。
- 选择某个室友，就获得更深的单人关系，也承担另外两个人的误读。
- 选择摆烂，就不是立刻爆炸，而是入口慢慢过期。
- 选择 5X，就进入唯一硬外流，获得轻松的新圈，也失去旧关系的重量。

这套方向比“每个 NPC 都写一条长外传”更稳。因为它把支线人物放回了游戏结构里：他们不是抢走主线，而是支撑林亦舟不同生活方向的血肉。

## 已经成立的部分

### 1. 分歧边界清楚

矩阵明确写了：主线事件一定发生，玩家改变的是林亦舟如何经过这些事件。这一点非常关键。

这能避免剧情漂移。周屿仍会进入活动圈，唐骁仍会走项目和流程，陆沉仍会有现实压力，晚风仍有自己的生活圈。玩家不能把别人改写成专属 NPC，只能改变林亦舟和他们之间的距离。

### 2. 路线不是奖励，而是生活选择

当前九条路线都有基本生活方向：

| 路线 | 生活方向 | 主要代价 |
|---|---|---|
| `DEFAULT-4XX` | 留在宿舍旧账里修补 | 被旧关系拖住，外部机会减少 |
| `R3-PERFECT` | 活动、流程、公开信用 | 被工具化，关系低温 |
| `R4-WORK` | 打工、钱、排班 | 错过、疲惫、被误读冷淡 |
| `R5-ROMANCE` | 亲密关系和外部陪伴 | 钱、时间、公开边界、宿舍缺席 |
| `R5-ZHOU` | 热闹、面子、活动圈 | 被误读偏袒，热闹也会伤人 |
| `R5-TANG` | 规则、项目、公平 | 被误读成审判，关系变窄 |
| `R5-LUCHEN` | 沉默、现实压力、边界 | 不能替他说，也不能救赎他 |
| `R5-LIEFLAT` | 回避、自保、低投入 | 入口过期，别人不再问 |
| `R5X-HARD` | 新圈子、无旧账、硬外流 | 新关系轻但薄，旧人远得重 |

这说明路线不是“谁好感最高谁结局好”，而是“林亦舟把大学生活投向哪里”。这个方向成立。

### 3. 5X 被正确隔离

5X 现在被定义为唯一硬外流，不由普通厌烦或软外流触发，进入后第六 / 七卷只走单线。这是必要的。

如果 5X 可以随时进出，它就会污染所有路线，让普通回避、湖边散步、夜市、外部朋友都变成硬分歧。当前设计把它锁在第五幕唯一外流点，合理。

### 4. 毕业结算温度比好感值更适合本项目

当前结算不是单纯成功 / 失败，而是：

- 说开一点
- 淡化一点
- 体面但不亲近
- 不和解也不追究
- 继续联系
- 错过
- 5X 遗憾

这比传统好感结局更符合母版调性。大学关系很多不是大胜大败，而是有些话说了，有些话没说，有些人还在联系人里，但很少再打开。

## 主要风险

### 风险一：体量目标容易爆炸

当前 S / A / B 级字数目标合理，但不能把它当成第一轮必须完成的正文长度。

如果一开始就给每条线写 8 万到 12 万字，开发会失控，而且后续转 JSON 时会重新拆一遍。

建议改成两层交付：

1. 第一轮交付“可玩长支线开发稿”：章节表、事件链、选择、变量、回调、毕业结算。
2. 第二轮再按开发优先级扩写关键剧情页，而不是整条线一次性小说化。

也就是说，现在先写足“可开发密度”，不急着写足“小说正文字数”。

### 风险二：`DEFAULT-4XX` 容易被误写成真结局

宿舍修补线是基准线，但不能被写成唯一正确答案。

它应该是“愿意留下来处理旧账”的路线，不是“大家终于恢复大团圆”的路线。修补线可以让四人有更多当面说话机会，也可以让有些事说开一点，但不能保证关系完全回到从前。

每一章都需要保留不圆满：

- 有人愿意说，但只说一半。
- 有人做事配合，但不再亲近。
- 有人到了饭局，却没有替过去道歉。
- 有人毕业后还联系，但频率很低。

### 风险三：`R5-ROMANCE` 候选来源和候选结构都需要更早固定

现在专注感情线选择沈嘉禾 / 夏知微是合理的，但它不能从主轴凭空生成。后续开发时必须先明确它来自第三幕 A3 父池里的哪条社团 / 活动生活半径，再明确候选拆分：

- 如果 A3 没有女生角色连续出现、承担任务压力和后续回声，就不能开启 `R5-ROMANCE`。
- 如果做双候选，就要在路线文件里拆成 `R5-ROMANCE-SHEN` 和 `R5-ROMANCE-XIA` 两个子方向。
- 如果先做一条样板线，就必须先选一个候选对象，另一个保留为对照和回声。

否则这条线最容易变糊：晚风、沈嘉禾、夏知微都在场，但谁都没有真正承载“亲密关系如何改变林亦舟生活”的主功能。

晚风也不能被写成失败选项。玩家没有和晚风更进一步，只代表她在这条线里退成朋友、网友、游戏搭子或低频回声。

### 风险四：`R3-PERFECT` 和 `R5-TANG` 有重叠

两条线都可能写成表格、流程、规则、项目。如果不区分，会变成同一种“做事正确线”。

建议明确区分：

- `R3-PERFECT` 写公开信用、活动履历、主持 / 融媒体 / 材料和“被看见”。
- `R5-TANG` 写公平、边界、项目责任、C407 演示和“规则保护人也伤人”。

完美线的压力来自外界不断把林亦舟当成可靠工具；唐骁线的压力来自关系被流程切开。

### 风险五：`R4-WORK` 和 `R5-LUCHEN` 有重叠

两条线都涉及钱、勤工、现实压力。如果不区分，会把陆沉写成兼职线默认陪跑。

建议明确区分：

- `R4-WORK` 的主语是林亦舟：生活费、班次、错过、疲惫、钱如何改写他的半径。
- `R5-LUCHEN` 的主语是陆沉关系：林亦舟能否看见陆沉的现实压力，但不替他说、不救赎他。

兼职线可以调用陆沉，但不能让陆沉自动成为路线核心。

### 风险六：三室友线必须写“另外两个人的后果”

周屿线、唐骁线、陆沉线都不能只写“被选中的室友和林亦舟关系变好”。

站队的本质是有后果：

- 站周屿，唐骁可能觉得林亦舟又把规则变成热闹，陆沉可能觉得自己又被绕过去。
- 站唐骁，周屿可能觉得被当众定规矩，陆沉可能觉得表格替他说完了话。
- 站陆沉，周屿可能觉得扫兴，唐骁可能觉得边界不清。

每条室友线至少每两章出现一次“未被选择者的误读、退场、短句、示威小动作或低温后果”。

### 风险七：摆烂线需要可玩，而不是少写

`R5-LIEFLAT` 很容易被写成“不选就是翻页”。这会让摆烂线不可玩。

摆烂线的选择应该是“怎么回避”：

- 静音群还是只发表情。
- 去买吃的还是刷手机。
- 不叫人上课还是不帮拿外卖。
- 想补救时找谁，但入口已经过期。

玩家仍然要操作，只是操作的是低成本逃避。它的乐趣来自“每次都好像没什么，最后却发现什么都没留下”。

### 风险八：5X 需要伏笔，但不能提前开放

5X 最后写是对的，但前面必须有伏笔。

伏笔可以是：

- 许澈远远出现一次。
- 后街 / 台球厅 / 新圈子在别人口中出现。
- 林亦舟在某次回避时短暂感觉“如果没人认识我就好了”。

但这些只能是影子，不能提前给玩家进入 5X 的选择。真正入口仍只能在第五幕唯一硬外流点。

## 必须补进开发模板的审查门

每条长支线文件正式开写前，建议先填以下检查表：

| 检查项 | 必须回答 |
|---|---|
| 林亦舟靠近了什么生活 | 宿舍、活动、兼职、亲密、某个室友、摆烂或 5X |
| 他因此少靠近了什么 | 至少写出一个被挤掉的人或生活入口 |
| 每章现实任务是什么 | 表格、排班、饭局、礼物、照片、材料、证书、清寝、离校等 |
| 每章关系压力是什么 | 谁等谁、谁不等谁、谁误读、谁缺席、谁被公开 |
| 未被选择者如何继续生活 | 不抹掉，只减少交集 |
| 毕业结算温度 | 说开、淡化、体面距离、不和解、继续联系、错过或 5X 遗憾 |
| 防奖励化措施 | 这条线给了什么甜，也拿走了什么 |
| 防漂移措施 | 哪些主线锁点绝不能取消 |

## 路线开发优先级建议

当前总表推荐顺序基本合理，但建议略微调整成“先定模板，再定同组，再定高风险”：

1. `DEFAULT-4XX`：先做模板线，检验章节表、选择密度、变量回调和毕业结算格式。
2. `R5-ZHOU`、`R5-TANG`、`R5-LUCHEN`、`R5-LIEFLAT`：同属第五幕站队组，最适合用同一套对照结构开发。
3. `R5-ROMANCE`：先确定 A3 女生候选来源，再拆沈嘉禾 / 夏知微双候选，不要让晚风默认失败，也不要让主轴直接跳其它感情线。
4. `R4-WORK`：补钱、排班、家庭暗线和错过。
5. `R3-PERFECT`：补活动 / 公开信用线，注意和唐骁线区分。
6. `R5X-HARD`：最后写，并回头检查伏笔。

## 是否可以继续开发

可以继续。

但下一步不建议立刻把九条线全部扩写成长正文。更稳的做法是先写 `DEFAULT-4XX` 的可玩长支线模板，让它具备：

- 10 章左右章节表
- 每章 2-4 个可拆选择
- 每章现实任务和关系压力
- 每 2-3 个选择有回调
- 第六卷承接方式
- 第七卷毕业结算温度

这条模板成立后，再复制结构到周屿线、唐骁线、陆沉线和摆烂线。这样不会让后续开发陷入“每条线都重新发明格式”。
