# IF 分支 P0-E 第五幕隐藏高难 / 5X 唯一硬外流线开发稿

## 路线定位

路线编号：`R5X-HARD`

路线名称：5X 唯一硬外流线 / 离场新圈线

入口时期：第五幕中后段，宿舍强冲突已经爆发，玩家处在旧宿舍线最难回头的位置。

路线核心：林亦舟没有回到 4XX 的当面关系现场，而是顺着后街、便利店、台球厅、校外小馆和许澈的新圈子离开。新圈子没有旧账，不追问军训手机事件、不追问公共桌、不追问谁站谁，但也不承担深责任。

核心收益：短时间内获得轻松、无审判感和新的生活半径。玩家仍能选择如何和许澈、新圈子、旧宿舍保持距离，但不能再回到原来的多线分歧里改写第五卷站队。

核心代价：一旦硬锁，4XX 宿舍修补机会断裂。第六卷和第七卷仍会经过相同校园流程，但旧人只以固定流程、偶然擦肩、群消息只读和远处同框出现，不再开放原多线抉择。

毕业倾向：新关系散得轻，旧关系远得重。结局不是“换个圈子就好了”，而是林亦舟发现自己从旧宿舍里走出来后，也没有真正被新圈子接住。

## 主线锁点

- 5X 是唯一硬外流。其它“去楼下坐坐”“去湖边走走”“找晚风 / 夏知微 / 沈嘉禾说几句”都只能是软回流或软外流，不得锁死路线。
- 许澈在第四卷只能作为远影、工牌、短句和便利店痕迹出现；正式成为新圈子核心，只能从 5X 后开始。
- 进入 5X 后，第六卷和第七卷只走 5X 单线。可以经过 C407、晨光体育场、青枫居、东区商业街、武生院站等同样场景，但不再参与 4XX、晚风、沈嘉禾、夏知微等原多线的关键选择。
- 原宿舍主线仍在世界里发生：周屿继续活动圈，唐骁继续项目和流程，陆沉继续勤工和现实压力。玩家改变的是林亦舟不再站在他们的关系现场中心。
- 5X 结局按本线变量判定，不回收为普通宿舍修补、晚风亲密、兼职线或摆烂线。

## 入口条件

建议前置变量：

- `hard_outflow_candidate >= 1`
- `dorm_trust <= 1`
- `avoidance >= 3`
- `door_slam_risk >= 2`
- `old_debt_unspoken >= 2`
- `xu_che_shadow >= 1`

硬锁条件：

- 玩家在唯一硬外流点选择继续离开，并明确不回 4XX 当面处理。
- 玩家选择切断或长期静音 4XX 事项群。
- 当前没有足够的 `zhou_trust`、`tang_trust`、`luchen_trust` 或 `dorm_repair` 把林亦舟拉回同质内流。

同源非 5X 出口：

- 如果宿舍互信足够，玩家即使选择出门，也应进入“同质外流后的冷静小结局”，最后并入第六卷普通入口。
- 如果玩家只是短暂逃避但没有进入许澈 / 新圈子，进入摆烂或回避倾向，不锁 5X。
- 如果玩家选择找晚风稳定靠近，回到主轴晚风线或亲密外部回声，不锁 5X。
- 如果玩家选择找沈嘉禾、夏知微等其它女生稳定靠近，只有在 A3 已经存在女生候选来源时才回到社团女生派生专注感情线；没有来源时只能作为软回流或低频照面，不锁 5X。

互斥提示：

- 5X 不和 P0-C 专注感情线并行。
- 5X 不和 P0-D 三室友站队线并行。
- 5X 不等于完美线或兼职线的强版本；它不是“去做更正确的事”，而是离开旧关系现场。

## 5X 新圈人物

### 许澈

功能：新圈子的入口人物，提供无旧账的轻松关系。

出现方式：便利店收银、后街买水、台球厅等人、外卖拼单、校外小馆临时凑桌。

边界：许澈不做人生导师，不替林亦舟判断宿舍。他的吸引力来自“不问”，他的危险也来自“不问”。

### 韩野

功能：台球厅和后街关系里的热闹人，负责把林亦舟拉进一张新桌子。

边界：韩野可以热情，可以请一局，也可以临时改约，但不会替谁负责。

### 蒋沐

功能：外卖拼单、校外小馆和兼职消息的连接人，负责展示新圈子的实用和松散。

边界：蒋沐熟得快，散得也快。她可以帮林亦舟找一份临时单，也会在事情麻烦时先保自己。

开发要求：5X 是唯一允许新增稳定小圈子的路线。新增人物必须有姓名和功能，不能出现一批无名“新朋友”替代关系。

## 核心事件链

### 事件一：最后一扇门

地点：青枫居楼梯间、宿舍门口、楼下便利店、后街入口。

参与人物：林亦舟、周屿、唐骁、陆沉、许澈。

现实触发：4XX 的强冲突已经从争吵变成短句、静音和不再等待。林亦舟站在回寝室和下楼之间，4XX 群里没有人再补第二句话。便利店或后街方向出现许澈的轻松入口：买水、搬货、等台球厅空桌、临时凑一份夜宵。

可拆选择：

- 回头敲门，说“我回来了，当面说”。
  - 结果：进入同质内流或宿舍修补线，不锁 5X。
  - 回响：旧冲突仍难处理，但第六卷还有修补入口。
- 下楼买水，但保留 4XX 群消息提醒。
  - 结果：`avoidance +1`，`hard_outflow_candidate +1`，暂不硬锁。
  - 回响：如果后续回去，仍可并入低温修补；如果继续切断，进入 5X 风险。
- 跟许澈去后街坐一会儿，不解释去向。
  - 结果：`hard_outflow_candidate +1`，`old_dorm_distance +1`，`new_circle_contact +1`。
  - 回响：4XX 不会立刻爆炸，但下一次事项群不再默认等他。
- 静音 4XX 群，回一句“你们先处理吧”后继续往外走。
  - 结果：`hard_outflow +1`，`route_lock = 5x`，`old_dorm_distance +2`。
  - 回响：进入 5X 单线；第六卷不再开放原宿舍多线抉择。

对话逻辑：

- 许澈只会问“去不去”“坐会儿吗”“喝什么”，不追问宿舍发生了什么。
- 周屿不会长篇挽留，只可能发一句“算了”或撤回一条玩笑。
- 唐骁会把事情转成“那我们先按当前版本处理”。
- 陆沉可能只读不回，这个沉默在 5X 里不再被解释。

### 事件二：新桌子

地点：后街烧烤、东区商业街小馆、台球厅、便利店后门。

参与人物：林亦舟、许澈、韩野、蒋沐。

现实触发：新圈子临时缺一个人拼桌、拼球局、拼外卖或凑兼职信息。这里没人知道 4XX 的旧账，没人知道谁没叫谁上课，没人知道公共桌上那张规则纸。

可拆选择：

- 加入新桌子，接住许澈递来的位置。
  - 结果：`new_circle_trust +1`，`old_dorm_distance +1`。
  - 回响：林亦舟第一次感到不用解释也能坐下，但 4XX 事项群开始变成只读背景。
- 只旁听，不把自己讲进去。
  - 结果：`thin_relation +1`，`avoidance +1`。
  - 回响：新圈子觉得他安静好相处，但不会真正等他。
- 提起自己宿舍的事，试探许澈反应。
  - 结果：`self_exposure +1`，`new_circle_trust` 按许澈反应变化。
  - 回响：许澈可以听，但不会替他追回旧关系。
- 想把周屿 / 唐骁 / 陆沉也叫出来。
  - 结果：`old_dorm_distance` 暂缓，`new_circle_fit -1`。
  - 回响：新圈子会笑着岔开，旧宿舍也未必有人来。

对话逻辑：

- 韩野热闹，但他的热闹是“今晚开心就行”，不是关系承诺。
- 蒋沐讲信息、兼职、店名和价格，熟得快，不代表交心快。
- 许澈会给林亦舟留位置，但不把这个位置说成归属。

### 事件三：薄关系显形

地点：台球厅、校外小馆、后街夜市、二手交易见面点。

参与人物：林亦舟、许澈、韩野、蒋沐，可软擦肩秦越或许棠。

现实触发：新圈子里出现一次需要承担责任的小事：有人临时改约、AA 账单不清、兼职信息过期、二手交易被放鸽子、台球厅订桌没人确认。与此同时，4XX 或班级流程也在同一天推进，但对林亦舟只保留消息提示或远处擦肩。

可拆选择：

- 替新圈子补钱或补确认，保住轻松关系。
  - 结果：`money_cost +1`，`new_circle_trust +1`，`thin_relation +1`。
  - 回响：新圈子会夸他靠谱，但不会因此承担他的旧事。
- 直接问清楚为什么没人提前说。
  - 结果：`thin_relation +2`，`new_circle_trust -1` 或转为真实。
  - 回响：许澈可能承认“大家就这样”，林亦舟第一次看见轻松背后的松散。
- 想回 4XX 看看，但发现关键时间已经过了。
  - 结果：`5x_regret +1`，不解除 `route_lock`。
  - 回响：旧线只保留错过事实，不开放补救选择。
- 装作没事，继续坐到散场。
  - 结果：`identity_drift +1`，`old_dorm_distance +1`。
  - 回响：林亦舟越来越像新圈子的人，也越来越不像 4XX 的人。

对话逻辑：

- 新圈子不是反派，不能写成集体欺骗。它的问题是轻、散、没有深责任。
- 许澈可以说“别想太多”，但这句话既是安慰，也是回避。
- 林亦舟不应突然醒悟，只能从账单、空位、迟到和没人追问里慢慢感到不对。

### 事件四：同场擦肩

地点：C407 项目演示外侧、晨光体育场毕业照外围、青枫居清寝窗口、东区商业街、武生院站。

参与人物：林亦舟、许澈、4XX 三人、晚风 / 夏知微 / 沈嘉禾 / 秦越 / 许棠作为软互动。

现实触发：第六卷和第七卷的学校固定流程仍然发生。林亦舟会在同一栋楼、同一张表、同一个体育场或同一条商业街看见旧人，但 5X 不开放“回去站队”“重新选择晚风”“转入沈嘉禾 / 夏知微”的关键抉择。

可拆选择：

- 看见旧人，点头但不走过去。
  - 结果：`soft_passing +1`，`old_dorm_distance` 保持。
  - 回响：毕业结算可出现体面擦肩。
- 在群里只读，不补解释。
  - 结果：`message_cut +1`，`5x_regret +1`。
  - 回响：第七卷旧饭局或毕业照只出现远处空位。
- 短句问一句“你们还好吗”。
  - 结果：`soft_passing +1`，`old_dorm_distance -0.5`，不解锁回流。
  - 回响：旧人可能礼貌回复，但不会把他拉回原线。
- 继续跟许澈去新圈子约定。
  - 结果：`new_circle_trust +1`，`old_dorm_distance +1`。
  - 回响：5X 单线稳定推进，遗憾也更明确。

对话逻辑：

- 旧人不应突然控诉林亦舟，也不应突然原谅。更多是“看见了，但没再叫住”。
- 晚风可以作为低频消息或朋友圈回声，不再承担本线亲密核心。
- 夏知微、沈嘉禾、秦越、许棠只能软擦肩，不能把玩家拉回她们对应路线。

### 事件五：新圈散场

地点：后街、青枫居清寝窗口、东区商业街、武生院站。

参与人物：林亦舟、许澈、韩野、蒋沐、4XX 远处回声。

现实触发：毕业流程推进到退寝、离校、行李、档案和去向复核。新圈子各自有新安排：许澈可能换岗或离校，韩野去另一个城市，蒋沐临时接了别的单。旧宿舍也完成自己的饭局、毕业照、清寝和离校，但林亦舟已经不在那张桌子中央。

可拆选择：

- 给 4XX 发最后一条旧消息。
  - 结果：`5x_regret -1`，`soft_passing +1`。
  - 回响：可能收到体面回复，也可能只显示已读；不强制和解。
- 给许澈说清“我当时是逃出来的”。
  - 结果：`self_exposure +1`，`new_circle_truth +1`。
  - 回响：许澈可以听见，但未必能留下。
- 删除 4XX 聊天置顶，保留新圈子群。
  - 结果：`old_dorm_distance +1`，`identity_drift +1`。
  - 回响：结局更偏“旧人渐远，新人也散”。
- 什么都不做，跟着行李离开。
  - 结果：`5x_regret +2`。
  - 回响：终章可写成宿命式遗憾：不是没人给过入口，是每个入口都被他走成了背影。

对话逻辑：

- 许澈不能突然深情挽留。他最多给出一句现实的话，例如“到站了就发个消息”，或者“以后来后街也不一定碰得到我”。
- 4XX 的回复要短，符合毕业语境：体面、克制、有人不回也合理。
- 终章不把 5X 写成惩罚，而写成一种选择后的温度。

## 变量设计

| 变量 | 含义 | 主要来源 | 后续用途 |
|---|---|---|---|
| `hard_outflow` | 是否进入 5X 硬外流 | 最后一扇门选择 | 锁定第六 / 七卷单线 |
| `route_lock` | 路线锁状态 | 硬锁条件满足 | 禁止回原多线 |
| `new_circle_trust` | 新圈子接纳度 | 新桌子、补钱、拼局、聊天 | 判断许澈是否仍保留联系入口 |
| `old_dorm_distance` | 旧宿舍距离 | 静音、不回、擦肩、错过 | 判断毕业旧人回响温度 |
| `thin_relation` | 新关系薄度 | 临时改约、AA 不清、没人等 | 判断外流失落强度 |
| `soft_passing` | 旧人软擦肩次数 | 第六 / 七卷同场景经过 | 判断是否有体面点头或短句 |
| `message_cut` | 群消息切断程度 | 静音、只读、删除置顶 | 决定终章旧群是否仍可回响 |
| `identity_drift` | 自我漂移 | 持续坐在新桌子，不再解释旧事 | 决定林亦舟是否两边都不像 |
| `5x_regret` | 外流遗憾 | 过期、散场、未发消息 | 第七卷结算核心 |

## 后续承接

### 第五卷承接

- 5X 必须从强冲突后出现，不能把普通厌烦都写成硬外流。
- 5X 入口前要让玩家明确看见还有回头方式：敲门、当面说、找某个室友、保留群提醒。
- 只有玩家主动选择继续离开，并且变量不足以支撑同质内流时，才锁定 5X。

### 第六卷承接

- 第六卷不再多线均摊，也不转晚风核心关系。
- 主要生活半径改为许澈、新圈子、后街、台球厅、便利店、外卖拼单、校外小馆。
- 4XX 只作为旧主轴回声：项目演示、毕业去向、材料归档、公共桌等事件仍发生，但林亦舟只能远处看见、短句回应或错过。
- 允许软互动：看见周屿在活动现场、唐骁在 C407、陆沉在图书馆、晚风朋友圈、夏知微镜头、沈嘉禾资料、秦越名单、许棠材料，但不开放转线。

### 第七卷承接

- 毕业固定流程仍发生：预登记、毕业照、答辩、清寝、饭局、离校手续。
- 5X 结算重点是“新圈散了，旧人远了”，不是惩罚玩家，也不是强行团圆。
- 可生成三种温度：
  - `5x_regret_high`：旧消息没发，新圈也散，林亦舟只留下过期入口。
  - `5x_polite_echo`：旧人有短句回复，新圈有一两个可联系的人，但都不再亲近。
  - `5x_truth_told`：林亦舟对许澈或旧人承认自己当时是逃出来的，遗憾仍在，但不再完全装作没事。

## 漂移防线

- 5X 不是许澈恋爱线。
- 5X 不是摆烂线。摆烂是低投入拖过去；5X 是主动离开旧关系现场，并进入新圈单线。
- 5X 不是兼职线。许澈和便利店可以带出现实感，但路线核心不是赚钱，而是无旧账的新关系。
- 5X 不是晚风线失败页。晚风可以低频出现，但不承担本线核心。
- 5X 不是隐藏好结局。它有轻松、有新鲜，也有薄、有散场和宿命感遗憾。
