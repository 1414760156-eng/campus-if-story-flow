# 开发长线_IF可玩支线R5-ZHOU周屿线

## 用途

本文用于把 `R5-ZHOU` 从第五幕站队分歧骨架扩展成可拆剧情页、选项页、变量回调和毕业结算的 A 级命运外流线。

它不是小说正文，也不是支线外传。它规定的是：玩家在第五幕强冲突后选择靠近周屿时，林亦舟的后续大学生活如何进入“热闹、面子、主持、活动圈、人情债和玩笑防御”的命运池。母本公共事件照常发生，但镜头、可操作内容和关系后果都按周屿线重新嫁接。

## 路线定位

路线编号：`R5-ZHOU`

路线池编号：`POOL-R5-ZHOU`

路线等级：A 级核心支线

共享范围：第一卷、第二卷共享；第三幕以后只按关键选择进入对应命运池；本线入口在第五幕站队分歧。

核心体验：林亦舟没有选择留在 4XX 公共桌做平均修补者，也没有选择唐骁的规则、陆沉的沉默边界或摆烂回避，而是先走向周屿。周屿会因此更愿意把玩笑背后的狼狈露给他看，也会更频繁把他拖进公开热闹、活动圈人情和面子账里。

本线重点不是“陪周屿热闹”，也不是“救一个爱开玩笑的人”。周屿真正的问题是：他习惯把难堪说成笑话，把低头说成扫兴，把请客和热闹当作补偿，把公开场面的顺滑误认为关系已经过去。玩家要在一次次场面里决定：是继续替他圆，还是让他把话落回真实处。

## 不改变的母本锁点

- 第五幕宿舍强冲突必然发生，且不会因为玩家靠近周屿就证明周屿完全正确。
- 唐骁、陆沉仍按母本生活半径推进：唐骁继续面对规则、项目、流程和实用压力；陆沉继续面对勤工、图书馆、家庭和发言边界。
- 第六卷的 C407 项目、课程演示、材料归档、关系定形仍发生，只是林亦舟更多从周屿的活动圈和公开场面一侧进入。
- 第七卷的预登记、毕业活动材料、开题 / 答辩、毕业照、清寝、最后一顿饭、离校手续仍发生。
- 5X 仍是唯一硬外流。本线内部回流不能临时打开 5X，也不能跳入唐骁线、陆沉线或感情线主分支。

## 核心变量

| 变量 | 含义 | 高值倾向 | 风险 |
|---|---|---|---|
| `zhou_trust` | 周屿是否愿意把玩笑背后的真话交给林亦舟 | 周屿愿意私下说难听话和真话 | 可能把林亦舟当成默认站边者 |
| `activity_link` | 林亦舟进入周屿活动圈的深度 | 活动、主持、秦越、许棠等事件增加 | 宿舍缺席和被工具化增加 |
| `public_smooth` | 公开场面是否被圆过去 | 场面好看、冲突暂时降温 | 真问题被拖后，唐骁 / 陆沉误读加重 |
| `face_debt` | 面子账和请客式补偿累积 | 周屿更依赖热闹解决尴尬 | 钱、时间、人情和旧账混在一起 |
| `joke_defense` | 周屿用玩笑防御的强度 | 短期降低尴尬 | 长期导致别人觉得他不认真 |
| `public_boundary` | 公开材料、照片、名字、朋友圈边界 | 授权清楚，毕业风险降低 | 场面不够好看，周屿会觉得被拆台 |
| `dorm_absence` | 林亦舟在 4XX 场域缺席程度 | 周屿线加深 | 唐骁、陆沉默认不等他 |
| `tang_misread` | 唐骁对林亦舟偏袒周屿的误读 | 唐骁减少非必要解释 | 第六卷专业协作变低温 |
| `luchen_distance` | 陆沉与林亦舟距离 | 陆沉减少被代言风险 | 林亦舟更难知道陆沉真实压力 |
| `qinyue_pressure` | 秦越活动流程 / 人情对林亦舟的挤压 | 活动圈资源上升 | 林亦舟被默认可调度 |
| `graduation_temperature` | 毕业结算温度 | 继续联系、说开、淡化、体面距离或不和解 | 由前述变量综合决定 |

## 体量与密度

建议 8 章开发体量，对应约 60,000-90,000 字正文容量。本文只提供可开发骨架，不直接写足正文。

最低配置：

- 至少 32 次有效选择。
- 至少 6 个内部回流点。
- 每个内部回流点至少 2 个回流方向。
- 每个回流方向至少 3-4 个连续剧情拍点。
- 至少 3 次强后果回调：第六卷 C407 / 活动留名 / 第七卷毕业照或最后一顿饭。

## 章节开发详表

### 第一章：先别逗笑

章节目标：确定玩家进入周屿线后的第一种代价。林亦舟靠近周屿，会让周屿觉得自己没有被当众定性；也会让唐骁和陆沉看到林亦舟先离开公共桌。

现实触发：
- 4XX 事项群只剩短句。
- 周屿在楼梯间或主持排练区把尴尬讲成笑话。
- 唐骁需要时间线，陆沉不想被人替他说。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `ZHOU-CH01-N01` | 读取站队入口 | 4XX 事项群 / 青枫居楼道 | 林亦舟、周屿、唐骁、陆沉 |
| `ZHOU-CH01-N02` | 决定先找周屿 | 楼梯间 / 食堂侧门 | 林亦舟、周屿 |
| `ZHOU-CH01-N03` | 玩笑第一次被截住或被圆过 | 主持排练区 / 活动区 | 林亦舟、周屿、秦越软照面 |
| `ZHOU-CH01-N04` | 4XX 回声 | 事项群 / 公共桌 | 唐骁、陆沉软回声 |

关键选择：

| 选择 ID | 选项 | 变量变化 | 即时反馈 | 后续回声 |
|---|---|---|---|---|
| `ZHOU-CH01-C01` | 帮周屿把场面圆过去 | `zhou_trust +1`, `public_smooth +1`, `tang_misread +1` | 周屿接住笑，唐骁只发截止时间 | 第二章唐骁不再主动解释流程 |
| `ZHOU-CH01-C02` | 轻声截住玩笑，说“这个别拿来逗” | `zhou_trust +1`, `joke_defense -1`, `conflict_heat +1` | 周屿脸色难看，但没有立刻走 | 第六章周屿更可能说真话 |
| `ZHOU-CH01-C03` | 私下提醒，不在众人面前拆他 | `zhou_trust +2`, `public_boundary +1`, `tang_misread +1` | 周屿接受度高，唐骁不知道发生过处理 | 第三章活动圈更愿意找林亦舟 |
| `ZHOU-CH01-C04` | 不拦，跟着笑过去 | `public_smooth +2`, `joke_defense +1`, `luchen_distance +1` | 场面轻松，陆沉减少后续发言 | 第六章旧账爆发时周屿更依赖玩笑 |

内部回流：调用 `RET-ZHOU-02` 玩笑过界。

防漂移：
- 不写成“周屿终于被理解”。
- 不让唐骁或陆沉当场变反派。
- 不让一次截住玩笑直接修好宿舍关系。

### 第二章：B206 的空白文档

章节目标：把“靠近周屿”从情绪选择推入现实任务。主持彩排缺人、名单空格、活动流程文档都会逼玩家判断：帮周屿补场，还是逼周屿自己面对流程。

现实触发：
- 秦越在活动群催主持队流程。
- B206 临时改彩排时间。
- 4XX 同时需要 C407 小组材料初版。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `ZHOU-CH02-N01` | 收到彩排缺人消息 | 微信活动群 / B206 门口 | 林亦舟、周屿、秦越 |
| `ZHOU-CH02-N02` | 空白文档分工 | B206 / 文档共享页 | 林亦舟、周屿 |
| `ZHOU-CH02-N03` | 4XX 材料催促 | 事项群 / C407 截止提醒 | 唐骁软回声 |
| `ZHOU-CH02-N04` | 彩排后回流 | 青枫居 / 活动群 | 周屿、秦越、4XX 软回声 |

关键选择：

| 选择 ID | 选项 | 变量变化 | 即时反馈 | 后续回声 |
|---|---|---|---|---|
| `ZHOU-CH02-C01` | 直接帮周屿补场串词 | `activity_link +2`, `dorm_absence +1`, `qinyue_pressure +1` | 秦越记住林亦舟，唐骁只回“收到” | 第三章活动圈默认林亦舟可调度 |
| `ZHOU-CH02-C02` | 帮周屿把流程补齐，但不上台 | `activity_link +1`, `public_boundary +1` | 周屿觉得少了点人情，秦越认可流程 | 第五章授权边界更清楚 |
| `ZHOU-CH02-C03` | 要求周屿自己给秦越回话 | `joke_defense -1`, `zhou_trust 0`, `conflict_heat +1` | 周屿不舒服，但第一次正面回复 | 第六章周屿更可能承认自己怕丢脸 |
| `ZHOU-CH02-C04` | 先回 C407 材料，再晚到彩排 | `project_stability +1`, `zhou_wait_cost +1`, `tang_misread -1` | 周屿把迟到当成“你也开始排表” | 第四章请客补偿更容易触发 |

内部回流：调用 `RET-ZHOU-01` 主持彩排缺人。

防漂移：
- B206 只作为教学楼/排练教室使用，不写成明德楼班会。
- 周屿活动圈不能吞掉 4XX 主线，只能改变林亦舟所在位置和后续误读。

### 第三章：热闹的人情

章节目标：让活动圈变成真实生活半径，而不是 NPC 任务板。秦越、许棠、东区商业街、健源体育场边的表演都要成为“热闹有用，但热闹也会欠账”的场。

现实触发：
- 东区商业街有学生乐队 / 舞蹈表演，周屿想去凑热闹。
- 秦越临时借人做活动表、搬设备或补签到。
- 许棠需要拍摄素材，但要求授权清楚。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `ZHOU-CH03-N01` | 东区小舞台邀约 | 东区商业街 / 健源体育场外 | 林亦舟、周屿 |
| `ZHOU-CH03-N02` | 秦越借人情 | 活动群 / B206 / 九龙壁附近 | 林亦舟、周屿、秦越 |
| `ZHOU-CH03-N03` | 许棠拍摄授权 | 小舞台边 / 融媒体软照面 | 许棠 |
| `ZHOU-CH03-N04` | 4XX 低温回声 | 事项群 / 公共桌 | 唐骁、陆沉软回声 |

关键选择：

| 选择 ID | 选项 | 变量变化 | 即时反馈 | 后续回声 |
|---|---|---|---|---|
| `ZHOU-CH03-C01` | 跟周屿去看表演，帮他拍一段 | `activity_link +1`, `zhou_trust +1`, `dorm_absence +1` | 周屿轻松，唐骁不再等材料确认 | 第五章公开素材风险增加 |
| `ZHOU-CH03-C02` | 看表演但不发朋友圈 | `public_boundary +1`, `zhou_trust 0` | 周屿嘴上说无所谓，实际有点失落 | 第五章周屿会提“你是不是怕被看见” |
| `ZHOU-CH03-C03` | 先帮秦越补活动表，再去找周屿 | `qinyue_pressure +1`, `activity_link +1`, `zhou_wait_cost +1` | 秦越把林亦舟当可靠人，周屿觉得被流程截胡 | 第七章活动留名默认加上林亦舟 |
| `ZHOU-CH03-C04` | 拉周屿回 4XX 处理材料 | `dorm_absence -1`, `zhou_trust -1`, `tang_misread -1` | 周屿觉得热闹被打断 | 第四章周屿更可能用请客补面子 |

内部回流：调用 `RET-ZHOU-03` 东区小舞台、`RET-ZHOU-04` 秦越借人情。

防漂移：
- 秦越不能变成新核心线，只负责活动压力、人情和流程。
- 许棠只负责公开素材边界，不开启感情线。

### 第四章：请客不是道歉

章节目标：处理周屿最典型的补偿方式：请客、开玩笑、拉热闹。玩家要决定这顿饭是让事情变软，还是把“请客不能替代道歉”说出来。

现实触发：
- 周屿想用一顿饭把尴尬盖过去。
- 可选地点：东区商业街东北饺子馆、后街烧烤、西园食堂二楼窗口、奶茶店。
- 林亦舟的钱、时间和 4XX 缺席都成为代价。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `ZHOU-CH04-N01` | 周屿提出请客 | 4XX 门口 / 食堂侧门 | 林亦舟、周屿 |
| `ZHOU-CH04-N02` | 选择地点和付款方式 | 东区商业街 / 后街 / 食堂 | 林亦舟、周屿 |
| `ZHOU-CH04-N03` | 是否谈刚发生的事 | 饭桌 / 奶茶店外 | 林亦舟、周屿 |
| `ZHOU-CH04-N04` | 饭后回声 | 4XX 事项群 / 支付记录 | 唐骁、陆沉软回声 |

关键选择：

| 选择 ID | 选项 | 变量变化 | 即时反馈 | 后续回声 |
|---|---|---|---|---|
| `ZHOU-CH04-C01` | 接受请客，先让场面软下来 | `zhou_trust +1`, `face_debt +1`, `money_boundary -1` | 周屿明显轻松 | 第六章旧账仍会爆发 |
| `ZHOU-CH04-C02` | 坚持 AA，并谈刚才那句玩笑 | `money_boundary +1`, `repair_depth +1`, `zhou_defense +1` | 周屿嘴硬，饭局变冷 | 第六章真话入口更早出现 |
| `ZHOU-CH04-C03` | 只喝饮料，不让饭局变成补偿 | `public_boundary +1`, `face_debt -1`, `zhou_trust 0` | 周屿尴尬但能听进去半句 | 第八章毕业饭局边界更清楚 |
| `ZHOU-CH04-C04` | 不谈冲突，只聊活动和近况 | `public_smooth +1`, `old_debt +1`, `zhou_trust +1` | 气氛最好，问题留下 | 第五章公开素材更容易被误读 |

内部回流：调用 `RET-ZHOU-05` 请客式补偿。

防漂移：
- 食物偏好要服务人物，不写菜单展览。
- 饭局不能变成和解仪式。
- 周屿请客不是坏心，也不是万能补丁。

### 第五章：被剪掉的停顿

章节目标：把周屿“公开顺滑”和“私人停顿”之间的裂缝做成可玩事件。玩家要处理活动素材、朋友圈、主持片段、公开署名和授权。

现实触发：
- 许棠或活动组需要一段主持 / 彩排素材。
- 周屿在片段里看起来很会圆场，但真实停顿被剪掉。
- 4XX 旧事可能被漂亮材料重新包装。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `ZHOU-CH05-N01` | 素材预览 | 融媒体门口 / 教学楼空教室 | 林亦舟、周屿、许棠 |

## 对话逻辑

### 周屿

- 先玩笑，后防御，最后才可能说真话。
- 他不怕热闹，怕的是自己在热闹里忽然被看穿。
- 他道歉通常不会完整说“对不起”，更可能是“那天我话说难听了点”“你还真记着啊”“行，下次我先问你”。
- 如果玩家长期帮他圆场，他会更依赖林亦舟；如果玩家长期当众拆他，他会觉得林亦舟也站到审判台那边。

### 林亦舟

- 不能变成圆场机器。
- 玩家每次靠近周屿，都要承担“唐骁和陆沉看见他先离开公共桌”的代价。
- 他可以帮周屿保住面子，但必须选择是否把真实问题留下来处理。

### 唐骁

- 在本线里不抢主轴，但会通过短句、截止时间、少解释和不再等人的方式形成误读回声。
- 他不是因为讨厌周屿，而是觉得林亦舟把“场面好看”放在“规则清楚”之前。

### 陆沉

- 在本线里更容易退开，因为林亦舟若持续替周屿圆场，陆沉会减少表达欲。
- 陆沉的回声要通过“只交自己的部分”“不补充解释”“不参与热闹”表现。

### 秦越

- 秦越是活动圈现实压力，不是攻略对象。
- 他认可能办事的人，也会天然把能办事的人继续调度。

### 许棠

- 许棠只负责公开材料、拍摄、授权和留存边界。
- 她可以提醒“好看的素材不等于完整事实”，但不能替 4XX 解决关系。

## 漂移防线

- 本线不是“周屿奖励线”。靠近周屿会增加信任，也会增加活动圈人情、公开误读和宿舍缺席。
- 本线不是“热闹社团线”。第三幕完美线写公开信用和被工具化；周屿线写站队后的人情、面子和玩笑防御。
- 本线不是“感情线”。晚风、沈嘉禾、夏知微可以作为软回声，但不抢周屿主轴。
- 本线不能把唐骁写成死板反派，也不能把陆沉写成沉默背景板。
- 本线不能让内部回流打开其他命运池。所有回流只改变本线关系温度、误读、资源代价和毕业结算。
- 本线不能取消母本公共锁点，只能改变林亦舟进入锁点的位置。

## 开发优先级

1. 先实现入口 `P0D_STAND_WITH_ZHOU` 与路由池锁定。
2. 再实现 8 章主事件链。
3. 再实现 6 个内部回流点，每个回流点至少 2 个方向。
4. 再写第六卷公共锁点嫁接：C407 / 项目 / 活动撞期。
5. 最后写第七卷毕业结算：毕业活动留名 / 饭局 / 清寝 / 离校。

本线完成后，可按同样结构开发 `R5-TANG`、`R5-LUCHEN`、`R5-LIEFLAT`，但三条线必须各自拥有不同的核心代价，不能只是把周屿名字替换成其他室友。
