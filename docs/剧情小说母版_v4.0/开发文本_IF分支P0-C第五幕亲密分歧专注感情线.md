# IF 分支 P0-C 第五幕亲密分歧 / 专注感情线开发稿

## 路线定位

路线编号：`R5-ROMANCE`

路线名称：专注感情线 / 稳定亲密线

入口时期：第五幕前段，宿舍矛盾开始变密、晚风关系可进一步但尚未锁定时。

路线核心：林亦舟没有选择和晚风更进一步，而是把主要情绪、时间和现实陪伴投向另一个已有铺垫的亲密对象。此线不新增无铺垫女主，优先从沈嘉禾和夏知微中二选一。

核心代价：亲密关系提供被理解、被等待、被偏爱的空间，但也会占用时间、钱、精力和解释机会。宿舍冲突仍发生，只是林亦舟更常把话先说给亲密对象，而不是回 4XX 当面处理。

毕业倾向：可能继续联系，也可能因为实习、考研、答辩、工作城市和公开边界分流而体面错过。结局温度取决于玩家是否长期承担亲密代价，而不是是否选了“恋爱正确答案”。

## 候选对象

### 沈嘉禾方向

适合主题：资料、边界、考试、共享文档、现实协作、未来选择。

有效吸引力：她把问题变具体，不替人背责任，也不把资料私聊自动理解成暧昧。林亦舟靠近她，是因为混乱被她整理成可以面对的东西。

适合场景：阳光书屋、图书馆、打印店、食堂侧门、教学楼、共享文档。

### 夏知微方向

适合主题：影像、观看角度、授权、表达、错过、被看见。

有效吸引力：她不会替林亦舟解释宿舍旧账，但会让他意识到自己如何被记录、如何站在别人故事边缘，以及什么东西不该被拍下来。

适合场景：摄影社、晨光湖、凌空栈道、东区商业街拍摄点、活动影像整理现场。

## 主线锁点

- 晚风仍存在，但可以停在朋友、游戏搭子、错过或低频联系状态。
- 第五卷宿舍强冲突仍发生，4XX 站队入口仍存在。
- 专注感情线不能取消宿舍关系，只改变林亦舟参与宿舍冲突的频率、立场和解释方式。
- 第六卷关系定形仍发生，但核心关系圈改为沈嘉禾 / 夏知微及其生活半径。
- 第七卷毕业流程仍通过预登记、毕业照、答辩、清寝、饭局和离校手续收束。

## 入口条件

建议变量门槛：

- `romance_focus >= 2`
- `dorm_absence >= 1`
- `wanfeng_boundary >= 1`
- `candidate_link >= 2`

候选对象变量：

- 沈嘉禾方向：`shen_jiahe_link >= 2`，`study_reality >= 1`，`boundary_awareness >= 1`
- 夏知微方向：`xia_zhiwei_link >= 2`，`image_boundary >= 1`，`expression_need >= 1`

推荐触发方式：

- 玩家在晚风可进一步节点选择保持朋友、说清边界或错过见面。
- 玩家在第五幕前段反复选择找沈嘉禾处理资料 / 未来问题，或找夏知微处理影像 / 表达问题。
- 玩家开始把原本该回 4XX 说的话，转向候选对象说，或选择用候选对象的场景消化情绪。

互斥提示：

- 如果玩家持续回 4XX 当面修补，保持 `DEFAULT-4XX`。
- 如果玩家继续推进晚风语音、见面、礼物、牵手和公开边界，则进入晚风亲密方向，不进入本线。
- 如果玩家只逃避，不稳定靠近任何对象，则滑向摆烂或回避线。

## 核心事件链

### 事件一：晚风停在门外

地点：青枫居楼下、游戏语音、微信聊天、地铁口未赴约。

参与人物：林亦舟、晚风、程小满、4XX 室友。

现实触发：晚风发来一次可以进一步的邀请。她可能问要不要线下见一面、要不要继续打完那局、要不要把最近的事说清楚。4XX 群里同时出现宿舍事项，候选对象也在另一个场景里给出可进入的现实入口。

可拆选择：

- 明确告诉晚风：现在不适合再往前。
  - 结果：`wanfeng_boundary +1`，`romance_focus_open +1`，`wanfeng_distance +1`。
  - 回响：晚风不会消失，但后续更少主动替林亦舟接情绪。
- 不解释，只把晚风消息按到后面。
  - 结果：`avoidance +1`，`wanfeng_misread +1`，专注感情线不稳定。
  - 回响：如果后续靠近沈嘉禾 / 夏知微，会留下“没有说清就转向”的道德压力。
- 继续和晚风保持朋友 / 游戏搭子状态，但不线下推进。
  - 结果：`wanfeng_boundary +1`，`intimacy_cost 0`，保留低频回声。
  - 回响：第六卷晚风只作为生活回声出现，不替代主亲密对象。
- 回 4XX 处理宿舍群消息。
  - 结果：`dorm_warmth +1`，`romance_focus_open 0`。
  - 回响：亲密分歧入口延后，默认宿舍主轴保留。

对话逻辑：

- 晚风不能被写成失败选项。她可以清楚、难过、体面，也可以不等林亦舟解释完。
- 程小满只作为晚风生活圈旁观，不要变成恶意挑拨。
- 4XX 不知道林亦舟如何处理晚风，只能感受到他有没有回来。

### 事件二：候选对象确认

地点：阳光书屋 / 打印店 / 食堂侧门；摄影社 / 晨光湖 / 凌空栈道。

参与人物：林亦舟、沈嘉禾或夏知微，视候选方向而定。

现实触发：林亦舟在不想立刻回寝室的节点，遇到一个能让他停下来的对象。这个入口不能随机出现，必须来自前文已有生活半径。

可拆选择：

- 进入沈嘉禾方向：找她核材料、聊考试、聊未来选择。
  - 结果：`romance_candidate = shen_jiahe`，`shen_jiahe_link +2`，`study_reality +1`。
  - 回响：后续亲密推进以共享文档、打印店、阳光书屋、食堂侧门为主。
- 进入夏知微方向：找她聊照片、授权、活动影像和“要不要被看见”。
  - 结果：`romance_candidate = xia_zhiwei`，`xia_zhiwei_link +2`，`image_boundary +1`。
  - 回响：后续亲密推进以摄影社、凌空栈道、拍摄点和毕业影像为主。
- 只短聊近况，不进入专注感情。
  - 结果：`candidate_link +1`，`romance_focus 0`。
  - 回响：保留异质回流功能，不锁路线。
- 不见任何候选对象，去便利店 / 湖边 / 刷手机。
  - 结果：`avoidance +1`，`missed_chance +1`。
  - 回响：更容易进入摆烂或回避。

对话逻辑：

- 沈嘉禾的靠近来自边界清楚：她会问文件、时间、停止条件，不会一上来问“你是不是心情不好”。
- 夏知微的靠近来自观看角度：她会问“这张能不能发”“你想不想留下来”，不替林亦舟定义情绪。
- 林亦舟在本线中的关键动作不是表白，而是开始把自己的选择解释给某个人听。

### 事件三：第一次现实陪伴增频

地点：按候选对象分流。

沈嘉禾方向：阳光书屋、打印店、食堂侧门、图书馆路线。

夏知微方向：摄影社、晨光湖、凌空栈道、东区商业街拍摄点。

参与人物：林亦舟、候选对象、4XX 室友、可能出现的同学或社团成员。

现实触发：候选对象连续出现在林亦舟处理压力的场景里。她不是来安慰，而是有自己的任务、作业、作品或考试。

可拆选择：

- 聊刚发生的宿舍事。
  - 结果：`relationship_truth +1`，`dorm_absence +1`，`intimacy_link +1`。
  - 回响：林亦舟获得被听见的感觉，但也更晚回 4XX。
- 只聊近况和未来，不说宿舍冲突。
  - 结果：`future_talk +1`，`emotional_delay +1`。
  - 回响：关系更轻，但宿舍旧账延迟。
- 帮对方完成具体任务。
  - 沈嘉禾方向：帮她核共享文档、搬打印材料、排队取准考证。
  - 夏知微方向：帮她拿脚架、确认授权、等她修片。
  - 结果：`intimacy_cost +1`，`time_debt +1`，`candidate_link +1`。
  - 回响：亲密从实际行动长出来，同时挤压宿舍时间。
- 中途回 4XX。
  - 结果：`dorm_warmth +1`，`romance_focus -1`。
  - 回响：专注感情线不锁，但候选对象会感到林亦舟仍被旧关系拽住。

对话逻辑：

- 候选对象不能只当情绪垃圾桶。她们必须有自己的任务和时间成本。
- 林亦舟如果只说“没事”，对方不会自动理解所有事。
- 宿舍那边的后果不需要当场爆炸，可以表现为下一次群消息变短、饭局不再默认问他。

### 事件四：礼物、节日和笨拙模仿

地点：校园快递站、蜜雪冰城 / 一点点、淘宝订单、摄影社、阳光书屋、4XX 公共桌。

参与人物：林亦舟、候选对象、周屿、唐骁、陆沉、许棠或秦越。

现实触发：特殊节日、生日、考试前、作品提交前或一次普通的“想买点东西”。大学生的亲密推进往往笨拙：问室友送什么、查攻略、买错规格、怕太正式、又怕太随便。

可拆选择：

- 送实用礼物。
  - 沈嘉禾方向：笔、文件夹、打印卡、书签、保温杯。
  - 夏知微方向：胶片贴纸、相机肩带、存储卡盒、照片夹。
  - 结果：`gift_debt +1`，`candidate_link +1`，`money_pressure +1`。
  - 回响：礼物不贵，但会被室友看见订单或包装。
- 不送礼物，只陪她完成一件事。
  - 结果：`time_debt +1`，`relationship_truth +1`。
  - 回响：更自然，但也更容易错过 4XX 当面节点。
- 问室友建议。
  - 结果：`dorm_visibility +1`，`teasing_risk +1`。
  - 回响：周屿可能乱教，唐骁会给预算表，陆沉只问“她用不用得上”。
- 临时退缩，不送也不解释。
  - 结果：`missed_chance +1`，`candidate_misread +1`。
  - 回响：关系不一定断，但会出现一次小冷。

对话逻辑：

- 周屿会把亲密关系说得很热闹，但建议不一定靠谱。
- 唐骁会把礼物也处理成预算和用途，制造好笑但真实的大学感。
- 陆沉的建议最短，也可能最准确。
- 许棠或秦越可以提供外部观察，但不能抢成恋爱导师。

### 事件五：公开边界和第一次明确靠近

地点：按候选方向分流。

沈嘉禾方向：共享文档权限、图书馆座位、打印店门口、食堂侧门。

夏知微方向：照片授权、凌空栈道、摄影社修片电脑、毕业影像素材夹。

参与人物：林亦舟、候选对象、4XX 室友、许棠或社团同学。

现实触发：亲密关系被别人看见。可能是一张照片、一条朋友圈评论、共享文档里反复出现的协作者名字、一起吃饭被室友撞见，或某个同学问“你们现在算什么”。

可拆选择：

- 明确靠近，但先说清公开边界。
  - 结果：锁定 `R5-ROMANCE`，`public_boundary +1`，`romance_focus +2`。
  - 回响：第六卷围绕候选对象生活圈深入，4XX 退位更明显。
- 暂时不命名关系，但保持稳定陪伴。
  - 结果：`romance_focus +1`，`ambiguity +1`。
  - 回响：关系有甜也有误读，毕业结算可能偏错过或体面分别。
- 因为宿舍冲突回头，暂停亲密推进。
  - 结果：回到 `DEFAULT-4XX` 或进入宿舍站队入口。
  - 回响：候选对象不消失，但关系停在“差一点”。
- 被别人起哄后否认。
  - 结果：`candidate_hurt +1`，`public_boundary 0`，`dorm_visibility +1`。
  - 回响：后续需要修补，否则专注感情线转为错过结算。

对话逻辑：

- 沈嘉禾方向不能把“共享文档频繁协作”直接写成恋爱，她会要求权限、署名和时间边界先清楚。
- 夏知微方向不能把“照片好看”直接写成告白，她会关心能不能发、给谁看、要不要删。
- 林亦舟真正的选择不是牵手或不牵手，而是愿不愿意承担被看见、被问、被误读后的解释成本。

## 变量建议

| 变量 | 用途 | 增长来源 | 过高后果 |
|---|---|---|---|
| `romance_focus` | 专注感情线投入度 | 现实陪伴、礼物、说清边界、稳定见面 | 宿舍缺席和旧账延迟 |
| `romance_candidate` | 主对象 | `shen_jiahe` 或 `xia_zhiwei` | 一旦锁定，第六卷不再多对象均摊 |
| `wanfeng_boundary` | 晚风关系边界 | 说清不进一步、保持朋友状态 | 晚风转为生活回声 |
| `dorm_absence` | 宿舍缺席 | 晚回、错过饭局、把话说给外人 | 第五卷误读加重 |
| `intimacy_cost` | 亲密代价 | 陪伴、礼物、时间、钱 | 压缩学习、宿舍和打工 |
| `public_boundary` | 公开边界 | 授权、朋友圈、共享文档、照片 | 过低会引发公开风险 |
| `future_talk` | 未来讨论 | 考试、作品、实习、城市选择 | 第七卷结算更具体 |
| `candidate_hurt` | 候选对象受伤 | 否认、失约、不解释 | 可能转体面分别或错过 |

## 候选方向差异

| 项目 | 沈嘉禾方向 | 夏知微方向 |
|---|---|---|
| 关系质感 | 安静、具体、边界清楚 | 被看见、轻微暧昧、表达压力 |
| 核心场景 | 阳光书屋、打印店、共享文档、食堂侧门 | 摄影社、晨光湖、凌空栈道、拍摄点 |
| 主要代价 | 容易把亲密处理成协作和流程 | 容易被照片、公开和观看角度误读 |
| 与宿舍关系 | 唐骁更能理解，周屿可能觉得无聊，陆沉觉得边界安全 | 周屿更容易起哄，唐骁担心公开风险，陆沉少评价 |
| 第七卷结算 | 现实去向、材料、考证、城市分流 | 照片、毕业影像、授权、留下或删掉 |

## 后续承接

### 第五幕

专注感情线进入第五幕中后段时，宿舍冲突仍会爆发，但林亦舟更可能：

- 先把情绪发给候选对象，而不是回 4XX。
- 因见面、礼物、拍摄或资料协作错过宿舍当面节点。
- 在站队时被室友认为“你最近根本不在寝室”。
- 把宿舍问题说成“我不知道怎么回去”，而不是直接进入争吵。

### 第六幕

第六卷不再多线均摊，而是围绕主对象和她的生活圈深入：

- 沈嘉禾方向：共享文档、考试、项目、材料署名、现实好感参照、未来选择。
- 夏知微方向：照片、授权、作品、活动影像、被看见与不被公开、毕业影像伏笔。
- 晚风只保留低频生活回声或错过文本，不再作为核心亲密对象。
- 4XX 作为旧主轴回声出现，事项群变短，不再默认等他。

### 第七幕

第七卷毕业结算偏向：

- 继续联系：长期承担时间和公开边界，未来城市或学习计划仍有接口。
- 体面分别：关系真实存在过，但实习、考研、工作和家庭分流让两人选择不拖住彼此。
- 错过：多次失约、否认、公开风险或没说清导致亲密关系停在“差一点”。
- 私人入口保留：不公开大团圆，但保留一个微信、照片、共享文档或旧书签式的联系入口。

## 开发备注

专注感情线不是“谁更适合当女主”的投票，而是玩家选择林亦舟把情绪交给哪种关系：沈嘉禾代表现实协作和边界，夏知微代表被观看、表达和影像。无论选谁，亲密都必须同时带来甜、时间代价、钱 / 精力消耗、宿舍缺席和后续误读。
