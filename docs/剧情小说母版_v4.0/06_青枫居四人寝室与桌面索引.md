# 06 青枫居四人寝室与桌面索引

## 核心原则

青枫居 4XX 不是普通背景板。四人间里的每个座位、桌面、床位、插排、电脑、手机、耳机、鼠标、旧充电线、外卖袋、床帘和公共桌，都应该和人物强相关。

寝室是大学里最小的社会。一个人的桌面往往比他说的话更诚实：谁把线卷好，谁把耳机随手扔在键盘上，谁把父母买的药塞进抽屉，谁的电脑贴着维修标签，谁的手机壳旧得发亮，谁的外设看起来花哨但真正贵的是一副耳机，谁的插排总被别人借走。

写寝室时遵守四条：

1. 座位不是随机分配的背景。它决定人物离门、窗、阳台、公共桌和水房动线有多近。
2. 桌面不是清单。每个物件都要能说明爱好、习惯、经济状态、家庭痕迹或关系旧账。
3. 电子设备不靠型号炫耀。重点写使用方式、磨损、升级、维修、借用、声音和谁会碰它。
4. 公共区负责冲突。插排、空调遥控器、垃圾桶、公共桌、阳台晾衣和门后挂钩，都可以让小事变成关系压力。

## 寝室空间默认布局

沿用第一卷已写事实：

- 周屿在靠门的床位，第一天就站在靠门床边拿着床帘杆。
- 唐骁在靠窗桌前，第一天把文件袋和校园卡放进抽屉。
- 陆沉在阳台门边，第一天把黑色背包放到床下。
- 林亦舟的床位相对空着，位于门、窗、公共桌和三个人之间，适合观察，也适合被迫成为缓冲。

后续如果写具体方位，默认使用这个关系，不需要每次画平面图。读者只要逐渐知道：周屿离门最近，唐骁离窗和光线近，陆沉靠阳台和墙，林亦舟处在能看见三个人的位置。

| 人物 | 座位位置 | 桌面与收纳 | 电脑 / 手机 / 外设 | 家庭与性格暗示 | 冲突用途 | 阶段变化 |
|---|---|---|---|---|---|---|
| 林亦舟 | 夹在门、窗和公共桌之间的观察位；坐下能看见周屿进出、唐骁桌面和陆沉阳台门边的动作 | 开学时桌面空，后面逐渐放进通知书、药袋、基础教材、便利店常温水、小票和不常翻出来的家庭物件；东西不乱，但也不够有主见 | 家里支持买的中端笔记本，普通手机，保护壳一开始新，后来边角磨旧；耳机常用来听晚风语音，鼠标和键盘不花哨；充电器容易被借 | 开学前家庭准备充足，后来生活费和物品补给变化会体现在旧手机壳、没换的数据线和反复使用的电脑包上；他的桌面像一张慢慢被大学写满的纸 | 适合写回避：手机扣在桌边、聊天框亮着、药袋露出一角、父母转账压在微信列表；也适合写他是否坐回自己的椅子 | 大一从空桌到课本堆叠；第四卷暑假桌面变空；第五卷 5A 可用旧充电线、手机、家庭小物压回宿舍线；终章清桌时最能回响 |
| 周屿 | 靠门位，最容易进出，也最容易把走廊声音带进来 | 桌面有耳机、手机支架、润喉糖、主持稿、吉他拨片、篮球护腕、没喝完的奶茶和临时塞回来的外卖袋；抽屉常有被忘掉的票根 | 手机使用频率最高，常开语音、短视频和群聊；外放、蓝牙音箱、便携麦或便宜声卡可以出现；键盘可能有颜色但收线随意 | 他想被看见，也怕被认真看见。桌面热闹说明社交广，乱说明情绪处理靠拖延；不是单纯花钱大手大脚，而是容易把生活摊开 | 适合引发外放、熄灯、进出门、忘收外卖和借线旧账；也适合修补：他把音量调小、把椅子推回去、把主持稿压在公共桌角 | 主持试训后桌上多稿纸和水杯；感情线后多礼物包装、送水和起哄痕迹；强冲突后桌面会短暂收干净，像装作没事 |
| 唐骁 | 靠窗桌位，光线好，适合摆书、计划表和电脑；他能看见房间大半，也容易注意谁动了公共物品 | 桌面分区明显：电脑、课本、计划表、便签、U 盘、移动硬盘、线夹、纸巾和水杯各有位置；抽屉里有备用笔、打印纸和证件复印件 | 笔记本配置不一定最高，但稳定耐用；机械键盘或鼠标偏实用，线会被理顺；手机日历、提醒事项、文件夹和截图命名整齐 | 家庭不一定富裕，但支持他把学习和设备准备好；他的安全感来自“东西有位置、事情有流程” | 适合引发规则冲突：公共桌谁占了、线谁拿了、文件谁改名、键盘声什么时候停；也适合修补：他把规则从管人改成共处工具 | 大一桌面像新生清单；大二实验报告和竞赛资料变多；第五卷 5A 可用规则纸、线夹、公共桌分区推动冲突；第六卷桌面转向实训、简历和毕业设计 |
| 陆沉 | 阳台门边和墙角位置，靠近晾衣、通风和可短暂躲开的边缘；不占中间动线 | 桌面东西少：旧笔记本、黑色背包、饭卡、小票、便宜水杯、一本夹票据的本子、旧耳机、勤工表截图打印件；物品常往里收半寸 | 笔记本可能旧、二手或配置一般，但维护得干净；鼠标垫磨边，耳机线缠过，充电器有胶带；手机不常亮，亮起来多是勤工群或家里消息 | 家庭压力和节省感藏在旧物里，但旧不等于脏乱。他用维护旧物来维持体面，也用不添置来减少需求 | 适合写长期忍耐：别人占了他的插孔、把东西压到他桌边、外放让他戴耳机；也适合写信任：他愿不愿把东西从桌里拿出来给别人看 | 第三卷勤工和材料使桌面多截图和文件；第四卷留校时其他桌空，他的桌更像房间最后的锚；第五卷 5A 必须让他的旧物被看见但不被替他说话 |

## 公共区细则

| 区域 / 物件 | 作用 | 可推动的关系 |
|---|---|---|
| 公共桌 | 临时放外卖、规则纸、电脑、材料、四个人一起改文档的地方 | 谁占得久，谁收得快，谁把别人东西挪开，都会变成宿舍温度 |
| 插排和充电线 | 最常见旧账之一，和军训手机袋、晚风夜聊、熄灯、上机作业都能相连 | 借线不还、插头被拔、谁先给手机充电、谁把坏线胶带缠好 |
| 路由器 / 网速 | 寝室娱乐和专业作业共同依赖 | 打游戏卡、上机提交卡、视频通话卡，会让学习和娱乐互相指责 |
| 空调遥控器 | 夏天、作息、身体感和公共规则的核心 | 谁怕冷，谁嫌热，谁半夜调温度，谁把遥控器放到公共桌 |
| 垃圾桶 | 外卖、奶茶、纸团、包装袋和懒得下楼的证据 | 周屿的外卖袋、唐骁的纸张、林亦舟的药盒、陆沉的饭团包装都可能在这里碰头 |
| 阳台晾衣 | 私人物品最容易暴露的公共地带 | 谁晾衣挡了风，谁袜子掉了，谁帮谁收衣服，谁的旧外套洗得发白 |
| 门后挂钩 | 出入、钥匙、雨伞、球拍、外套的交汇点 | 周屿进出快，唐骁会把东西挂正，林亦舟常站在门口，陆沉会把自己的东西往里挪 |
| 椅子 | 每个人最小的领地 | 坐谁的椅子、把椅子推不推回去、争吵后椅背朝向哪里，都是关系信号 |

## 电子设备使用规则

不要写“他用某某高端型号”来代替人物。设备要通过以下方式进入剧情：

- 声音：键盘声、外放、耳机漏音、手机震动、风扇转速。
- 磨损：手机壳旧角、数据线胶带、鼠标垫磨边、电脑包拉链卡住。
- 借用：充电线、耳机、U 盘、鼠标、电脑、移动硬盘。
- 升级：大二换内存、买新硬盘、换手机、旧电脑送修。
- 家庭：父母开学时买的电脑、生活费紧后没换的手机、父亲寄来的旧充电头、母亲提醒别买贵外设。
- 爱好：周屿的麦克风和耳机，唐骁的线夹和硬盘，陆沉的旧耳机和勤工表，林亦舟的游戏语音和晚风聊天框。

## 阶段使用建议

第一卷：四张桌子都是空白。重点写谁先把什么放上去，谁的家庭痕迹最多，谁的东西少得不合适。

第二卷：军训服、手机袋、充电线、热水、晾衣和汗味进入桌面。手机被收走时，桌上的充电器和空手机壳会更明显。

第三卷：专业课、机房、共享文档、主持稿、新闻素材和水房报告让桌面变密。键盘声、文件命名、外放和公共桌边界开始制造摩擦。

第四卷：暑假让桌子空下来。谁的桌空得干净，谁留下旧耳机线，谁的勤工表还压着，谁的家庭小物没有带走，都能写出空宿舍的重量。

第五卷：5A 宿舍修补线必须回到四张桌子。争吵不只发生在嘴上，也发生在椅子、线、公共桌、遥控器、旧外卖袋和谁没有把东西放回原位上。

第六卷：桌面转向去向。实习电脑、考研书、简历、竞赛证书、毕业设计材料、旧手机和外设升级，能让选择变得具体。

第七卷：清桌是终章重点。四年里的旧线、坏耳机、过期药、准考证、旧小票、游戏账号、毕业照衬衫和没带走的小物件，都可以成为最后回响。

## 禁令

- 不要把桌面写成物品陈列清单。每个物件必须连到人或事件。
- 不要用设备贵贱直接判断家庭背景。贵、旧、新、坏都要看使用方式。
- 不要让宿舍桌面只服务男生爱好。它还要承载家庭、学业、亲密关系、情绪回避和公共生活。
- 不要让林亦舟永远站在门口。门口可以是回避起点，但最终要让他坐回自己的位置。
- 不要让陆沉的旧物被别人替他解释。旧物可以被看见，但话要留给陆沉自己。
