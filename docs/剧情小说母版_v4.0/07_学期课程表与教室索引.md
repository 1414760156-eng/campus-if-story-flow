# 07 学期课程表与教室索引

本文件用于给正文写作和后续互动拆分提供“课程-地点-时间”索引。它不是玩家可见的系统表，而是保证大学时间线真实、教室使用合理、人物日常动线可被复用的底层设定。

## 使用原则

1. 同一门课在同一学期一般固定一个教室、机房或实验室。连续多章出现同一教室时，必须让读者知道它属于同一门课，而不是所有课程都在同一地点。
2. 不同课程、不同学期、班会、答疑、补考、期末考试、等级考试和社团咨询会应使用不同地点。考试地点按考务安排走，不默认等于平时上课教室。
3. 课程地点要服务日常剧情：离宿舍近就能赶早饭，离八号教学楼远就会压缩吃饭和补觉，机房排队会影响晚饭和社团，B204/B206 班会会让人从食堂或宿舍被通知牵走。
4. 明德楼仍只用于学工、融媒体、行政材料和相关说明会，不承接班会、普通课程、机房课、答疑或专业导论。

## 动线写法

- 青枫居到 A101/B204/B206：可以写成普通赶课路线，适合早饭、楼梯、人流、忘带材料。
- 青枫居到 C201/C305：要有楼层和教学区内部寻找感，适合迟到边缘、临时打印、从食堂赶过去。
- 青枫居到八号教学楼机房：距离更远，适合“先吃饭还是先占机位”的小争执，也适合让周屿、唐骁、陆沉和林亦舟分散就座。
- 教学楼到东区商业街、健源体育场或晨光体育馆：适合下课后被社团、运动、排练和吃饭拉走。
- 教学楼到图书馆、阳光书屋、星辰书屋：适合期末周、证书考试、材料整理、关系降温后的安静回声。

## 回改要求

上册 1-4 卷回改时，优先补三类信息：

1. 课程表入口：班群课表、寝室群截图、唐骁的计划表、学习委员通知。
2. 固定课说明：第一次出现某课程地点时说明它属于哪门课，本学期是否固定在这里。
3. 路线差异：同一天出现两门课时，不要只写“去教学楼”，要写从 A101 到 B204、从西园到 C201、从青枫居到八号教学楼机房等具体移动。
