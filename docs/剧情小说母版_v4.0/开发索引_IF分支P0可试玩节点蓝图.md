# IF 分支 P0-D 第五卷站队节点蓝图

## 定位

本文件用于把“时期外流点 + IF 命运分歧”落实成第一批可试玩节点。它不是支线外传目录，也不是小说正文重写，而是互动剧情开发时的路线骨架。

矩阵复核后，本文件仅作为 `开发索引_IF命运分歧开发矩阵.md` 中 P0 的第五卷站队子模块使用，不代表完整 P0。完整 P0 还需要补齐第三幕 A3 社团活动父池 / 完美子外流、第四幕暑假 / 兼职线、第五幕主轴晚风线与 A3 女生派生感情线和 5X 唯一硬外流。

核心原则：

- 主线锁点继续发生，玩家不能取消军训手机事件、汇演、班会、主持招新、第五卷站队、第六卷关系定形和第七卷毕业流程。
- 玩家改变的是林亦舟把自己投向哪里：宿舍、周屿、唐骁、陆沉、摆烂、活动、打工、亲密关系或 5X 外流。
- 被选择的方向获得更多当面事件、更多代价、更多关系回声和更明确的毕业结算。
- 未被选择的人仍按母版生活，只是与林亦舟的交集减少、语气变淡、误读变多或只在固定流程中重聚。
- 5X 仍是唯一硬外流。其他路线都是软外流：生活重心偏移，但主线世界不被重写。

### 路线判定

| 判定层 | 说明 | 例子 |
|---|---|---|
| 微变量 | 单次选择只改关系、心态和后续语气 | `zhou_trust +1`、`avoidance +1`、`time_debt +1` |
| 路线倾向 | 同一方向连续选择 2-3 次后出现路线提示 | 多次替唐骁核流程，进入 `tang_route_hint` |
| 软锁 | 一个时期外流点末尾确认生活重心 | 第五卷站队后进入周屿 / 唐骁 / 陆沉 / 摆烂线 |
| 硬锁 | 只用于 5X 唯一硬外流 | 进入 5X 后第六、七卷不回原多线 |
| 毕业结算 | 第七卷按长期投入方向输出结局温度 | 说开、淡化、体面不亲近、不和解、继续联系、错过 |

## 全局变量建议

| 变量 | 用途 | 增减来源 |
|---|---|---|
| `dorm_trust` | 宿舍整体互信 | 当面解释、替人确认、保留公共桌责任上升；缺席、阴阳怪气、代填下降 |
| `old_debt` | 旧账积累 | 没说清、私下吐槽、故意不帮、消息不回上升 |
| `avoidance` | 回避值 | 不进门、刷手机、绕路、只回短句上升 |
| `time_debt` | 时间欠账 | 赴约、兼职、活动、项目占用导致缺席上升 |
| `money_pressure` | 钱压力 | 兼职、生活费拖延、礼物、聚餐、交通消耗改变 |
| `public_boundary` | 公开边界敏感度 | 摄影、融媒体、朋友圈、主题材料、毕业照相关选择改变 |
| `route_focus` | 当前路线重心 | `default_4xx`、`zhou`、`tang`、`luchen`、`lieflat` 等 |
| `primary_bond` | 最靠近的人 | `zhou`、`tang`、`luchen`、`wanfeng`、`shen_jiahe`、`xia_zhiwei`、`xu_che` |
| `graduation_temperature` | 毕业结算温度 | 第七卷根据变量和路线生成 |

## P0-D：第五卷站队分歧

本子模块先开发第五卷站队分歧，因为它最直接影响第六卷关系窄化和第七卷毕业收束。路线包括：

- `DEFAULT-4XX`：宿舍主轴修补线
- `R5-ZHOU`：周屿线
- `R5-TANG`：唐骁线
- `R5-LUCHEN`：陆沉线
- `R5-LIEFLAT`：摆烂 / 不站队线

### 公共入口节点

#### ACT5_STAND_00_GROUP_MESSAGE：事项群变短

| 项目 | 内容 |
|---|---|
| 场景 | 4XX 事项群、公共桌、青枫居走廊 |
| 触发 | 第五卷宿舍裂痕进入显性阶段 |
| 功能 | 让玩家第一次感到“大家已经不默认解释” |
| 主线锁点 | 宿舍强冲突必定继续发生 |

选择：

| 选项 | 效果 | 路线倾向 |
|---|---|---|
| 在群里补一句“等我回去当面说” | `dorm_trust +1`、`repair_intent +1` | DEFAULT-4XX |
| 私聊周屿，问他是不是又没看完整消息 | `zhou_trust +1`、`tang_misread +1` | R5-ZHOU |
| 私聊唐骁，让他先把事实顺一遍 | `tang_trust +1`、`rule_pressure +1` | R5-TANG |
| 私聊陆沉，问他要不要先别进群 | `luchen_trust +1`、`others_misread +1` | R5-LUCHEN |
| 不回，先把手机扣下 | `avoidance +1`、`old_debt +1` | R5-LIEFLAT |

#### ACT5_STAND_01_DOOR_OR_OUTSIDE：进门还是拖开

| 项目 | 内容 |
|---|---|
| 场景 | 青枫居门口、楼下长椅、便利店、湖边路线 |
| 触发 | 林亦舟回到 4XX 门口前 |
| 功能 | 区分“当面处理”和“短暂偏移”，但除 5X 外不硬外流 |

选择：

| 选项 | 效果 | 后续 |
|---|---|---|
| 直接进门 | `repair_intent +1`，进入公共桌冲突 | DEFAULT-4XX |
| 去楼下坐两分钟，等一个室友出来 | 根据遇见对象加对应室友信任 | 同质回流 |
| 去便利店买水，拖到群里再响一次 | `avoidance +1`、`time_debt +1` | 摆烂或异质回流 |
| 去湖边 / 凌空栈道绕路，碰见夏知微或收到晚风消息 | `public_boundary +1` 或 `intimacy_pull +1` | 异质回流，不改主线 |
| 继续往后街走，不回消息 | 若前置条件满足，开启 5X 风险提示 | 5X 候选，不立刻锁 |

#### ACT5_STAND_02_PUBLIC_TABLE：公共桌爆发

| 项目 | 内容 |
|---|---|
| 场景 | 青枫居 4XX 公共桌 |
| 触发 | 门口节点后必达 |
| 功能 | 站队分流核心，四人缺点正面暴露 |

选择：

| 选项 | 效果 |
|---|---|
| 先让每个人把事实说完 | `dorm_trust +1`、`repair_intent +1`，DEFAULT 倾向 |
| 接住周屿的面子，说“别先把他写成故意的” | `zhou_trust +2`、`tang_misread +1`、`luchen_silence +1` |
| 支持唐骁按证据和时间线来 | `tang_trust +2`、`rule_pressure +1`、`zhou_face_loss +1` |
| 让陆沉先说完，不许别人替他解释 | `luchen_trust +2`、`others_misread +1` |
| 说“随便，你们定”，然后退出讨论 | `lieflat_score +2`、`old_debt +2`、`dorm_trust -1` |

### DEFAULT-4XX：宿舍主轴修补线

#### 事件链

| 节点 ID | 场景 | 核心事件 | 可操作选择 | 代价 |
|---|---|---|---|---|
| DORM_01_TABLE_AFTER | 公共桌散场后 | 林亦舟留下收拾被压皱的表格和外卖袋 | 收周屿的东西 / 留唐骁的纸 / 不碰陆沉的袋子 | 谁被照顾，谁也会觉得自己被判断 |
| DORM_02_WATER_ROOM | 水房 / 洗衣房 | 两个室友错位出现，话说一半 | 问事实 / 只递东西 / 装没听见 | 问太多会被防备，不问会错过修补 |
| DORM_03_DINNER_ATTEMPT | 东北饺子馆或食堂二楼 | 试图让四人坐下吃饭 | 先定位置 / 只填自己 / 等最后一个人 | 饭局可能失败，钱和时间被耗掉 |
| DORM_04_PROJECT_ECHO | 第六卷 C407 | 四人能专业协作，但不亲密 | 主动补一句解释 / 只交付任务 / 留空接收人 | 影响第六卷事项群语气 |
| DORM_05_GRAD_ECHO | 第七卷清寝 / 最后一顿饭 | 旧账以物件和流程回响 | 说开 / 淡化 / 仅留存 / 不追究 | 结算不是大团圆 |

#### 结算文本方向

- 高 `dorm_trust` + 高 `repair_intent`：有机会说开，但仍保留各自去向。
- 中等 `dorm_trust` + 高 `old_debt`：体面但不亲近。
- 高 `avoidance`：毕业时很多话变成“算了”。

### R5-ZHOU：周屿线

#### 路线主题

林亦舟选择靠近周屿，不是简单替他辩护，而是进入“热闹、面子、公开表达和玩笑后果”的生活半径。周屿线要让玩家看见：热闹不是无辜的，玩笑也不是轻的；但周屿也不是只会逃避，他只是害怕被严肃场合当众定罪。

#### 事件链

| 节点 ID | 场景 | 核心事件 | 可操作选择 | 代价 |
|---|---|---|---|---|
| ZHOU_01_BACKSTAGE | 九龙壁舞台 / 主持队边上 | 周屿被活动圈起哄，仍想把宿舍事打成笑话 | 当众帮他圆 / 私下戳破 / 什么都不说 | 圆场会增加唐骁误读，戳破会伤周屿面子 |
| ZHOU_02_NIGHT_FOOD | 后街烧烤 / 东区商业街 | 周屿想用夜宵把事情盖过去 | 陪他坐下 / 逼他说实话 / 拍照发群 | 陪坐会缺席 4XX，发群会触发公开边界 |
| ZHOU_03_QINYUE_PULL | 秦越活动圈 | 秦越临时拉人救场，周屿默认林亦舟会帮 | 去帮 / 拒绝 / 只帮到截止时间 | 去帮增加活动债，拒绝影响周屿信任 |
| ZHOU_04_PUBLIC_STORY | 许棠 / 融媒体材料 | 周屿担心自己被写成漂亮故事里的笑点 | 支持删改 / 让他自己确认 / 交给许棠 | 公开边界改变毕业材料 |
| ZHOU_05_GRAD_ECHO | 最后一顿饭 / 武生院站 | 周屿用玩笑留半句道歉 | 接住玩笑 / 认真回应 / 装没听见 | 决定结算是“热闹还在”还是“只剩群里偶尔响” |

#### 关系后果

- 唐骁会更容易觉得林亦舟偏向“讲面子不讲规则”。
- 陆沉可能觉得自己的沉默又被热闹盖过去。
- 周屿好感上升不等于宿舍修复，上升的是“他愿不愿意在玩笑后面多说一句”。

### R5-TANG：唐骁线

#### 路线主题

林亦舟选择靠近唐骁，就进入规则、流程、项目、边界和秩序的生活半径。唐骁线不能写成“正确答案线”，它要显示规则能保护人，也能伤人；能让事情推进，也会让关系变冷。

#### 事件链

| 节点 ID | 场景 | 核心事件 | 可操作选择 | 代价 |
|---|---|---|---|---|
| TANG_01_TIMELINE | 4XX 公共桌 / 打印店 | 唐骁要求把事实按时间线列出来 | 跟他列时间线 / 提醒他别像审人 / 拒绝做表 | 列表提高项目稳定，降低宿舍温度 |
| TANG_02_C407_FIX | C407 / 机房 | 项目演示前有人又没按时交 | 支持唐骁锁版本 / 留一分钟等人 / 自己补缺口 | 锁版本伤关系，等人伤进度，补缺口伤自己 |
| TANG_03_DUMPLING_TABLE | 东北饺子馆 | 唐骁用稳定正餐替代道歉 | 坐下吃 / 问他是不是只会安排 / 替别人占位 | 饭桌能缓和，但不能替说开 |
| TANG_04_BOUNDARY_DOC | 毕业材料确认 | 唐骁坚持所有人本人确认 | 支持本人确认 / 替缺席者留言 / 留空 | 影响第七卷毕业流程冲突 |
| TANG_05_GRAD_ECHO | 离校手续 | 唐骁给出最实用的照看 | 收下 / 开玩笑缓和 / 说清以前的不舒服 | 结算偏清楚、稳，但温度不一定高 |

#### 关系后果

- 项目、毕业材料和流程更稳。
- 周屿可能把林亦舟看成“和唐骁一起审他的人”。
- 晚风或亲密线会被截止时间、演示、签字挤压。

### R5-LUCHEN：陆沉线

#### 路线主题

林亦舟选择靠近陆沉，进入勤工、钱、沉默、家庭压力、证书和边界的生活半径。陆沉线不是拯救线，而是玩家学会不替他发声、不替他解释，只给他一个能自己开口的位置。

#### 事件链

| 节点 ID | 场景 | 核心事件 | 可操作选择 | 代价 |
|---|---|---|---|---|
| LUCHEN_01_SHIFT | 图书馆勤工 / 星辰书屋 | 陆沉因为排班错过当面讨论 | 去找他 / 在群里替他说 / 不问 | 找他会缺席宿舍现场，替他说会改写边界 |
| LUCHEN_02_RECEIPT | 食堂二楼 / 快递站 | 小票、生活费或快递暴露家庭压力 | 直接问 / 只帮他拿东西 / 转移话题 | 问太急会退缩，不问会错过信任 |
| LUCHEN_03_CERT_CHECK | 明德楼窗口 / 打印店 | 证书或材料核验卡住 | 陪他排队 / 让唐骁处理 / 让他自己说 | 陪排队提高信任，也增加别人误读 |
| LUCHEN_04_QUIET_CONFLICT | 4XX | 周屿或唐骁误解林亦舟总替陆沉想 | 解释 / 让陆沉自己说 / 先不回应 | 决定陆沉线是变深还是变重 |
| LUCHEN_05_GRAD_ECHO | 旧书 / 补登记 | 陆沉用旧物说迟来的在意 | 接过 / 问清楚 / 只说谢谢 | 结算偏低声、重，但不一定圆满 |

#### 关系后果

- 林亦舟更懂现实压力，但也会被其他人误读为偏向沉默的一边。
- 陆沉信任上升时，不应变得外向，只是愿意多说一句。
- 这条线的甜或暖必须轻，不能把沉默人物写成突然倾诉机器。

### R5-LIEFLAT：摆烂 / 不站队线

#### 路线主题

摆烂线不是搞笑线。它的核心是：没有强烈爆炸，但入口一个个关掉。玩家每次都选省力，事情也许不会立刻坏，但别人会越来越少问他。

#### 事件链

| 节点 ID | 场景 | 核心事件 | 可操作选择 | 代价 |
|---|---|---|---|---|
| LIE_01_UNREAD | 手机 / 外卖架 | 群消息一直没回，外卖到了 | 先吃 / 回个表情 / 假装没看见 | `old_debt +1`，宿舍语气变短 |
| LIE_02_GAME_OR_SLEEP | 寝室床帘 / 游戏语音 | 冲突后选择游戏或睡觉跳过现场 | 打游戏 / 刷短视频 / 关机 | 资源和关系入口减少 |
| LIE_03_BACKSTREET_IDLE | 后街 / 台球厅 / 便利店 | 外面很轻松，没人问旧账 | 跟着坐一会 / 早点回 / 继续拖 | 可能触发 5X 风险，但不默认硬外流 |
| LIE_04_MISSED_HELP | 材料 / 饭局 / 清寝 | 因没确认错过一次补救 | 道歉 / 装作不知道 / 让别人处理 | 影响第六卷事项群和第七卷毕业流程 |
| LIE_05_GRAD_ECHO | 离校 | 很多话没有合适语气 | 发群消息 / 私聊一个人 / 什么都不发 | 结算偏“没有大事，但很多入口过期” |

#### 关系后果

- 弱冲突多，强冲突少，但后果不断积累。
- 宿舍不会一直骂他，而是慢慢不等、不问、不把他算进去。
- 适合做低投入路线和隐藏成就，但不能让摆烂没有成本。

## P1 预留：第三幕、第四幕、第五幕晚风线 / 社团女生派生感情

P1 不抢 P0，但需要先留好入口，避免后续接 JSON 时路线打架。

### A3-ACTIVITY-PUBLIC：第三幕社团活动父池

第三幕入口不能直接理解为“选融媒体”或“进入完美线”。先用父池记录玩家持续靠近的通道，再判断是否进入 `R3-PERFECT`。

试玩拆分时，A3 父池固定为三次社团 / 活动机会：

| 机会 | 节点功能 | 选择后果 |
|---|---|---|
| 第一次 | 全开放试探 | 所有社团 / 活动方向都可选，记录 `a3_first_focus`。 |
| 第二次 | 维持或试新 | 可以维持第一次方向，也可以去别的方向试试，记录 `a3_second_focus`。 |
| 第三次 | 结算 | 只能选坚持第一个、坚持第二个、两个都要、两个都不要。 |

第三次后才允许生成子外流。坚持第一个 / 第二个才锁定单一社团方向；两个都要自动归为压力模式；两个都不要进入宿舍线。`R3-PERFECT` 的条件不是“选择某个社团”，而是坚持单一方向且 `mode_perfect` 成立。

| focus | 典型场景 | 后续影响 |
|---|---|---|
| `focus_hosting` | 主持招新 / 九龙壁舞台 / 秦越临时物资桌 | 周屿线前置、活动人情债。 |
| `focus_newsroom` | 新闻中心 / 融媒体 / 明德楼材料窗口 | 公开边界、材料流向。 |
| `focus_photo` | 摄影社 / 夏知微 / 正脸授权 | 影像边界和被看见的代价。 |
| `focus_backstage` | 签到表 / 串场稿 / 班会归档 | 完美线或项目线前置。 |
| `focus_dorm_return` | 回 4XX 处理作业和公共桌 | 保持宿舍主轴。 |
| `focus_public_avoid` | 拒绝曝光 / 不上台 / 借热闹逃开 | 回避、摆烂或 5X 前置。 |

### R3-PERFECT：A3 父池下的完美 / 自我管理线

## 验收标准

第五卷站队子模块写入后续开发数据前，至少满足以下条件：

1. 每条 P0-D 站队路线至少有 5 个节点，其中 1 个入口、3 个中段事件、1 个毕业结算入口。
2. 每个节点至少有 3 个可操作选择，且不全是“继续/返回”。
3. 每个路线都必须有代价：时间、钱、关系误读、错过、公开风险或资源流失。
4. 宿舍四人仍优先。外部人物只能补位，不能替 4XX 处理主线旧账。
5. 5X 只允许从唯一硬外流点进入，进入后不再回原多线。
6. 晚风不是必选路线；亲密方向可以靠近，也可以停在朋友、错过或转向沈嘉禾 / 夏知微。
7. 第七卷结算不强迫团圆，要允许说开、淡化、体面不亲近、不和解和继续联系并存。
