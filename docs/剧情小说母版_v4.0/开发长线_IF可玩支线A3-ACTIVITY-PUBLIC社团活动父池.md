# 开发长线_IF可玩支线A3-ACTIVITY-PUBLIC社团活动父池

## 用途

本文用于把 `A3-ACTIVITY-PUBLIC` 从“第三幕社团 / 活动选择骨架”扩展成可拆剧情页、选项页、权限判断和后续路线分发的 B 级父池开发稿。

它不是 `R3-PERFECT` 完美线本体，也不是 `R5-ROMANCE` 感情线入口本体。它规定的是：第三幕里，玩家让林亦舟把时间投向主持队、新闻中心 / 融媒体、摄影社、幕后物资、志愿服务、回 4XX 或回避公开时，系统怎样记录生活半径、公开边界、女生候选权限、完美子外流来源和后续白名单。

本父池的核心任务不是“让玩家参加更多社团”，而是把第三幕的公共生活第一次切成可追踪的选择：他到底被谁看见、替谁做事、把什么材料放进公开场、有没有形成可持续的女生候选来源，以及他是否还回得去 4XX 当面说话。

## 路线定位

父池编号：`A3-ACTIVITY-PUBLIC`

路线池编号：`POOL-A3-ACTIVITY-PUBLIC`

路线等级：B 级父池 / 分发池

入口时期：第三幕，汇演、班会、主持招新、摄影社、融媒体和活动圈开始成形时。

共享范围：第一卷、第二卷共享；第三幕进入父池；第五幕普通长线锁定前仍必须满足最低主线选择要求；若第三幕只形成父池倾向而未确认子外流，第六卷可按 `POOL-A3-ACTIVITY-PUBLIC` 的父池回声承接。

核心体验：林亦舟不是立刻变成某个社团的人，也不是立刻进入完美线或感情线。他只是第一次发现，公共生活会把人分走：一张签到表、一次照片授权、一份归档材料、一个主持队临时群、一次后台物资签收，都可能让他被活动圈记住，也让 4XX 少等他一次。

本父池重点不是热闹，而是分发。热闹只是表面，真正的玩法是：三次机会之后，玩家必须承认自己把时间交给了哪里。

## 三次父池选择规则

第三幕父池一共有三次社团 / 活动机会。

| 窗口 | 规则 | 可选范围 | 结果 |
|---|---|---|---|
| `A3-CHOICE-01` | 第一次全开放试探 | 主持、新闻 / 融媒体、摄影、幕后、志愿、回 4XX、回避公开 | 写入 `a3_first_focus` |
| `A3-CHOICE-02` | 第二次可维持或试新 | 维持第一次方向，或选择另一个方向 | 写入 `a3_second_focus` |
| `A3-CHOICE-03` | 第三次必须收束 | 坚持第一个、坚持第二个、两个都要、两个都不要 | 写入 `a3_final_choice` |

第三次结算规则：

- `keep_first` / `keep_second`：锁定单一父池方向，进入对应 `route_focus_tag`。
- `take_both`：自动进入 `mode_pressure`，不能锁 `R3-PERFECT`，也不能直接锁 `R5-ROMANCE`。
- `take_neither`：回到宿舍主轴或回避 / 摆烂倾向，不能凭空生成社团女生候选。
- 若第三次前没有形成连续任务级接触，女生角色只算软照面，不算 `female_candidate_present`。

## 父池 focus

| focus | 内容 | 主要人物 | 默认权限 |
|---|---|---|---|
| `focus_hosting` | 主持招新、串场、候场、秦越流程、周屿活动圈 | 周屿、秦越、许棠软照面 | `club_without_female_candidate` |
| `focus_newsroom` | 新闻中心 / 融媒体、采访、公开文案、授权材料 | 许棠、辅导员、沈嘉禾可作为材料支线出现 | 默认无候选；若沈嘉禾连续承担材料任务，可升为 `club_with_female_candidate` |
| `focus_photo` | 摄影社、照片授权、活动影像、晨光湖和凌空栈道 | 夏知微、摄影社成员 | 满足连续任务后可为 `club_with_female_candidate` |
| `focus_backstage` | 幕后物资、签到、道具、节目单、归档 | 秦越、唐骁、周屿、许棠 | 默认无候选；需额外候选来源才能升级 |
| `focus_volunteer` | 志愿服务、服务型社交、秩序维护 | 志愿负责人、同学、老师 | `club_without_female_candidate` |
| `focus_dorm_return` | 不接长期活动，回 4XX 当面处理 | 周屿、唐骁、陆沉 | 回 `DEFAULT-4XX` |
| `focus_public_avoid` | 拒绝曝光、不上台、不入群、借热闹逃开 | 4XX、活动群软回声 | `R5-LIEFLAT` 或低温回避倾向 |

## 父池 mode

| mode | 触发方式 | 允许后续 |
|---|---|---|
| `mode_normal` | 正常参与，有活动回声但不持续自控或强压力 | 父池回声、宿舍主轴、晚风、站队组 |
| `mode_perfect` | 流程、低风险、按时、公开可控持续成立 | `R3-PERFECT` |
| `mode_pressure` | 两个都要、救场、人情债、多头补位持续累积 | 活动压力回声、周屿前置、低温父池承接 |
| `mode_avoid` | 拒绝公开、不入群、逃开活动又不回寝 | `R5-LIEFLAT` 或 5X 阴影 |
| `mode_dorm` | 连续回 4XX，当面处理关系 | `DEFAULT-4XX` |

## 女生候选权限

本父池只负责“候选来源是否合法”，不直接写成完整感情线。

| 来源 | 可否成为 `R5-ROMANCE` 候选 | 必要条件 | 不足时处理 |
|---|---|---|---|
| 沈嘉禾 / 材料协作 | 可以 | `shen_jiahe_link >= 2`，至少两次材料、共享文档、打印店或阳光书屋任务级接触；她承担现实压力，不只是帮忙改格式 | 只作为材料边界软照面 |
| 夏知微 / 摄影社 | 可以 | `xia_zhiwei_link >= 2`，至少两次照片授权、活动影像、晨光湖或凌空栈道任务级接触；她承担观看和公开边界压力 | 只作为观看角度软照面 |
| 许棠 / 融媒体 | 当前不开放为 romance 候选 | 可提供公开叙事、采访、授权和活动材料压力 | 只作为公开叙事软照面；不写入 `romance_candidate` |
| 主持队 / 秦越 / 周屿活动圈 | 不开放 | 可加强 `focus_hosting`、`activity_link`、周屿线前置 | 不得凭空跳沈嘉禾 / 夏知微 |
| 幕后 / 志愿 | 默认不开放 | 除非另有沈嘉禾或夏知微连续任务介入 | 只流向 `R3-PERFECT`、宿舍线、晚风线或摆烂线 |

权限判定：

```json
{
  "route_parent_pool_id": "POOL-A3-ACTIVITY-PUBLIC",
  "outflow_permission_class": "club_without_female_candidate",
  "female_candidate_present": false,
  "romance_outflow_allowed": false,
  "candidate_source_tag": "none",
  "allowed_romance_candidates": []
}
```

若沈嘉禾或夏知微达标，才允许改为：

```json
{
  "outflow_permission_class": "club_with_female_candidate",
  "female_candidate_present": true,
  "romance_outflow_allowed": true,
  "candidate_source_tag": "a3_photo_or_material",
  "allowed_romance_candidates": ["shen_jiahe", "xia_zhiwei"]
}
```

实际拆 JSON 时，`allowed_romance_candidates` 不能同时把两个候选都写成已锁定对象；第五幕 `P0C_ACT5_CANDIDATE_CONFIRM` 必须再选择一个 `romance_candidate`。

## 不改变的母本锁点

- 汇演、班会、主持招新和活动圈成形仍按母本发生。
- 周屿仍会被活动场面推到更显眼的位置，秦越仍负责流程和现场调度。
- 许棠、沈嘉禾、夏知微都按各自生活半径存在，但不能抢宿舍四人和晚风主轴。
- 第四幕暑假、生活费、课程压力仍发生；A3 不取消 `R4-WORK` 的可能。
- 第五幕宿舍强冲突仍发生；A3 只改变林亦舟进入冲突时的位置、公开信用和活动牵引。
- 第五卷普通长线锁定前仍必须满足最低 5 个主线选择窗口。
- 第六卷若仍停留在父池，必须使用 20 个剧情段、5 个四段组、5 个二方向选择窗口；不能重新横跳所有社团。
- 5X 仍是唯一硬外流。A3 的公共生活不能提前变成“新圈子无旧账”。

## 第五 / 第六卷硬约束接入

`A3-ACTIVITY-PUBLIC` 虽然是第三幕父池，但后续若要锁普通完整长线，仍必须服从第五卷最低主线选择要求：

- 至少 5 个主线选择窗口：宿舍站队、亲密 / 晚风、项目 / 规则、活动 / 兴趣、回避 / 外流。
- 每个计入窗口都必须有 `act6_echo_hook`。
- 未达标时，A3 只能作为活动倾向、候选来源或低温回声，不能直接锁完整第六卷。
- 第六卷进入 `POOL-A3-ACTIVITY-PUBLIC` 后，固定 20 个剧情段、5 个四段组、5 个二方向选择窗口；每个窗口只给两个当前父池内方向。

## 核心变量

| 变量 | 含义 | 高值倾向 | 风险 |
|---|---|---|---|
| `a3_activity_choice_count` | 第三幕父池选择次数 | 达到 3 后必须结算 | 不能无限横跳社团 |
| `a3_first_focus` | 第一次全开放选择 | 记录初始兴趣 | 不能直接当作锁线 |
| `a3_second_focus` | 第二次维持或试新选择 | 判断持续性或摇摆 | 试新会增加解释成本 |
| `a3_final_choice` | 第三次收束选择 | `keep_first` / `keep_second` / `take_both` / `take_neither` | 决定是否可进子外流 |
| `a3_focus_votes` | 各 focus 累计 | 判断最终主通道 | 多头过高进入压力模式 |
| `a3_mode_votes` | 各 mode 累计 | 判断普通、完美、压力、回避或回寝 | 过分自控会降低宿舍温度 |
| `activity_link` | 活动圈牵引 | 秦越、周屿、许棠、摄影社更常找他 | 宿舍误读他“不在” |
| `public_credit` | 公开场可靠度 | 老师、活动圈默认他能处理事 | 容易被继续加活 |
| `self_control` | 自我管理与流程优先 | 可进入 `R3-PERFECT` | 情绪表达延迟 |
| `emotional_delay` | 把关系问题推迟处理 | 先交材料、先保现场 | 第五卷旧账更重 |
| `public_boundary` | 公开材料、照片、采访、署名边界 | 降低公开风险 | 低值引发授权误读 |
| `candidate_source_tag` | 候选来源 | `shen_material` / `xia_photo` / `none` | 防止凭空开感情线 |
| `female_candidate_present` | 是否有女生候选连续出现 | 允许候选核验 | 一次擦肩不得记 true |
| `romance_outflow_allowed` | 是否允许晚风外感情线 | 可接 `R5-ROMANCE` | 不得从主轴直接跳线 |
| `dorm_warmth` | 4XX 日常温度 | 保留当面解释入口 | 活动过高会降温 |
| `time_debt` | 时间欠账 | 活动、排练、归档占用时间 | 挤压宿舍、晚风和课程 |
| `graduation_temperature` | 毕业结算温度 | 公开信用、候选来源、低频联系或体面淡化 | 由前述变量综合决定 |

共享回声变量：

| 变量 | 来源 | 在本父池中的用途 |
|---|---|---|
| `shen_jiahe_link` | 沈嘉禾材料协作 | 判断沈嘉禾是否能成为 `R5-ROMANCE` 合法候选。 |
| `xia_zhiwei_link` | 夏知微摄影社 / 影像授权 | 判断夏知微是否能成为 `R5-ROMANCE` 合法候选。 |
| `zhou_trust` | 周屿活动圈 | 判断主持 / 活动牵引是否转向周屿线前置。 |
| `project_stability` | C407 / 项目协作 | 判断公开流程是否影响唐骁或项目线回声。 |
| `old_debt` | 4XX 旧账 | 判断公共生活是否把宿舍旧账推迟到第五卷。 |

共享回声变量不把父池改成对应子线，只用于后续白名单和文本回声。

## 体量与密度

建议 7 章开发体量，对应约 45,000-70,000 字正文容量。本文只提供可开发骨架，不直接写足正文。

最低配置：

- 至少 24 次有效选择。
- 至少 6 个内部回流点。
- 三次父池选择窗口必须全部可玩。
- 每个 focus 至少有一次可识别的现实任务。
- 至少 2 次候选来源核验节点：沈嘉禾材料方向、夏知微摄影方向。
- 至少 3 次强后果回调：活动圈继续找他、4XX 少等他一次、公开授权或材料边界影响第五卷。

## 章节开发详表

### 第一章：招新桌不是答案

章节目标：建立第三幕父池的第一次全开放选择。玩家可以帮主持队、看融媒体、进摄影社、做幕后、做志愿、回 4XX 或回避公开，但第一次只记录兴趣，不直接锁线。

现实触发：

- 主持招新咨询点临时缺人，秦越把签到表压在水杯下面。
- 周屿在台前试词，嘴上轻松，手指一直折节目单边角。
- 许棠在旁边问公开照片和采访授权。
- 夏知微背着相机从晨光湖方向过来，问哪一块不能拍。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `A3-CH01-N01` | 父池入口说明 | 晨光大道 / 招新桌 | 林亦舟、秦越、周屿 |
| `A3-CH01-N02` | 第一次全开放选择 | 招新桌 / 摄影社摊位 / 4XX 群 | 林亦舟、许棠、夏知微、4XX |
| `A3-CH01-N03` | 记录初始 focus | 活动群入群页 | 林亦舟 |
| `A3-CH01-N04` | 当晚回声 | 青枫居 4XX | 周屿、唐骁、陆沉 |

可拆选择：

| 选择 ID | 选择 | 变量变化 | 后果 |
|---|---|---|---|
| `A3-CH01-C01` | 帮秦越接主持招新签到表 | `a3_first_focus = focus_hosting`、`activity_link +1`、`public_credit +1` | 秦越记住他能处理现场，周屿开始把他当活动同路人 |
| `A3-CH01-C02` | 去许棠那边确认采访和授权 | `a3_first_focus = focus_newsroom`、`public_boundary +1` | 公开叙事入口打开，但不自动生成 romance 候选 |
| `A3-CH01-C03` | 跟夏知微确认摄影社照片边界 | `a3_first_focus = focus_photo`、`xia_zhiwei_link +1`、`public_boundary +1` | 夏知微成为候选来源伏笔，但还不足以开线 |
| `A3-CH01-C04` | 去后台帮物资和节目单 | `a3_first_focus = focus_backstage`、`self_control +1`、`time_debt +1` | 后台记住他靠谱，4XX 晚饭缺席概率上升 |
| `A3-CH01-C05` | 只做一次志愿服务，不进长期群 | `a3_first_focus = focus_volunteer`、`public_credit +1` | 服务型回声，不自动通向感情线 |
| `A3-CH01-C06` | 回 4XX 处理作业和当面话 | `a3_first_focus = focus_dorm_return`、`dorm_warmth +1` | 活动父池变弱，宿舍主轴保温 |
| `A3-CH01-C07` | 不入群，不上台，也不回去解释 | `a3_first_focus = focus_public_avoid`、`emotional_delay +1`、`old_debt +1` | 回避倾向上升 |

章节钩子：晚上 4XX 公共桌上多出一张招新传单，周屿问“你今天到底算哪边的人？”林亦舟发现自己答不上来。

### 第二章：第二张报名表

章节目标：第二次机会让玩家维持第一次方向，或去另一个方向试试。系统开始区分“持续生活半径”和“一次擦肩”。

现实触发：

- 班会材料、活动复盘和摄影社图包同一天要确认。
- 秦越问他下次还来不来。
- 许棠说公开稿需要删掉私人寝室信息。
- 夏知微问上一组照片是否能留背影。

可拆选择：

| 选择 ID | 选择 | 变量变化 | 后果 |
|---|---|---|---|
| `A3-CH02-C01` | 维持第一次方向，继续做同一类任务 | `a3_second_focus = a3_first_focus`、`a3_focus_votes +1`、`route_confidence +1` | 单一路线稳定度上升 |
| `A3-CH02-C02` | 试一个新方向，看看是否更合适 | `a3_second_focus = new_focus`、`time_debt +1`、`emotional_delay +1` | 第三次必须二选一，解释成本上升 |
| `A3-CH02-C03` | 两边都答应，先不拒绝任何人 | `a3_mode_votes.mode_pressure +1`、`activity_link +2`、`dorm_warmth -1` | 压力模式伏笔，完美线门槛下降 |
| `A3-CH02-C04` | 只保最低参与，晚上回寝室 | `a3_mode_votes.mode_dorm +1`、`dorm_warmth +1` | 活动线降为回声 |

候选来源提示：
