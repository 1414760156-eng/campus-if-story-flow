# 开发长线_IF可玩支线R4-WORK暑假兼职线

## 用途

本文用于把 `R4-WORK` 从“第四幕暑假 / 生活费 / 勤工兼职倾向”扩展成可拆剧情页、选项页、变量回调和毕业结算的 B 级生活方式外流线。

它不是小说正文，也不是陆沉线的前置版本。它规定的是：玩家让林亦舟更早被钱、排班、生活费波动和家庭暗线切出宿舍现场时，后续大学生活如何进入“工时、成本、迟到、错过、节省和现实优先”的命运池。陆沉可以在本线中成为现实压力的镜像，但本线不自动站陆沉，也不把勤工写成救赎或励志。

## 路线定位

路线编号：`R4-WORK`

路线池编号：`POOL-R4-WORK`

路线等级：B 级生活方式外流线

共享范围：第一卷、第二卷共享；第三幕以后保留生活方式倾向；第四幕暑假形成钱压和排班入口；第五幕普通长线结算前仍必须满足最低主线选择要求。

核心体验：林亦舟开始把“够不够用”“能不能换班”“这顿饭要不要去”“这条消息能不能晚点回”放到更靠前的位置。他不是突然变穷，也不是变冷漠，而是第一次发现大学生活里很多选择都要用钱、时间、体力和可解释性来换。玩家要处理的不是“要不要努力赚钱”，而是：现实压力如何慢慢改变他出现在哪里、错过谁、被谁误读、还能不能坐回公共桌。

本线重点不是苦情，也不是励志。钱解决一部分问题，也制造一部分沉默；排班带来稳定，也切碎关系现场；节省是一种能力，也可能让别人误以为他不在乎。

## 进入条件

建议满足以下条件进入本线：

| 条件 | 建议值 | 说明 |
|---|---:|---|
| `money_pressure` | `>= 2` | 生活费延迟、饭卡余额、交通费或礼物成本已经可见 |
| `work_shift` | `>= 2` | 玩家至少两次选择勤工、快递站、打印店、校内兼职或现实任务优先 |
| `family_signal` | `>= 1` | 家庭消息、父母转账、紧急联系人或暑假去向登记出现压力 |
| `dorm_absence` | `>= 1` | 因排班、证明、取件或疲惫错过 4XX 当面现场 |
| `route_lock` | `false` | 未进入 `R5X-HARD` 唯一硬外流 |

## 不改变的母本锁点

- 第四卷暑假、家庭变化、返校压力和本人签字仍按母本发生。
- 第五幕宿舍强冲突仍发生；林亦舟更常不在第一现场，但不能因此取消冲突。
- 陆沉仍有自己的勤工和家庭压力；本线不能把陆沉写成默认同伴或默认导师。
- 周屿、唐骁、晚风、沈嘉禾 / 夏知微仍按各自母本生活半径推进，只能作为软回声或同场误读出现。
- 第六卷 C407、材料归档、关系定形仍发生；只是林亦舟更多从成本、排期和是否请假进入。
- 第七卷预登记、毕业活动材料、清寝、饭局、离校手续仍发生。
- 5X 仍是唯一硬外流。本线不能用“新兼职圈轻松无旧账”提前打开 5X。

## 第五 / 第六卷硬约束接入

`R4-WORK` 虽然从第四幕暑假开始积累，但普通长线完整锁定前仍必须服从第五卷最低主线选择要求：

- 至少 5 个主线选择窗口：宿舍站队、亲密 / 晚风、项目 / 规则、活动 / 兴趣、回避 / 外流。
- 每个计入窗口都必须有 `act6_echo_hook`。
- 未达标时，`R4-WORK` 只能作为生活方式倾向或低温软回声，不能直接锁完整第六卷。
- 第六卷进入 `POOL-R4-WORK` 后，固定 20 个剧情段、5 个四段组、5 个二方向选择窗口；每个窗口只给两个当前池内方向。

## 核心变量

| 变量 | 含义 | 高值倾向 | 风险 |
|---|---|---|---|
| `money_pressure` | 生活费、饭卡、交通、礼物、打印和医疗费用压力 | 玩家更重视成本和节省 | 容易被误读为冷淡、抠或不愿投入关系 |
| `work_shift` | 排班、勤工、兼职、临时加班和换班次数 | 工作生活半径稳定 | 关系现场被切碎，错过公共桌和饭局 |
| `family_signal` | 父母转账、联系人、家庭备注、生活用品和电话暗线 | 家庭压力更具体 | 不能写成单纯卖惨，也不能替家庭问题一笔解决 |
| `time_debt` | 因排班、交通、证明、返工产生的时间欠账 | 现实任务更可信 | 迟到、缺席、消息延迟增加 |
| `dorm_absence` | 不在 4XX 当面现场的次数 | 宿舍默认不等他 | 高值滑向低温毕业或摆烂误读 |
| `missed_chance` | 错过晚风、活动、饭局、补话入口 | 生活半径变窄 | 后续只能软回声，不再完整打开旧入口 |
| `reality_bond` | 与勤工负责人、打印店老板、快递站人员等现实关系的熟悉度 | 工作场景更有支持 | 不能变成新圈子奖励线 |
| `dorm_misread` | 宿舍对林亦舟缺席的误读 | 别人以为他不在乎或站外部现实 | 需要具体回声，不能只写旁白 |
| `energy_low` | 疲惫、低烧、夜归、精神消耗 | 可触发校医院或请假节点 | 不能变成苦情滤镜 |
| `public_boundary` | 证明材料、工资截图、请假说明和授权是否清楚 | 少替人、少默认 | 低值会引发证明和代签风险 |
| `graduation_temperature` | 毕业结算温度 | 独自处理流程、体面淡化、错过、继续低频联系 | 由前述变量综合决定 |

共享回声变量：

| 变量 | 来源 | 在本线中的用途 |
|---|---|---|
| `dorm_warmth` | `DEFAULT-4XX` 宿舍修补变量 | 标记 4XX 是否仍愿意给林亦舟留公共桌、饭局和当面解释入口。 |
| `repair_depth` | `DEFAULT-4XX` 修补深度变量 | 标记林亦舟是否把缺席、钱压或家里压力说成具体事实，而不是只留下沉默。 |
| `old_debt` | 母本旧账变量 | 标记未解释的旧误会是否继续拖到第五 / 第六卷。 |
| `proxy_risk` | 公开边界 / 代签风险变量 | 标记凑单、带话、代交、代签等现实便利是否越界。 |
| `project_stability` | C407 / 项目协作变量 | 标记排班和证明是否影响 C407 或公共材料协作。 |
| `dorm_trust` | 宿舍信任变量 | 标记毕业后是否还能保持低频真实联系。 |

说明：上述共享回声变量不把本线改回宿舍修补线，只用于承接母本事件回声；正式拆 JSON 时可作为 cross_route_echo 写入，核心路由仍由 `POOL-R4-WORK` 的 11 个局部变量控制。

## 体量与密度

建议 8 章开发体量，对应约 45,000-70,000 字正文容量。本文只提供可开发骨架，不直接写足正文。

最低配置：

- 至少 26 次有效选择。
- 至少 6 个内部回流点。
- 每个内部回流点至少 2 个方向，建议 3 个方向。
- 每个回流方向至少 3-4 个连续剧情拍点。
- 至少 3 次强后果回调：排班撞饭局、证明材料返工、第五幕不在现场、第七卷毕业流程独自处理。

| 2 | 去向登记那一栏 | `ME-SUMMER-SIGN` | 暑假留校、实习 / 兼职、本人签字和紧急联系人 | 真实去向不能由别人代填 |

## 章节开发详表

### 第一章：这周晚点

章节目标：建立钱压不是突然出现的设定。林亦舟第一次把父亲短备注、母亲生活用品、饭卡余额和晚风 / 宿舍消息放在同一个屏幕上。

现实触发：
- 父亲转账备注从完整解释变成“这周晚点”。
- 母亲问生活用品够不够，语气仍像原来那样细。
- 饭卡余额、地铁费、打印费和下一顿饭同时变得具体。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `WORK-CH01-N01` | 读取钱压入口 | 青枫居4XX / 微信转账 | 林亦舟、父亲、母亲 |
| `WORK-CH01-N02` | 决定要不要问家里 | 公共桌 / 阳台 | 林亦舟 |
| `WORK-CH01-N03` | 处理第一笔节省 | 西园食堂 / 东区商业街 | 林亦舟、室友软回声 |
| `WORK-CH01-N04` | 回到暑假去向 | 4XX 群 / B204 通知 | 林亦舟、学习委员 |

关键选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `WORK-CH01-C01` | 直接问父亲生活费是不是有变化 | `family_signal +1`、`money_pressure +1` | 第四章证明材料更早出现 |
| `WORK-CH01-C02` | 不问，先压支出 | `money_pressure +1`、`missed_chance +1` | 第五章饭局入口降温 |
| `WORK-CH01-C03` | 问陆沉勤工怎么申请 | `work_shift +1`、`reality_bond +1`、`luchen_soft_echo +1` | 第三章快递站或图书馆回声 |
| `WORK-CH01-C04` | 接一个临时跑腿 / 兼职群任务 | `work_shift +1`、`time_debt +1` | 第二章去向登记更偏留校 |

本章结尾钩子：学习委员发来暑假去向登记补充说明，留校、实习、兼职和校外住宿都要本人签字。

### 第二章：去向登记那一栏

章节目标：把“兼职 / 留校”写成流程，而不是一句倾向。玩家必须决定把真实去向写到哪里、给谁看、是否让家庭消息进入学校表格。

现实触发：
- 教学楼 B204 暑假去向登记纸质签字。
- 表格分成返家、留校、实习 / 兼职、校外住宿几摞。
- 有同学试图让室友代签，被辅导员拦下。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `WORK-CH02-N01` | 去向栏选择 | B204 | 林亦舟、辅导员、学习委员 |
| `WORK-CH02-N02` | 联系人和家庭备注 | 走廊 / 手机 | 林亦舟、父亲、母亲 |
| `WORK-CH02-N03` | 兼职说明材料 | 打印店 / 辅导员桌边 | 林亦舟、打印店老板 |
| `WORK-CH02-N04` | 回 4XX 解释去向 | 青枫居4XX | 林亦舟、室友软回声 |

关键选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `WORK-CH02-C01` | 如实填留校兼职 | `public_boundary +1`、`work_shift +1` | 第三章排班入口更稳 |
| `WORK-CH02-C02` | 只填留校，不写兼职细节 | `family_signal +1`、`dorm_misread +1` | 第四章证明材料返工 |
| `WORK-CH02-C03` | 先问父亲能不能写太细 | `family_signal +2`、`time_debt +1` | 第八章毕业联系人回声 |
| `WORK-CH02-C04` | 请同学转告自己晚点补签 | `public_boundary -1`、`missed_chance +1` | 后续代签风险被点名 |

本章结尾钩子：辅导员把“兼职”那摞纸推过来，说：“写清楚，不是为了管你，是为了出事时找得到你。”

### 第三章：快递站半天班

章节目标：让工时第一次真正切走关系现场。玩家不是选择“去赚钱”那么简单，而是在换班、饭局、宿舍消息和站长加件之间做取舍。

现实触发：
- 校园快递站暑期晚班缺人。
- 4XX 饭局、公共桌讨论或晚风消息撞上排班。
- 站长临时加件，取件码和未签收单压到手边。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `WORK-CH03-N01` | 接班或换班 | 校园快递站 | 林亦舟、站长、陆沉软回声 |
| `WORK-CH03-N02` | 饭局撞期 | 4XX 群 / 快递站后门 | 林亦舟、周屿、唐骁、陆沉 |
| `WORK-CH03-N03` | 加件和迟到 | 快递站窗口 | 林亦舟、快递站工作人员 |
| `WORK-CH03-N04` | 下班后回寝 | 青枫居楼下 | 林亦舟、宿管阿姨 |

关键选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `WORK-CH03-C01` | 尝试换班去 4XX 饭局 | `dorm_warmth +1`、`work_shift -1`、`money_pressure +1` | 工时不稳，第四章钱压更高 |
| `WORK-CH03-C02` | 推掉饭局照常排班 | `work_shift +1`、`dorm_absence +1` | 第七章被说“不在现场” |
| `WORK-CH03-C03` | 让陆沉帮问排班但不让他代替 | `reality_bond +1`、`public_boundary +1`、`luchen_soft_echo +1` | 陆沉线软回声，不锁陆沉线 |
| `WORK-CH03-C04` | 下班后补一句解释 | `dorm_misread -1`、`time_debt +1` | 第五章饭局仍可低温打开 |

本章结尾钩子：林亦舟回到青枫居时，公共桌上已经没有他的那份饭盒，只剩一双没拆的筷子。

### 第四章：打印店那枚章

章节目标：把证明材料、勤工表和项目材料绑在一起。证明不是卖惨，格式也不是小事；玩家必须处理“让别人知道多少”和“材料能不能按时交”之间的冲突。

现实触发：
- 勤工 / 兼职材料缺章，打印店和学院办公室开放时间撞期。
- C407 材料或课程证明也需要打印、签名、归档。
- 母亲问他要不要补生活费，父亲只发来一句“别写太细”。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `WORK-CH04-N01` | 打印证明 | 教学楼旁打印店 | 林亦舟、打印店老板 |
| `WORK-CH04-N02` | 学院盖章 | B204 / 辅导员办公室 | 林亦舟、辅导员 |
| `WORK-CH04-N03` | C407 材料撞期 | C407 / 共享盘 | 林亦舟、唐骁软回声 |
| `WORK-CH04-N04` | 家庭消息压回 | 阳台 / 微信 | 林亦舟、父亲、母亲 |

关键选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `WORK-CH04-C01` | 找打印店老板先打证明 | `work_shift +1`、`money_pressure +1` | 打印店成为固定生活半径 |
| `WORK-CH04-C02` | 回学院先盖章 | `public_boundary +1`、`time_debt +1` | C407 材料可能延迟 |
| `WORK-CH04-C03` | 问母亲要不要临时补生活费 | `family_signal +1`、`money_pressure -1`、`old_debt +1` | 第八章家庭联系人回声 |
| `WORK-CH04-C04` | 用 C407 材料压住勤工证明 | `project_stability +1`、`work_shift -1` | 工作线倾向下降，但钱压不消失 |

本章结尾钩子：打印店老板把章盖歪了一点，说“能用”，林亦舟看着那两个字，忽然不知道是材料能用，还是自己只能这样先用着。

### 第五章：食堂二楼省下的三块钱

章节目标：让节省进入人际场。三块钱不是大事，但如果每次都因为三块钱改地点、拒绝聚餐或不点饮料，别人会慢慢停止问他。

现实触发：
- 饭卡余额低，室友约东区商业街小饭馆。
- 食堂二楼固定窗口便宜但远，排队也久。

## 防漂移总则

- 本线不是苦情线。钱压要具体到饭卡、打印、排班、交通、证明、AA 和工时，不写抽象可怜。
- 本线不是励志线。工作能力增强不等于关系自动变好，也不等于毕业开挂。
- 本线不是陆沉线。陆沉可以理解工时和沉默，但玩家不能借他进入 `POOL-R5-LUCHEN`。
- 本线不是摆烂线。林亦舟在工作池里是在承担现实任务，不是单纯不回消息；但现实任务仍会造成错过。
- 本线不是 5X。兼职场景可以出现许澈、便利店、快递站工作人员等软照面，但不能提前打开无旧账新圈子完整线。
- 每一章都必须有现实任务、关系压力和可回收后果；环境描写只服务任务、选择、阻碍和后果。
