# IF 分支 P0-A 第三幕社团活动父池 / 完美子外流开发稿

## 路线定位

父池编号：`A3-ACTIVITY-PUBLIC`

路线编号：`R3-PERFECT`

路线名称：A3 父池下的完美线 / 自我管理线

入口时期：第三幕，汇演、班会、主持招新和活动圈开始成形时。

上层纠偏：本文件不是“第三幕一定选择融媒体”的路线，也不是第三幕唯一外流。第三幕原本逻辑是社团、主持招新、新闻中心 / 融媒体、摄影社、幕后物资、公开表达、志愿服务和回 4XX 共同构成父池分发。许棠和融媒体只是 `focus_newsroom`，周屿 / 秦越主持队是 `focus_hosting`，夏知微摄影社是 `focus_photo`，幕后物资和归档是 `focus_backstage`。只有玩家在父池内持续选择流程、低风险、按时、公开可控和自我管理，才进入本文件的 `R3-PERFECT` 子外流。

路线核心：林亦舟更早把生活重心投向流程、履历、公开表达和自我管理。他不一定更优秀，但更习惯把事情安排好、把情绪压低、把关系处理成可控事项。

核心代价：宿舍关系不会立刻破裂，但温度降低。周屿会觉得他像活动里的工作人员，唐骁会认可他的效率却更少找他商量情绪，陆沉会觉得他越来越像“能处理事的人”，而不是“能听完话的人”。

毕业倾向：主线稳定、去向清晰、公开评价较好，但很多关系停在体面和可联系，不一定能说开。

## 主线锁点

- 军训手机事件已经发生，陆沉的公开边界和宿舍互信仍是暗线。
- 汇演、班会、主持招新必须发生。
- 周屿仍会进入活动圈，秦越仍是活动流程里的关键人物。
- 第五卷宿舍站队和强冲突仍会发生，只是林亦舟在冲突中更倾向于控制损失，而不是当场卷入情绪。
- 晚风仍可存在，但在本路线中不默认更进一步。

## 入口条件

建议变量门槛：

- `self_control >= 3`
- `public_credit >= 2`
- `emotional_delay >= 2`
- `dorm_warmth <= 1`

推荐触发方式：

- 玩家先进入 `A3-ACTIVITY-PUBLIC` 父池，并在第三幕连续选择“流程优先”“公开风险可控”“少卷入宿舍情绪”的选项。
- 玩家在主持招新或班会材料节点选择接下具体事务，而不是只跟随宿舍行动。
- 玩家没有在第三幕把主要时间投入晚风、宿舍修补或摆烂逃避。

互斥提示：

- 如果玩家多次选择替周屿兜情绪，会更容易进入周屿线前置，而不是完美线。
- 如果玩家主要跟随许棠 / 新闻中心 / 融媒体处理材料，只能先标记 `focus_newsroom`，不自动锁 `R3-PERFECT`。
- 如果玩家主要跟随夏知微 / 摄影社处理镜头和授权，只能先标记 `focus_photo`，不自动锁 `R3-PERFECT`。
- 如果玩家多次选择回 4XX 处理当面矛盾，会保持默认宿舍主轴。
- 如果玩家选择逃开公共任务且不承担任何方向，会进入回避或摆烂倾向。

## 核心事件链

### 事件一：主持招新协助表

地点：教学楼门口咨询点、晨光大道、主持队临时物资桌。

参与人物：林亦舟、周屿、秦越、唐骁。

现实触发：主持招新现场临时缺人，秦越把签到表、物资表和候场顺序交给林亦舟。周屿在台上补串词，唐骁在群里催课程作业分工。

可拆选择：

- 接签到和物资表，先保证招新现场不乱。
  - 结果：`self_control +1`，`public_credit +1`，`time_debt +1`。
  - 回响：秦越记住林亦舟靠谱；唐骁第一次在作业分工里不等他确认。
- 只帮周屿撑到咨询结束，不接长期流程。
  - 结果：`zhou_trust +1`，`activity_link +1`，完美线门槛小幅推进。
  - 回响：周屿会觉得林亦舟是来帮他的，不是来接活动圈的。
- 拒绝临时加活，回 4XX 处理作业确认。
  - 结果：`dorm_warmth +1`，`public_credit 0`，完美线入口延后。
  - 回响：唐骁信任上升，秦越不生气但不再主动把流程交给他。
- 主动补秦越流程漏洞，重新整理签到和候场顺序。
  - 结果：`public_credit +2`，`self_control +1`，`dorm_warmth -1`。
  - 回响：秦越会直接说“下次还找你”，周屿会用玩笑遮住一点不舒服。

对话逻辑：

- 秦越只讲截止、人数、谁负责，不说漂亮话。
- 周屿会开玩笑：“你这表格做得像要接管主持队。”
- 唐骁不会吵，只会在群里发“我先按没人补的版本排了”。
- 林亦舟如果选择完美线，不解释太多，只把事情先做完。

### 事件二：班会材料归档

地点：教学楼 B204 / B206、打印店、明德楼材料窗口。

参与人物：林亦舟、辅导员、沈嘉禾、唐骁。

现实触发：班会结束后，签到、照片授权、主题材料和主持记录需要当天归档。辅导员只说“今晚能交最好”，但窗口第二天上午排队。

可拆选择：

- 整理签到和照片授权，优先规避公开风险。
  - 结果：`public_boundary +1`，`public_credit +1`。
  - 回响：许棠或融媒体后续更愿意找他确认授权边界。
- 找沈嘉禾核对材料格式，只让她看格式不让她替写。
  - 结果：`shen_jiahe_link +1`，`self_control +1`。
  - 回响：形成“现实好感参照”的早期伏笔，但不直接开感情线。
- 找唐骁拆分作业和班会材料时间。
  - 结果：`tang_trust +1`，`project_stability +1`。
  - 回响：唐骁认可效率，但会默认林亦舟以后也能按流程处理所有冲突。
- 只交最低要求，保留晚上回寝室时间。
  - 结果：`dorm_warmth +1`，`public_credit 0`。
  - 回响：完美线入口变弱，但宿舍主轴保温。

对话逻辑：

- 沈嘉禾边界清楚，只会指出文件名、页边距、附件顺序，不替人背责任。
- 唐骁会把事情拆成表格，并认为这样最公平。
- 林亦舟如果偏完美线，会把情绪问题推迟到“交完再说”。

### 事件三：公开表达低风险化

地点：汇演复盘会、主持队旁听区、课堂展示现场、东区商业街小舞台边缘。

参与人物：林亦舟、周屿、秦越、许棠、夏知微。

现实触发：汇演后需要复盘稿、照片素材、主持队口播样稿或课堂展示总结。林亦舟可以选择被看见，也可以选择只做幕后。

可拆选择：

- 做幕后串场稿，不上台。
  - 结果：`self_control +1`，`public_credit +1`，`activity_link +1`。
  - 回响：别人知道他有用，但不一定记得他本人。
- 上台做简短说明。
  - 结果：`public_credit +2`，`emotional_delay +1`。
  - 回响：周屿会调侃他越来越稳；林亦舟更习惯把紧张压成流程。
- 拒绝夏知微拍正脸，只允许拍手稿和背影。
  - 结果：`public_boundary +1`，`xia_zhiwei_link +1`。
  - 回响：夏知微会尊重边界，也可能在后续异质回流里提供观察角度。
- 接受许棠采访，但要求删寝室号和私人细节。
  - 结果：`public_boundary +2`，`public_credit +1`。
  - 回响：公开叙事风险降低，后续毕业材料线更顺。

对话逻辑：

- 许棠关心“能不能公开”，不是单纯八卦。
- 夏知微会从拍摄角度提醒他：不露脸也能被人看见。
- 周屿在这个节点会半真半假地说“你别把我也写进流程里”。

### 事件四：宿舍热度被管理

地点：青枫居 4XX、公共桌、外卖架、阳台洗衣区。

参与人物：林亦舟、周屿、唐骁、陆沉、宿管阿姨。

现实触发：招新、班会、课程作业和生活杂事挤在一起。4XX 公共桌上开始出现未签字表、外卖小票、洗衣卡、取件码和课程分工。

可拆选择：

- 把公共桌事项列成清单，谁的东西谁确认。
  - 结果：`dorm_order +1`，`dorm_warmth -1`。
  - 回响：唐骁认可，周屿觉得没劲，陆沉压力变小但话更少。
- 只提醒截止，不追问原因。
  - 结果：`self_control +1`，`old_debt +1`。
  - 回响：后续冲突里别人会觉得他“不问就是不在意”。
- 替大家错峰安排洗衣、取外卖、签字。
  - 结果：`dorm_order +1`，`time_debt +1`。
  - 回响：短期效率上升，长期容易被默认成“他会处理”。
- 不管公共桌，转去活动或学习任务。
  - 结果：`activity_link +1` 或 `project_stability +1`，`dorm_warmth -1`。
  - 回响：第五卷站队时，4XX 会更快进入“各自处理”状态。

对话逻辑：

- 宿管阿姨只认签字和物品归属，不替他们判断谁对谁错。
- 陆沉不会直接表达不满，但会把自己的东西移出公共区。
- 唐骁会说“这样省事”，周屿会说“你们真把寝室过成办公室了”。

### 事件五：完美线确认节点

地点：教学楼课后、明德楼材料窗口、主持队复盘桌、C407 机房外。

参与人物：林亦舟、秦越、周屿、唐骁、沈嘉禾。

现实触发：秦越提出长期协助活动流程；老师或辅导员希望林亦舟继续做材料归档；唐骁需要他确定课程项目分工；周屿希望他别把所有事都变成表格。

确认条件：

- `self_control >= 4`
- `public_credit >= 3`
- `emotional_delay >= 2`
- 玩家选择“继续把事情安排好”，而不是优先修补情绪。

可拆选择：

- 接长期协助，但只做职责内事项。
  - 结果：锁定 `R3-PERFECT`。
  - 回响：第六卷项目和毕业材料更稳，第七卷结算偏体面和清晰。
- 拒绝长期协助，回 4XX 处理当面关系。
  - 结果：回到 `DEFAULT-4XX`。
  - 回响：宿舍主轴保留更强，活动线降为支撑场景。
- 接下协助，同时继续替周屿兜情绪。
  - 结果：进入周屿线前置，完美线不锁。
  - 回响：第五卷站队时更容易站周屿。
- 接下协助，并找沈嘉禾或唐骁建立协作边界。
  - 结果：保留完美线，附带学业 / 项目倾向。
  - 回响：第六卷可承接到项目归档、材料边界和现实好感参照。

对话逻辑：

- 秦越会直接问“你能不能固定接这个”，不是暧昧邀请。
- 周屿会在旁边说“你别真把自己招进去”，语气轻，但有一点被甩开的感觉。
- 唐骁会问具体交付，不问他为什么接。
- 沈嘉禾只提醒“你要写清楚停止条件”。

## 变量建议

| 变量 | 用途 | 增长来源 | 过高后果 |
|---|---|---|---|
| `self_control` | 自我管理、流程优先 | 接表格、控风险、按时交付 | 情绪表达减少 |
| `public_credit` | 活动圈和老师眼中的可靠度 | 招新、班会、汇演材料 | 更容易被继续加活 |
| `emotional_delay` | 把关系问题推迟处理的倾向 | 先交材料、先做流程、少解释 | 第五卷旧账爆发更重 |
| `dorm_warmth` | 4XX 日常温度 | 回寝室、当面聊、替室友留时间 | 过低会导致宿舍退位提前 |
| `activity_link` | 活动圈牵引 | 秦越、主持队、东区小舞台 | 宿舍误读“他不在” |
| `public_boundary` | 公开叙事边界 | 授权、采访、照片处理 | 有助于第七卷公开材料风险控制 |

## 后续承接

### 第四幕

完美线进入第四幕时，林亦舟的暑假不一定更轻松。他会更早习惯把生活费、作业、活动材料、家庭消息分成几类处理。若生活费暗线加压，他会先压缩支出和时间，而不是第一时间找人倾诉。

### 第五幕

第五卷冲突爆发时，完美线的林亦舟更倾向于做“损失控制”：

- 先确认谁有材料、谁有截止、谁要签字。
- 不轻易站队，但这种“不站队”会被别人理解成冷。
- 强冲突中，他可能没有摔门，也没有吵赢，但会让关系更像事项群。

### 第六幕

第六卷承接为项目、归档、公开边界和去向清晰：

- 唐骁更愿意和他协作项目，但不一定更亲近。
- 沈嘉禾或老师线可以提供现实协作参照。
- 晚风若存在，更多是提醒他别把亲密关系也处理成项目。
- 4XX 会更早退位为“必要事项同路”。

### 第七幕

第七卷毕业结算偏向：

- 毕业去向明确，手续最少出错。
- 公开材料、毕业照、归档边界处理较稳。
- 宿舍关系可能体面、可联系，但不一定有强烈和解。
- 若 `dorm_warmth` 长期过低，结局容易变成“大家都承认他靠谱，但很少再私下找他”。

## 开发备注

本路线不是奖励线，也不是“正确人生”。它的可玩性来自稳定背后的代价：林亦舟越会处理事，越可能把自己也处理成一个不容易被靠近的人。
