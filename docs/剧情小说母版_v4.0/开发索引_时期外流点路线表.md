# 时期外流点路线表

## 用途

本表是从小说母版进入互动剧情开发的第一张路线表。它不把全卷拆成无数支线外传，也不把每个事件都做成完整 IF 树，而是只在几个大学阶段的关键时期设置外流点。

玩家选择改变的是林亦舟把时间、情绪和行动投向哪里。世界不会因此重写：周屿仍会进入活动圈，唐骁仍会走项目和秩序，陆沉仍要面对勤工、家庭和边界，晚风仍有自己的朋友和生活圈。改变的是林亦舟和这些人之间的距离、误读、投入、代价和毕业时的结算温度。

## 开发原则

1. 主线锁点不取消：军训手机事件、汇演、班会、主持招新、第五卷站队爆发、5X 唯一外流、第六卷关系定形、第七卷毕业收束仍然发生。
2. 普通选项只改微变量：关系温度、回避值、互信、钱/时间压力、是否记住某件事、后续台词和场景露出。
3. 大路线只从时期外流点进入：第三幕、第四幕、第五幕前段、第五幕中后段、5X 隐藏点。
4. 被选择的方向获得更多剧情容量：3-5 个核心事件、2-3 个现实代价、1 个第七卷毕业结算入口。
5. 未被选择的人不消失：他们按母版世界线继续生活，只是和林亦舟的交集变少，或只以消息、场景擦肩、固定流程重聚出现。
6. 第一卷、第二卷全路线共享；第三幕以后，所有长支线都按外流线处理。玩家一旦通过关键抉择进入某条路线，后续公共锁点只嫁接到当前路线版本，其它路线不再同时完整展开。
7. 主轴外流只保留宿舍站队线和晚风线。其它感情线不能从默认宿舍主轴直接跳出，必须先由第三幕社团 / 活动父池中“有女生连续出现”的方向自然派生。
8. 第五卷和第六卷的玩法节奏按硬约束执行：普通长线必须满足第五卷最低主线选择窗口；第六卷承接固定为 20 个剧情段、每 4 段 1 个二方向选择窗口，详见 `开发规则_IF第五第六卷玩法节奏硬约束.md`。

## 共享序章与蝴蝶效应

本表中的“支线”实际都是外流线。第一卷和第二卷提供共同起点：入校、青枫居 4XX 初识、军训手机事件、初始晚风联系、规则感和公开边界。第三幕开始，玩家的关键选择产生蝴蝶效应：

- 选社团 / 活动 / 公开表达，先进入 `A3-ACTIVITY-PUBLIC` 父池；再按持续选择分发到主持、新闻中心 / 融媒体、摄影、幕后、志愿、回寝室或回避等子方向，其中 `R3-PERFECT` 只是 `mode_perfect` 子外流。
- 选暑假兼职 / 勤工 / 钱压优先，进入 `R4-WORK`。
- 从主轴选择晚风关系，进入 `R5-WANFENG`。
- 只有在社团 / 活动线里已有女生角色连续出现并形成生活半径时，才允许派生进入 `R5-ROMANCE`。
- 选周屿、唐骁、陆沉或不站队，进入对应第五幕站队外流线。
- 选唯一硬外流，进入 `R5X-HARD`。

进入外流线后，内部怎么回流、怎么补支线、怎么安排短照面都可以丰富；但不能再让其它外流线同时作为完整主线参与。

## 总路线表

| 路线 ID | 路线名 | 入口时期 | 入口动作 | 路线核心 | 毕业结算倾向 |
|---|---|---|---|---|---|
| DEFAULT-4XX | 宿舍主轴线 | 默认 | 未进入大外流点，或多次回到 4XX 处理当面关系 | 宿舍四人关系优先，冲突、修补、误读和毕业回声都围绕 4XX 展开 | 有机会说开，但不保证圆满 |
| A3-ACTIVITY-PUBLIC | 第三幕社团活动父池 | 第三幕 | 主持招新、新闻中心 / 融媒体、摄影社、幕后物资、志愿服务、公开表达或回寝室之间持续选择 | 先判断林亦舟把时间和责任交给哪个活动半径，不直接锁融媒体或完美线 | 父池不直接结算，分发到子外流或回 DEFAULT-4XX |
| R3-PERFECT | A3 子外流：完美 / 自我管理线 | 第三幕 | 在 A3 父池内连续选择流程、低风险、按时、公开可控和自我管理 | 林亦舟更早进入流程、履历、活动和自我管理；情感温度偏低 | 去向清楚，关系淡一些，遗憾偏轻但余味冷 |
| R4-WORK | 兼职 / 生存线 | 第四幕 | 暑假、生活费、留校、勤工、快递站或打印店选择现实优先 | 林亦舟更早理解钱、排班、时间和错过；宿舍当面机会减少 | 更能自立，但许多话过了时间 |
| R5-WANFENG | 晚风线 | 第五幕前段 | 从主轴选择继续推进晚风关系，承担游戏、微信、抖音、见面、礼物、公开边界和时间代价 | 林亦舟把主要情绪交给一个已经存在的外部亲密生活圈 | 可能继续联系，也可能因毕业去向和公开边界留下温柔错过 |
| R5-ROMANCE | 社团女生派生感情线 | 第五幕前段 | 仅当第三幕社团 / 活动线已有女生角色连续出现时，才可转向另一个稳定亲密对象 | 林亦舟把主要情绪和时间投给由社团生活半径自然长出来的关系 | 可能继续联系，也可能因现实分流留下错过 |
| R5-ZHOU | 周屿线 | 第五幕中后段 | 宿舍冲突爆发后明显站到周屿一侧 | 活动圈、面子、表达和旧玩笑被认真处理 | 可能说开热闹背后的受伤，也可能被认为偏袒 |
| R5-TANG | 唐骁线 | 第五幕中后段 | 宿舍冲突爆发后明显站到唐骁一侧 | 规则、项目、表格、秩序和边界成为主轴 | 去向最稳，但容易被认为只讲流程 |
| R5-LUCHEN | 陆沉线 | 第五幕中后段 | 宿舍冲突爆发后明显站到陆沉一侧 | 勤工、沉默、家庭压力、边界和迟来的解释成为主轴 | 关系更深也更重，容易被其他人误读 |
| R5-LIEFLAT | 摆烂 / 不站队线 | 第五幕中后段 | 冲突爆发后不站队、不追问、不修补，只选省力路径 | 弱冲突、低投入、资源慢慢流失，很多事被拖过最佳时间 | 遗憾最明显，毕业时像被生活推出去 |
| R5X-HARD | 5X 唯一硬外流线 | 第五幕隐藏高难 | 在唯一硬外流点进入许澈 / 新圈子，不再回原多线 | 新关系轻松、旧宿舍渐远，后期按单线推进 | 新人未必真靠得住，旧关系留下宿命感遗憾 |

## DEFAULT-4XX：宿舍主轴线

| 项目 | 内容 |
|---|---|
| 入口 | 玩家没有进入明确大外流，或在多次选择后仍持续回到 4XX 当面处理问题。 |
| 主线锁点 | 所有宿舍关键事件照常发生。第五卷站队爆发仍是核心，第六卷关系定形，第七卷毕业收束。 |
| 核心事件 | 1. 晚归、外卖、公共桌、自查表等小事积累成冲突。2. 第五卷爆发时必须当面处理旧账。3. 第六卷专业协作变成“能合作但不亲密”。4. 第七卷毕业照、清寝、最后一顿饭、离校手续被动重聚。 |
| 代价节点 | 1. 外部亲密关系或活动机会被压缩。2. 反复处理旧账带来疲惫。3. 有些修补只能做到体面，不能做到圆满。 |
| 变量 | `dorm_trust`、`avoidance`、`repair_intent`、`old_debt`、`public_table_memory`。 |
| 第六卷承接 | 4XX 不再天天同路，但公共桌、C407、事项群仍能把四人短暂拉回。 |
| 第七卷结算 | 生成说开、淡化、体面但不亲近、不和解但不再追究等不同毕业文本。 |

## A3-ACTIVITY-PUBLIC：第三幕社团活动父池

| 项目 | 内容 |
|---|---|
| 入口 | 第三幕汇演、班会、主持招新、社团招新和公开表达开始成形时。 |
| 父池作用 | 不直接决定结局，只负责分发林亦舟第三幕以后主要靠近的生活半径。 |
| 不是融媒体线 | 许棠、新闻中心 / 融媒体只是 `focus_newsroom`，不能代替主持队、摄影社、幕后物资、志愿服务和回寝室方向。 |
| 不等于完美线 | `R3-PERFECT` 只是父池内部的 `mode_perfect` 子外流；如果玩家主要陪周屿和秦越，就是主持 / 活动圈方向；如果主要处理照片授权，就是摄影 / 公开边界方向；如果连续回 4XX，则不锁第三幕外流。 |

### 父池 focus 分发

| focus | 方向 | 典型选择 | 后续用途 |
|---|---|---|---|
| `focus_hosting` | 主持队 / 周屿 / 秦越 | 帮主持招新、补串场、陪周屿处理活动场面 | 周屿线前置、活动人情债、公开场面压力。 |
| `focus_newsroom` | 新闻中心 / 融媒体 / 许棠 | 接受采访、核材料、处理授权和公开边界 | 公开叙事风险、材料流向、毕业主题材料回声。 |
| `focus_photo` | 摄影社 / 夏知微 | 拒绝或接受正脸、确认照片用途、处理镜头边界 | 影像支线、公开边界、被看见的代价。 |
| `focus_backstage` | 幕后 / 签到 / 物资 / 归档 | 接签到表、整理候场顺序、归档班会材料 | 完美线、项目线、流程信用。 |
| `focus_volunteer` | 志愿 / 服务 / 秩序 | 去青志协、维持活动秩序、帮生活人物处理现场 | 服务型社交和责任感回声。 |
| `focus_dorm_return` | 回 4XX | 不接长期活动，回寝室处理作业、公共桌和室友当面关系 | 保持 DEFAULT-4XX。 |
| `focus_public_avoid` | 回避公开 | 不上台、不露脸、借热闹逃开 | 摆烂 / 回避 / 5X 前置。 |

### 父池 mode 分发

| mode | 形成条件 | 子外流 |
|---|---|---|
| `mode_normal` | 参与活动但责任不连续，仍能回到 4XX | 只保留活动回声，不锁长线。 |
| `mode_perfect` | 连续流程优先、低风险、按时、公开可控 | 进入 `R3-PERFECT`。 |
| `mode_pressure` | 连续接救场、临时补位、人情债 | 活动压力子外流，可作为周屿线或完美线前置。 |
| `mode_avoid` | 连续拒绝曝光、不上台、短句逃离 | 回避前置，后续更容易摆烂或 5X。 |
| `mode_dorm` | 连续回寝室当面处理关系 | 回 DEFAULT-4XX。 |

### 三次选择判定窗口

第三幕父池一共开放三次参加社团活动的机会。三次机会结束后，必须结算林亦舟是否进入某个社团 / 活动半径，不能继续在所有方向之间无限摇摆。

| 选择次数 | 功能 | 结果 |
|---|---|---|
| 第一次 | 全开放试探 | 所有方向都可选：主持队、新闻中心 / 融媒体、摄影社、幕后物资、志愿服务、回 4XX、不参加公开活动。 |
| 第二次 | 维持或试新 | 玩家可以维持第一次方向，也可以去另一个方向试试。 |
| 第三次 | 结算 | 只能选择：坚持第一个、坚持第二个、两个都要、两个都不要。 |

结算规则：

- 坚持第一个：进入第一次选择的社团 / 活动方向。
- 坚持第二个：进入第二次选择的社团 / 活动方向。
- 两个都要：自动归为 `mode_pressure`，形成活动压力子外流；不开放两个完整社团主线。
- 两个都不要：进入 `DEFAULT-4XX` 宿舍主轴，社团只留下错过、回声或低频照面。
- 如果被坚持的方向本身是回 4XX / 不参加，则也结算为 `join_none`。
- `mode_perfect` 只能来自“坚持一个方向并把流程、边界、时间处理得稳定”，不能来自“两个都要”。

多选时，以“三次选择后的责任承担”而不是“一次选择”判定。没有胜出的 focus / mode 只作为关系变量、台词回声、软照面或后续文本，不再打开完整主线。

### 社团线后续外流权限

| A3 结果 | 可继续外流方向 | 禁止方向 |
|---|---|---|
| 社团 / 活动线中有女生连续出现，例如摄影社夏知微、材料协作中的沈嘉禾等 | 可在第五幕前段派生 `R5-ROMANCE`，也可流向 `R3-PERFECT`、回宿舍修补 / 站队、`R5-WANFENG` 或 `R5-LIEFLAT` | 不能同时保留多个完整感情主线。 |
| 社团 / 活动线中没有女生连续出现，例如纯主持、幕后、志愿、流程自控等 | 只能流向 `R3-PERFECT`、回 `DEFAULT-4XX` / 宿舍站队、`R5-WANFENG` 或 `R5-LIEFLAT` | 不能凭空开启沈嘉禾 / 夏知微等其它感情线。 |
| 主轴未进入社团女生候选 | 只能在晚风线和宿舍站队线之间外流 | 不能直接进入其它感情线。 |

这里的“女生连续出现”不是一次擦肩，而是至少有连续事件、现实任务、可拆选择和后续回声。否则只能作为软照面存在。

## R3-PERFECT：A3 父池下的完美 / 自我管理子外流

| 项目 | 内容 |
|---|---|
| 入口 | 在 `A3-ACTIVITY-PUBLIC` 父池内，玩家连续选择公开表达、流程执行、履历积累和低风险处理。 |
| 主线锁点 | 汇演、班会、主持招新仍发生；周屿仍进入活动圈；宿舍关系仍会积累裂痕，只是林亦舟更少卷入强冲突现场。 |
| 核心事件 | 1. 主持招新时选择跟进流程，不替人情绪兜底。2. 活动材料和公开表达得到老师或学长认可。3. 周屿活动圈和林亦舟自我管理形成镜像，一个热闹，一个克制。4. 第五卷冲突爆发时，林亦舟倾向用表格、时间表和边界处理。 |
| 代价节点 | 1. 宿舍觉得他总能“安排好自己”，但不一定愿意靠近。2. 情绪表达减少，晚风或其他亲密对象会觉得他太会留余地。3. 完美路线弱冲突更多，但冷感更重。 |
| 变量 | `self_control`、`public_credit`、`dorm_warmth`、`activity_link`、`emotional_delay`。 |
| 第六卷承接 | 项目、归档和毕业去向更顺，但和 4XX 的当面解释减少。 |
| 第七卷结算 | 文本偏“清楚、体面、少遗憾大事，但也少了几句真正难听的话”。 |

## R4-WORK：兼职 / 生存线

| 项目 | 内容 |
|---|---|
| 入口 | 第四幕暑假、生活费波动、家庭暗线、留校和勤工选择中，玩家连续选择赚钱、排班、节省和现实优先。 |
| 主线锁点 | 暑假、家庭变化、返校压力仍发生；宿舍冲突仍在第五卷爆发，只是林亦舟经常不是第一现场。 |
| 核心事件 | 1. 校园快递站、打印店、勤工岗位、食堂窗口或校医院形成固定生活半径。2. 陆沉线获得更多共鸣，但不等于自动站陆沉。3. 因排班错过聚餐、彩排或宿舍当面说明。4. 生活费、红包、取件码和工时表成为现实压力。 |
| 代价节点 | 1. 时间被排班切碎。2. 宿舍容易把缺席解释成冷漠。3. 钱解决一部分问题，也制造一部分沉默。 |
| 变量 | `money_pressure`、`work_shift`、`missed_meal`、`reality_bond`、`dorm_misread`。 |
| 第六卷承接 | 进入项目协作时更讲成本和排期，和陆沉有更多现实对话。 |
| 第七卷结算 | 离校时更能独自处理流程，但错过很多本来能当面说的话。 |

## R5-ROMANCE：社团女生派生专注感情线

| 项目 | 内容 |
|---|---|
| 入口 | 只能由第三幕 `A3-ACTIVITY-PUBLIC` 中“有女生连续出现”的社团 / 活动方向派生。主轴玩家不选择晚风时，不能凭空跳进沈嘉禾 / 夏知微线。 |
| 候选方向 | 候选必须来自父池里已有生活半径的人：例如摄影社 / 夏知微、材料协作 / 沈嘉禾。不新增无铺垫女主，也不把偶遇写成路线入口。 |
| 主线锁点 | 晚风仍存在，但可以停留在朋友、游戏搭子、错过或低频联系；宿舍冲突仍发生。 |
| 核心事件 | 1. 亲密对象必须通过 A3 已选方向自然增加出现频率。2. 送礼、节日、散步、第一次牵手或告白必须带来时间和钱的代价。3. 宿舍消息延迟、缺席、误读与甜感同场出现。4. 第六卷关系不只谈“喜欢”，还要谈未来、边界、公开范围和各自生活。 |
| 代价节点 | 1. 宿舍修补机会减少。2. 情绪依赖上升。3. 如果公开边界处理不好，会影响活动、材料或朋友圈观感。 |
| 变量 | `romance_focus`、`intimacy_cost`、`public_boundary`、`time_debt`、`dorm_absence`。 |
| 第六卷承接 | 第六卷不再多线均摊，而是深入这段关系及其生活圈。 |
| 第七卷结算 | 生成继续联系、体面分别、错过、保留私人入口等亲密关系毕业文本。 |

## R5-ZHOU：周屿线

| 项目 | 内容 |
|---|---|
| 入口 | 第五幕宿舍强冲突后，玩家多次替周屿解释、保留面子、进入活动圈或承认他热闹背后的受伤。 |
| 主线锁点 | 周屿仍会在活动、主持、公开场合中被看见，也仍会因玩笑、面子和不及时解释制造误会。 |
| 核心事件 | 1. 林亦舟陪周屿处理活动圈里的误读。2. 周屿用玩笑掩饰受伤，被玩家选择拆穿或接住。3. 唐骁会认为林亦舟偏向“讲热闹不讲规则”。4. 毕业材料中，周屿更在意自己是否被写成漂亮故事。 |
| 代价节点 | 1. 唐骁互信下降。2. 陆沉可能觉得自己的沉默又被热闹盖过去。3. 周屿线容易把林亦舟拖进活动人情债。 |
| 变量 | `zhou_trust`、`activity_debt`、`tang_misread`、`luchen_silence`、`public_face`。 |
| 第六卷承接 | 周屿仍能热闹，但不再默认等林亦舟；关系靠少数活动和材料节点重合。 |
| 第七卷结算 | 可能有半句道歉、玩笑式说开，也可能只保留群里的热闹。 |

## R5-TANG：唐骁线

| 项目 | 内容 |
|---|---|
| 入口 | 第五幕宿舍强冲突后，玩家多次选择规则、流程、项目和边界，支持唐骁不再替别人圆场。 |
| 主线锁点 | 唐骁仍会推进项目、表格、课程设计和毕业材料；他的正确不必自动等于温暖。 |
| 核心事件 | 1. 林亦舟和唐骁一起拆分任务、补材料、查流程。2. 周屿觉得两人把宿舍变成审核表。3. 陆沉感谢规则减轻压力，但也可能害怕被规则问到无处可躲。4. 第六卷项目演示中，专业配合更稳但不亲密。 |
| 代价节点 | 1. 宿舍温度下降。2. 林亦舟被认为“只站有理的一边”。3. 亲密关系会被流程和截止时间挤压。 |
| 变量 | `tang_trust`、`rule_pressure`、`project_stability`、`dorm_warmth`、`emotional_blindspot`。 |
| 第六卷承接 | C407、项目归档、毕业去向表更顺，但 4XX 事项群更短。 |
| 第七卷结算 | 去向清楚，照看实用，情感说开较少。 |

## R5-LUCHEN：陆沉线

| 项目 | 内容 |
|---|---|
| 入口 | 第五幕宿舍强冲突后，玩家多次听陆沉说完、替他保留边界、理解勤工和家庭压力。 |
| 主线锁点 | 陆沉仍不会突然变外向；他的压力通过排班、生活费、证书材料、图书馆和沉默体现。 |
| 核心事件 | 1. 林亦舟陪陆沉处理勤工、证书、材料或家庭消息。2. 周屿和唐骁会对林亦舟“总替陆沉想”产生不同解释。3. 陆沉不是被拯救，而是在少数节点里愿意说一句真话。4. 第七卷旧书、补登记和清寝成为迟来的在意。 |
| 代价节点 | 1. 林亦舟承担更多情绪和现实压力。2. 容易被误解为偏向弱势或沉默的一边。3. 关系变深但不一定变轻松。 |
| 变量 | `luchen_trust`、`family_pressure`、`work_reality`、`quiet_bond`、`others_misread`。 |
| 第六卷承接 | 陆沉和林亦舟有更多现实对话，但四人不因此恢复天天同路。 |
| 第七卷结算 | 可能有迟来的解释和旧物归还，也可能只是更体面地分开。 |

## R5-LIEFLAT：摆烂 / 不站队线

| 项目 | 内容 |
|---|---|
| 入口 | 第五幕强冲突后，玩家连续选择不追问、不解释、不替任何人确认，只走省力路径。 |
| 主线锁点 | 冲突仍爆发，但更常以错过、冷掉、没人再问的形式存在。完美线和摆烂线都适合弱冲突，但弱冲突不等于无后果。 |
| 核心事件 | 1. 不进门、刷手机、外卖、游戏、后街或湖边消耗时间。2. 宿舍问题没人立刻爆炸，却一点点变轻、变远。3. 关键活动和材料可以低分通过，但人际入口减少。4. 第七卷被固定流程拉回时，很多话已经没有合适语气。 |
| 代价节点 | 1. 资源流失。2. 宿舍互信低。3. 亲密关系浅，外部机会也不稳定。 |
| 变量 | `avoidance`、`lieflat_score`、`lost_chance`、`low_conflict`、`late_regret`。 |
| 第六卷承接 | 林亦舟更多作为旁观者出现，别人不再默认等他。 |
| 第七卷结算 | 最像“没有大事发生，但许多入口都关了”。 |

## R5X-HARD：5X 唯一硬外流线

| 项目 | 内容 |
|---|---|
| 入口 | 第五幕隐藏高难外流点。玩家在旧宿舍线最难回头的位置选择进入许澈或新圈子。 |
| 主线锁点 | 原宿舍主线不被取消，但从林亦舟视角退成远处的固定流程和偶然擦肩。第六卷、第七卷不再回原多线。 |
| 核心事件 | 1. 新圈子提供轻松关系，不问旧账。2. 后街、便利店、台球厅或新活动空间成为新半径。3. 旧宿舍在毕业流程中偶然出现，但不再承担主要选择。4. 新关系逐渐显露逢场作戏或薄关系本质。 |
| 代价节点 | 1. 宿舍修补机会断裂。2. 新圈子没有旧账，也没有深责任。3. 毕业时遗憾更宿命。 |
| 变量 | `hard_outflow`、`new_circle_trust`、`old_dorm_distance`、`thin_relation`、`5x_regret`。 |
| 第六卷承接 | 只走 5X 单线，允许经过同样校园场景，但不再参与原多线抉择。 |
| 第七卷结算 | 新人散场，旧人渐远，林亦舟发现自己从两边都被拉开。 |

## 开发顺序

1. 先做 DEFAULT-4XX、R5-ZHOU、R5-TANG、R5-LUCHEN、R5-LIEFLAT。第五幕站队是最核心的可玩性来源，也是第六卷、第七卷承接最强的部分。
2. 再做 R4-WORK 和 A3-ACTIVITY-PUBLIC 父池，其中 `R3-PERFECT` 只是 A3 父池下优先开发的完美 / 自我管理子外流。这两类线影响林亦舟的生活方式和前期资源结构，但不应抢走宿舍主轴。
3. 再做 R5-ROMANCE。先确认第三幕 A3 父池里是否已经形成女生候选来源，再确定沈嘉禾 / 夏知微等对象；没有 A3 来源时，只能走晚风线、宿舍站队、完美线或摆烂线。
4. 最后做 R5X-HARD。它是隐藏高难、单线外流，不适合过早开发，否则会扰乱第五幕多线收束。

## 暂不开发的内容

- 不做每个事件都开完整 IF 树。
- 不做人人一篇长支线外传。
- 不默认晚风是唯一亲密答案。
- 不让玩家选择改写周屿、唐骁、陆沉、晚风等人的人生方向。
- 不做无限平行宇宙式重写；只做林亦舟视角、投入方向和毕业结算的差异。

## 待确认

1. R5-ROMANCE 的专注感情线对象需要先绑定 A3 父池来源，再定稿候选。建议优先候选：沈嘉禾或夏知微，但不能从主轴直接跳入。
2. A3 父池内哪些子外流需要优先做长线，需要在第三幕入口试玩时确认；`R3-PERFECT` 先作为 `mode_perfect` 样板，不再代表整个第三幕。
3. 摆烂线是否保留轻量娱乐玩法，例如游戏、外卖、后街、刷短视频和低强度社交成就。
4. 5X 外流线中许澈的新圈子需要单独列最少 3 个核心事件，避免只剩“离开宿舍”这个动作。
