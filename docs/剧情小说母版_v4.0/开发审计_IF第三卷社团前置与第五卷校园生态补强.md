# IF 第三卷社团前置与第五卷校园生态补强审计

更新时间：2026-05-29

## 结论

用户判断成立：当前 `R4-WORK` 的 Act4 -> Act5 试玩已经能跑通勤工、材料、宿舍和晚风压力，但玩家实际看到的校园人际仍然偏窄。第五卷应该是全游戏最精华、最好玩的可玩核心，而不是把第四卷的工作 / 材料压力延长一遍。

问题不在母版没有支线人物。总控和 A3 父池已经明确要求：夏知微、沈嘉禾、秦越、许棠等人都有自己的学院、寝室、社团办公室、图书馆座位、活动路线和生活半径。问题在于当前 `R4-WORK` 试玩层没有把第三卷社团 / 活动前置足够兑现到第五卷微选项里。

当前试玩缺口：

- `playtest/r4-work-act4-act5-full-preview-data.js` 中没有命中夏知微、沈嘉禾、秦越、许棠、姜宁、梁思予、蒋策等命名校园人物。
- Act5 玩家侧主要看见的是 4XX、晚风、老师、窗口、快递站、活动群和普通场景人物。
- 旧版 `act5_bridge_nodes` 曾经有沈嘉禾、许棠、秦越的桥接影子，但玩家现在跑的是 `act5_detail_blocks`，这些影子没有进入实际试玩主层。
- 因此前四卷积累的“公开生活半径”没有真正参与第五卷站队，导致第五卷体量和趣味感接近第四卷，且所有剧情像围着固定几个人转。

## 规则依据

总控已规定：

- 校园生活不能只有青枫居，支线人物的寝室楼、取餐点、社团办公室、图书馆座位和通勤路线要反复出现。
- 第五卷是下册可玩性核心，必须把前四卷积累的宿舍、家庭、晚风、学业、活动、支线人际和外流诱惑压到同一阶段。
- 第五卷每个主要冲突簇都必须回答“他站哪边”，站队不只是帮室友，也包括选择规则、前途、情感关系、活动 / 兴趣、家庭压力、沉默或不站队。
- 室友退位后，主角不能进入人物真空。夏知微、沈嘉禾、秦越、许棠和短暂生活人物都可以补位，但必须从各自生活半径自然出现。

A3 父池已规定：

- 第三卷应有三次社团 / 活动选择窗口：第一次试探、第二次维持或试新、第三次收束。
- `focus_hosting` 对应主持招新、秦越流程、周屿活动圈、许棠软照面。
- `focus_newsroom` 对应新闻中心 / 融媒体、许棠、辅导员、沈嘉禾材料支线。
- `focus_photo` 对应摄影社、夏知微、照片授权和影像边界。
- `focus_backstage` 对应幕后物资、签到、道具、节目单、秦越、周屿、许棠。
- 沈嘉禾、夏知微只有连续任务级接触达标，才可成为合法候选来源；一次软照面不能直接开亲密线。

## 应补的第三卷前置

第三卷需要先补成“可被第五卷读取的前置池”，否则第五卷临时塞人会显得硬。

这里的前置不是简单决定“人物能不能出现”，而是决定人物在后续剧情中的权限层级：

- 玩家在第三卷明确选择过某个社团 / 活动方向时，对应人物可以成为“核心边缘人物”。他们仍不抢 4XX 和主线位置，但可以真正改变第五卷的选择结构：多出一条主选择方向、改变某个站队成本、提前触发一段误读，或让某个后果变得更重。
- 玩家没有选择该方向，但第一至三幕曾经和这些人有过照面、短暂协作或公共场合互动时，对应人物仍然可以作为“软回声”进入微心态选项。他们不能改变主选择结构，也不能突然开路线，但可以提供一瞬间的比较、犹豫、羞耻感、被看见感或逃开的理由。
- 完全没有前置互动的人物，不应在第五卷突然承担情绪出口或关键建议。他们可以作为普通校园背景出现，但不能参与玩家判断。

建议新增 / 强化以下前置记录：

| 前置方向 | 第三卷可玩事件 | 记录变量 | 第五卷可兑现内容 |
| --- | --- | --- | --- |
| 主持 / 活动 | 秦越临时缺人，周屿在台侧试词，林亦舟是否接签到表 | `a3_first_focus=focus_hosting`, `activity_link`, `zhou_trust` | 第五卷活动群不是抽象群，而是秦越 / 周屿活动圈继续找他 |
| 融媒体 / 公开叙事 | 许棠问采访授权、寝室号能不能写、公开稿如何处理 | `focus_newsroom`, `public_boundary`, `public_credit` | 第五卷材料 / 外部舆论可由许棠引出，不只由老师窗口引出 |
| 摄影 / 观看边界 | 夏知微问正脸、背影、4XX 私人物品是否能入图 | `focus_photo`, `xia_zhiwei_link`, `public_boundary` | 第五卷缺席、晚归、活动散场可被影像边界照见 |
| 材料 / 安静协作 | 沈嘉禾只核格式、不替写、不替签 | `shen_jiahe_link`, `candidate_source_checked` | 第五卷学业 / 材料压力可出现图书馆、打印店、共享文档补位 |
| 幕后 / 志愿 | 后台物资、节目单、志愿签到、活动归档 | `focus_backstage`, `time_debt`, `public_credit` | 第五卷“可靠”会变成被继续加活，而不是单纯好人缘 |
| 回宿舍 / 拒公开 | 拒入长期群，回 4XX 当面处理 | `focus_dorm_return`, `dorm_warmth` | 第五卷保留宿舍热度，但活动圈低频回声仍存在 |
| 回避公开 | 不入群、不上台，也不回寝解释 | `focus_public_avoid`, `old_debt`, `emotional_delay` | 第五卷回避 / 外流倾向更自然 |

## 第五卷扩容原则

第五卷扩容不应只是加页数，而应增加“可站队的人际场”。

人物进入第五卷时应分为两档：

| 权限层级 | 触发条件 | 在第五卷中的作用 | 限制 |
| --- | --- | --- | --- |
| 核心边缘人物 | 第三卷明确选择过对应方向，或连续两次任务级互动达标 | 可影响主选择、改变后果权重、提供替代行动入口 | 不抢宿舍主轴，不直接替玩家解决冲突 |
| 微心态软回声 | 第一至三幕有照面、短协作、群消息、公开场合互动 | 只进入微选项，影响心态、语气、解释成本和短反馈 | 不新增路线，不改变主选择数量，不临时升级候选 |
| 背景人物 | 无明确前置，仅符合校园场景 | 提供真实校园质感 | 不承担情绪判断，不给关键建议 |

举例：

- 选过主持 / 活动方向：秦越可以在第五卷成为真正的压力源，问他能不能固定、要不要接流程、是否把活动优先写清。没选过但见过秦越：只作为群里一个名字或现场一句催促，进入“想到活动那边也有人等回复”的微心态。
- 选过融媒体 / 公开叙事方向：许棠可以影响公开材料口径，让玩家选择删寝室号、保留班级、是否把事件写进活动稿。没选过但有照面：她只在微选项里带来“这件事会不会被别人讲成另一种样子”的犹豫。
- 选过摄影方向且夏知微任务级互动达标：她可以用照片授权、删图、背影边界改变一段冲突的公开后果。没达标：她只能作为一次被看见的软回声，不开亲密线。
- 选过材料 / 共享文档方向且沈嘉禾达标：她可以提供格式、图书馆、打印店和安静谈话入口。没达标：她只能提醒“格式可以帮看，内容不能代写”，不承担安抚功能。

Act5 每个主要块至少应有一个校园生态兑现点：

| Act5 块 | 当前功能 | 需要补的校园生态 |
| --- | --- | --- |
| B01 宿舍站队 / 晚班入口 | 勤工和留校本人签字 | 秦越或许棠从活动 / 材料侧发来现实请求，让“先去快递站”变成对另一个校园圈子的缺席 |
| B02 学业 / 证件 / 材料 | 补考、证明、本人确认 | 沈嘉禾以共享文档 / 打印格式出现，只帮到边界，不替他承担 |
| B03 晚风 / 活动 / 兼职冲突 | 亲密、活动和工作撞车 | 夏知微或许棠在活动散场 / 影像授权里照见他的缺席，不把晚风写成唯一出口 |
| B04 规则 / 项目 / C407 | 项目规则和宿舍责任 | 秦越 / 活动同学和 C407 同时要他确认，让“可靠”变成两边都可消耗 |
| B05 固定班 / 新半径 | 工作线锁定或降密度 | 活动群、4XX、晚风、学业共同争夺同一周，不再只是抽象活动群 |

## 微选项补强口径

后续微选项必须遵守以下口径：

- 命名人物不是装饰，出现时必须带着自己的任务：夏知微带照片和授权，沈嘉禾带格式和共享文档，秦越带名单和流程，许棠带公开叙事和材料口径。
- 社团选择过的人物，可以进入主选择层；只照面过的人物，只进入微心态层。两者都能让校园变大，但权重不同。
- 不要把所有支线人物写成等权候选。沈嘉禾 / 夏知微需满足两次任务级接触才开放候选来源；许棠当前只做公开叙事压力，不升级 romance。
- 普通校园人物也要保留：打印店老板、驿站阿姨、窗口老师、活动同学、助教、食堂窗口、奶茶店员可以提供生活质感，但不能替代命名人物的结构作用。
- 每个微选项不只是“说一句话”，而要改变一个后果：少一次解释、被一个圈子默认、保住一次边界、错过一次当面照面、留下一个公开材料痕迹。
- 第五卷的强度来自多线压迫，不来自更多表格。连续两段都是材料 / 窗口 / 签字后，下一段必须切入人际事件：缺席、误读、帮或不帮、谁在外人面前护了一句、谁回寝后又刺了一句。

## 日常校园人物补强

这里要单独立规则：第五卷不能只补社团人物，也不能只让老师、店员、老板、宿管、驿站人员存在于场景描述里。真实校园的压力很多时候不是来自“大人物”，而是来自那些反复见到你、记得你、又不完全理解你的人。

这些人物不一定开路线，但要有“场景权力”：

- 他们可以记住林亦舟：打印店老板认出他又来补身份证复印件，驿站工作人员知道他今天第一次上晚班，食堂窗口记得他最近总是压点来。
- 他们可以误解林亦舟：窗口老师以为说明是别人帮他写的，宿管阿姨以为他又让室友带话，店员以为他是在等女朋友电话。
- 他们可以制造微后果：多退一张表、少问一句话、给一个不舒服的方便、把一句提醒说得像批评。
- 他们可以成为外部见证：不是替主角判断，而是让玩家意识到他的生活已经被校园系统记住。

建议给日常人物分成以下功能层：

| 人物类型 | 可承担的剧情功能 | 可以影响什么 | 不应怎么写 |
| --- | --- | --- | --- |
| 陈老师 / 辅导员 / 任课老师 | 制度边界、延期许可、点名、补考提醒、材料退回 | 主选择的规则成本、学业压力、公开羞耻感 | 只当通知机器，或每次都温和解释 |
| 助教 / 课程同学 | 实验进度、作业互查、补交窗口、群内提醒 | 唐骁 / C407 / 学业线的可信压力 | 只在群里发一句模板消息 |
| 勤工负责人 / 驿站工作人员 | 排班、迟到、扫码枪、签收错误、被默认可靠 | 工作线是否变成生活半径 | 只说“负责人提醒”，没有态度和记忆 |
| 打印店老板 / 文印店员 | 复印件、身份证、证明、材料顺序、排队时间 | 材料线的生活质感和尴尬成本 | 只作为“去打印店”的地点背景 |
| 宿管阿姨 / 门禁值班 | 晚归登记、代取边界、谁替谁说话 | 宿舍线的外部见证和小型审判 | 只负责说一句门禁规则 |
| 食堂窗口 / 便利店店员 / 奶茶店员 | 余额、熟脸、借充电线、错过饭点、活动散场 | 经济压力、晚风见面、回寝时间 | 只当环境音或消费动作 |
| 书屋老板 / 店员 / 常客 | 安静落脚点、二手书、旧票根、见过谁来过 | 回避、学习、亲密或外流的低声入口 | 只当“星辰书屋”氛围板 |
| 活动老师 / 场地管理员 | 场地钥匙、证明、授权、活动事故责任 | 活动 / 公共信用的硬边界 | 只让秦越一个人承担全部活动压力 |

落到第五卷时，每个主要块都应该至少有一个“日常人物动作”，而不是只有环境描写：

- B01 晚班入口：勤工负责人不只发通知，而是第一次把林亦舟的名字写进晚班表；驿站工作人员提醒他扫码枪放错位置。
- B02 材料 / 补考：打印店老板认出同一份证明又改了一版；窗口老师退回一栏，让他自己解释用途。
- B03 晚风 / 活动 / 夜晚：便利店店员借线、奶茶店活动散场、店门口有人发剩下的手环，这些都要对他的回复和迟疑形成压力。
- B04 C407 / 项目：助教、任课老师或实验室值班同学让项目规则变成可见后果，不只靠唐骁提醒。
- B05 固定班 / 新半径：宿管阿姨、食堂熟脸、驿站工作人员都能看出他这一周的动线变了，让玩家感到生活半径被重新登记。

这类人物的写法原则：不抢戏，但要能留下针脚。读者不一定记住他们全名，却要记住那句“你又改证明啊”、那次退回来的表、那根借来的充电线、那一次别人少问了半句。

## 下一步施工顺序

1. 先补第三卷 A3 / 社团前置规则表：从三次社团选择窗口产出可读变量和人物权限。
2. 把规则写入正式 JSON，形成 `act3_to_act5_campus_echo_rules`，作用类似现有 `act4_to_act5_carryover_rules`。
3. 扩写 Act5 结构：从 5 个块扩到核心卷标准，按 12-15 个主选择窗口规划；第一版试玩可先落 12 个窗口，但蓝图保留 15 个槽位。每个冲突簇包含公开爆点、寝室回声、小动作修补 / 报复、下一次更难开口。
4. 重写 `R4-WORK` Act5 微选项，让 A3 前置决定谁能自然出现、出现到什么程度、是否只作软照面。
5. 重新生成 4 -> 5 全链试玩，检查玩家从第三卷前置、第四卷勤工、第五卷站队之间是否能读到连续因果。

## 当前判断

现在不宜继续只在第五卷末尾加剧情。正确顺序是先补第三卷社团 / 校园人物前置，再扩第五卷。否则第五卷即使加到很多页，也会继续围着 4XX、晚风、勤工和材料打转，缺少真正的校园群像和可玩站队。
