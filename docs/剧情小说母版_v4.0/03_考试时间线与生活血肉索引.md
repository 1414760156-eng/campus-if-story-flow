# 03 考试时间线与生活血肉索引

## 核心原则

大学时间不是只靠学期标题推进。真正让人感到四年过去的，往往是一次又一次报名、查分、缺考、补考、二战、抢座、打印准考证、找耳机和身份证。英语四级考了一次又一次，六级报名窗口又开了，普通话证书快过期了，教师资格证笔试和面试在群里刷屏，计算机二级成绩截图被发到朋友圈，运动会报名表又贴到学院群里，大学就在这些不一定成为主事件的东西里往前走。

这些内容的用途不是制造新的主线，而是给人物对话、时间流逝、精神疲惫和支线生活提供背景。有人认真准备，有人临时抱佛脚，有人嘴上说不考了又偷偷报名，有人每次查分都说“这次随缘”，有人把准考证塞进书里，到毕业收拾行李时才翻出来。

具体角色偏好、地点映射、年级节奏和集体记忆交点见 `04_角色偏好与场景索引.md`。本文件提供考试、证书、生活人物和场景素材库；`04` 负责说明谁更可能使用这些素材、何时使用、为什么使用，以及这些个人偏好如何变成宿舍或群体共同记忆。

## 考试与证书压力表

| 项目 | 出现方式 | 对人物的影响 | 适合使用的阶段 |
|---|---|---|---|
| 英语四级 | 报名通知、听力耳机、准考证打印、考试周占座、查分、二战 | 最适合作为时间流逝标尺；一次没过不一定大事，但会让人开始计算“还有几次机会” | 大一下至大三 |
| 英语六级 | 四级过后的下一层压力、室友互相问分、考研/就业简历焦虑 | 让“已经过四级”的人也继续被考试追着走 | 大二至大四 |
| 期末周 / 期末考试 | 第十五周后的占座、复习资料、老师划重点、打印店排队、外卖夜宵、宿舍作息冲突 | 每学期都要存在；它压缩娱乐、约会、活动和运动时间，也让讲题、占座、买夜宵变成互信来源 | 每学期末 |
| 补考 / 重修预警 | 开学初成绩、辅导员提醒、补考教室、临时借笔记、补考前夜失眠 | 不能当笑料；它影响自尊、聚餐出席、家庭通话和新学期心态 | 寒暑假后、开学初 |
| 计算机二级 | 非计科同学刷证、计科学生被亲戚误问“你们是不是不用考”、报名费和备考群 | 可以照出专业身份误会，也能让支线人物与计科宿舍产生对话 | 大一至大二 |
| 普通话水平测试 | 报名名额、练读材料、播音主持/教资需求、室友听别人练绕口令 | 适合牵出周屿主持、秦越活动、教师资格证和普通同学的就业备选 | 大二至大三 |
| 教师资格证 | 笔试、面试、普通话证书、网上课程、家长劝考、同学结伴报名 | 不必人人参加，但会影响时间分配和“多拿一个证”的焦虑 | 大二下至大四 |
| 驾照 | 暑假练车、科目二挂科、教练电话、晒黑、返校后缺课补材料 | 可以和家庭、暑假、生活费、交通独立性绑定 | 寒暑假 |
| 软考 / 技术证书 | 软件设计师、网络工程相关备考、题库、报名费、证书截图 | 适合计科专业里“想多拿点东西”的焦虑，但不能取代真实项目能力 | 大二至大四 |
| 考研 / 考公 / 考编证书材料 | 报名、照片审核、缴费、确认、打印准考证 | 第六卷强化去向成形，不提前吞掉第五卷 | 大三下至大四 |

使用规则：

- 考试不必每次写成章节核心。更多时候，它应出现在群通知、饭桌闲聊、图书馆座位、打印店排队、耳机坏了、身份证找不到、查分后沉默等细节里。
- 四六级可以反复出现，但每次状态要不同：第一次是新鲜，第二次是烦，第三次是怕被问，最后一次可能只是毕业前整理材料时的一张旧准考证。
- 考试压力要影响时间分配：谁不能去东区商业街，谁错过寝室饭局，谁在星辰书屋待到闭馆，谁把约会改到查分之后，谁把活动推给别人。
- 考试也能制造普通对话：食堂桌上问“你听力耳机买了吗”，奶茶店排队时背单词，KTV 门口有人说“明天四级还唱这么晚”，后街烧烤摊边有人查分。
- 期末周要改变校园质感：图书馆和星辰书屋变挤，食堂窗口更快收摊，青枫居夜里键盘声变密，楼下外卖架堆满盖饭和奶茶，浴室热水和洗衣机更容易引发小摩擦。
- 补考要写出具体学科和具体代价：高数、线代、大学英语、程序设计、离散数学、数据结构都可以成为压力来源。补考不是“笨”的证明，而是缺课、拖延、活动过载、家庭压力或心理状态失衡留下的账。

## 竞赛、项目与运动会

竞赛和运动会不同于证书背景。它们需要属性强、人物愿望强或关系压力强，才适合成为强剧情点。

| 类型 | 可写内容 | 强剧情条件 |
|---|---|---|
| 数学竞赛 | 高数老师提醒、刷题、宿舍晚间讨论、唐骁计划表、林亦舟被拉去凑人数 | 必须和人物自我证明或团队关系绑定 |
| 数学建模 | 三天熬夜、论文模板、数据处理、队友分工、截图和提交时间 | 适合制造宿舍作息冲突和合作边界 |
| 蓝桥杯 / 程序设计竞赛 | 机房训练、题目卡住、排名、错一道题后的沉默 | 适合计科专业压力和唐骁/陆沉能力差异 |
| 大学生创新创业训练计划 | 立项书、指导老师、答辩 PPT、队友挂名、经费报销 | 适合第六卷项目/创业尝试型预结局 |
| 挑战杯 / 创新创业类校赛 | 路演、商业计划书、活动资源、人情承诺、证明材料 | 适合秦越、许棠、陆沉和林亦舟的材料压力 |
| 校运动会 | 方阵、短跑、接力、跳远、铅球、啦啦队、摄影、主持、学院积分 | 适合强剧情点，因为身体、荣誉、围观和关系会同场发生 |
| 院系友谊赛 | 篮球、羽毛球、乒乓球、网球、跨学院组队 | 适合暧昧、误会、受伤、和解和临时替补 |

运动会可以成为大事件，但不能写成单纯热血。它要有报名被推、临时替补、腿伤瞒着、学院积分、摄影照片、主持串场、看台上的关系暗流和比赛后一起吃饭。它可以让周屿被看见，让夏知微有镜头，让秦越有活动资源，让唐骁把规则和秩序抓得太紧，让陆沉在看台边缘被人喊上场或选择继续做后勤。

竞赛要保留现实疲惫：题不会、队友不回、模板错误、提交失败、老师只给保守评价、比赛结束后还要补课。它们能丰富人物履历，但不能让故事变成竞赛爽文。

## 无名生活人物总则

这个世界不能只有主角和命名支线。无名生活人物不需要姓名，也不需要影响结局，但他们构成大学的血肉。他们可以是食堂阿姨、小饭馆老板、奶茶店店员、校医院护士、后街水果摊大哥、修自行车的一对老夫妻、打印店老板、快递驿站工作人员、KTV 前台、台球厅老板、宿管阿姨、保安叔叔。

他们的作用：

- 让校园像有人在生活，而不是只为剧情搭景。
- 给大学生提供情绪出口：吃顿好的、喝杯奶茶、唱歌、打台球、吃烧烤、买水果、修车、看病、打印准考证。
- 形成重复记忆：某个阿姨记得你不要香菜，某个店员知道你总点少冰，某个水果摊大哥看见你失恋那天买了很多橘子。
- 给遗憾增加重量：毕业前再去那家店，老板说“好久没来了”，人物才会知道时间真的走过去了。

正文中不要把这些人称为 NPC。他们就是学校和周边生活的一部分。

## 可复用生活人物与场景

| 场景 | 人物类型 | 可写细节 | 情绪出口 |
|---|---|---|---|
| 西园餐厅 / 北园食堂 / 东园餐厅 / 桃园餐厅 | 打饭阿姨、收盘叔叔、窗口师傅 | 记得少辣、多饭、不要葱；饭点吵闹；考试周队伍变长 | 不开心时吃一份热饭，和阿姨短短聊两句 |
| 东区商业街小饭馆 | 老板、端菜阿姨、兼职学生 | 常坐角落桌；老板问“还是老样子”；毕业前桌子换了新贴纸 | 聚餐、和解、争吵后补一顿饭 |
| 蜜雪冰城 / 一点点等奶茶店 | 店员、排队学生 | 少冰、三分糖、第二杯半价、节日纸袋、查分当天排长队 | 约会、失恋、等人、躲雨 |
| 后街夜市 | 烧烤摊主、水果摊大哥、炒饭摊夫妻 | 烤串油烟、水果袋、临时加辣、熟客打招呼 | 半夜散心、分手后乱吃、宿舍和解 |
| 校医院 | 护士、值班医生、取药窗口 | 体温枪、挂号单、药袋、输液椅、排队咳嗽声 | 生病露怯、被朋友照顾、装大人失败 |
| 修自行车摊 | 老夫妻、修车师傅 | 打气筒、补胎水盆、旧扳手、电动车铃声 | 迟到、摔车、有人陪着等修好 |
| 打印店 | 老板、排版熟手、排队同学 | 准考证、报名表、彩印照片、U 盘打不开 | 考试焦虑、材料截止、证件照尴尬 |
| 快递驿站 | 驿站工作人员、取件队伍 | 生活用品、父母寄来的药、礼物退货、取件码 | 家庭线、送礼物、分手退东西 |
| KTV / 台球厅 | 前台、老板、隔壁包间 | 麦克风电量、低消、台球杆、散场结账 | 生日、失恋、醉酒前的热闹、装大人 |
| 便利店 | 夜班店员、临时兼职 | 常温水、创可贴、雨伞、电池、排班表 | 外流感、深夜短暂停靠、陌生轻松 |
| 宿舍楼下 | 宿管阿姨、保安叔叔、外卖小哥 | 门禁、外卖架、查寝、丢钥匙、拖箱子 | 回寝前的最后一口气 |

这些人物可以没有名字，但可以有固定动作和话。比如食堂阿姨不必叫名字，只要她第三次问“今天又不吃青椒啊”，读者就会记住她。修车老夫妻不必有完整前史，只要他们在雨后给唐骁的车胎补好、又多收一块钱找不开，就足够像真的。

## 写作禁令

- 不要把无名生活人物写成教训主角的工具人。
- 不要让每个店员都说金句。
- 不要让品牌名变成广告。蜜雪冰城、一点点、便利店、小饭馆可以出现，但只服务生活真实感。
- 不要为了热闹塞满场景。一个短问候、一只塑料袋、一张小票，有时比长篇说明更有用。
- 不要让无名人物承担命名支线人物的弧光。他们可以有温度，但不抢宿舍四人、晚风、夏知微、沈嘉禾、秦越、许棠和许澈的位置。

## 各卷使用建议

- 第一卷：报到当天的食堂窗口、宿管阿姨、快递驿站和校门保安，建立“住进来”的现实触感。
- 第二卷：军训期间校医院、食堂窗口、便利店买水和晨光体育场边的小摊，让身体疲惫落地。
- 第三卷：早八后的打印店、四级报名提醒、东区商业街表演、奶茶店排队、机房外小卖部，以及第一次期末周的占座、夜宵和复习摩擦，让热闹分流更真实。
- 第四卷：暑假空校园里的便利店夜班、后街夜市、水果摊、修车摊和校医院，承接家庭、晚风、引导式外流，以及成绩查询、补考预警和新学期返校压力。
- 第五卷：同质回流和异质软回流可以借助外卖架、食堂侧门、奶茶店、KTV、台球厅、图书馆打印机和便利店完成，但只有 5X 才进入硬外流。
- 第六卷：证书、期末成绩、补考记录、竞赛、大创、实习材料、软考、考研报名、教资和普通话成绩，成为去向形成的现实证据。
- 第七卷：毕业前最后一次去老店、最后一次打印材料、最后一次路过水果摊、四级准考证、补考通知截图和运动会号码布从抽屉里翻出来，让遗憾有具体重量。
