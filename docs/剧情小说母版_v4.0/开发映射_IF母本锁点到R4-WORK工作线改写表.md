# IF 母本锁点到 R4-WORK 工作线改写表

更新时间：2026-05-23

## 用途

本文用于在正式扩写 `R4-WORK` 前锁定一条硬规则：工作线不能自由创作新主线，只能在小说母版公共事件内做镜头改写。

`R4-WORK` 的写法是：

```text
母本事件不变
-> 林亦舟在该事件中的生活半径改为钱、排班、证明、工时、饭卡、家庭转账和缺席解释
-> 选择窗只改变当前工作池内的处理方式、变量和毕业温度
-> 其它路线人物只作为软回声或微影响出现
```

因此，后续写正式剧情页时不能把工作线写成独立外传、苦情励志线、陆沉陪跑线、晚风替代线或 5X 前置奖励线。

## 全局改写原则

| 项 | 规则 |
|---|---|
| 母本事件 | 必须保留同一时期的公共锁点：暑假、补考、返校、站队、C407、项目归档、毕业预登记、最后一个新年、开题、答辩、毕业照、清寝、饭局、离校。 |
| R4 镜头 | 把同一事件改写为林亦舟被钱压、排班、证明、家庭备注、饭卡余额、打印费、工资确认和晚归登记切碎的生活方式。 |
| 选择逻辑 | 第四 / 五幕保留三项选择，进入第六 / 七幕后按收束卷规则改为双选。 |
| 其它人物 | 周屿、唐骁、陆沉、晚风、秦越、许棠、沈嘉禾等只能作为母本生活半径里的软回声，不重新开启完整路线。 |
| 现实代价 | 每次缺席都必须有具体原因：班次、证明、窗口、工资、疲惫、饭卡、家庭转账或请假边界；不能只写“忙”。 |
| 毕业温度 | 工作完成度增强不等于关系自动变好；独立办流程、体面距离、错过饭局、家庭说开和低频联系都必须能从前文变量读出来。 |

第四幕正式写作最低要求：

- 至少 14 个剧情段。
- 至少 6 个选择剧情点。
- 每个选择窗至少 3 项：工作主轴推进 + 宿舍 / 晚风 / 陆沉 / 家庭 / 学业中的两个微内流点。
- 末尾必须接 `P0B_ACT4_ROUTE_CONFIRM -> ACT5-WORK-B01`，不能直跳 Act6。
