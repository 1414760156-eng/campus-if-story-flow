# 开发长线_IF可玩支线R5-LUCHEN陆沉线

## 用途

本文用于把 `R5-LUCHEN` 从第五幕站队分歧骨架扩展成可拆剧情页、选项页、变量回调和毕业结算的 A 级命运外流线。

它不是小说正文，也不是支线外传。它规定的是：玩家在第五幕强冲突后选择靠近陆沉时，林亦舟的后续大学生活如何进入“勤工、沉默、家庭压力、可公开边界、现实任务和迟来的解释”的命运池。母本公共事件照常发生，但镜头、可操作内容和关系后果都按陆沉线重新嫁接。

## 路线定位

路线编号：`R5-LUCHEN`

路线池编号：`POOL-R5-LUCHEN`

路线等级：A 级核心支线

共享范围：第一卷、第二卷共享；第三幕以后只按关键选择进入对应命运池；本线入口在第五幕站队分歧。

核心体验：林亦舟没有选择留在 4XX 公共桌做平均修补者，也没有选择周屿的热闹、唐骁的规则或摆烂回避，而是先走向陆沉。陆沉不会因此突然变得外向，也不会把家里、钱、排班和沉默都解释清楚。他只会在林亦舟没有替他说完、没有把理解写成施舍、没有把沉默当成默认答案时，留出一个更窄但更真实的入口。

本线重点不是“拯救陆沉”，也不是“沉默就天然正确”。陆沉真正的问题是：他习惯把不可说的压力折成短句和行动，别人就容易把他的缺席、冷淡、不解释理解成不在乎。玩家要在一次次现实任务里决定：是替他把话说完整、让场面暂时好看，还是只守住事实边界，把能说的部分交还给陆沉自己。

## 不改变的母本锁点

- 第五幕宿舍强冲突必然发生，且不会因为玩家靠近陆沉就证明陆沉完全无错。
- 周屿、唐骁仍按母本生活半径推进：周屿继续面对活动圈、玩笑、面子和公开误读；唐骁继续面对流程、项目、责任和表格边界。
- 陆沉仍面对勤工、排班、家庭暗线、钱压和发言边界。玩家不能把这些压力一句话解决。
- 第六卷的 C407 项目、课程演示、材料归档、关系定形仍发生，只是林亦舟更多从陆沉的现实压力侧和沉默边界侧进入。
- 第七卷的预登记、毕业活动材料、开题 / 答辩、毕业照、清寝、最后一顿饭、离校手续仍发生。
- 5X 仍是唯一硬外流。本线内部回流不能临时打开 5X，也不能跳入周屿线、唐骁线或感情线主分支。

## 核心变量

| 变量 | 含义 | 高值倾向 | 风险 |
|---|---|---|---|
| `luchen_trust` | 陆沉是否愿意把最低事实和少量真实压力交给林亦舟 | 陆沉会多说一句，或在关键节点等林亦舟一下 | 高信任不等于高表达，不能写成外向化 |
| `proxy_risk` | 林亦舟是否替陆沉说完整心声或代交决定 | 低值时边界更稳 | 高值会让陆沉退回更深的沉默 |
| `work_reality` | 排班、勤工、钱和现实任务是否被看见 | 林亦舟理解陆沉的时间不是空白 | 容易把本线误写成兼职陪跑 |
| `family_pressure` | 家庭消息、补材料、生活费和不可公开压力 | 陆沉的沉默更有现实重量 | 不能用家庭压力替他免除所有关系责任 |
| `quiet_bond` | 沉默中的信任和共处温度 | 能坐一会儿、等一下、留一个入口 | 温情必须很低声，不能突然热烈 |
| `repair_temperature` | 宿舍修补的当面温度 | 旧账有机会说到最低事实 | 可能只做到不再误会，不保证和解 |
| `dorm_trust` | 4XX 是否仍愿意把陆沉算进同一张桌 | 还有同场、同桌、同流程机会 | 低值时陆沉线更深但宿舍更冷 |
| `money_pressure` | 钱、饭卡、打印、交通和排班成本 | 现实任务更具体 | 不能把贫穷写成煽情标签 |
| `tang_misread` | 唐骁对林亦舟靠近陆沉的误读 | 唐骁觉得边界不清、责任不明 | 可能推高表格代言压力 |
| `zhou_distance` | 周屿对林亦舟偏向陆沉的距离感 | 周屿少用玩笑拉他进热闹 | 活动圈照面变少，毕业同框低温 |
| `graduation_temperature` | 毕业结算温度 | 低频联系、迟来的解释、体面淡化或空白离校 | 由前述变量综合决定 |

## 体量与密度

建议 8 章开发体量，对应约 60,000-90,000 字正文容量。本文只提供可开发骨架，不直接写足正文。

最低配置：

- 至少 32 次有效选择。
- 至少 6 个内部回流点。
- 每个内部回流点至少 2 个回流方向。
- 每个回流方向至少 3-4 个连续剧情拍点。
- 至少 3 次强后果回调：第六卷排班 / C407 分工 / 第七卷清寝旧书或补登记。

## 章节开发详表

### 第一章：不替他说

章节目标：确定玩家进入陆沉线后的第一种代价。林亦舟靠近陆沉，会让陆沉知道有人没有急着把他解释成冷漠；也会让周屿觉得林亦舟在帮沉默的人避开场面，让唐骁担心责任又变得模糊。

现实触发：
- 4XX 事项群只剩短句。
- 陆沉没有在冲突现场补充解释，只留下一个“嗯”或“我知道”。
- 周屿急着要一句态度，唐骁要每个人把责任写清。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `LUCHEN-CH01-N01` | 读取站队入口 | 4XX 事项群 / 公共桌 | 林亦舟、陆沉、周屿、唐骁 |
| `LUCHEN-CH01-N02` | 决定先找陆沉 | 青枫居楼下 / 走廊 | 林亦舟、陆沉 |
| `LUCHEN-CH01-N03` | 处理第一句代言风险 | 公共桌 / 私聊窗口 | 林亦舟、陆沉、周屿、唐骁 |
| `LUCHEN-CH01-N04` | 回到主线锁点 | 4XX / 事项群 | 4XX |

关键选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `LUCHEN-CH01-C01` | 只说事实，不替陆沉讲感受 | `proxy_risk -1`、`luchen_trust +1`、`others_misread +1` | 陆沉下次愿意多留一分钟 |
| `LUCHEN-CH01-C02` | 替他说完整结论，让场面先过去 | `proxy_risk +2`、`dorm_trust +1`、`quiet_bond -1` | 第三章陆沉更少补充 |
| `LUCHEN-CH01-C03` | 把问题交回陆沉，只问能不能说 | `luchen_trust +2`、`repair_temperature +1`、`conflict_heat +1` | 第六章陆沉可能多说一句 |
| `LUCHEN-CH01-C04` | 什么都不说，只陪他下楼 | `quiet_bond +1`、`zhou_distance +1`、`tang_misread +1` | 周屿和唐骁的误读提前出现 |

本章结尾钩子：陆沉到楼下后没有解释冲突，只问林亦舟：“你明天早八后有十分钟吗？”

### 第二章：排班撞上公共桌

章节目标：把陆沉的现实压力放到桌面上，但不让玩家用“他很辛苦”直接盖过宿舍冲突。排班是事实，不是免罪牌；钱压是现实，不是让所有人必须让步的理由。

现实触发：
- 快递站或勤工排班和 4XX 公共桌讨论撞期。
- 饭局、材料截止或课堂小组任务同时压来。
- 陆沉只说“我晚点到”，没有说为什么。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `LUCHEN-CH02-N01` | 排班撞期 | 快递站 / 4XX 群 | 林亦舟、陆沉 |
| `LUCHEN-CH02-N02` | 公共桌等待 | 青枫居 4XX | 周屿、唐骁、林亦舟 |
| `LUCHEN-CH02-N03` | 说明最低事实 | 事项群 / 楼下 | 林亦舟、陆沉、4XX |

关键选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `LUCHEN-CH02-C01` | 先处理排班，告诉公共桌只等十分钟 | `work_reality +2`、`dorm_trust -1`、`luchen_trust +1` | 第七章 C407 时间表更紧 |
| `LUCHEN-CH02-C02` | 先回宿舍，把陆沉缺席原因说成“有事” | `proxy_risk +1`、`dorm_trust +1`、`quiet_bond -1` | 陆沉觉得自己被柔化处理 |
| `LUCHEN-CH02-C03` | 问陆沉哪一部分能公开 | `luchen_trust +2`、`repair_temperature +1`、`conflict_heat +1` | 唐骁要求“能公开就写清楚” |
| `LUCHEN-CH02-C04` | 不解释，让所有人自己猜 | `others_misread +2`、`quiet_bond +1`、`dorm_trust -2` | 第五章误读升温 |

本章结尾钩子：陆沉把排班截图发给林亦舟，却撤回了最后一行备注。

### 第三章：阳光书屋的空位

章节目标：写陆沉线的低声共处。阳光书屋不是浪漫化沉默的地方，而是一个让玩家意识到“坐在旁边”也可能有选择代价的场景。

现实触发：
- 陆沉在阳光书屋复习、补材料或等打印店开门。
- 林亦舟发现他旁边有空位，但桌面只留了半张草稿纸。
- 家里消息或补材料提示在手机上短暂亮起。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `LUCHEN-CH03-N01` | 阳光书屋照面 | 阳光书屋 | 林亦舟、陆沉 |
| `LUCHEN-CH03-N02` | 最低可公开事实 | 阳光书屋 / 走廊 | 林亦舟、陆沉 |
| `LUCHEN-CH03-N03` | 家庭压力边界 | 私聊 / 书屋门口 | 林亦舟、陆沉 |

关键选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `LUCHEN-CH03-C01` | 坐远一点，只问他要不要一起走 | `quiet_bond +2`、`luchen_trust +1` | 陆沉第六章可能主动等他 |
| `LUCHEN-CH03-C02` | 直接问家里是不是出事 | `family_pressure +1`、`luchen_trust -1`、`conflict_heat +1` | 陆沉会用更短句挡回去 |
| `LUCHEN-CH03-C03` | 问“这件事能不能让他们知道一点” | `proxy_risk -1`、`repair_temperature +1`、`family_pressure +1` | 第五章可用于抵消误读 |
| `LUCHEN-CH03-C04` | 只借一支笔，不问任何事 | `quiet_bond +1`、`missed_chance +1` | 第七章材料节点缺少解释 |

本章结尾钩子：陆沉把笔还给林亦舟，说“你别拿这个去证明什么。”

### 第四章：快递站错件

章节目标：把“帮忙”写成有边界的可玩选择。玩家可以帮陆沉处理现实任务，但不能把陆沉的责任、钱压或沉默全部接走。

现实触发：
- 快递站包裹错件，补登记表和课程材料同时催。
- 陆沉在排班间隙确认收件信息，林亦舟看见他手背上的胶带痕。
- 站长、同学或 4XX 群都在催不同的事。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `LUCHEN-CH04-N01` | 快递站错件 | 快递站 | 林亦舟、陆沉、生活人物 |
| `LUCHEN-CH04-N02` | 补登记与课程材料撞期 | 打印店 / 快递站 | 林亦舟、陆沉 |
| `LUCHEN-CH04-N03` | 帮忙边界选择 | 路上 / 事项群 | 林亦舟、陆沉、唐骁 |

关键选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `LUCHEN-CH04-C01` | 帮搬件，但让陆沉自己确认登记 | `work_reality +1`、`luchen_trust +1`、`proxy_risk -1` | 清寝旧书节点更稳 |
| `LUCHEN-CH04-C02` | 直接替他跑补登记窗口 | `proxy_risk +2`、`money_pressure +1`、`quiet_bond -1` | 第八章补登记可能出现授权问题 |
| `LUCHEN-CH04-C03` | 先问唐骁能不能延后小组材料 | `tang_misread +1`、`project_stability -1/+1`、`repair_temperature +1` | 唐骁认为边界不清 |
| `LUCHEN-CH04-C04` | 只把错件事实发群，不讲陆沉压力 | `dorm_trust +1`、`proxy_risk -1`、`zhou_distance +1` | 周屿觉得林亦舟“说一半” |

本章结尾钩子：陆沉在快递站关灯前把错件单递给林亦舟，说“这个不用替我收。”

### 第五章：被理解成偏袒

章节目标：让未选择者误读真正发生。林亦舟靠近陆沉，不会让周屿和唐骁自动理解；他们会用各自的方式退开、试探或把问题推回现实流程。

现实触发：
- 周屿活动圈缺人，林亦舟没有去补场。
- 唐骁要求每个人确认分工，陆沉只确认最低事实。
- 林亦舟的“等他自己说”被读成偏袒沉默。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|
| `LUCHEN-CH05-N01` | 周屿误读 | 东区活动 / 4XX 群 | 林亦舟、周屿 |
| `LUCHEN-CH05-N02` | 唐骁表格代言风险 | C407 / 共享文档 | 林亦舟、唐骁、陆沉 |
| `LUCHEN-CH05-N03` | 误读是否当场解释 | 走廊 / 群聊 | 林亦舟、周屿、唐骁、陆沉 |

关键选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `LUCHEN-CH05-C01` | 向周屿说明只能说最低事实 | `zhou_distance -1`、`proxy_risk -1`、`conflict_heat +1` | 周屿玩笑变少但仍不热 |
| `LUCHEN-CH05-C02` | 让唐骁把表格留给陆沉自填 | `proxy_risk -2`、`tang_misread +1`、`luchen_trust +1` | 第七章分工边界更清 |
| `LUCHEN-CH05-C03` | 为了效率替陆沉填完 | `project_stability +1`、`proxy_risk +2`、`quiet_bond -2` | 陆沉第六章不会多说 |
| `LUCHEN-CH05-C04` | 不解释误读，只保留现场不吵 | `dorm_trust -1`、`others_misread +2`、`quiet_bond +1` | 毕业同框温度降低 |

本章结尾钩子：周屿在群里发了一个笑脸表情，又撤回，唐骁把共享表格的“代填”栏删掉了。

### 第六章：陆沉多说一句

章节目标：给陆沉线一个转深节点，但不把它写成大段倾诉。陆沉多说一句，是因为前面多次没有被代言；这一句也不该被玩家拿去证明他终于“变好”。

现实触发：
- 楼下、阳光书屋或快递站关门后只剩两个人。
- 家里消息、补登记或排班临时变化把陆沉逼到必须解释最低事实。
- 林亦舟有机会追问，也有机会停住。

可拆节点：

| 节点 ID | 功能 | 场景 | 人物 |
|---|---|---|---|

## 对话逻辑

### 陆沉

- 台词短，不是为了神秘，而是因为很多事确实不能公开、来不及公开或说了也没有马上有用。
- 他不会因为玩家靠近就突然倾诉。信任的表现可以是多等一分钟、把截图发来、把一句话说完、把旧书交给林亦舟看着。
- 他对“替他说完整心声”很敏感。玩家越代言，他越会退回只确认事实。
- 他表达照看通常不说“谢谢你理解我”，更可能说“这个不用你跑”“你先回去”“那栏别填”。

### 林亦舟

- 不能变成陆沉的翻译器。
- 玩家每次靠近陆沉，都要承担“周屿觉得被冷处理、唐骁觉得边界不清”的代价。
- 他可以帮陆沉保留入口，但不能把“我理解你”变成替陆沉消除责任。

### 周屿

- 在本线里不抢主轴，但会通过玩笑撤回、活动不再叫人、饭局少问一句形成距离。
- 他不是讨厌陆沉，而是不知道怎么面对一个不解释的人，也会觉得林亦舟把沉默看得比当场热闹更重要。

### 唐骁

- 唐骁会成为本线重要压力源：他要清楚、要分工、要提交，陆沉只给最低事实。
- 他不是反派。他的问题是怕边界不清导致责任漂移，所以会误读林亦舟在“纵容不说话”。

### 晚风 / 沈嘉禾 / 夏知微

- 只能作为软回声或边界照面。
- 晚风可以通过消息低频提醒“你最近总是在等别人说话”，不能转入晚风主线。
- 沈嘉禾可以提醒资料边界，夏知微可以提供观看角度，但都不能替陆沉线完成关系修复。

## 漂移防线

- 本线不是“陆沉救赎线”。靠近陆沉会增加信任和现实理解，也会增加宿舍误读、唐骁边界压力和周屿距离。
- 本线不是“兼职线”。快递站、排班、钱压和家庭暗线只是现实载体，真正可玩的是不代言、可公开事实和沉默边界。
- 本线不是“沉默正确线”。沉默能保护人，也会伤人；玩家不能用“他不说就是有苦衷”取消关系责任。
- 本线不能把周屿写成只会起哄，也不能把唐骁写成只会逼问。
- 本线不能让内部回流打开其他命运池。所有回流只改变本线关系温度、误读、资源代价和毕业结算。
- 本线不能取消母本公共锁点，只能改变林亦舟进入锁点的位置。

## 开发优先级

1. 先实现入口 `P0D_STAND_WITH_LUCHEN` 与路由池锁定。
2. 再实现 8 章主事件链。
3. 再实现 6 个内部回流点，每个回流点至少 2 个方向。
4. 再写第六卷公共锁点嫁接：排班 / C407 / 可公开边界 / 关系定形。
5. 最后写第七卷毕业结算：旧书 / 补登记 / 饭局 / 离校。
