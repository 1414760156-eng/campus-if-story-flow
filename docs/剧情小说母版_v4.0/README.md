# 武生院沉浸版剧情小说母版 v4.0

本目录用于承载《武生院沉浸版》全游戏剧情小说母版。

它不是 5A 模板，也不是试玩 JSON，而是覆盖序幕/第一幕至第六幕、终章《最后一个新年》的长篇小说底稿。它在游戏之上，首先要作为一部长篇校园群像小说成立。后续所有支线、人物对话、选项、回流、外流、心理暗线、资源变量和结局回响，都可以从这里取材。

最高优先级是文学性和吸引力。后续互动分支、选项、变量、回流和成就，都应从小说中拆分，而不是反过来压缩小说的叙事节奏、人物复杂度和章节悬念。

## 目标

总字数不少于 40 万字，目标区间为 42-48 万字。

母版要像一部长篇校园群像小说，而不是剧情表。文学性、悬念、人物吸引力和阅读连续性优先；游戏拆分只是后续用途。主线关键事件一定发生并继续向前推进，但主线之间的日常也要充分展开：宿舍生活习惯差异、生活费变化、家庭压力、被骗、感情暧昧、老师误会、小组合作、小算计、活动资源、错过和修补，都要成为人物成长和关系变化的现实来源。

## 使用方式

后续改具体试玩节点时，先查本母版对应场景，再拆成：

- 路线级选择
- 心理级微选择
- 关系变化
- 资源/生活费/证明材料变化
- 回流段
- 唯一硬外流
- 第六幕入口标签
- 终章软回响

这样可以保证单个节点好玩，同时不让全局剧情漂移。
