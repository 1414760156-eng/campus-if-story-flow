# IF 命运分歧开发矩阵

## 核心纠偏

本游戏不应以“支线外传”为主要开发目标。支线人物、地点和生活碎片的作用，是给主线事件提供血肉、动机、关系回声和可选场景；真正要开发的是：玩家在关键事件点做出不同选择后，林亦舟会走向哪一种生活方式、关系状态和结局温度。

换句话说：

- **主线事件一定发生**：军训手机事件、汇演、班会、主持招新、第五卷站队、5X 唯一外流、第六卷关系定形、第七卷毕业收束不能被取消。
- **玩家改变的是林亦舟如何经过这些事件**：他修补、逃避、站队、讨好、沉默、靠近外部、投入学业、进入活动圈、打工自保或外流。
- **玩家真正选择的是“靠近谁/靠近哪种生活”**：被选择靠近的人或方向会获得更多事件容量、互动频率、现实代价和毕业结局文本；未被选择的人仍按母版主线生活，不被抹掉，也不为了玩家选择而改写人生。
- **支线不是目的**：夏知微、沈嘉禾、秦越、许棠、晚风、许澈和无名生活人物，主要是不同 IF 生活方式的支撑场景，不是每个人都要独立做成长篇外传。
- **晚风不是必选更进一步**：晚风线只是“亲密外部”方向的一种可能。玩家可以靠近，也可以保持网友/朋友状态，也可以错过，也可以把时间投向宿舍、学业、活动、打工或外流。

## 分歧边界

IF 分歧不等于重写整个世界。母版提供一条稳定运行的“世界真实线”：四人同寝、关键事件、支线人物生活半径、毕业流程都会继续发生。玩家选择改变的是林亦舟把自己投向哪里，以及他因此获得或错过哪些当面机会。

开发时按三条边界处理：

1. **主线人物不被玩家重写**：周屿仍会进入活动圈，唐骁仍会走向项目和秩序，陆沉仍会面对勤工和现实压力，晚风仍有自己的朋友和生活圈。玩家只能改变林亦舟和他们之间的距离、信任、误读、亲密或疏远。
2. **被选择方向要长出更多故事**：选择靠近晚风，就要有更多游戏、微信、抖音、见面、送礼、牵手、公开边界和时间代价；选择靠近宿舍，就要有更多公共桌、饭局、清寝、站队、修补和旧账；选择学业、活动、打工，也必须各自长出完整事件链。
3. **毕业结局按投入方向结算**：不是简单加好感，而是根据林亦舟长期靠近谁、错过谁、承担什么代价，生成不同毕业温度：说开、淡化、体面不亲近、不和解、继续联系、错过、外流遗憾。

## 简化开发模型：时期外流点

为控制开发复杂度，本项目不按“每个事件都开 IF 树”处理，而按**不同时期的外流点**处理。这里的“外流”多数是软外流：林亦舟的生活重心偏离默认宿舍线，进入另一条更有剧情容量的路线；不是每次都彻底离开主线。真正硬外流仍只保留 5X 唯一外流。

默认主轴是 **4XX 宿舍线**。如果玩家没有在关键时期选择外流，就继续沿宿舍主轴推进，支线只作为短事件、微心态和关系温度存在。

| 时期 | 外流点 | 触发选择 | 形成路线 | 路线核心 | 不变内容 |
|---|---|---|---|---|---|
| 第三幕 | 社团 / 活动入口 | 主持招新、活动圈、履历与公开表达优先 | 完美线 / 自我管理线 | 林亦舟更早偏向履历、流程、公开表现和“把自己安排好”，宿舍线变成回声 | 汇演、班会、主持招新仍发生 |
| 第四幕 | 暑假入口 | 留校、生活费压力、勤工、快递站 / 打印店优先 | 兼职线 / 生存线 | 林亦舟更早理解钱、时间和现实压力，错过部分宿舍当面机会 | 暑假、家庭暗线、返校压力仍发生 |
| 第五幕前段 | 亲密分歧入口 | 不选择和晚风更进一步，转向另一个女主 / 现实陪伴关系 | 专注感情线 | 林亦舟把主要情绪和时间投给新的亲密对象，宿舍冲突仍发生但参与方式变少 | 晚风仍存在，可停在朋友 / 错过状态 |
| 第五幕中后段 | 宿舍站队入口 | 强冲突爆发后选择周屿、唐骁、陆沉或不选 | 周屿线 / 唐骁线 / 陆沉线 / 摆烂线 | 选择一个室友就获得更多对方故事、矛盾、修补和毕业结算；不选则进入低投入、低修补的摆烂线 | 第五幕站队、宿舍裂痕、第六卷关系定形仍发生 |
| 第五幕隐藏高难 | 5X 唯一硬外流 | 在唯一外流点离开旧线，进入新圈 | 5X 外流线 | 进入许澈 / 新圈子单线，旧宿舍渐远，新关系轻松但薄 | 第六 / 七卷只走外流单线，不回原多线 |

开发优先级以这张表为准。下方“关键事件 IF 分歧总表”只作为素材库和变量来源，不再要求每个事件都扩成独立大路线。

## 开发层级

| 层级 | 名称 | 开发重点 | 支线人物作用 |
|---|---|---|---|
| L0 | 主线锁点 | 关键事件必定发生，保证全局剧情不漂移 | 提供现场或回声，不改变事件存在 |
| L1 | IF 命运分歧 | 玩家选择林亦舟靠近的人和生活方向，改变关系强度、事件容量和结局温度 | 按选择方向进入不同人物生活半径 |
| L2 | 支撑场景 | 用连续事件、对话、地点和物件支撑 IF 的真实感 | 让分歧不是凭空数值变化，而是多出来的生活 |
| L3 | 微心态选择 | 不改主线，但改心理、语气、关系温度、后续文本 | 提供可操作细节和重复出现的回声 |
| L4 | 结局结算 | 第七卷按前面 IF 累积结算说开、淡化、不和解、外流遗憾 | 只回响，不强行团圆 |

## 林亦舟命运方向标签

| 标签 | 方向 | 典型选择 | 主要代价 | 结局倾向 |
|---|---|---|---|---|
| IF-DORM-REPAIR | 宿舍修补 | 留在 4XX、当面说清、承担公共桌烂摊子 | 外部机会减少，情绪被旧账拖住 | 有机会说开，但不一定完全和解 |
| IF-DORM-BREAK | 宿舍裂痕 | 站队过硬、拒绝解释、故意不帮、错过饭局 | 宿舍互信下降，误读恶化 | 体面但不亲近，或不再追究 |
| IF-STUDY | 学业项目 | 去机房、找老师、竞赛/项目优先、按流程处理 | 情绪表达变少，被认为只讲结果 | 去向清晰，但关系可能变窄 |
| IF-ACTIVITY | 活动舞台 | 主持、融媒体、摄影、商业街小舞台、活动圈 | 人情债增加，宿舍觉得他“不在” | 被看见，但旧关系淡化 |
| IF-WORK | 打工生存 | 勤工、快递站、打印店、省钱、推掉聚餐 | 时间少，别人误读冷淡或抠 | 更懂现实，但错过很多当面机会 |
| IF-INTIMACY | 外部亲密 | 回晚风语音、见面、送礼、散步、牵手 | 宿舍缺席、钱/时间消耗、旧账延迟 | 关系有形状，但不能替代宿舍结算 |
| IF-AVOID | 回避自保 | 不进门、刷手机、去湖边/便利店、只回短句 | 问题被拖大，别人替他解释 | 遗憾多，很多话过期 |
| IF-PERFECT | 完美自控 | 每件事都按时、少犯错、少卷入强冲突 | 关系温度不足，别人觉得他总在管理自己 | 主线稳，但情感回响偏淡 |
| IF-LIEFLAT | 纯摆烂 | 不争、不问、不补救，顺手选最省力路径 | 资源和关系慢慢流失 | 弱冲突多，最后像被生活推走 |
| IF-5X | 唯一外流 | 在第五卷唯一外流点离开旧线，进入新圈 | 宿舍渐远，新关系轻松但薄 | 单线遗憾结局，不回原多线 |

|---|---|---|---|
| 宿舍修补 | 周屿、唐骁、陆沉、宿管阿姨 | 用门、钥匙、公共桌、饭局、查寝制造当面机会 | 不能让外部人物替室友和解 |
| 宿舍裂痕 | 周屿、唐骁、陆沉、许棠 | 用私下吐槽、公开材料、群消息、缺席制造误读 | 不能让主线事件消失 |
| 学业项目 | 唐骁、沈嘉禾、老师/助教 | 用机房、C407、打印店、项目归档承接选择 | 不能把沈嘉禾写成万能资料员 |
| 活动舞台 | 周屿、秦越、许棠、夏知微 | 用主持队、融媒体、摄影、商业街小舞台承接热闹 | 不能让活动圈吞掉宿舍主线 |
| 打工生存 | 陆沉、快递站、打印店、校医院、宿管 | 用排班、小票、生活费、错过聚餐承接现实压力 | 不能把穷写成标签或笑料 |

| 回避自保 | 晚风/夏知微/无名生活人物/便利店 | 用不进门、湖边绕路、买水、刷手机承接逃避 | 不能让回避没有后果 |
| 完美自控 | 唐骁、老师、沈嘉禾 | 用按时、表格、项目、成绩承接稳定但低温 | 不能写成唯一正确路线 |

| 5X 外流 | 许澈、新圈子、后街、台球厅、便利店 | 用轻松关系和无旧账感承接硬外流 | 不能再流回原多线 |

## 开发优先级

### P0：先做 IF 骨架

1. 先做时期外流点：第三幕社团 / 完美线，第四幕暑假 / 兼职线，第五幕亲密分歧 / 专注感情线，第五幕站队 / 三室友线与摆烂线，5X 唯一硬外流。
2. 每条路线只补关键事件链：3-5 个核心事件、2-3 个代价节点、1 个第七卷毕业结算入口。
3. 普通事件不再开大分支，只做微心态、关系温度、台词和后续文本变化。
4. 第六卷承接规则：不同路线如何让 4XX 退位、亲密关系、项目 / 打工 / 活动方向发生差异。
5. 第七卷结算规则：说开、淡化、体面不亲近、不和解、继续联系、错过、5X 遗憾。

### P1：再做支撑场景

1. 夏知微：摄影/授权/观看角度，服务兴趣表达和异质回流。
2. 沈嘉禾：资料/项目/边界，服务学业项目和现实好感对照。
3. 秦越：主持/活动圈，服务周屿和活动舞台。
4. 许棠：融媒体/公开边界，服务公共叙事风险。
5. 家庭/考试/打工/被骗/恋爱对照，服务现实压力和时间流逝。

### P2：最后做碎片和成就

1. 宿管阿姨晚归簿。
2. 快递站取件码。
3. 食堂/商业街/后街吃饭。
4. 校医院和运动受伤。
5. 东区小舞台、KTV、台球厅。

这些不必每条做成完整外传，适合做短事件、隐藏成就、地点回响和微心态选择。

## 结论

开发方向应从“支线外传”调整为“时期外流点 + IF 命运分歧”。支线人物仍然重要，但他们是路线支撑材料，不是主目标。玩家不是在每个事件都把剧情分成一棵树，而是在第三幕、第四幕、第五幕几个关键时期选择林亦舟靠近哪种生活。被选择的方向获得更多事件、更多互动、更多代价和更明确的毕业结算；未被选择的人仍按母版主线生活，只是在林亦舟的视角里交集变少、关系温度不同。
