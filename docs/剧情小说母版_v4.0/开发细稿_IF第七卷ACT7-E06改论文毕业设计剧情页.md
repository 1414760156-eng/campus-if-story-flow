# IF 第七卷 ACT7-E06 改论文 / 毕业设计剧情页级细稿

## 用途

本文用于把 `ACT7-E06-THESIS-REVISION` 拆成可开发剧情页、选项页、变量回调和后续 JSON 事件结构。

本文件不是小说正文，也不是试玩器脚本。它服务于第七卷毕业结算层：同一个“改论文 / 改毕业设计”事件点，所有已锁定路线都必须经过，但每条路线读取自己的 `route_pool_id`、`act7_variant_id`、`act6_final_direction` 和 `graduation_temperature`，输出不同的帮看关系、版本边界、截止压力和结局温度。

## 接入位置

| 字段 | 建议值 |
|---|---|
| `event_id` | `ACT7-E06-THESIS-REVISION` |
| `act` | `act7` |
| `mother_chapter` | 第七卷第四章《开题》、第五章《三四月》、第七章《答辩以后》嫁接 |
| `previous_event` | `ACT7-E05-MARCH-QUEUE` |
| `next_event` | `ACT7-E07-DEFENSE` |
| `route_switch_allowed` | `false` |
| `same_mother_event` | `true` |
| `main_location` | 教学楼 C407、打印店、西园餐厅二楼、图书馆、阳光书屋 |
| `base_scene` | 导师红批、开题 / 毕业设计方向初表、修改稿版本、十点前提交 |

本事件必须承接 `ACT7-E03-LAST-NEW-YEAR` 末尾那张“开题 / 毕业设计方向初表”：上一事件把压力从旧物公开推到“下一张表不能一起空着”，本事件则把这张表落成红批、版本命名、谁能看稿、谁不能替他说理由。

它不能写成纯学业流程，也不能只说“论文改完了”。改稿的功能是检验当前路线关系：谁还愿意看，谁只能回一个“收到”，谁能提醒边界，谁已经不等。

## 共用硬规则

1. 每条路线必须出现一条具体导师 / 指导老师修改意见，至少包含红批、批注、版本命名、数据来源、格式或图表问题之一。
2. 每条路线必须出现一个明确截止时间，推荐使用“今晚十点前”“九点四十五前到 C407”“查重前版”等现实节点。
3. 每条路线必须让玩家做一次核心二方向选择：自己改 / 找当前路线人物帮看，发完整稿 / 只发格式问题，等一个人 / 不再等，交最低版 / 重改关键页。二方向按路线语境改名，但不能扩成新路线。
4. 每条路线必须写出“帮看不等于替写”。当前路线人物可以看格式、边界、图表、来源和版本，不能替林亦舟决定毕业设计要讲什么。
5. 每条路线必须留下 `ACT7-E07-DEFENSE` 的回声：答辩门口谁在、谁不在、谁只发消息、谁赶流程、谁等到最后一分钟。
6. `R5X-HARD` 仍保持 `route_lock = "5x"`，旧线人物只能作为软照面、消息、同场流程或低温回声，不得回原多线。

禁止写法：

- 不允许把改论文写成“大家一起熬夜所以和好”。
- 不允许让晚风、沈嘉禾、夏知微、许澈或任何路线人物替 4XX / C407 旧项目收尾。
- 不允许用“导师通过了”跳过版本、格式、签名、上传和补交。
- 不允许因为某人帮看稿而在第七卷重新进入另一条完整路线。

## 从开题方向初表到修改稿

| 承接项 | 初表阶段 | 修改阶段 | 可回调变量 |
|---|---|---|---|
| 方向 | 可写“待定”，但需本人签字 | 必须落成题目、数据来源和展示范围 | `thesis_topic_confirmed`、`project_stability` |

| 公开 | 最后一个新年材料可空白 | 课程成果和主题材料必须分清 | `public_scope_checked`、`graduation_temperature` |

## 共用事件拍点

这四拍为所有路线共享骨架。路线变体只能改变谁能看稿、在哪改、改哪一类问题、消息温度和后续回声，不能改变事件功能。

| 拍点 | 页面 ID | 母版动作 | 玩家功能 |
|---|---|---|---|
| 第1拍：拿到红批 | `ACT7-E06-COMMON-B01` | C407 或导师组发回修改意见，红笔点在数据来源、格式、版本或图表上。 | 确认本路线的毕业设计问题是什么。 |
| 第2拍：决定给谁看 | `ACT7-E06-COMMON-B02` | 林亦舟面对“自己改 / 找当前路线人物帮看 / 只发格式问题”的选择。 | 检验亲密、信任和边界。 |
| 第3拍：改稿现场 | `ACT7-E06-COMMON-B03` | 西园餐厅、打印店、图书馆、阳光书屋、宿舍公共桌或新圈桌面成为改稿现场。 | 把路线关系压进具体动作。 |
| 第4拍：提交与回声 | `ACT7-E06-COMMON-B04` | 修改版上传、纸质版补交、文件名定版，消息回到答辩等待。 | 进入 `ACT7-E07-DEFENSE`。 |
