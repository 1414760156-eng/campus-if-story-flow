# IF R4-WORK 母本小说横向关联对比审计

更新时间：2026-05-29

## 审计结论

用户判断成立：`R4-WORK` 不能只按自己这条工作线纵向顺完，还必须回看母本第五幕，尤其是已经成熟的 `5A` 宿舍修补线，做横向关联对比。

当前 `R4-WORK` Act5 已从旧 5 窗扩为 12 个主选择窗口，已经能试玩，也已经修掉一轮说明腔和重复模板句。但它仍只是“第一版可玩细稿”，还没有达到母本第五幕的核心可玩密度。母本 `5A` 的强度不只来自页数，而来自四件事：

- 多线压力开场后迅速拆成单线。
- 主线场景中有同质回流和异质软回流。
- 晚风、夏知微、沈嘉禾、秦越等边缘人物带着各自载体出现，不替主线解决问题。
- 第五幕结果必须能沉到第六幕和终章，而不是停在本幕“选择完成”。

因此，`R4-WORK` 下一步不应只继续润句，也不应盲目加窗。它需要按母本 5A 的玩法骨架补“工作线版本”的回流层、复发层和横向人物镜面。

## R4-WORK 下一轮补强建议

### 第一优先：补 4 个“工作线软回流组”

不新增大路线，只在 12 窗内插入或扩写短回流：

| 回流组 | 建议落点 | 功能 |
|---|---|---|
| 班前门口回流 | `B01 -> B03` | 进 4XX / 下楼快递站 / 楼下停住；对应母本门口空间选择。 |
| 图书馆 / 打印店沈嘉禾回流 | `B02` | 格式可以帮看，说明不能代写；对应资料软回流。 |
| 活动名单秦越 / 许棠回流 | `B07/B10` | 活动可靠和公开叙事同时消耗他；对应资源 / 公开镜面。 |
| 晚风 / 夏知微夜晚回流 | `B04/B09` | 私人联系和影像边界照见缺席，但最后推回工作 / 学习债。 |

### 第二优先：补 2 个“母本级波峰”

| 波峰 | 建议落点 | 目标 |
|---|---|---|
| 工作线规则会 | `B05/B08` | 让 C407、4XX、快递站、材料证明同桌冲突，不只是各讲各的。 |
| 期末 / 补考复发 | `B09/B12` | 工作排班撞上学习债，验证前面边界是否有效，接第六幕稳定 / 低温。 |

### 第三优先：调整第六幕输出

R4 不一定需要母本 5X 那样的新人物硬外流，但必须有同等级风险：

- `lock_work_radius`：固定班成为主表，4XX / 活动 / 晚风 / 学习都开始绕开他。
- `reduce_shift`：少接一晚，保留多个入口，但收入和证明材料变薄。
- `stop_conditions`：排班、活动、作业、回寝都写停止条件，进入第六幕的稳定 / 修补倾向。
- `silent_low_completion`：若一直用工作解释所有缺席，进入第六幕低完成风险。

## 禁止照搬

- 不把 `5A` 的宿舍冲突照搬成工作线吵架。
- 不让晚风、夏知微、沈嘉禾、秦越替林亦舟讲道理。
- 不把 73 节点作为 R4 的硬指标；R4 的指标是“有回流层、有复发层、有第六幕沉淀”。
- 不因为补校园人物就把 `POOL-R4-WORK` 打散成晚风线、学业线、活动线。
- 不在玩家正文出现“母本 / 横向关联 / 回流层 / 变量 / 第六幕会”等开发词。

## 当前结论

`R4-WORK` Act5 现在可以试玩，但还不能称为母本级第五幕。它的下一步应该是“在现有 12 窗上补横向回流和母本波峰”，而不是继续逐句润色或直接进入 Act6。

建议执行顺序：

1. 先做 `r4_work_act5_mother_alignment_pass_01`，补 B01/B03/B05/B07/B09/B12 的回流和波峰种子。
2. 再导出 4->5 全链试玩，让玩家确认第五幕体量是否明显超过第四幕。
3. 然后才进入 Act6 接口，因为 Act6 需要读取这些第五幕沉淀，而不是只读取当前 12 个方向标签。
