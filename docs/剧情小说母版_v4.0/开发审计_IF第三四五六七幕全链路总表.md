## 总结论

3/4/5/6/7 全链路还不能冻结。

准确表述如下：

- 第一、二幕共享，不在这里开长线命运池。

- 第五幕承担主轴亲密、宿舍站队和 5X 硬外流：P0-C / P0-D / P0-E 均已接入。

- IF 结构层面不应只看第五幕；早期外流线不能从第三 / 四幕直接跳到第六幕。A3 父池与 `R3-PERFECT` 已补明第四幕生活半径和第五幕承接；当前只剩 `R4-WORK` 第五幕承接仍需处理后才能冻结 3/4/5/6/7 全链路。

## 入口层判断

### 第三幕：P0-A

第三幕已经不是“选融媒体”或“直接进完美线”。当前结构为：

- `A3-ACTIVITY-PUBLIC` 是父池，负责承接主持队、新闻中心 / 融媒体、摄影社、幕后物资、志愿服务、公开表达和回 4XX 等方向。
- `R3-PERFECT` 是父池内部持续低风险、流程化、自我管理后的子外流。
- 有女生连续出现并承担事件压力的 A3 方向，才允许在第五幕派生 `R5-ROMANCE`。

当前判断：第三幕入口层已接入；A3 父池与 `R3-PERFECT` 均已补第四幕生活半径和第五幕桥接。`R3-PERFECT` 已通过 `ACT4-PERFECT` / `ACT5-PERFECT` 把公开无错、私人延迟、工具化风险和宿舍低温交给 Act6。

### 第四幕：P0-B

第四幕已经不是普通支线外传，而是现实压力外流点。当前结构为：

- 生活费、留校、快递站、打印店、勤工和错过照面逐步累积。
- 进入 `R4-WORK` 后，第六幕继续用工时、证明、钱账、排班和宿舍缺席定形。
- 第七幕以独立生活、第一周安排和低频联系收束。

当前判断：第四幕入口层已接入，但仍需复核 / 补出第五幕承接层。工作线不能从第四幕暑假直接跳到第六幕，必须在第五幕看到钱压、排班、错过照面和宿舍缺席如何继续发酵。

### 第四幕：全路线生活半径分化

第四幕不应只被当成 `R4-WORK` 的入口。它同时承担所有路线的生活半径预热：

- 已在第三幕外流的 `A3-ACTIVITY-PUBLIC` / `R3-PERFECT`，第四幕必须写当前池内的活动任务、公开材料、低风险自控、宿舍缺席和私人延迟。
- 工作线 `R4-WORK` 的第四幕是主场，钱、工时、排班、家庭暗线和错过照面应成为路线主体。
- 第五幕才锁定的 P0-C / P0-D / P0-E 组，第四幕不直接开完整主分支，但必须埋变量：晚风联系频率、宿舍温度、周屿活动牵引、唐骁项目压力、陆沉现实任务、摆烂回避值和 5X 离场疲惫。

当前判断：3/4/5/6/7 冻结前，需要先把“第四幕生活半径”作为独立检查项补进早期外流线审计，不再只检查第五幕入口是否存在。

第四幕选择密度按 `开发规则_IF第四幕生活半径与池内流硬约束.md` 执行：

- 普通生活半径层：每条路线最低 4 个选择剧情点。
- 第四幕主场线：最低 5 个选择剧情点；当前 `R4-WORK` 属于主场线。
- 最低线只用于验收，不是正式写作目标；普通生活半径层推荐 10-12 个剧情段，早期外流池内层推荐 12-14 个剧情段，第四幕主场线推荐 14-16 个剧情段。
- 每个选择剧情点必须包含选择前铺垫、选择窗、选择后反馈、变量变化和第五幕回声。
- 进入当前 `route_pool_id` 后，第四幕选择窗至少 3 项：1 个当前支线主轴推进 + 2 个池内微内流点，不允许写成 3-4 个完整路线方向。
- 三项内流的标准结构是“推进本支线主轴 / 处理微内流点 1 / 处理微内流点 2”；微影响只能改变软回声、误读、边界、资源压力和第五幕入口偏置，不能改变 `route_pool_id` 或提前开启新完整路线。
- 已外流线必须让三项都留在当前池；未外流线只能记录候选倾向和 `act5_entry_bias`，不能提前写死 `active_route_id`。
- 进入支线后必须保持小说母版主轴的写法和玩法规格：母版公共锁点、人物对话、选择前铺垫、选择后反馈、现实代价和其它人物回声都不能因外流被压缩成概要。
- 第四幕不得直接跳 Act6，必须把变量交给第五幕。

### 第五幕：P0-C / P0-D / P0-E

第五幕入口层分三组：

- P0-C：亲密分歧。主轴只能直接进晚风线；其它感情线必须来自 A3 女生候选来源。
- P0-D：宿舍站队。共用站队入口先分发到 DEFAULT、周屿、唐骁、陆沉和摆烂，再由各自 Act6 接续。
- P0-E：5X 唯一硬外流。满足硬锁条件后进入不可回流单线。

当前判断：第五幕入口层已完成。后续如说“第五幕还没做”，应只指正文 P2 润色或旧稿残留清理，不再指结构缺口。

## 漂移风险复核

本轮从现有索引和规格中确认的防漂移口径：

- `A3-ACTIVITY-PUBLIC` 不等于融媒体，也不等于 `R3-PERFECT`。
- `R5-ROMANCE` 不从默认宿舍主轴凭空生成，必须有 A3 女生候选来源。
- P0-D 不为每条三室友线重复写入口；共用入口是正确结构。
- `R5-LIEFLAT` 不是省内容线，必须把回避作为可操作选择并在 Act7 结算入口过期。
- `R5X-HARD` 只能从第五幕唯一硬外流点触发，不能由第三幕、第四幕或普通厌烦提前触发。

当前发现的结构级缺口收窄为：`R4-WORK` 仍需复核 / 补第五幕承接层。`A3-ACTIVITY-PUBLIC` 与 `R3-PERFECT` 已补齐，不需要改 Act7。
