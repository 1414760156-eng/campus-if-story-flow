# 上册阶段总结：第一至四卷

## 上册定位

第一至四卷可以作为《武生院沉浸版剧情小说母版 v4.0》的上册：它写的是“人怎样来到同一张桌子旁边”。这一部分不急着把路线收窄，而是让武生院、青枫居、教学楼、食堂、快递站、书屋、体育场、东区商业街和地铁动线慢慢长出重量。上册的重点不是结局，而是关系基础、生活旧账、现实压力和支线人物的可信存在。

上册的主要读感应是现实主义校园群像：大事不多，但每件小事都能留下后果。军训手机袋、汇演、班会、主持招新、补考、实验分工、材料边界、生活费、网购包裹、晚风现实见面、留校申请和纸质签字，共同把宿舍四人从“刚认识”推到“彼此知道一点痛处，但还不能完全替对方说话”的位置。

上册的课程地点已经按时间固定：大一上从 A101 专业导论、B204 班会、C201 高数、B307 程序设计理论课和八号教学楼机房展开；第四卷的数据结构实验长期在 A302，是同一门课的连续压力，不代表所有课程都挤在同一间教室。后续开发拆节点时，应先查 `07_学期课程表与教室索引.md`，再决定一段剧情发生在哪条课表动线上。

## 分卷功能

第一卷《行李落在青枫居》负责把人和地方落地。它建立武生院站、青枫居 4XX、四张桌子、四个人的初始物件和入学第一晚的陌生感。这里的关键不是冲突爆发，而是让读者第一次看清周屿靠门、唐骁靠窗、陆沉靠阳台、林亦舟夹在门窗和公共桌之间。

第二卷《把手机交出去》负责把集体规则压到个人身上。军训、手机袋、汇演和照片授权让四个人第一次被同一套规则挤在一起。它也让晚风、夏知微、秦越、许棠等支线人物以真实媒介进入，不再只是功能人物。

第三卷《热闹把人带走》负责展示“公开表达”的代价。班会、主持招新、水房观察、匿名照片、融媒体推文、上机检查和主持队试岗，让周屿的表达欲、唐骁的规则感、陆沉的沉默边界和林亦舟的协调习惯互相撞上。第三卷的核心不是热闹本身，而是热闹怎样把人从宿舍桌边带出去，又把后果带回来。

第四卷《夏天没有停下来》负责把热闹降温到现实生活里。暑假、补考、生活费、父母错位消息、晚风光谷见面、数据结构实验、本人版本、紧急联系人和暑假去向签字，把上册所有“帮忙”和“替人说话”的问题推向更具体的生活层面。第四卷结束时，四个人没有彻底和解，也没有崩开，他们只是学会在一些表上签自己的名字。

## 宿舍四人状态

林亦舟的上册弧光是从“协调别人”走向“只替自己能确认的部分签字”。他仍然习惯把事压在自己手里，但第四卷后段通过实验材料、票务投诉、联系人顺序和返家截图，让他开始把家庭、宿舍和晚风分开放置。

周屿的上册弧光是从“用玩笑抢回场面”走向“知道有些场合不能抢”。主持队和东区商业街小舞台给了他被看见的机会，也让他被误写成进度问题的来源。到第四卷末，他仍然嘴快，但已经能把“送室友优先”说出口并照做。

唐骁的上册弧光是从“用规则替所有人省事”走向“规则只能约束自己能做的部分”。他最容易替别人改顺、改准、改得像样，也最早被迫承认代写、代签和代确认不是同一种帮忙。

陆沉的上册弧光是从“能不说就不说”走向“必要时用自己的名字留下记录”。图书馆勤工、留校申请、换班截图和签字表让他的沉默变得有边界。他不负责解释全组，也不再让别人替他确认。

## 支线状态

晚风线在上册完成了从游戏、微信、抖音到光谷现实见面的转换。她不是心理出口，也不是答案提供者；她和林亦舟的关系靠语音、截图、避雷链接、被骗善后和到家报平安慢慢变实。

父母线通过生活费、红包备注、包裹、袜子、药、联系人顺序和车票截图形成暗线。它还没有爆发成家庭大事件，但已经能支撑第五卷更集中的家庭压力。

夏知微、沈嘉禾、秦越、许棠都在上册获得了独立生活半径：摄影、共享文档、主持名单、融媒体材料和各自寝室空间。她们不抢宿舍四人主轴，但已经能为后续支线对话、回流段和关系温度提供素材。

许澈在上册只保留远影和轻松空气，不进入稳定关系。这保证第五卷唯一 5X 外流出现时，有预感，但不提前破坏结构。

## 上下册玩法区别建议

上册适合做“探索与积累”：大量心理级微选择、关系温度、生活费/材料/考试/活动小变量、支线人物反复露面和同质回流。冲突多为弱到中等，重点是让玩家相信这些人、这些地方和这些旧账真实存在。

下册适合做“收束与分流”：第五卷把前期多线压成几条更清晰的单线，允许强冲突和唯一 5X 硬外流；第六卷形成九个主预结局；第七卷《最后一个新年》做毕业活动、关系温度和遗憾回响。下册的选择不应简单增加数量，而应让上册积累的物件、签字、旧消息、活动名单和未说出口的话变成后果。
