# IF 第四幕 ACT4-A3 活动父池生活半径剧情页

更新时间：2026-05-23

## 用途

本文补齐 `A3-ACTIVITY-PUBLIC` 在第三幕外流后、第五幕承接前的第四幕生活半径。

本线不是从第三幕直接跳第六幕。林亦舟已经进入 `POOL-A3-ACTIVITY-PUBLIC`，第四幕必须在当前池内继续演化：活动任务、公开材料、宿舍缺席、候选来源和返校前压力都要落成可读事件。

## 硬规则

- 路线：`A3-ACTIVITY-PUBLIC`
- 当前池：`POOL-A3-ACTIVITY-PUBLIC`
- Act4 体量：14 个剧情页，5 个选择剧情点。
- 每个选择窗至少 3 项：1 个当前支线主轴推进 + 2 个池内微内流点。
- 三项都留在当前池，不开启完整新路线。
- 每个微内流点必须有触发、人物对话 / 动作、小抉择和回声；不能写成 1-2 页消息过场。
- 第四幕末尾必须交给第五幕，不得直接跳 Act6。

## 总链路

```text
P0A_ACT3_ROUTE_CONFIRM
-> ACT4-A3-B01 暑假活动群留名
-> ACT4-A3-B02 公开材料和照片授权

-> ACT4-A3-B04 候选来源核验
-> ACT4-A3-B05 返校前交接
-> ACT5-A3-B01
```
