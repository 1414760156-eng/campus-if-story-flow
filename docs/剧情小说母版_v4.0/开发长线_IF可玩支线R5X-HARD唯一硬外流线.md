# 开发长线_IF可玩支线R5X-HARD唯一硬外流线

## 用途

本文用于把 `R5X-HARD` 从“第五幕隐藏高难外流点”扩展为可拆 JSON、可写剧情页、可校验硬约束的 S 级独立命运池。

它不是普通软外流，也不是许澈奖励线。它规定的是：玩家在第五幕最难回头的位置，连续错过宿舍、亲密、项目、活动和回避窗口后，明确选择跟许澈或新圈离开旧多线，林亦舟的第六卷和第七卷如何只走 5X 单线。母本公共锁点仍发生，但旧宿舍、晚风、社团候选和站队组只能作为软照面、消息、同场流程或低温回声。

## 路线定位

路线 ID：`R5X-HARD`

路线池：`POOL-R5X-HARD`

入口时期：第五幕隐藏高难点。

入口节点建议：`P0E_5X_LAST_DOOR`

入口选择建议：`P0E_5X_FOLLOW_XUCHE`

二次确认选择建议：`P0E_5X_CONFIRM_NO_RETURN`

核心体验：玩家不再选择“站谁”“修补谁”“靠近谁”，而是让林亦舟把原有多线整体放下，进入许澈、韩野、蒋沐等人构成的新圈子。新圈轻松、没有旧账、不会追问他为什么晚回，也不会要求他解释 4XX 发生了什么；代价是这种轻松本来就薄，散得也快。5X 的痛点不是新人坏，而是新人没有义务替旧人承担重量。

本线难解锁，原因有三层：

1. 它必须发生在第五幕唯一硬外流点，不能由第三幕、第四幕或普通厌烦提前触发。
2. 它需要前置变量证明林亦舟不是偶尔想散心，而是长期不回头、不解释、不补入口。
3. 它一旦进入就设置 `route_lock = "5x"` 和 `hard_outflow = true`，后续不能再回原多线完整路线。

## 进入条件

### 基础硬锁条件

`R5X-HARD` 可以覆盖普通长线的第五卷最低 5 个主线选择窗口要求，但不能无证据触发。建议正式 JSON 判定时同时满足：

| 条件 | 建议值 | 说明 |
|---|---:|---|
| `current_act` | `act5` | 必须在第五幕隐藏点。 |
| `entry_node_id` | `P0E_5X_LAST_DOOR` | 只能在最后一扇门判定。 |
| `hard_outflow_candidate` | `>= 3` | 玩家多次选择离开现场、静音、不进门或跟新圈走。 |
| `old_dorm_distance` | `>= 4` | 旧宿舍距离已经真实拉开。 |
| `missed_chance` | `>= 2` | 至少两个可补入口已过期。 |
| `message_cut` | `>= 2` | 事项群、私聊或电话被切断过。 |
| `hard_outflow_warning_seen` | `true` | 玩家看过不可回退提示或剧情警告。 |
| `player_choice` | `P0E_5X_FOLLOW_XUCHE` | 明确跟许澈 / 新圈离开。 |
| `confirm_choice` | `P0E_5X_CONFIRM_NO_RETURN` | 二次确认不回头敲门。 |

### 可降低门槛但不可直接锁线的前置影子

以下内容只能提高 `hard_outflow_candidate` 或 `hard_outflow_shadow`，不能直接锁 5X：

- 第四卷便利店许澈远影、工牌、小票、冷柜、搬货。
- `R5-LIEFLAT` 中后街、台球厅、便利店的许澈软照面。
- A3 `mode_avoid` 的公开回避。
- R4 兼职线中的便利店、快递站、后街工作人员软照面。
- 晚风或感情线中的“想逃开一下”。

### 入口互斥

| 既有状态 | 处理 |
|---|---|
| `route_lock = "5x"` | 已锁 5X，不重复判定。 |
| `hard_outflow = true` | 第六卷、第七卷只走 `POOL-R5X-HARD`。 |
| `route_pool_id = POOL-R5-LIEFLAT` 且 `hard_outflow_shadow < 3` | 仍是摆烂线，不能临时跳 5X。 |
| `route_pool_id = POOL-R5-WANFENG` 且亲密投入持续有效 | 不允许把一次争执直接改成 5X。 |
| `route_pool_id = POOL-R5-ROMANCE` 且候选关系已锁定 | 只允许软照面新圈，不开 5X。 |
| `route_pool_id = POOL-A3-ACTIVITY-PUBLIC` | 只能形成回避影子，必须等 P0-E 硬锁。 |
| `route_pool_id = POOL-R4-WORK` | 工作场景不是 5X 入口，除非回到第五幕 P0-E。 |

## 不改变的母本锁点

- 第五幕站队爆发仍发生，只是林亦舟在最后一轮选择不回到旧多线。
- 第六卷关系定形仍发生，但旧宿舍、晚风、项目、活动和感情候选都退成同场回声。
- 第七卷毕业照、清寝、离校手续和武生院站仍发生，但林亦舟不再拥有“回原线完整修补”的按钮。
- 许澈、新圈和后街场景不能替代母本世界；它们只是林亦舟新的生活半径。

## 第五 / 第六卷硬约束接入

`R5X-HARD` 是唯一硬外流例外：

- 普通长线必须满足第五卷最低 5 个主线选择窗口；`R5X-HARD` 可在唯一硬外流点覆盖这个限制。
- 覆盖不等于轻易触发。必须有 `hard_outflow_candidate`、`old_dorm_distance`、`missed_chance`、`message_cut` 和二次确认。
- 进入后 `route_lock = "5x"`，`hard_outflow = true`，`active_route_id = "R5X-HARD"`。
- `closed_route_ids` 必须包含 `DEFAULT-4XX`、`R5-ZHOU`、`R5-TANG`、`R5-LUCHEN`、`R5-LIEFLAT`、`R5-WANFENG`、`R5-ROMANCE`、`R3-PERFECT`、`R4-WORK`。
- 第六卷固定 20 个剧情段、5 个四段组、5 个二方向选择窗口；每次选择只能在 `POOL-R5X-HARD` 内处理“留在新圈 / 看见旧人但不回旧线”等方向。

## 核心变量

### 局部变量

| 变量 | 含义 | 增长来源 | 主要后果 |
|---|---|---|---|
| `hard_outflow_candidate` | 硬外流候选强度 | 不进门、跟新圈走、静音离开 | 达标后允许 P0-E 二次确认 |
| `route_lock` | 不可回退路线锁 | 二次确认 | `route_lock = "5x"` 后关闭其它完整路线 |
| `hard_outflow` | 是否进入唯一硬外流 | 5X 锁定 | 第六 / 七卷只走 5X |
| `new_circle_trust` | 新圈熟悉度 | 一起吃饭、打球、游戏、帮忙补钱 | 新圈事件开放，但不等于深关系 |
| `thin_relation` | 薄关系显形 | 临时爽约、AA 不清、玩笑带过 | 高值触发 5X 遗憾 |
| `old_dorm_distance` | 旧宿舍距离 | 不回消息、不参与饭局、不进门 | 旧线只能软照面 |
| `message_cut` | 消息切断程度 | 群静音、电话不接、只读不回 | 第七卷低温和错过 |
| `money_cost` | 新圈现实成本 | 临时聚餐、台球、打车、垫钱 | 薄关系和现实压力显形 |
| `soft_passing` | 旧人擦肩次数 | 食堂、教学楼、毕业流程同场 | 只给低温回声，不回线 |
| `new_circle_truth` | 对新圈性质的认知 | 许澈说清、韩野散局、蒋沐改约 | 决定毕业时是否承认“这不是答案” |
| `identity_drift` | 林亦舟自我漂移 | 长期用新圈身份遮旧账 | 毕业结算偏空、偏遗憾 |
| `5x_regret` | 5X 遗憾强度 | 想回头但入口关闭 | 第七卷 5X 遗憾结局 |
| `graduation_temperature` | 毕业温度 | 旧人擦肩、新圈散场、是否发消息 | 结算文本温度 |

### 共享回声变量

| 变量 | 来源 | 在 5X 中的作用 |
|---|---|---|
| `dorm_trust` | 4XX 旧互信 | 只用于判断旧人擦肩时的语气 |
| `old_debt` | 宿舍旧账 | 不再修补，只影响遗憾强度 |
| `wanfeng_link` | 晚风亲密回声 | 只能作为朋友圈、消息或同场擦肩 |
| `project_stability` | 项目 / C407 | 只影响公共流程是否仍能最低完成 |
| `activity_link` | 活动圈关系 | 可带来新圈场景，但不开活动完整线 |

## 体量与密度

- 等级：S 级主轴等量线。
- 建议体量：10 章，后续正文容量约 80,000-120,000 字。
- 有效选择下限：最低 42 次，建议 45-50 次。
- 每章至少 3-5 个可拆选择，且至少 1 个现实任务和 1 个关系压力。
- 内回流点：至少 8 个，本稿配置 9 个。
- 第六卷承接：固定 20 个剧情段、5 个四段组、5 个二方向选择窗口。

## 章节开发详表

### 第一章：最后一扇门

章节目标：在第五幕强冲突之后，让玩家清楚知道这不是普通“出去冷静一下”。林亦舟站在最后一扇门前，旧宿舍仍有可敲门的机会，但玩家也能选择下楼、静音群消息、跟许澈或新圈离开。真正锁 5X 前必须给二次确认。

核心事件：

- 4XX 门内还有声音，事项群出现最后一条可回消息。
- 许澈在楼下或便利店门口发来短句，不追问原因。
- 玩家第一次选择是否离开。
- 系统提示或剧情提示：跟新圈走会关闭旧多线完整回路。
- 玩家二次确认是否“不回头敲门”。

可拆节点：

| 节点 ID | 场景 | 人物 | 功能 |
|---|---|---|---|
| `R5X-CH01-N01` | 4XX 门外 | 林亦舟、4XX 室友远声 | 最后一扇门 |
| `R5X-CH01-N02` | 楼梯间 | 林亦舟 | 群消息静音 |
| `R5X-CH01-N03` | 楼下 | 林亦舟、许澈 | 新圈邀请 |
| `R5X-CH01-N04` | 便利店门口 | 林亦舟、许澈 | 二次确认 |

选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `R5X-CH01-C01` | 回头敲门 | `hard_outflow_candidate -1`、`old_dorm_distance -1` | 回普通旧池，不锁 5X |
| `R5X-CH01-C02` | 下楼冷静但不跟走 | `hard_outflow_candidate +1`、`message_cut +1` | 仍可回 `R5-LIEFLAT` |
| `R5X-CH01-C03` | 跟许澈去便利店 | `hard_outflow_candidate +2`、`old_dorm_distance +1` | 进入二次确认 |
| `R5X-CH01-C04` | 确认今晚不回头 | `route_lock = "5x"`、`hard_outflow = true`、`old_dorm_distance +2` | 锁 `POOL-R5X-HARD` |
| `R5X-CH01-C05` | 看完提示后取消 | `5x_regret +1`、`message_cut +1` | 退回低温旧池 |

### 第二章：新桌子不用解释

章节目标：写出新圈吸引力。许澈、韩野、蒋沐在后街小馆或便利店外桌吃东西，他们不问 4XX，不问晚风，不问谁对谁错。玩家第一次体会到“没有旧账”的轻。

核心事件：

- 新桌子上的话题都短、快、无责任。
- 许澈替林亦舟拿饮料，但不追问他为什么来。
- 韩野约台球，蒋沐改约游戏局。
- 旧宿舍群消息在手机上亮起，被玩家选择处理或跳过。

选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `R5X-CH02-C01` | 接许澈递来的饮料，不解释 | `new_circle_trust +1`、`message_cut +1` | 新圈熟悉度提升 |
| `R5X-CH02-C02` | 主动讲一点 4XX | `new_circle_truth +1`、`thin_relation +1` | 新圈有人岔开话题 |
| `R5X-CH02-C03` | 把旧群消息划掉 | `old_dorm_distance +1`、`5x_regret +1` | 第五章旧人擦肩加重 |
| `R5X-CH02-C04` | 回一句“晚点说” | `message_cut -1`、`soft_passing +1` | 只保留低温回声 |

### 第三章：第一笔没算清的钱

章节目标：让新圈薄关系第一次显形。轻松不是坏，但它没有深责任。台球费、夜宵钱、外卖拼单或临时打车让玩家看到：这里没有旧账，也没有谁必须替谁兜底。

核心事件：

- 韩野说“先你垫一下”，蒋沐临时改约。
- 许澈说可以 AA，但语气像习惯了散局。
- 林亦舟发现旧宿舍的钱账麻烦，新圈的钱账也不自动清楚。
- 玩家选择要不要问清。

选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `R5X-CH03-C01` | 直接垫钱 | `money_cost +2`、`new_circle_trust +1`、`thin_relation +1` | 后续被默认好说话 |
| `R5X-CH03-C02` | 当场 AA 说清 | `new_circle_truth +1`、`thin_relation -1` | 许澈认可边界 |
| `R5X-CH03-C03` | 不问，装没事 | `thin_relation +2`、`5x_regret +1` | 第八章散局更重 |
| `R5X-CH03-C04` | 想起 4XX 旧账但不发消息 | `old_dorm_distance +1`、`identity_drift +1` | 旧线低温加深 |

### 第四章：不用回群的人

章节目标：把 5X 的“自由”变成具体日常。林亦舟不再需要在 4XX 事项群里解释晚归、材料、外卖和饭局，但这也意味着别人开始不再等他。

核心事件：

- 事项群里出现一个本来需要林亦舟确认的表格。
- 新圈游戏局开麦，许澈问他来不来。
- 唐骁或周屿在群里短句收尾，没有追问。
- 玩家决定是否切掉旧群。

选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `R5X-CH04-C01` | 继续开麦，不回旧群 | `message_cut +2`、`new_circle_trust +1` | 旧人不再默认等他 |
| `R5X-CH04-C02` | 给旧群发最低确认 | `message_cut -1`、`soft_passing +1` | 只留流程回声 |
| `R5X-CH04-C03` | 私聊一个旧人但不解释 | `old_dorm_distance +1`、`5x_regret +1` | 第五章短句尴尬 |
| `R5X-CH04-C04` | 把旧群免打扰置顶下移 | `identity_drift +1`、`message_cut +1` | 第十章置顶结算 |

### 第五章：旧人同场

章节目标：旧人不是消失，而是近到能看见、远到不能再进入完整线。食堂、教学楼、活动现场或 C407 外，林亦舟会遇到周屿、唐骁、陆沉、晚风或 A3 候选，但只给软照面。

核心事件：

- 周屿在活动现场和别人开玩笑，看到林亦舟停半拍。
- 唐骁把材料交给老师，不再让林亦舟确认完整流程。
- 陆沉点头但不替他说话。
- 晚风或候选对象只以消息、朋友圈或同场出现。

选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `R5X-CH05-C01` | 点头就走 | `soft_passing +1`、`old_dorm_distance +1` | 旧线保持体面低温 |
| `R5X-CH05-C02` | 发一句“最近还好吗” | `5x_regret +1`、`message_cut -1` | 只回短句，不回线 |
| `R5X-CH05-C03` | 借口新约离开 | `new_circle_trust +1`、`old_dorm_distance +1` | 新圈优先 |
| `R5X-CH05-C04` | 站在原地听完旧人说话 | `soft_passing +2`、`identity_drift -1` | 第七章旧流程更重 |

### 第六章：许澈不是答案

章节目标：把许澈从“新关系核心”写回“没有旧账的轻关系”。他可以好聊，可以帮林亦舟从旧关系中短暂脱身，但他不是拯救者，也不会替林亦舟解释人生。

核心事件：

- 许澈下班后换掉工牌，像把一段关系也顺手收起来。
- 林亦舟试探性讲旧宿舍，许澈听，但不接深承诺。
- 韩野约下一局，蒋沐临时拉别人。
- 玩家选择是否把许澈当答案。

选择：

| 选择 ID | 玩家动作 | 变量变化 | 后续回调 |
|---|---|---|---|
| `R5X-CH06-C01` | 对许澈讲清旧事 | `new_circle_truth +2`、`5x_regret +1` | 许澈听见但不承诺 |

## 对话逻辑

### 林亦舟

- 早期少解释，像终于不用承担旧账。
- 中期开始发现“不被追问”也可能是“不被真正接住”。
- 后期说话要更短、更清醒，不忏悔式自虐，也不把自己写成赢家。

### 许澈

- 轻松、好聊、动作快，能给林亦舟一个无旧账入口。
- 不承诺深责任，不替林亦舟解释旧关系。
- 他的好处是真实的，但有限；他的有限也不是恶意。

### 韩野 / 蒋沐

- 负责新圈的热闹、临时性和散局感。
- 可以提供台球、游戏、夜宵、外卖、短行程。
- 不能写成稳定新宿舍。

### 旧线人物

- 周屿：看见林亦舟时可以停半拍，但不追上来。
- 唐骁：把流程做完，不再要求林亦舟确认每一步。
- 陆沉：点头、让开、短句，不替他讲旧事。
- 晚风：可以有消息、朋友圈、远处同场，但不重开亲密线。
