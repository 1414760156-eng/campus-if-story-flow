# IF 第五幕 ACT5-WORK 工作线桥接剧情页

更新时间：2026-05-26

## 用途

本文补齐 `R4-WORK` 的第五幕桥接层，把第四幕暑假 / 兼职入口交给第六幕工作线定形层。

本桥接不是把工作线写成苦情奖励，也不是把陆沉、晚风、默认宿舍线重新打开。它让工作线按母版第五幕五类压力完成一次本池版本的锁池：宿舍站队、亲密 / 晚风、项目 / 规则、活动 / 兴趣、回避 / 外流都必须出现，但都从钱、排班、证明、错过照面和第一周现实安排处理。

## 硬规则

- 路线：`R4-WORK`
- 当前池：`POOL-R4-WORK`
- Act5 桥接骨架：5 个桥接节点，20 个选择前剧情页，5 个宏观选择窗。
- 5 个宏观选择窗只代表五类主压力已经覆盖，不代表整幕只有 5 个可玩选择点；正式黄金样板需要继续补方向反馈链、选择后承接剧情和必要微心态。
- 每个选择窗 3 项，全部留在当前池。
- 每个选择窗对应第五幕最低主线选择窗口之一。
- 末尾接 `ACT6-WORK-B01`。

### ACT5-WORK-B01 黄金样板正文

本节是 `ACT5-WORK-B01` 的修订样板。核心承接不是“新道具出现”，而是第四幕 L06 已经由林亦舟本人签下兼职 / 留校说明，第五幕开始让他第一次承担这份签字带来的缺席后果。

玩家页只呈现现场剧情、可点击动作和选择后反馈；变量、路线池、四步走校验和回声钩子只放在写回字段建议中。

#### `ACT5-WORK-S01` 青枫居 4XX / 晚班后补说明

签完暑假去向登记以后，4XX 安静了几天。

透明文件夹被塞回书架，黑色签字笔滚到公共桌边角，没人再提 B204 那一笔本人签字。林亦舟也以为，兼职 / 留校说明暂时只是系统里多出来的一行字。

周五下午，这一行字先从快递站群里亮起来。

负责人发了晚班试岗名单，林亦舟的名字排在第三个，后面写着：19:20 前到岗，带身份证复印件。

他还没来得及回，唐骁推门进来，把 C407 分工表放到桌上：“今晚把现场分工过一遍。”

周屿拎着水杯从阳台回来，扫了一眼表格：“今晚？林亦舟不是也在吗？”

唐骁没有马上接。他看见林亦舟手机屏幕上的快递站名单，笔尖停在分工表最后一栏。

那一栏还空着。

#### `ACT5-WORK-S02` 快递站 / 排班表

林亦舟下楼去快递站确认。

负责人把纸质排班表夹进透明板，指了指他的名字：“你留校说明过了，今晚先试第一班。”

工资不高，但能按周结一部分，够补掉打印费和接下来几天饭钱。负责人说得很平，像在讲普通规则：“证件复印件带齐。真有事提前讲，别到了点才不来。”

林亦舟点头。

手机在这时亮了一下。

唐骁：今晚 C407 要把分工表定下来。

后面跟着一句：你那栏我先空着。

林亦舟握着笔，笔尖在排班表上停了两秒。

负责人把复印件要求写到他名字后面，又把排班表夹回透明板里。

林亦舟把学生证复印件收进包侧，手机又震了一下。唐骁没有催第二句，只发来一张照片：C407 分工表最后一栏空着，纸角被压在一只装线材的纸箱下面。

他站在快递站门口看完那张照片。楼梯间的风从上面灌下来，带着一点纸箱和旧设备的灰味。

#### `ACT5-WORK-S03` C407 门口 / 不问理由

林亦舟没有先回 4XX。

他按唐骁发来的教室号走到 C407，门口果然堆着两只纸箱。照片里那只压住分工表纸角的箱子被挪开了一半，里面是演示要用的线材和旧设备。唐骁蹲在地上核对清单，周屿正把一卷线绕回原来的塑料圈。

唐骁没有问他为什么接晚班，也没有问他是不是家里又出了什么事。他只把分工表翻到最后一页：“你今晚到不到？”

林亦舟看着那一栏空白，想起快递站负责人刚在他名字后面写下的证件要求。第一班还没开始，C407 已经先空出一个位置。

周屿从教室里探头：“要不我们先把你那栏空着？”

唐骁没回头：“空着也要写原因。”

林亦舟低头看手机。快递站群里，负责人又发了一遍到岗名单；C407 群里，唐骁把新版分工表传了上去；4XX 群还没有人说话。

三条消息压在一起，没有一句像问题。

林亦舟把手机收回口袋，离开 C407 时，纸箱边角蹭到他的裤脚。楼道尽头能看见青枫居那边的灯，六点五十快到了。

#### `ACT5-WORK-S04` 4XX 门口 / 今晚先说哪一句

傍晚六点五十，林亦舟回到 4XX 门口，门半开着。

周屿在收外卖袋，唐骁把分工表摊在膝盖上，陆沉正在给自己的文件袋贴姓名。没人催林亦舟进来，也没人替他把那一栏填上。

林亦舟站在门口，背包带勒在肩上。

他今晚确实有班。钱也确实紧。父亲的转账备注、补考后的打印费、母亲一直没接通的电话都在背后，可它们不可能在这几分钟里被讲完整。

六点五十三，唐骁手机震了一下；六点五十四，快递站群又刷出“新同学提前十分钟到”；六点五十五，母亲问他今天怎么没接电话。

林亦舟的手还搭在门把上，屋里的灯光落到他鞋尖。门里是 4XX，楼下是快递站，手机里还亮着 C407 那一栏空白。

#### `ACT5-WORK-CHOICE-01`

抉择引导文：4XX 的门半开着，C407 的分工表还空着一栏，快递站七点半点名。林亦舟已经亲手把兼职 / 留校说明签进系统，今晚第一次要把这份签字带回真实的人面前。

### 方向 A：先在群里写清晚班时间和 C407 能交的部分

#### `ACT5-WORK-C01-A-F01` 走廊 / 群消息

林亦舟没有先进门。

他站在走廊尽头，把背包换到另一边肩上，在 4XX 群里打字：今晚七点半到十点快递站固定班，我十点半前回。C407 我能交文档和说明页，现场线材今晚不一定到。

发送前，他删掉了“抱歉”两个字。

不是不该道歉，是这两个字太轻，轻得像能替他把签字后的第一场缺席盖过去。

消息发出去后，群里安静了十几秒。

周屿先回：收到。你别把自己寄出去就行。

唐骁回得短：文档今晚十一点前要有。

陆沉最后发了一句：门卡带了没。

三句话都不热。可那一栏不再只是空着。

微心态抉择：这条群消息怎么补完整？

| 微选项 | 玩家侧动作 | 点击后反馈 |
|---|---|---|
| A1 | 再补一句：我能交的文档今晚发，线材明早补。 | 唐骁把他那栏改成“文档先到”，没有再追问理由。 |
| A2 | 再补一句：十点半前回寝，回来补签说明。 | 陆沉回“别过点”，晚归和签字变成今晚的硬边界。 |
| A3 | 再补一句：别替我交，缺的部分我自己补。 | 周屿停了玩笑，回了一个“行”，责任没有再被别人顺手接走。 |

#### `ACT5-WORK-C01-A-F02` 快递站扫码台 / 第一次固定班

七点半，负责人点名。

林亦舟把学生证复印件递过去，负责人核对完，在他名字后面画了一个勾。扫码枪的提示音一下一下响，快件从传送带上滑过来，他得低头看编号，再按楼栋分进不同筐里。

手机在口袋里震了几次，他没有马上看。

到九点五十，周屿发来一句：你那栏没空着，等你回来签。

林亦舟把手机扣回去，继续分最后一车件。

扫码枪响了一下，他把下一件快递分进筐里。那条路是自己签下的，今晚终于有人知道它通向哪里。

### 方向 B：先去问能不能请半小时，再回 4XX 当面说

#### `ACT5-WORK-C01-B-F01` 快递站柜台 / 请半小时

林亦舟先去快递站。

他没有绕开 4XX，只是知道自己如果要当面说，就得先问清这半小时从哪里来。

负责人正在把晚班表夹进透明板，听他说完，只问了一句：“第一班就请？”

这句话没有责备，但足够让后背僵一下。

林亦舟把学生证复印件递过去：“我七点半能到岗，八点到八点半想回寝处理一张说明表。后面半小时我补到十点半。”

负责人看了他一眼，把表推回来：“写清楚。临时换班提前说，补时也要签到。”

林亦舟在备注栏写下：20:00-20:30 回寝补晚班说明，22:00-22:30 补时。

笔画落下去，半小时不再是“回去解释一下”，而是被写进了工作规则里。要当面说清，他先付出了第一笔现实成本。

微心态抉择：请假时先把哪句话说实？

| 微选项 | 玩家侧动作 | 点击后反馈 |
|---|---|---|
| B1 | 说清：我七点半先到岗，八点回寝半小时，十点后补回来。 | 负责人把补时写进备注，工作入口变稳，但今晚会拖到更晚。 |
| B2 | 说清：4XX 那边要我本人确认，不能让室友替我写。 | “本人确认”被快递站也听见了，规则不再只属于学校表格。 |
| B3 | 说清：这次只请半小时，后面固定班照常。 | 负责人没有多问，只提醒他别把第一次例外变成习惯。 |

#### `ACT5-WORK-C01-B-F02` 青枫居 4XX / 当面说

八点零六，林亦舟推开 4XX 的门。

屋里比他想的安静。外卖盒已经收走，公共桌上只剩分工表、晚班后补说明和一只黑色签字笔。周屿抬头看他，像想开玩笑，又把话咽回去。

唐骁先问：“请到假了？”

林亦舟点头：“半小时。十点以后补。”

他把快递站备注拍给他们看，又把晚班后补说明拉到自己面前：“今晚我不在 C407，不是躲。固定班能按周结一部分，补考打印和接下来几天饭钱要先顶上。”

说到“饭钱”时，他停了一下。

这不是最体面的说法，却把第四幕那些没说完的压力接了起来：签字、联系人、补考、兼职，都不是独立事件。

陆沉把黑笔递给他：“那就写清楚。别让我们猜。”

微心态抉择：钱压说到哪一步？

| 微选项 | 玩家侧动作 | 点击后反馈 |
|---|---|---|
| B1 | 只说：生活费没完全到账，我先接固定班。 | 4XX 知道钱压存在，但不知道家里细节，关系修补留在低温位置。 |
| B2 | 说明：补考材料、打印和饭钱这周都要现金。 | 压力被说具体，周屿没再玩笑，唐骁把分工表往他那边推近一点。 |
| B3 | 补一句：我不想让你们替我解释。 | 陆沉看了他一眼，把“本人确认”四个字圈出来，没人再替他代签。 |

#### `ACT5-WORK-C01-B-F03` 走廊 / 补回晚班

林亦舟只坐了二十多分钟。

分工表写完能交的部分，晚班后补说明也签了本人确认。他起身时，周屿把剩下半瓶水推到桌边：“别跑太急，水拿走。”

林亦舟没有说谢谢。

他说：“我十点半前回来。”

楼道灯亮得很白，快递站的扫码声像已经从楼下传上来。请半小时没有让钱压消失，反而让它变得更具体：他今天多走了一趟，也多欠了半小时。

但这一次，4XX 至少知道他往哪张表里去了。

### 方向 C：先进门说今晚有晚班，家里的事先不展开

#### `ACT5-WORK-C01-C-F01` 4XX 门口 / 只说今晚有班

林亦舟先进了门。

他没有把父亲转账、母亲消息和补考打印费一起摊开，只把背包放到椅背上，说：“今晚我有快递站晚班，七点半走，十点多回。”

屋里停了一下。

周屿先反应过来：“那 C407 呢？”

“我能交文档。”林亦舟说，“线材和现场调试今晚不一定到。”

唐骁把分工表翻到他那栏，写下“文档先交”。笔尖划过纸面的时候很响，像把没有说完的部分也一并划开。

陆沉问：“家里知道你接班了吗？”

林亦舟没有马上答。

他说：“还没说到那一步。”
