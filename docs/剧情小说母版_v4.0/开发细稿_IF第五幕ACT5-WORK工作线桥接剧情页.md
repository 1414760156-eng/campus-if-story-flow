# IF 第五幕 ACT5-WORK 工作线桥接剧情页

更新时间：2026-05-26

## 用途

本文补齐 `R4-WORK` 的第五幕桥接层，把第四幕暑假 / 兼职入口交给第六幕工作线定形层。

本桥接不是把工作线写成苦情奖励，也不是把陆沉、晚风、默认宿舍线重新打开。它让工作线按母版第五幕五类压力完成一次本池版本的锁池：宿舍站队、亲密 / 晚风、项目 / 规则、活动 / 兴趣、回避 / 外流都必须出现，但都从钱、排班、证明、错过照面和第一周现实安排处理。

## 硬规则

- 路线：`R4-WORK`
- 当前池：`POOL-R4-WORK`
- Act5 桥接骨架：5 个桥接节点，20 个选择前剧情页，5 个宏观选择窗。
- 5 个宏观选择窗只代表五类主压力已经覆盖，不代表整幕只有 5 个可玩选择点；正式黄金样板需要继续补方向反馈链、选择后承接剧情和必要微心态。
- 每个选择窗 3 项，全部留在当前池。
- 每个选择窗对应第五幕最低主线选择窗口之一。
- 末尾接 `ACT6-WORK-B01`。

## 总链路

```text
P0B_ACT4_ROUTE_CONFIRM

```

### ACT5-WORK-B01 黄金样板正文

本节是 `ACT5-WORK-B01` 的正式扩写样板。玩家页只呈现现场剧情、可点击动作和选择后反馈；变量、路线池、回声钩子只放在写回字段建议中。

#### `ACT5-WORK-S01` 青枫居 4XX / 晚班后补说明

开学后第一个周五，4XX 的公共桌又被表格占满。

C407 分工表压在左边，补考安排和证书材料夹在中间，右边多了一张辅导员刚发的晚班后补说明。表头很短，只有“缺席原因、补交时间、本人确认”三栏，林亦舟的名字被唐骁用铅笔圈出来。

周屿拎着水杯从阳台回来，看见那个圈，笑意没完全起来：“你今晚又不在？”

“不是又。”唐骁把铅笔放下，“是第一次正式缺。”

陆沉没有接话。他把自己的学生证复印件抽出来，放到透明文件夹最上面。林亦舟看见那张复印件，才想起快递站负责人下午也提醒过，固定班第一次到岗要带学生证和身份证复印件。

公共桌没有责备谁。它只是把所有需要本人出现的东西摊开，让林亦舟看见：今晚他不在，不会只少一个人。

#### `ACT5-WORK-S02` 快递站 / 排班表

快递站门口的卷帘门拉到一半，负责人把新表贴在扫码台旁边。

周五晚班从七点半到十点，试岗三次后固定。工资不高，但能按周结一部分，够补掉打印费和接下来几天的饭钱。负责人说得很平，像在讲一条普通规则：“固定班最怕临时空。真有事提前讲，别到了点才不来。”

林亦舟把自己的名字写上去时，手机亮了一下。

唐骁：今晚 C407 要把分工表定下来。

后面跟着一张照片。照片里，公共桌那张晚班后补说明已经被摊开，空着的位置正好在他名字旁边。

林亦舟握着笔，笔尖在排班表上停了两秒。第四幕里那些“先问清楚”“本人签字”“带回公共桌”的事，到这里不再是准备动作。它们开始向他要今晚的时间。

#### `ACT5-WORK-S03` C407 门口 / 不问理由

C407 门口堆着两只纸箱，里面是演示要用的线材和旧设备。

唐骁蹲在地上核对清单，没有问林亦舟为什么缺钱，也没有问他是不是又被家里什么事绊住。他只把分工表翻到最后一页：“今晚能不能到场？”

这个问题比追问更难答。

如果说能，他就要从快递站请假，第一班还没站稳就先开口。如果说不能，C407 的分工表会先按他缺席排，后面每一个材料口径都要别人替他留位置。

周屿从教室里探头：“要不我们先把你那栏空着？”

唐骁没回头：“空着也要写原因。”

林亦舟低头看手机。快递站排班、4XX 补说明、C407 分工表，三张图在聊天框里连成一排。它们不像三件事，更像同一张表被拆成了三份。

#### `ACT5-WORK-S04` 4XX 门口 / 今晚先说哪一句

傍晚六点五十，4XX 的门半开着。

公共桌上的晚班后补说明还在。周屿把外卖袋往旁边挪，给那张纸留出地方；陆沉正在把材料夹进文件袋，夹到林亦舟那一页时停了一下；唐骁坐在椅子上，分工表摊在膝盖上。

林亦舟站在门口，背包带勒在肩上。

他今晚确实有班。钱也确实紧。家里的消息、父亲的转账、补考后的打印费都在背后，但它们不可能在这几分钟里被讲完整。

快递站七点半点名，C407 等一个明确答复，4XX 也不能继续用“晚点再说”填空。

他要先决定，今晚缺席这件事，第一句话落在哪里。

#### `ACT5-WORK-CHOICE-01`

抉择标题：今晚缺席，先把哪句话落下？

抉择引导文：快递站七点半点名，C407 的分工表要当面定，4XX 的公共桌还等着晚班后补说明。林亦舟站在门口，知道今晚不可能同时把钱、排班和家里的事说完整。

| 方向 | 玩家侧选项 | 玩家侧说明 |
|---|---|---|
| A | 先把班次和回寝时间发到群里 | 人可以不在场，但今晚几点上班、几点回来、材料放在哪，先让 4XX 看见。 |
| B | 先去问能不能请半小时，回寝当面说 | 第一班还没站稳就开口请假，会多一笔现实成本，但至少这次不是让公共桌替你解释。 |
| C | 先进门说今晚有班，家里的事先不展开 | 先承认今晚缺席，钱和家里的部分暂时不摊开，只把能说清的一句说出来。 |

### 方向 A：先把班次和回寝时间发到群里

#### `ACT5-WORK-C01-A-F01` 快递站后门 / 群消息

林亦舟没有先进门。

他站在走廊尽头，把背包换到另一边肩上，在 4XX 群里打字：今晚七点半到十点快递站固定班，我十点半前回。C407 那份分工表我能做的部分先按文档写，纸质材料放公共桌右上角。

发送前，他删掉了“抱歉”两个字。

不是不该道歉，是这两个字太轻，轻得像能替他把晚班、钱和缺席都盖过去。

消息发出去后，群里安静了十几秒。

周屿先回：收到，你别把自己寄出去就行。

唐骁回得短：文档今晚十一点前要有。

陆沉最后发了一句：门卡带了没。

三句话都不热，但至少没有人再等一个空白位置替他说话。

微心态抉择：这条群消息怎么补完整？

| 微选项 | 玩家侧动作 | 点击后反馈 |
|---|---|---|
| A1 | 再补一句：材料我放公共桌右上角，别替我交。 | 唐骁回了一个“知道”，分工表不再空着，但责任也明确落回林亦舟自己身上。 |
| A2 | 再补一句：十点半前回寝，回来补签晚班说明。 | 陆沉发来“别过点”，门卡和晚归登记成了今晚的硬边界。 |
| A3 | 再补一句：今晚我只能先交文档，线材明早补。 | C407 的缺口被写清楚，唐骁没追问理由，只把他那栏改成“文档先到”。 |

#### `ACT5-WORK-C01-A-F02` 快递站扫码台 / 第一次固定班

七点半，负责人点名。

林亦舟把学生证复印件递过去，负责人核对完，在他名字后面画了一个勾。扫码枪的提示音一下一下响，快件从传送带上滑过来，他得低头看编号，再按楼栋分进不同筐里。

手机在口袋里震了几次，他没有马上看。

到九点五十，周屿发来公共桌照片。照片里，他的文件袋被放在右上角，旁边压着一张便签：回来签。字迹是陆沉的，旁边还有唐骁补的一行：文档别拖到明早。

林亦舟把手机扣回去，继续分最后一车件。

这个方向没有让宿舍变暖。它只是让 4XX 先知道：他不是消失，他是真的被排进了另一张表里。

### 方向 B：先去问能不能请半小时，回寝当面说

#### `ACT5-WORK-C01-B-F01` 快递站柜台 / 请半小时

林亦舟先去了快递站。

负责人正在把晚班表夹进透明板，听他说完，只问了一句：“第一班就请？”

这句话没有责备，但足够让后背僵一下。

林亦舟把学生证复印件递过去：“我七点半能到岗，八点到八点半想回寝处理一张说明表。后面半小时我补到十点半。”

负责人看了他一眼，把表推回来：“写清楚。临时换班提前说，补时也要签到。”

林亦舟在备注栏写下：20:00-20:30 回寝补晚班说明，22:00-22:30 补时。

笔画落下去，半小时不再是“回去解释一下”，而是被写进了工作规则里。

微心态抉择：请假时先把哪句话说实？

| 微选项 | 玩家侧动作 | 点击后反馈 |
|---|---|---|
| B1 | 说清：我七点半先到岗，八点回寝半小时，十点后补回来。 | 负责人把补时写进备注，工作入口变稳，但今晚会拖到更晚。 |
| B2 | 说清：4XX 那边要我本人确认，不能让室友替我写。 | “本人确认”被快递站也听见了，规则不再只属于学校表格。 |
| B3 | 说清：这次只请半小时，后面固定班照常。 | 负责人没有多问，只提醒他别把第一次例外变成习惯。 |

#### `ACT5-WORK-C01-B-F02` 青枫居 4XX / 当面说

八点零六，林亦舟推开 4XX 的门。

屋里比他想的安静。外卖盒已经收走，公共桌上只剩分工表、晚班后补说明和一只黑色签字笔。周屿抬头看他，像想开玩笑，又把话咽回去。

唐骁先问：“请到假了？”

林亦舟点头：“半小时。十点以后补。”

他把快递站备注拍给他们看，又把晚班后补说明拉到自己面前：“今晚我不在 C407，不是躲。固定班能按周结一部分，补考打印和接下来几天饭钱要先顶上。”

说到“饭钱”时，他停了一下。

这不是最体面的说法，却是最能落地的一句。

陆沉把黑笔递给他：“那就写清楚。别让我们猜。”

微心态抉择：钱压说到哪一步？

| 微选项 | 玩家侧动作 | 点击后反馈 |
|---|---|---|
| B1 | 只说：生活费没完全到账，我先接固定班。 | 4XX 知道钱压存在，但不知道家里细节，关系修补留在低温位置。 |
| B2 | 说明：补考材料、打印和饭钱这周都要现金。 | 压力被说具体，周屿没再玩笑，唐骁把分工表往他那边推近一点。 |
| B3 | 补一句：我不想让你们替我解释。 | 陆沉看了他一眼，把“本人确认”四个字圈出来，没人再替他代签。 |

#### `ACT5-WORK-C01-B-F03` 走廊 / 补回晚班

林亦舟只坐了二十多分钟。

分工表写完能交的部分，晚班后补说明也签了本人确认。他起身时，周屿把剩下半瓶水推到桌边：“别跑太急，水拿走。”

林亦舟没有说谢谢。

他说：“我十点半前回来。”

楼道灯亮得很白，快递站的扫码声像已经从楼下传上来。请半小时没有让钱压消失，反而让它变得更具体：他今天多走了一趟，也多欠了半小时。

但这一次，4XX 至少知道他往哪张表里去了。

### 方向 C：先进门说今晚有班，家里的事先不展开

#### `ACT5-WORK-C01-C-F01` 4XX 门口 / 只说今晚有班

林亦舟先进了门。

他没有把父亲转账、母亲消息和补考打印费一起摊开，只把背包放到椅背上，说：“今晚我有快递站晚班，七点半走，十点多回。”

屋里停了一下。

周屿先反应过来：“那 C407 呢？”

“我能交文档。”林亦舟说，“线材和现场调试今晚不一定到。”

唐骁把分工表翻到他那栏，写下“文档先交”。笔尖划过纸面的时候很响，像把没有说完的部分也一并划开。

陆沉问：“家里知道你接班了吗？”

林亦舟没有马上答。

他说：“先这样。”

这句话把今晚能说的部分落下了，也把不能说的部分留在了屋里。

微心态抉择：只说今晚有班时，怎么留下一点可见位置？

| 微选项 | 玩家侧动作 | 点击后反馈 |
|---|---|---|
| C1 | 把 C407 文件袋放到公共桌上，说：我能交的都在这里。 | 唐骁接过文件袋，分工表不再全空，但缺席原因仍然没人知道。 |
| C2 | 给唐骁发两项今晚能交的文档，不解释钱。 | 项目部分有了交付，宿舍误读没有解除，只是被压到下一次。 |
| C3 | 对陆沉说：十点后我会回来签晚班说明。 | 陆沉点头，晚归和签字有了落点，家里的问题仍然没被打开。 |

#### `ACT5-WORK-C01-C-F02` 快递站后门 / 没说出的部分

七点半后，林亦舟站在扫码台旁边，手机屏幕一直亮。

母亲问：今天怎么没接电话？

父亲的转账备注还停在上一条，只有“先用”。晚风也发来一句：你今晚是不是又很晚？

他都没有点开。

不是因为不重要。是因为刚才在 4XX 门口，他已经把今晚能承认的部分用完了。再多说一句，就会把钱、家里、补考、晚风和工作全都拉进同一条解释里，而他还没有办法把它们排好。

十点过后，周屿在群里问：你是真有班，不是躲我们吧？

林亦舟看着那句话，扫码枪在旁边响了一声。

他回：不是躲。今晚先把班站完，回去签说明。

这句回复不够完整，但比沉默多了一点位置。
