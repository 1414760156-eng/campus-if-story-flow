## 验收规则

- 5A 主轴允许短回流，但每个回流必须明确 `回流节点`。
- 每个回流源点至少 2 种回流方式，`ACT5_5A01_CORRIDOR_RETURN` 至少 3 种。
- 每条回流路线至少 3 页：离开现场、发生具体人物/消息/物件讨论、带着变化回主轴。
- 非 5X 回流段不得出现无名互动人物；外部软回流人物必须是前文已出现的命名人物。
- 当前异质软回流人物使用晚风、夏知微、沈嘉禾、秦越。边缘人物用于补充强关系之外的世界厚度，不改变第五幕强关系收束中心。
- 异质软回流必须设置 `禁止解锁分线: true`，不得解锁 5B-5F 或 5X。
- `同质收束` 是 5A 内部小结局，不等同于 `5X_离场外流线`。
- 5A 只能有一个硬外流点：`ACT5_5A15_DOOR_SLAM_RISK`。
- 5X 外流线唯一新人命名为许澈；后续可写许澈外流单线，但不得反向开放回流抉择。
- 心理级微选择只改变状态、人物关系和心理暗线，不新增分线，不新增外流点。
- v3.4 不新增节点和选项数量，重点验收玩家可见文本：选项、反馈和输出页不得出现系统调试口吻，输出页必须改成场景内动作和人物反应。
- v3.7 在 1-7 全局跑线审计后执行 `chinese-novelist` 小说化重排：入口同屏压力必须包含生活费/父母、晚风微信/抖音、共享文档、活动证明；旧账节点必须回收到军训手机袋、热水卡、水费、外放、空调和陆沉“没事”；5X 跨幕延续统一为第六幕与终章，不再沿用旧跨幕口径。
- v3.8 不扩节点，只做关键路径试玩阅读精修：玩家可见文本不得出现抽象认知总结、游戏元文本或地图旧口径；期末复发节点的第六幕入口信息保留在结构化字段，不直接写成上帝视角说明。
- v3.9 不改节点结构，只在既有框架内全量重写玩家可见表达：晚风、夏知微、沈嘉禾、秦越等软回流人物只能通过消息、照片、资料、活动名单等具体物件把林亦舟推回 5A 场景，不替宿舍主线下结论；玩家可见文本不得出现“资源线”“回主线”“问题消失”“这个回答/这个沉默”等元叙事或作文式总结。
