# 第四卷第四十二章《不能确认的地方》设计说明

## 目标

承接第四十一章旧文档转出课程群后的二次撤回成本，把“帮别人做示例”和“替别人写生活理由”的边界继续压实，同时为第四卷后段作为上册收束积累更重的宿舍关系、课程协作和现实余波。

## 必守约束

- 不开启第五卷正式外流线；许澈不进入林亦舟关系核心。
- 主场仍在武生院：青枫居、教学楼 A302、阳光书屋、校园快递站和西园餐厅。
- 明德楼不承载班会、课程、机房或答疑。
- 晚风只作为现实镜像和微信侧线，不替宿舍四人处理问题。
- 正文不出现玩家可见系统词、变量词、上帝视角预告句或“林亦舟意识到”式总结。

## 章节结构

1. 清晨青枫居：邓鸣发来外班复习群截图，旧文件已经被改名传播；四人把“确认不了的部分”写出来。
2. 教学楼 A302：陈老师和助教要求 22 组、罗泽组、邓鸣、梁嘉明各自说清事实、来源、删除确认和不可确认项。
3. 走廊短场：梁嘉明拒绝被反复道歉和解释，邓鸣学着把道歉缩短，林亦舟不插手替他们说完。
4. 校园快递站和西园餐厅：生活碎片回到现实，家庭包裹、取件码、二级成绩入口、小舞台排练和图书馆排班并排存在。
5. 阳光书屋：四人和相关同学整理空表，把“无法确认”写成正式栏目，而不是用情绪补足。
6. 夜晚青枫居：课程群要求明天各组带空表互换填写，压力转向“别人拿着空表会怎么写”。

## 文风要求

- 用手机震动、打印纸、取件码、空表、删除确认、电脑屏幕和桌面物件承载心理。
- 冲突以现实行动推进，不写长篇道理。
- 章末保留下一章钩子：空表被互换，规则从 22 组扩散到全班。
