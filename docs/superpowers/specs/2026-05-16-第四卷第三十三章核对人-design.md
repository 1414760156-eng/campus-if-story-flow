# 第四卷第三十三章《核对人》设计说明

## Summary

承接第三十二章末尾的《实验四合并审核说明模板》，第三十三章写“审核别人”这件事如何从课堂要求变成青枫居 4XX 的新矛盾。章节重点不是增加技术操作，而是让四个人在公共桌前尝试核对彼此：能提醒、能退回、能写清楚看过什么，但不能把对方的东西改成自己的样子。

## Chosen Approach

采用“三路分散 - 十点半公共桌 - 第一份互审 - 新抽查通知”的结构。

1. 分散段：周屿去东区商业街听被黎旻顶掉的开场，唐骁在西园旁边补计算机二级回放，陆沉到青枫居一楼补门禁，林亦舟把模板发进四人群并带着电脑回宿舍。
2. 汇合段：十点半四人回到青枫居公共桌。审核说明模板三栏成为新的桌面物件，与二级错题、主持稿、补签纸和仓库页面同时出现。
3. 冲突段：唐骁核对周屿异常说明时越过边界，试图把措辞改成自己的版本；周屿拿“你又替我讲了”顶回去。陆沉补出“审核可以退回，不能代写”的操作边界。林亦舟不再当主持式圆场，而是把审核人和被核对人配对重新排出来。
4. 钩子段：第一份互审说明完成后，助教临时补发现场合并抽查要求，要求审核人能说清楚自己核过哪一部分、不接受只签名。下一章进入机房现场合并与成员互审说明被公开问到。

## Boundaries

- 章节主场在青枫居 4XX、青枫居一楼宿管窗口、西园旁自习区、东区商业街小舞台，不使用明德楼承载课程。
- 不使用系统、节点、判定、主线等游戏开发词。
- 不使用“让林亦舟意识到”“下一句话会”等上帝视角句。
- 不写“周屿怕/唐骁怕/陆沉怕”的并列心理总结。
- 许澈不出现，避免第四卷提前加重外流线。
- 晚风最多作为一条轻消息存在，不替四人关系给结论。

## Character Work

- 林亦舟：从“把模板发出去”到“把人按可核对内容重新配对”，学会让别人说清楚自己核对什么。
- 周屿：小舞台开场被顶后带着面子伤口回宿舍，在唐骁改他措辞时第一次把“别替我讲”说得较硬。
- 唐骁：补二级回放失败后更急于把可控的实验材料修整齐，因此越界；本章让他把“指出问题”和“代替修改”分开。
- 陆沉：门禁补签完成后，把宿管阿姨“别替人签”的现实规则带回公共桌，成为互审说明的操作底线。
