# 第三卷第十八章《署名》设计

## Summary

本轮继续《武生院沉浸版剧情小说母版 v4.0》第三卷《热闹把人带走》，承接第十七章结尾“学院融媒体推文邀请”。目标是把匿名传播之后的正式发布压力写实，不把它处理成荣誉奖励。

第三卷当前已超过原定 55,000 字符目标，但结构还需收束。第十八章应承担“正式渠道比匿名号更干净，也更难拒绝”的现实矛盾：他们可以阻止别人偷拍，却不能轻易说清自己是否愿意把经历主动发布出去。

## Chosen Approach

采用“西园餐桌讨论 -> 楼层授权 -> 许棠改稿规则 -> 周屿 B206 对照”的结构。

1. 西园餐桌上四人围绕融媒体推文讨论署名方式：四个人实名、计科一班水房观察小组、匿名小组、或不发。
2. 唐骁要求先问楼层同学，陆沉整理可公开材料，林亦舟负责把话发给罗泽/丁祺/何书南，周屿既想让事情被看见，又不想被写成“热水机主持人”。
3. 罗泽等人提出现实条件：不露脸、不写楼层具体寝室、不把四楼写成故事背景、不把“半杯水”写成励志。
4. 许棠给出融媒体稿件规则：正式推文不是匿名热闹，但也会压扁人，标题、署名、图片和事实范围都要提前定。
5. 章末周屿到教学楼 B206，秦越要求他重新讲一个开场，不能把“推文邀请”当成荣耀，也不能把自己当成受害者。

## Style Constraints

- 不写系统词和策划词。
- 不写自我认知总结，不写“下一句话会如何”。
- 用餐盘、手机、截图、打印稿、标题草稿、B206 灯光承载压力。
- 明德楼只作为许棠/融媒体渠道，不承接班会、课程或主持试训。
