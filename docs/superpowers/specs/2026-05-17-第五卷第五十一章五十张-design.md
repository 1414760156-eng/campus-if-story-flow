# 第五卷第五十一章《五十张》设计

## 目标

承接第五十章结尾：第一批五十张节目单仍印着旧导语。把线上材料残留转为实体纸张、打印流程、经费和旧纸去向的现实处理，让冲突落在可见行动上。

## 写作原则

- 不开启 5X，不引入许澈。
- 不把五十张纸写成大灾难，重点写“未发放前如何收住”。
- 宿舍四人仍是核心：周屿确认底线，唐骁拆流程，陆沉留档，林亦舟传话和跑动。
- 秦越、姜宁、许棠、打印店老板各自做具体事，不做道理输出。
- 保留成本和人情：谁付重印钱、旧纸怎么处理、谁负责不让旧纸散出去。

## 事件链

1. 五十张节目单残留旧导语，现场短暂失声。
2. 唐骁先问“有没有发出去”，陆沉统计份数和去向。
3. 姜宁解释昨晚发了 2307 版，但打印店用的是早前保存的“最终版”。
4. 周屿拒绝用贴纸遮旧话，认为遮掉后反而更像把它做成展示。
5. 众人去东区商业街打印店核文件、重打，打印店老板指出文件名“最终版”太多。
6. 秦越想自己付钱，周屿要求走活动重印记录，不把钱变成私人情面。
7. 旧五十张装袋，留一张作证，其余回收但不随手丢在外面。
8. 新节目单正确打印。
9. 章末：主持手卡或舞台侧边流程夹仍沿用旧开场，下一章继续处理“手里拿着念的东西”。

## 禁区

- 不使用开发者口吻。
- 不写“让某人意识到”式自我总结。
- 不使用明德楼、A302、公交、火车到武生院站等错误地点动线。
- 不把支线人物写成 NPC 传声筒。
