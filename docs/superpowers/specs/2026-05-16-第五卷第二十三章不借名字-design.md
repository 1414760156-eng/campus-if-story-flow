# 第五卷第二十三章《不借名字》设计

## 写作目标

承接第二十二章《袋口》章末课程群通知，推进周五 C305 的“别人坚持要看内容时如何拒绝，且不把拒绝推给不在场的人”。本章继续属于第五卷 5A 宿舍修补线，不开启 5X，不改变主线锁点。

## 核心冲突

22 组已经封住旧文件袋，但“不开”本身会被外部解读为心虚、麻烦、遮掩。新的难点不是袋子能不能打开，而是谁站出来说“不提供”，并且不把压力推给缺席者、涉及人或某个更软的人。冲突要落在具体动作上：唐骁起草说法，周屿差点借“他不在”挡问，陆沉把拒绝写成主体承担，林亦舟把晚风票务线里的“她不让给”改成“我们选择不提交私人内容”。

## 场景安排

1. 青枫居 4XX 清晨：四人试写拒绝话术，围绕“涉及人不在场”与“我们不提供内容”争执。
2. 西园餐厅或走廊：丁祺、罗泽、何书南再次进入，用椅背材料和旧账追问“能不能借一句话挡过去”，暴露借名风险。
3. 教学楼 C305：陈老师设置临场追问，要求 22 组面对“我就是要看内容”的同学，展示目录、封条、登记纸、撤回记录，但不打开袋子。
4. 支线镜像：秦越小舞台设备原始视频、晚风票务申诉聊天截屏都出现“别拿别人名字挡”的相同压力。
5. 夜里 4XX：楼层群误传“4XX 说不让看，因为人不在”，四人发现拒绝也会被改写，留下下一章钩子。

## 文风约束

- 使用 chinese-novelist 原则：展示而非讲述、冲突驱动、章末留钩。
- 正文不出现系统、节点、变量、主线、回流、触发、判定等玩家可见策划词。
- 不写“让林亦舟意识到”“下一句话会”“周屿怕/唐骁怕/陆沉怕”等解释性或上帝视角句式。
- 地点只使用青枫居、西园餐厅、教学楼 C305、东区商业街小舞台、北园二楼/微信消息等合理空间；不使用明德楼承接课程。
