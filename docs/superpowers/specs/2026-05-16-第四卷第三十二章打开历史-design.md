# 第四卷第三十二章《打开历史》设计说明

## Summary

承接第三十一章课程群通知，第三十二章写“仓库历史被公开打开”的课堂压力。重点不是把实验四做成技术说明，而是让每个人前一晚留下的记录在教学楼 A302 被看见、被追问、被同学旁观，从而推动青枫居四人的协作边界继续变硬。

## Chosen Approach

采用“课前准备 - 教室公开抽查 - 课后余波 - 下一轮审核要求”的四段式。

1. 青枫居清晨先把四人的现实压力摆出来：唐骁的计算机二级错题、周屿的小舞台录音、陆沉的门禁补签纸、林亦舟的合并权限。
2. 阳光书屋或西园餐厅短暂停留，让四人围绕“能不能补改昨晚记录”发生轻冲突。结论是不改旧记录，只补今天的新说明。
3. 教学楼 A302 课堂投影打开仓库历史。陈老师和助教抽查两名成员，再顺带追问组长责任和同组核对边界。公开压力要具体：提交说明写了什么，日志能不能对应工作，README 的口语化约定能否解释规则。
4. 课后回到西园或青枫居，不写胜利。唐骁仍错过二级讲题，周屿的小舞台被别人顶了一段，陆沉还要去补门禁记录，林亦舟收到新的“合并前需成员互审说明”通知。

## Boundaries

- 课程、抽查、答疑统一放在教学楼 A302，不使用明德楼。
- 不出现正文禁用词和系统提示口吻。
- 不写“让林亦舟意识到”“下一句话会”等上帝视角表达。
- 不让周屿、唐骁、陆沉用“怕……”并列句解释心理。
- 许澈不出现，避免第四卷远影频率过高。
- 晚风可以只作一条消息，不替宿舍或课堂给结论。

## Character Work

- 林亦舟：从只想把四个人记录整理顺，转为在公开追问下承认组长只负责核对，不能把别人的经历修成好看材料。
- 周屿：面对“未整理成最终讲稿”这句被投出来的尴尬，不靠完整玩笑遮住，而是把缺席 B206 的原因说成实验材料的一部分。
- 唐骁：README 从规则纸变成公开可读材料。他要解释“说一声”为什么比“统一审批”更适合寝室协作，同时承担错过二级错题的代价。
- 陆沉：日志里的测试记录被追问。他不让门禁补签纸替自己说话，只回答自己实际核对过什么、什么时候核对。
