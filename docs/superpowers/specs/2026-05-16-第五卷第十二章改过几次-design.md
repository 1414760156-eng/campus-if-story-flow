# 第五卷第十二章《改过几次》设计稿

## 任务目标

续写《武生院沉浸版剧情小说母版 v4.0》第五卷 5A 宿舍修补线。承接上一章结尾“宿管阿姨要看公共桌说明纸改过几次”，把公共桌的内部争执推到青枫居一楼宿管窗口，让外部目光检验四人最近几天的临时修补是否真的能落地。

## 固定约束

- 宿舍四人优先：周屿、林亦舟、唐骁、陆沉必须共同在场并各有动作。
- 第五卷仍保持多线收束：晚风线只能做现实镜像，不抢走 5A 宿舍修补线重心。
- 许澈不进入核心关系；本章不启用 5X。
- 计算机组成原理相关地点继续使用教学楼 C305，不出现 A302。
- 明德楼不承担班会、课程、答疑、机房或上机。
- 正文避免系统口吻、抽象总结和被点名禁用表达。

## 本章事件设计

1. 青枫居清晨：四人带公共桌说明纸下楼。说明纸上的透明胶、灰、铅笔补充、周一补充时间都作为可见物件出现。
2. 青枫居一楼宿管窗口：宿管阿姨不审“谁对谁错”，只看改动痕迹、时间、是否本人知道。楼长和别寝室同学在场，公共目光带来压力。
3. 外部规则落地：阿姨要求增加一张“小改动表”，只写时间、改了什么、谁在场，不把公共桌变成追责墙。
4. 早饭或阳光书屋：周屿向唐骁要“缺人版本”照片。唐骁把文件递出去，但双方围绕“提前准备是不是默认对方会缺席”发生更深对话。
5. 晚风镜像：晚风那边的票务材料也出现“要不要留原记录”的问题，林亦舟只给现实建议，不替她决定。
6. 回到 4XX：四人真正把小改动表贴到公共桌下沿。章末用 C305 新通知或宿管二次抽查继续压迫下一章。

## 写法要求

- 用物件推进心理：透明胶、灰、铅笔字、记录夹、窗口印章、手机照片、父亲红包备注。
- 对话要像大学寝室里的真实拧巴：有人嘴硬，有人怕麻烦，有人想把表格做完整，有人只想别再扩大。
- 不直接写人物“意识到”“害怕”“成长”，只写他们怎么递纸、删字、停顿、看手机、换说法。
- 冲突要有影响力但合理，不写成校园审判。

## 章末钩子

建议以“周三 C305 要交换查看原始改动记录”和“宿管阿姨明晚抽看小改动表是否照写”双压力收束，让第十三章继续把课程协作与寝室修补并排推进。
