# 第五卷第七十四章《那边》设计

## 目标

承接第七十三章结尾“秦越那边催手卡”，把一句含混转述拆成现场压力、临时省事、借名授权和材料边界四层，不让秦越成为唯一背锅者，也不让责任被“大家都以为”冲淡。

## 章节位置

- 场景仍在明德楼融媒体工作室，符合明德楼只用于融媒体、材料复核和行政说明的规则。
- 继续第五卷 5A 宿舍修补线，不打开 5X，不让晚风吞掉本章中心。
- 宿舍四人仍占叙事核心：周屿挡住替罪羊式追责，唐骁追问具体授权，陆沉拆表，林亦舟控制外部消息进入范围。

## 冲突设计

1. “秦越那边”不是秦越直接下令，而是活动现场催手卡的压力被陈景和摄影社小群转述成了放人进群的依据。
2. 秦越承认自己昨晚确实催过手卡，也说过“缺什么快补”，但没有说过“让临时人扫码进素材群”。
3. 陈景通过电话说明，他听见台侧在催手卡，又见邵闻帮忙扛架子，便把“秦越那边催”写成省事句子。
4. 老师要求所有人不要再用“那边”，必须写清人名、原话、转述和执行动作。
5. 章末陈景补发完整截图，露出更早的一句“二维码在上面，手卡补字的人先扫”，把下章压力推向二维码如何被默认当成入口。

## 文风约束

- 使用 chinese-novelist 原则：展示而非讲述，动作和对话推动。
- 禁止系统口吻、游戏提示、路线提示和上帝视角总结。
- 不写“让林亦舟意识到”等解释性句式。
- 不写班会、课堂或普通教学进入明德楼。

## 验证

写完后运行字数统计，并扫描第五卷正文中的禁用表达、明德楼误用、交通误写、A302 和许澈。
