# 第三卷第二十三章《没能删掉的》设计记录

## 目标

承接第二十二章末尾“截图进入公共课群”和陈老师要求水房观察小组在教学楼 A101 说明材料使用边界的后果。章节重点不是扩大舆论，而是把热闹带回课堂、带回宿舍四人的学业和疲惫，让公开表达的代价以具体细节呈现。

## 必守约束

- 班级说明、专业导论与课堂讨论发生在教学楼 A101，不使用明德楼。
- 明德楼只保留为学工、融媒体、行政材料相关空间，本章不写明德楼场景。
- 宿舍四人必须同场推进：林亦舟负责删改材料与开口，周屿面对公共表达的语气变化，唐骁把边界拆成表格，陆沉负责证据文件与不可展示材料。
- 方苒与沈嘉禾作为柳园洗衣房事件的支线人物继续出现，但不被拉成课堂中心，不把当事人姓名公开化。
- 章节必须回到理工科大学生活：专业导论、程序设计基础实验、提交压力，为第三卷收束和后续学业线提供支点。

## 文学处理

- 使用 chinese-novelist 原则：用教室投影、手机群消息、删掉的图片框、A101 的粉笔灰、课后通知等具体物象推进。
- 不写游戏系统口吻，不写“下一句话会”“这一步会被记”等说明性句子。
- 不用抽象总结替代冲突，同学追问、材料删改、老师收束和作业通知构成事件压力。

## 章节结构

1. 青枫居清晨，四人修改“材料使用说明”：删姓名、删截图、用文件名和马赛克框替代照片。
2. 去教学楼 A101 的路上，公共课群继续转截图，沈嘉禾发来方苒的态度：别把她名字放进任何表里。
3. 专业导论课上，小组说明删掉了什么、为什么删、哪些内容已经无法撤回。同学提出现实质疑。
4. 陈老师把讨论压回课程训练：材料不是情绪附件，公开表达需要留下可核查的说明。
5. 课后，助教发布程序设计基础实验提交通知，四人被迫从舆论热闹回到自己的代码、作业和下一轮生活压力。

## 本章钩子

章节末尾以“程序设计基础实验周日晚提交”的通知收束，提醒四人：水房报告还没完全过去，新的学业压力已经进门。
