# 第四卷第三十八章《未完成项》设计稿

## 写作边界

- 承接第三十七章结尾：周日晚 22:00 前提交本周日志、补测截图与审核说明，课堂将展示“未完成项如何记录”。
- 第四卷仍处于引导式外流阶段，不开启 5X；许澈不进入本章关系核心。
- 宿舍四人优先，晚风票务线只能作为现实支线映照林亦舟的边界处理，不进入实验四材料。
- 明德楼不承载课堂、班会、考试、机房、上机或答疑；本章主要使用青枫居、阳光书屋、八号教学楼机房、西园餐厅、图书馆西侧、东区商业街消息余音。
- 正文避免系统口吻、上帝视角预告和解释性心理总结；用文件夹、截图、签退照片、主持流程表和微信聊天框呈现压力。

## 本章事件骨架

1. 青枫居周日清晨：四人各自补日志。唐骁被第 19 题保存路径的不确定感拖住；周屿改主持流程表；陆沉整理图书馆签退照片；林亦舟整理晚风票务证据。
2. 西园餐厅短饭桌：四人约定下午去阳光书屋对齐材料，不把“看起来有用但不属于实验四”的东西塞进提交包。
3. 阳光书屋与八号教学楼机房：核对补测截图、审核说明和个人日志。唐骁想把不确定写得更短，陆沉坚持签退只写自己的时间，周屿把小舞台证明从实验日志里拿掉。
4. 晚风票务后续：室友想发朋友圈曝光，晚风来问怎么写。林亦舟帮她拆事实，但不代写结论；票务表必须和实验四文件分开。
5. 青枫居夜间提交：林亦舟差点把票务证据表拖进实验四文件夹，唐骁拦住，形成一次不激烈但有重量的冲突。四人重新分出“提交材料”和“不提交材料”。
6. 21:58 提交完成后，助教公布周一课堂展示组别，22 组被点名展示“未完成项如何记录”。

## 人物推进

- 林亦舟：从能拆清别人票务证据，推进到能把“别人的事”和课程材料真正分开放。
- 唐骁：不确定第 19 题后更想把材料写干净，但最终接受“未确认就是未确认”。
- 周屿：把主持和小舞台证明从实验日志里移出，承认热闹不能替课程贡献。
- 陆沉：用签退照片和个人日志守住自己的时间，不接他人额外解释。
- 晚风：通过朋友圈曝光犹豫，把大学生被骗后的面子、愤怒和证据边界带出来。
