# 第五卷第四十四章《谁创建的》设计

## 目标

承接第四十三章结尾“旧截图来自主持队公共盘，创建者：秦越”。本章不把秦越写成单一替罪羊，而是拆开旧文件夹创建、共享权限、志愿者取用、现场素材包再进入今天活动的责任链。

## 写作边界

- 仍属于第五卷 5A 宿舍修补线，不进入 5X。
- 周屿不回到主持展示位，只参与边界确认和现场后勤。
- 秦越有责任和愧疚，但不能用“我全责”掩盖具体流向。
- 姜宁继续处在活动执行压力里，不能写成恶意泄露者。
- 许棠负责查材料来源和关权限，口吻保持新闻中心/素材管理经验，不写成裁判。
- 禁止出现系统口吻、作者总结、上帝视角分岔提示，以及 A302、许澈等旧问题词。

## 场景链

1. 青枫居 4XX：四人看到“创建者：秦越”，先压住直接定罪冲动，讨论“创建”和“今天使用”不是同一个动作。
2. 活动材料群与秦越私聊：秦越急着认下旧文件夹问题，周屿和 4XX 要求他说清共享状态、取用人、当前素材包，不许用一句“我全责”把链条盖住。
3. 青枫居一楼宿管窗口：宿管阿姨用旧钥匙和借用登记做生活类比，强调配钥匙的人、拿钥匙的人、今天开门的人要分开。
4. 东区商业街小舞台：秦越、姜宁、许棠处理现场素材和二维码，周屿在台后不补位，只核对“这次不展示”是否被执行。
5. 支线镜像：夏知微处理摄影社旧公开相册；晚风处理旧群邀请链接和以前聊天被拉出来当证据。
6. 章末钩子：旧链接关闭后，图片已被打进今天的“现场花絮待剪”压缩包，打包人显示姜宁。

## 本章功能

- 把第五卷“材料边界”从单张截图推进到“旧权限和今天传播”的责任拆分。
- 让秦越、姜宁、许棠从功能角色变成有压力、有补救动作的人。
- 继续让宿舍四人围绕同一张表、同一扇门、同一条群消息修补说话方式。
