# 第四卷第四十四章《只讲依据》设计

## 目标

承接第四十三章末尾陈老师要求 22 组说明“为什么划掉这一行”，让周屿亲自把“别替我写”说出来。章节重点不是再讲表格规则，而是把公开说明落到人物弧光：周屿收住玩笑，唐骁收住代答，林亦舟收住圆场，陆沉用一句短话守住“本人确认”的边界。

## 写作边界

- 场景限定在武生院内：青枫居、教学楼 A302、北园餐厅、东区商业街小舞台、阳光书屋、图书馆西侧。
- 明德楼不承载课堂、班会、答疑、机房或上机。
- 晚风票务投诉只做现实镜像，不抢宿舍四人主轴。
- 许澈不出现。
- 第四卷继续作为上册收束前压力汇合，不开启第五卷硬外流。

## 事件设计

1. 青枫居清晨：周屿把小舞台排班表背面的“别替我写”折进文件夹；唐骁准备发言提纲却被周屿拦下；父亲分笔转账、母亲袜子消息、二级成绩延迟和陆沉勤工表同桌出现。
2. A302 公开说明：周屿说明自己确实参加主持、小舞台和运动会替补，但空表里看不出这些和实验进度的因果；十七组魏同学承认是根据课堂印象写的。陈老师要求把“印象”改成“无法从本表确认”。
3. 走廊余波：魏同学道歉，周屿拒绝小作文，只要对方下次别替他写；唐骁没有替周屿补充，陆沉说“他已经说了”。
4. 北园餐厅：四人短饭桌缓冲，唐骁二级成绩依然延迟，林亦舟收到父亲第二笔小额转账，母亲不再追问缺什么。
5. 东区商业街小舞台：周屿把串场词改得更朴素，秦越看见他的变化；台下热闹仍在，但他不再用别人的故事撑场。
6. 阳光书屋/青枫居夜：晚风票务审核要求补授权证明，她选择写“无法提供”；四人收到课程群下一步通知：整理“不能由别人替写”的三类句子，章末把压力推进到第四卷上册收束前的最后一组课堂整理。

## 文风要求

- 用动作和对话推进心理，不使用系统提示式总结。
- 禁止出现“这个回答”“这个沉默”“让林亦舟意识到”“下一句话会”等旧问题表达。
- 结尾保留钩子：下一章可围绕“三类不能替写的句子”继续收束。
