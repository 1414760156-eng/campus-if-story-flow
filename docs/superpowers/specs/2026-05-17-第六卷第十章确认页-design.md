# 第六卷第十章《确认页》设计记录

## 断点

第九章《点一遍》结尾，C407 现场复测中页面核心动作通过，但说明页页脚残留“最终确认：林亦舟”。陈老师和冯助教指出复测人、改动人、说明页检查人、材料来源核对人、最终确认人的边界不能混写，要求下周一提交“最终确认页”。

## 本章目标

- 把“最终确认”从单人背书改为可检查的分栏确认：页面改动、复测记录、说明页文字、材料来源、提交负责人。
- 宿舍四人优先：唐骁承认页面与提交责任，林亦舟只签复测结果，周屿只签说明页文字范围，陆沉只签公开材料来源。
- 继续第六卷准结局形成阶段：项目能跑通只是第一层，责任能否写清才是毕业前硬结局的前置条件。
- 让晚风通过微信短句形成现实镜像：签名不是替别人担保，而是写清自己看过什么。
- 结尾钩子推进下一章：周一 C407 通过确认页，但陈老师追加演示分工表和“维护到毕业”责任栏，逼近第六卷后段的去向和硬结局。

## 场景与动线

- 青枫居 4XX：周五夜拆页脚、做确认页，加入寝室作息、耳机、复习、图书馆排班和公共桌细节。
- 教学楼旁打印店：周一早上打印确认页，穿插期末材料、四级、普通话、项目打印潮。
- C407：提交确认页并被追问“最终”到底是谁负责。
- 不使用明德楼承载课程或班会，不写公交，不写 A302，不写许澈。

## 写法约束

- 不写上帝视角总结，不写“让林亦舟意识到”等抽象认知句。
- 不使用玩家可见术语：主线、副线、分线、节点、flag、触发、权重、判定、回主线、资源线、系统记录等。
- 不使用“你怕我怕他怕”式并列心理。
- 用纸张、签名、页脚、打印机卡纸、微信短句、公共桌物件来呈现人物状态。
