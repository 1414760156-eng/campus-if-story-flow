# 第五卷第二章《自查表》设计记录

## 目标

本章承接第五卷第一章结尾的周五教学楼 B206 班会通知，把第五卷正式推入 5A 宿舍修补入口。核心不是让矛盾立刻爆炸，而是让自查表、课程协作时间表和晚归登记把四个人的旧习惯压到同一张公共桌上。

## 章节功能

1. 让青枫居 4XX 的公共区成为下册第一个可见压力点：插排、旧线、外卖袋、空调遥控器、阳台晾衣和门后挂钩。
2. 让周屿主持队、唐骁课程计划、陆沉图书馆晚班、林亦舟家庭半额转账和晚风票务补证同时占用周五晚上。
3. 让教学楼 B206 班会只承担班级和寝室安全说明，不放进明德楼。
4. 支线人物以现实方式出现：秦越通过录制通知拉走周屿，沈嘉禾用短提醒补充表格边界，晚风只通过微信票务材料存在。
5. 继续不启动正式 5X，不让许澈进入关系核心。

## 冲突设计

- 外层冲突：班会、自查、实验时间表、主持录制、图书馆晚班同时挤压。
- 寝室冲突：周屿想把“会回来”写成在寝，唐骁坚持写当时事实；陆沉不想把勤工原因写细；林亦舟夹在晚风补证和寝室表格之间。
- 低强度但有效：旧插排和胶带数据线是否算隐患，公共桌是否可以临时堆主持物料，谁能替谁下楼取自查表。

## 文风约束

- 使用具体物件和动作，不写抽象心理总结。
- 避免“意识到/觉得/突然/忽然/发现/下一句话会”等高风险表达。
- 正文不出现游戏策划术语。
- 章末留钩子：自查照片和现实桌面不完全一致，推动下一章继续处理“拍给别人看”和“自己真实生活”的落差。
