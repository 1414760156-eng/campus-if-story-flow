# 第四卷第十章《出口》设计

## 目标

正式写周六光谷现实见面，但不让它抢走第四卷主轴。重点不是把晚风写成“答案”，而是把线上关系第一次落到武汉真实动线、等待、雨、礼物、尴尬、边界和返校成本里。

## 叙事边界

- 这是第四卷引导式外流和晚风现实化，不是第五卷 5X 硬外流。
- 许澈不出现或只保留远处名字，不进入本章关系核心。
- 宿舍四人通过出发前准备、寝室群消息、返校接应和夜归门禁贯穿本章。
- 光谷作为武汉现实外部场域，只承载一次具体见面；章节必须把武生院站出发、阳逻线进城、换乘、等人、雨和返校时间写实。

## 场景安排

1. 青枫居午后：出发前四人短对话、伞、钥匙扣、路线截图、母亲/父亲消息。
2. 武生院站与地铁换乘：高架站、闸机、雨伞、车厢、换乘人流，写出时间成本。
3. 光谷出口：晚风迟到，林亦舟避开出口正中间等待，第一次看见白色防晒衣和黑色帆布包。
4. 光谷步行街：笨拙交换小礼物、奶茶 AA、室友被骗善后、补考空窗和线上/线下语气差异。
5. 返校：阵雨、换乘、寝室群提醒、青枫居门禁，章末用宿舍灯和新挂件收束。

## 写法要求

- 使用 `chinese-novelist` 原则：展示而非讲述，冲突驱动，章末留钩子。
- 不写系统词，不写抽象成长总结，不写“晚风治愈主角”。
- 现实细节要具体：路线截图、闸机、扶梯、湿伞、奶茶小票、手机电量、末班前返校、宿管阿姨登记本。
