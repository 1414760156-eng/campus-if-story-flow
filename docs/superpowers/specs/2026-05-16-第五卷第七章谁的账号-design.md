# 第五卷第七章《谁的账号》设计说明

## 目标

续写《武生院沉浸版剧情小说母版 v4.0》第五卷第七章，承接第六章末尾“本人账号登录记录”和晚风付款人不愿出面的双重压力，同时补入青枫居 4XX 更具体的生活习惯摩擦，避免 5A 宿舍修补线只剩材料表格。

## 核心原则

- 不改主线锁点：第五卷仍是多线收束，5A 仍优先处理宿舍修补，晚风只作为现实支线镜面，不抢宿舍四人主轴。
- 使用课程表索引：计算机组成原理相关课堂继续使用教学楼 C305；账号登录记录可以通过机房/课程平台/本人设备呈现，但本章不把地点错写回 A302。
- 保持小说化：用桌面、插排、洗衣液、外卖袋、床帘、电脑登录框、微信消息和宿管阿姨夜归登记推动，不写系统口吻、变量解释或上帝视角预告。
- 增强冲突影响力：生活习惯摩擦不是笑料，而是“谁替谁方便、谁占用公共区、谁把别人习惯当默认”的具体旧账。

## 章节事件设计

1. 清晨青枫居 4XX：课程群要求本人账号登录记录，唐骁提前列登录步骤；周屿嫌他像检查寝室，陆沉要赶图书馆早班，林亦舟被晚风消息打断。
2. 生活习惯摩擦：公共插排被周屿昨晚补录设备占着，唐骁电脑电量告急；洗衣液箱子挡住陆沉拿旧包，母亲寄来的东西被误当公共物品；林亦舟的外卖袋没有及时扔，宿舍开始从“材料边界”滑向“生活边界”。
3. C305 前后：本人登录记录检查不看截图转发，助教要求现场登录或提交平台记录。四人分开完成，不互相代登；有人想借 22 组账号流程，被唐骁挡回去。
4. 晚风线镜像：付款人不愿用自己账号，想让林亦舟代写代发。林亦舟不看私发截图，只让晚风把“付款人是否愿意出面”作为第一栏。
5. 回到青枫居：四人因插排、外卖、洗衣液和夜间键盘声起更明的争执。冲突不爆炸，但比前几章更刺耳；最后他们不做和解宣言，只把公共桌使用和账号登录分开写成很短的寝室纸条。

## 章末钩子

章末由宿管阿姨或楼层群引出下一章压力：青枫居要抽查公共区和晚归记录，且课程平台明天会核验登录 IP / 设备信息。宿舍修补从“写清楚”进入“能不能长期照做”。
