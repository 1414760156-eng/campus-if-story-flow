# 第四卷第四十章《没有示例》设计

## 使用的技能

- superpower-bootstrap
- superpower-brainstorming
- planning-with-files
- superpower-writing-plans
- chinese-novelist
- superpower-verification

## 章节目标

承接第三十九章末尾罗泽追问“没有示例怎么写”，把空目录从“帮同学”写成新的现实压力：别人要的不是结构，而是可以直接照抄的答案。四个人需要拒绝代写，同时不把同学推成敌人；这让第四卷的“边界、审核、信任”继续贴着宿舍四人走。

## 锁定边界

- 场景主轴在青枫居 4XX、青枫居一楼外卖架、西园餐厅、星辰书屋/阳光书屋、教学楼 A302。
- 明德楼不承载班会、课程、答疑、机房、试训或课堂展示。
- 晚风票务投诉只作现实支线余音，不替四人解决问题。
- 唐骁二级成绩查询只制造个人压力，不改变实验四材料边界。
- 第四卷继续只埋外流感，不开启 5X；许澈不出现或只作远影。

## 事件设计

1. 夜里青枫居，罗泽继续要示例。四人围绕“回不回、怎么回、谁回”发生低强度争执。
2. 楼下外卖架发生拿错饭的小插曲，宿管阿姨一句“看清编号再拿”把材料照抄的风险生活化。
3. 西园餐厅或星辰书屋短场，罗泽当面拿出他们组写好的草稿，里面有照抄 22 组口吻的痕迹和私人原因外溢。
4. 四人不替罗泽重写，只问他材料能证明什么；唐骁守住规则，周屿避免把问题变成嘲讽，陆沉提醒不要写别人的原因，林亦舟把“帮忙”限成看栏目。
5. A302 课前或课后，陈老师/助教点出空目录外传后的问题：可参考目录，不可复制说明。下一步要求各组带一份自己修改过的未完成项说明来当场互看。
6. 章末钩子：课程群里出现一份匿名转发的“示例版”，里面有明显模仿 22 组的话，且夹进了一个同学的私人理由。

## 文风执行

- 用饭袋编号、文件名、聊天气泡、划掉的句子和桌面空栏承载冲突。
- 不写“意识到/明白/下一句话会”等解释性句式。
- 人物台词保持当下情境：唐骁直接、周屿收住玩笑、陆沉短句、林亦舟避免主持腔。
- 结尾留钩子，不把模板传播问题一次解决。
