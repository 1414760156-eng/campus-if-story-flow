# 第五卷第十三章《作废页》设计稿

## 任务目标

续写第五卷 5A 宿舍修补线。承接第十二章结尾“C305 周三交换查看原始记录”，把唐骁记录夹里的作废页、周屿本人保存的照片、陆沉对私人信息的敏感和林亦舟从晚风票务线学到的遮挡方式放进同一个课堂场景里检验。

## 固定约束

- 宿舍四人仍为第一优先级，所有外部人物只作为压力和镜像。
- 第五卷继续保持多线收束，不让晚风线成为核心。
- 不启用 5X，不让许澈进入关系核心。
- 计算机组成原理相关课堂仍在教学楼 C305，不出现 A302。
- 明德楼不承载班会、课程、答疑、机房或上机。
- 正文不得出现玩家可见系统口吻和禁用表达。

## 本章事件设计

1. 青枫居清晨：四人带原始记录去 C305。公共桌小改动表仍贴着，空格还空，提醒他们昨晚不是结束。
2. 教学楼 C305：陈老师安排相邻组交换查看原始记录，要求不收重新誊抄版。
3. 沈嘉禾组进入镜像：她所在组有校医院挂号/排班改动材料，遮住名字但保留划线和时间；她不替 22 组讲道理，只问“这张真的给我们看吗”。
4. 作废页公开压力：唐骁带出“缺人版本”作废页，周屿发现别人会看见“未确认是否到场”，出现不适和反弹。
5. 边界重定：陈老师允许遮住无关私人信息，但不能重写事实。陆沉借图书馆交接表和沈嘉禾组材料补出“原始不等于裸露”的现实规则。
6. 晚风镜像：晚风票务材料也要提交给辅导员，林亦舟把课堂经验转化为“遮私人信息、不重做故事”的建议。
7. 夜里回宿舍：小改动表第一次被新外卖袋压住，四人决定是否连这种小事也写。通过写与不写的争执推进 5A。

## 写法要求

- 不写心理总结，用纸页、遮挡、划线、拍照、沉默和椅子移动表达。
- 课堂冲突要真实：尴尬、别扭、被看见，不写成审判。
- 沈嘉禾出现要有自己的材料和小组压力，不当工具人。
- 晚风线轻触，不抢宿舍四人主轴。

## 章末钩子

建议以小改动表出现第一条新的“生活类”记录收束，同时让课程群通知下次抽查“被查看组也要回应对方提出的疑问”，把第十四章推向更具体的互相质询。
