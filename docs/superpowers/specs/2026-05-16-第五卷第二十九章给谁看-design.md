# 第五卷第二十九章《给谁看》设计记录

## 目标

承接第二十八章结尾陌生头像询问“要不要跟他道歉，还是在群里说一下”，继续推进 5A 宿舍修补线。章节不打开 5X，不引入外流选择，只把“道歉”从善后动作写成新的现实压力：道歉也可能再次把人推到灯下。

## 写法原则

- 使用 chinese-novelist 原则：用动作、删改痕迹、消息停顿和场景细节表现人物变化，不写系统口吻，不直接替人物下结论。
- 宿舍四人优先：周屿负责接收外部消息，唐骁负责压缩措辞，陆沉负责确认边界，林亦舟把晚风线中的相似问题带回桌面。
- 事件驱动：每一场都围绕“道歉给谁看”推进，从深夜私聊到宿管窗口、C305 课堂、阳光书屋、东区商业街小舞台，再回到 4XX 公共桌。
- 支线只作现实镜像：秦越、夏知微、晚风、沈嘉禾用于照出同类难题，不抢走宿舍四人的中心位置。

## 章节结构

1. 深夜 4XX：陌生头像自报姓名，四人围绕“群里公开道歉”与“私下询问能否接收”产生分歧。
2. 一楼宿管窗口：宿管阿姨用夜归登记的生活经验提醒他们，补救不能堵人门口。
3. 西园早餐：罗泽、丁祺、何书南围绕“我也要不要道歉”继续施压，显示事件外溢。
4. C305 课堂：陈老师让各组改写“道歉说明”，沈嘉禾用校医院材料作参照，22 组提出不得把道歉对象再写出来。
5. 支线镜像：秦越的小舞台设备组、夏知微的摄影社、晚风的票务聊天分别出现同类处理。
6. 夜晚回桌：梁嘉明发来公开道歉草稿，草稿里仍写出“被我打扰的同学”，四人发现补救语言仍可能制造第二次指认。

## 边界

- 不使用明德楼承载课堂、班会、答疑或机房内容。
- 不新增校外交通误写，不写公交、大巴、客运等离校动线。
- 不出现许澈，不开启 5X。
- 正文避免解释式、高高在上的总结句，避免反复出现“意识到”“下一句话会”“这一步会被记”等表达。
