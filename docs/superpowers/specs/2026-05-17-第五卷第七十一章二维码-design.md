# 第五卷第七十一章《二维码》设计

## 目标
承接第七十章末“这个群谁建的”，把责任从“冯喻建群”继续拆开到群二维码、成员进入方式和临时协作习惯。章节不把冯喻、姜宁或任何单个人写成唯一背锅位，而是让 5A 宿舍修补线继续围绕“顺手、临时、默认方便”产生现实压力。

## 场景与人物
场景继续固定在明德楼融媒体工作室。宿舍四人仍在桌面核心：周屿守住不推出替罪羊，唐骁推动逐项追问，陆沉记录建群、二维码、成员和材料流向，林亦舟只递入晚风外部回声。许棠、梁思予、秦越、冯喻进入现场或电话，姜宁通过截图和语音继续补材料。

## 事件走向
冯喻承认群由自己创建，但二维码曾被发进主持队临时群，方便舞台、字幕、摄影和文案快速进来。许棠发现二维码有效期和群名变更记录，秦越承认当时为了让排练继续，默认“扫码进群”比逐个确认快。唐骁想按人头问责，老师把问题压回进入方式和材料种类。陆沉写出四列表：谁拉入、何时进入、看过什么、有没有外发。

## 章末钩子
导出的聊天记录里出现一条 01:06 的消息：“我存一下，明天问宣传口。”这句话来自一个没有实名备注的头像，且头像不在正式活动名单。章末停在老师要求找出这个账号的对应人。
