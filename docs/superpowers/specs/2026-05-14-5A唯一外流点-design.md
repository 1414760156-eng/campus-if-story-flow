# 5A 唯一外流点设计

## Summary

5A 可以增加多个回流点来提高可玩性，但外流点必须只有一个。外流点不是普通绕路，也不是提前跳回原来的家庭、晚风、学业、作品、资源等多条线；它应该进入一条新的外流线，让“原本的一切”在一段时间里真正远离，并把这种离场代价交给第六幕读取。

## Chosen Approach

采用“多回流、单外流”的结构。

- 回流点：用于短暂离开现场，触发心理暗线、晚风消息、室友偶遇或证据补充，随后回到 5A 主轴。
- 外流点：全 5A 只允许一个，用于玩家明确离开修补现场，进入新的 `5X_离场外流线`。
- 外流点不能设在开头走廊，因为那会让逃避成本太低。
- 外流点应设在中后段强冲突处，让玩家已经看过旧账、陆沉表达、规则纸和站队代价之后再决定是否离场。
- 外流线不能回到原先的多线入口，也不能被晚风线接管；晚风消息可以作为远处的屏幕信号出现，但不能成为本线主角。

## Exact Placement

唯一外流点放在：

```text
ACT5_5A15_DOOR_SLAM_RISK
```

新增外流选项：

```text
不再劝了，推门出去把这场谈话留在身后
```

该选项进入：

```text
ACT5_5X01_STAIRWELL_MUTE
```

外流线是一条新的短线，不回到 5A 主轴，也不回到原来的多条线。它写“离开原本一切”的具体经验：楼梯间静音、校园外沿、陌生场域、外流输出。它的气质应接近第七幕：一切像已经被安排好了，玩家仍然能选择，但选择只影响人物关系、心理状态、记忆标签和第六/终章回响，不再改变外流方向。

## Route Shape

```text
正常路线：
ACT5_5A15_DOOR_SLAM_RISK -> ACT5_5A16_CANTEEN_SEAT -> ... -> ACT5_5A99_DORM_OUTPUT_FULL

唯一外流：
ACT5_5A15_DOOR_SLAM_RISK
  -> ACT5_5X01_STAIRWELL_MUTE
  -> ACT5_5X02_SOUTH_GATE_EDGE
  -> ACT5_5X03_STRANGE_SEAT
  -> ACT5_5X99_OUTFLOW_OUTPUT
```

外流线内部是定轨结构。所有选项都顺序推进，不再解锁其他场景，不回 5A，不回 5B-5F，也不分裂成新分支。选项的意义只体现在：

- `宿舍线外流`
- `关系降温`
- `自我麻木`
- `晚风远距牵引`
- `家庭消息未读`
- `终章疏离回响`
- `记忆标签`

## Source-Specific Ending

外流线的结局必须根据来源线本身判定，不能使用通用“离开/低温”结局。5A 的来源是宿舍修补线，因此外流结局应围绕“旧室友渐远”和“新关系不扎根”展开：

- 林亦舟认识新的人，起初觉得还不错，因为对方不问旧账，也不需要他解释宿舍里的矛盾。
- 宿舍关系逐渐远离：群聊变少、饭桌位置不再自然、毕业活动同框密度下降。
- 林亦舟自己也被一切拉远，短期轻松变成找不清方向。
- 最后发现新人关系更像逢场作戏：轻松、顺手、没有根。
- 外流输出应写入 `新人逢场作戏_宿舍渐远_遗憾回响`，第六幕和终章读取遗憾，而不是给玩家一个爽快的新开始。

## Cross-Act Continuity

外流线一旦成立，第六幕和第七幕也只走自己的外流单线。它可以经过和其他线相同的场景，也可以让其他线角色出现在过场中，但这些都属于软互动，不再提供回到旧线的抉择入口。

允许的软互动：

- 在毕业活动、食堂、宿舍楼下、家中饭桌、校园门口等同场景中经过。
- 和周屿、唐骁、陆沉、晚风、父母、老师、夏知微、秦越等角色短暂擦肩、寒暄、互相看见。
- 群聊已读、朋友圈状态、抖音状态、活动合照边缘、饭桌一句问候。
- 过场剧情中表现关系温度、遗憾、疏离或偶然缓和。

禁止的回流互动：

- 不允许出现“重新处理宿舍修补线 / 晚风线 / 家庭线 / 学业线”等抉择按钮。
- 不允许通过第六幕或第七幕选择把外流线改回原线。
- 不允许让其他线角色在外流线中重新成为主处理对象。
- 不允许把软互动升级成新的多线混乱。

第六、第七幕外流线的选择逻辑延续第七幕式定轨：玩家还能选择，但只影响心理状态、人物温度、遗憾强度、是否承认自己走远、终章疏离或低温回响，不改变外流方向。

## Acceptance Criteria

- 5A 完整试玩数据中只能有 1 个 `外流节点`。
- 5A 完整试玩数据中只能有 1 个 `外流点` 选项。
- 外流点必须位于 `ACT5_5A15_DOOR_SLAM_RISK`。
- 外流点必须解锁 `ACT5_5X01_STAIRWELL_MUTE`。
- 新外流线必须使用 `分线: "5X_离场外流线"`。
- 新外流线必须包含 `ACT5_5X01_STAIRWELL_MUTE`、`ACT5_5X02_SOUTH_GATE_EDGE`、`ACT5_5X03_STRANGE_SEAT`、`ACT5_5X99_OUTFLOW_OUTPUT`。
- 外流线节点必须标记 `外流线节点: true`。
- 外流线节点必须标记 `定轨节点: true`。
- 外流线内部选项不得设置 `解锁` 到其他旧线；允许不写 `解锁`，按 chain 顺序推进。
- 外流线内部选择只允许影响人物关系、心理状态、记忆标签或终章回响，不允许改变外流方向。
- 普通 5A15 选项必须跳过外流节点，回到 `ACT5_5A16_CANTEEN_SEAT`。
- 外流线必须进入 `ACT5_5X99_OUTFLOW_OUTPUT`，不再回到修补主轴。
- 外流线不能解锁 5B/5C/5D/5E/5F 任何旧分线入口。
- 外流输出页必须标记 `外流来源线: "5A_宿舍修补线"`。
- 外流输出页必须标记 `外流结局类型: "新人逢场作戏_宿舍渐远_遗憾回响"`。
- 外流输出页必须标记 `跨幕延续: "第六第七幕仅走外流单线"`。
- 外流输出页必须标记 `软互动规则: "可同场景经过和过场互动，不开放回流抉择"`。
- 正常路线仍能完整抵达输出页，且不误进外流节点。

## Open Risks

- 外流点如果写得像“爽快离开”，会奖励逃避；因此反馈必须写清代价：宿舍线进入低温、修补机会损失、第六幕毕业活动读取疏离状态。
- 外流线可以写新的剧情质感，但不能变成另一个大型主线；本轮控制在 3 个推进节点 + 1 个外流输出页。
