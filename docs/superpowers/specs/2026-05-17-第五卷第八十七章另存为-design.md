# 第五卷第八十七章《另存为》设计

## 本章定位

承接第八十六章结尾：`导语候选_第一版.docx` 顶端灰字既像提醒，也像入口。本章继续复原第一版如何变成第二版，不新开事故，不转场离开东区商业街后台。重点写“另存为”这个看似普通的动作：陈景不是凭空写出旧句变体，而是在第一版灰字、文件夹名、冯喻转发句和旧稿预览残影之间，把提醒改成了更像推文的开头。

## 场景与动线

- 场景固定在东区商业街小舞台后台，承接商铺关灯、收线、旧电脑和话筒箱。
- 具体物件：陈景电脑、旧一体机、胡嘉宁电脑、白卡、Word 属性、最近文档、另存为记录、截图框、灯线箱。
- 不进入明德楼，不新增校外交通，不写 A302，不引入许澈。

## 人物功能

- 林亦舟：负责在两台电脑和手机截图之间压住节奏，不把晚风消息写成解决问题。
- 周屿：继续反对把问题压成“陈景一个人写错”，也不允许把“大家赶时间”变成免责。
- 唐骁：追问“删掉灰字”和“留下意思”的差别，火气落在具体操作。
- 陆沉：把第一版、另存为、删灰字、第二版正文、后续发送逐项落表。
- 胡嘉宁：说明为什么写灰字、为什么让陈景接着写；她不是反派，但她的提醒确实打开了口子。
- 陈景：承认自己将第一版另存为第二版，删掉灰字以后仍沿着灰字的顺序写。
- 老师：要求以文件时间和截图为准，把“想法”压回“动作”。

## 叙事目标

1. 老师要求打开第一版和第二版的文件属性、最近编辑时间、另存路径。
2. 陈景承认自己从第一版另存为第二版，先删掉顶端灰字，再写导语。
3. 唐骁追问：删掉的是字，还是边界。
4. 周屿把话压回文件动作，不让陈景被单独推出去，也不让“我只是润色”轻轻过去。
5. 陆沉将链条写成：第一版灰字、另存为第二版、灰字被删、意思留下、正文成句。
6. 林亦舟收到晚风只发“回寝记得报平安”的消息，外部生活在远处等着，不抢现场。
7. 章末发现第二版发出前还有一次“终版候选”复制记录，创建者不是陈景本人，而是 `WENAN-LAPTOP`，为下一章留下压力。

## 禁止写法提醒

- 不写系统、分支、判定、触发、变量等游戏口吻。
- 不写“让林亦舟意识到”“他突然发现”“下一句话会”等解释性旁白。
- 不写“你怕我怕”式心理并列。
- 不让晚风、老师或边缘人物替宿舍四人完成关系推进。
- 不写明德楼课程/班会误用，不写武生院站交通误用。
