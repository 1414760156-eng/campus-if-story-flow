# 第六卷第二章《票根》设计记录

## 调用技能

- planning-with-files：继续记录本轮写作进度和断点。
- superpower-brainstorming：先确定第六卷第二章的时间段、事件功能和篇幅边界。
- superpower-writing-plans：将本章写作拆成执行计划。
- chinese-novelist：使用中文小说式叙事，展示动作和对话，避免总结式说教。
- superpower-verification：写完后运行字数统计、禁用表达和地点逻辑扫描。

## 本章定位

第六卷已锁定 14-16 章，第二章必须推进一个完整时间段，而不是继续追表格细节。本章承接《第一格》中的项目实训方向表，进入寒假与春节，把“未来方向”从 C407 和公共桌带到离校登记、车票、家庭饭桌和微信消息里。

## 本章目标

1. 推进时间段：寒假前离校到春节夜。
2. 展示四人方向差异：
   - 唐骁把“项目/竞赛/实习”带回家，被家里问是否考研。
   - 周屿把普通话训练册和主持旧事带回家，不再让别人替他写经历。
   - 陆沉因排班和票价拖到最后离校，生活费压力继续露出。
   - 林亦舟在母亲买票、父亲红包、晚风消息和方向表之间迟迟不填完整。
3. 保留晚风现实化：只通过微信、抖音状态、短语音和春节问候出现，不替林亦舟处理家庭或宿舍问题。
4. 为下一章大三下春季铺垫：寒假结束前需要上传方向表修订版和一份能支撑第一意向的材料。

## 场景和动线

- 青枫居 4XX：离校前收拾、留校/离校登记、公共桌方向表。
- 武生院站：作为地铁站使用，只写地铁和换乘，不写成国铁站。
- 汉口站/武汉站：只作为国铁出发点的一闪，不展开旅行线。
- 家庭饭桌：现实追问、红包备注、生活用品减少和春节亲戚问去向。
- 微信/抖音：晚风和宿舍群的假期回声。

## 冲突

本章冲突是低烈度但持续的现实追问：你到底准备往哪走？人物不需要吵翻，但每个人都要被一句饭桌话、一次转账、一个票价或一个消息逼着改动纸上的字。

## 结尾钩子

除夕夜后，陈老师在班群发寒假修订要求：开学前上传方向表修订版和一份支撑第一意向的材料。林亦舟打开表格，发现自己那一栏仍然只有“项目实训”，材料附件是空的。
