# 第五卷第七十六章《退群记录》设计

## 目标

承接第七十五章结尾“宣传口小邵已退出群聊”，继续追查临时备份群里短暂入群的人到底是谁、谁给了他“宣传口”的身份感、他在退群前看过什么材料。章节要把“退了”写成一段仍需要核对的可见时间，而不是把邵闻直接写成替罪羊。

## 场景位置

- 场景仍在明德楼融媒体工作室，属于融媒体复核和材料核对，符合地点规则。
- 本章继续第五卷 5A 宿舍修补线，不开启 5X，不引入许澈，不让晚风或夏知微抢走本章核心。
- 宿舍四人仍在桌边：周屿继续拦住简单定罪，唐骁追问退群前的可见范围，陆沉记录入群时间、退群时间、群昵称和可见材料，林亦舟只负责把外部联系压回事实核对。

## 冲突推进

1. 姜宁说明“宣传口小邵”是群昵称，不是她私下备注；她当时因为昵称和一句“明天问宣传口”误以为对方来自宣传口。
2. 梁思予和许棠核对群成员导出记录，确认“宣传口小邵”就是摄影社临时外援邵闻，入群时间早于保存失效页截图。
3. 邵闻通过电话说明自己不是宣传口，只是想明天去问宣传口；他把群昵称改成“宣传口小邵”，是为了让别人回他、也为了让自己在临时工作里看起来有位置。
4. 唐骁追问：不是宣传口为什么要写宣传口；周屿挡住把邵闻单独推出去，因为二维码、昵称误认、手卡备份、截图可见范围都是多人动作叠出来的。
5. 陆沉把表格扩成：入群来源、群昵称、昵称来源、可见材料、保存动作、退群时间、本人说明。
6. 章末导出记录显示邵闻退群前发过一句“我先存一下，明天问宣传口”，并且这句话下面跟着一张“发布端预览_不要外传.png”的缩略图，把下一章压力推向“不要外传”的文件到底是谁发进备份群。

## 文风要求

- 用电话停顿、手机导出记录、群昵称、缩略图、表格空栏和现场物件推进。
- 不写系统口吻、游戏术语、上帝视角预告和解释性自我认知。
- 不出现“让林亦舟意识到”“下一句话会”等禁用句式。
- 不写许澈，不把 5X 提前带入。

## 验证

写完后运行字数统计，扫描第五卷禁用表达、明德楼误用、交通误写和新章新增文本。
