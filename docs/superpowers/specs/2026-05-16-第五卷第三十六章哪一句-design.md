# 第五卷第三十六章《哪一句》设计

## 断点

第三十五章末，拍照者在楼层群写“已经和 4XX 沟通过，他们也知道我不是故意的”，群里追问：“所以 4XX 是同意他说法了吗？”

## 章节目标

- 延续 5A 宿舍修补线，不开启 5X，不引入许澈。
- 把“沟通过”纠偏为“回过哪一句、没回哪一句”，避免私聊被拿来当背书。
- 周屿承担自己回过话的后果，但不把私聊截图整页甩到群里。
- 唐骁守住引用边界，陆沉补“遮掉别人求助句”的实际办法，林亦舟把晚风线里“承认听见不等于承认内容”的镜像带回来。
- 支线镜像使用秦越小舞台、夏知微摄影社、晚风微信和 C305 课堂，服务宿舍四人中心。

## 场景安排

1. 青枫居 4XX 夜里：四人围绕是否发截图、更正“沟通过”起争执，最后只发 4XX 自己回过的三句，不暴露对方完整求助。
2. 青枫居一楼窗口：楼长和宿管阿姨登记“4XX 回过什么”，拍照者试图坚持“沟通过就是知道”。
3. 西园餐厅：罗泽、丁祺、何书南处理小群里“截图呢”的追问，讨论凭什么要求别人公开私聊。
4. 教学楼 C305：陈老师用“引用自己的话”和“引用别人的话”训练，要求只展示自己可承担的那一句。
5. 东区商业街小舞台、阳光书屋、晚风微信：分别出现“我跟你说过，所以你知道”的镜像。
6. 章末钩子：拍照者提出“那我把完整私聊发出来”，把下一章压力推向私聊截图是否外发。

## 文风约束

- 不写系统提示、选择节点、变量、判定等游戏化词。
- 不使用“让林亦舟意识到”“下一句话会”“周屿怕/唐骁怕/陆沉怕”等解释性模板。
- 不使用 A302 或许澈；C305 继续作为第五卷计算机组成原理课堂地点。
- 明德楼不承载课程、班会、机房或答疑。
