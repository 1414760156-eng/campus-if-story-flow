# 人物外貌与穿着索引 Design

## Summary

用户指出当前母版缺少稳定的人物外貌与穿着描写，导致人物形象模糊。本次补强不改变人物设计和主线锁点，而是建立一套可持续调用的视觉规则：让外貌、发型、衣服、鞋、包、配饰、磨损、换季和场合选择，成为人物性格、经济状态、关系变化和时间流逝的可见证据。

## Chosen Approach

采用“总控规则 + 独立索引 + 进度登记”的方式。

总控只写原则，避免把正文写成设定表；独立索引负责给主要人物、支线人物、父母、老师和无名生活人物建立视觉锚点；进度表和 README 登记该索引，保证后续续写时会先查用。

不直接重写既有正文，因为当前任务是世界观补强。后续续写和总修时，再把这些视觉锚点自然织入章节。

## Requirements

- 外貌穿着不能只服务“好不好看”，必须服务性格、处境和事件。
- 不一次性给人物贴长相标签，不写成游戏立绘说明。
- 每个重要人物至少有初始视觉锚点、阶段变化、场景使用方式和禁忌。
- 宿舍四人优先，晚风、夏知微、沈嘉禾、秦越、许棠、许澈次级展开。
- 视觉细节必须可被后续剧情反复调用：背包、鞋、外套、发圈、胸牌、相机带、资料夹、工牌、旧小票等。
- 穿着要随年级、季节、经济压力、亲密关系、活动场合、考试周和毕业临近发生变化。
- 避免“高冷”“阳光”“女神”“男神”“清冷感”等空泛人设词。

## Open Risks

- 视觉锚点过强可能把人物固定成标签，因此索引要强调“重复中的变化”。
- 过度外貌化可能削弱现实感，因此正文使用时以动作和物件为主，不集中堆脸。
- 许澈在第四卷仍只能远影化，不能因为视觉索引提前变成稳定关系核心。
