# 第五卷第二十六章《谁能删》设计

## Summary

承接第二十五章末楼层群追问“楼长那里留到一周，谁有权提前删”，以及晚风询问“对方道歉了，是不是也可以提前删”。本章继续推进第五卷 5A 宿舍修补线，重点把“提前删除”从道歉、和解、逃避和抹平事实里拆出来。

## Chosen approach

采用“水房停顿 - 4XX 草稿 - 宿管窗口确认 - 西园餐桌现实质疑 - C305 删除条件 - 支线镜像 - 夜里删除申请”的结构。

这样能守住宿舍四人主轴：周屿想尽快把旧事从群里拿掉，唐骁担心提前删会让记录失去可信度，陆沉用图书馆换班记录说明删除也要有记录，林亦舟把晚风那边的道歉问题带回 4XX，但不让晚风线吞掉 5A。

## Scene beats

1. 青枫居水房门口，四人面对楼层群和晚风的同步追问，先不急着回，回到 4XX 写“提前删除条件”草稿。
2. 4XX 里围绕“道歉后能不能删”“当事人想删但旁观者还想看”“谁提出谁承担”发生低强度争执。
3. 一楼宿管窗口中，宿管阿姨要求提前删除不能靠群里投票，必须由保存人、涉及人、登记人共同确认，并留下删除时间。
4. 西园餐厅中，罗泽、丁祺、何书南把椅背旧照片和洗衣机材料拿来，提出“当事人想删，别人说还没看完”的现实疑问。
5. 教学楼 C305 中，陈老师让各组写删除申请样例，只看：谁提出、为什么删、谁确认、删后留什么记录。
6. 秦越小舞台、夏知微摄影社、晚风票务分别照出：道歉不等于可以替所有人删材料，提前删除也不能成为逃避后续责任。
7. 夜里 4XX 写好楼长登记的提前删除申请模板，章末真正有人私聊楼长要求“现在就删”，但没有说明理由。

## Constraints

- 不开启 5X，不让许澈出现。
- 不写“让林亦舟意识到”等解释性句子。
- 不写“下一句话会”式预告。
- 正文不出现游戏系统词。
- 课堂地点继续使用教学楼 C305；明德楼不承载课程、班会、答疑、机房或上机。
- 交通不新增校外动线。
- 晚风只作为镜像和手机现实压力，不开启 5C。
