# 第五幕可玩性系统与 5A 宿舍修补线样板 Design

## 背景与边界

本轮方向已经确认：下一步先优化可玩性系统骨架，再写一条具体样板分线，最后按样板批量展开其他分线。第五幕不是晚风线核心，也不是持续混乱堆叠，而是把前期并行积累的多条线整理成几条清晰可玩的单线。

必须锁住的主线内容不变：宿舍四人关系为周屿、林亦舟、唐骁、陆沉；关键事件为军训手机袋事件、汇演、班会、主持招新、毕业活动。第五幕新增的现实冲突只改变心理压力、关系温度、资源状态和第六幕输入，不改写这些主线事件。

## 方案比较

### 方案 A：一次写完 5A-5F 全部分线

优点是产出量大，能很快看到完整第五幕。风险是每条线的互动规格不一致，容易再次把某一条线写成核心，导致“多线收束”变回“多线堆叠”。

### 方案 B：只先做系统，不写具体剧情

优点是规则干净，便于 JSON 实现。风险是文本层仍然抽象，缺少样板后，后续扩写容易变成机制表而不是沉浸剧情。

### 方案 C：系统骨架 + 5A 样板线

这是推荐方案。先建立第五幕单线合同，再用 5A 宿舍修补线跑通完整节点、选择、状态、强弱冲突分歧和输出。5A 最适合作为样板，因为宿舍矛盾既现实高频，又不会天然抢走晚风、家庭或学业线的主题位置。

## 第五幕单线合同

每条第五幕分线必须满足同一套结构：

1. 入口盘点：先在 `ACT5_00_CONFUSION_AUDIT` 显示玩家前四幕真实触发过的压力源，只显示已触发内容。
2. 分线整理：在 `ACT5_01_LINE_ROUTER` 选择一条主处理线，最多允许标记一条次处理线。未选择线进入低温保留，不在本幕频繁插入打断。
3. 单线推进：进入分线后，至少包含“引爆、取证、对谈、复发、关键选择、输出”六类节点。
4. 状态记录：每个节点必须写入关系、心理、资源或记忆池变化，避免只有对白没有游戏后果。
5. 回响输出：每条线结尾必须生成第六幕可读取的输出标签。

每个节点建议使用以下字段：

| 字段 | 用途 |
|---|---|
| `节点ID` | 与 JSON 节点保持一致 |
| `场景` | 限定在五神院沉浸版空间与合理动线 |
| `出场角色` | 支线人物必须真实出现或以消息、语音、物件留下可互动痕迹 |
| `事件` | 具体可感的现实动作，不写抽象心理总结 |
| `玩家选项` | 至少 3 个，关键节点 4-5 个 |
| `即时反馈` | 当场对白、动作、关系温度变化 |
| `延迟回响` | 第六幕或终章可调用的记忆、资源、关系状态 |

## 5A 宿舍修补线设计

5A 的核心不是“宿舍吵架”本身，而是把大学里最常见的生活习惯差异写成有后果的关系压力：外放、空调温度、公共桌面、借物不还、夜里作息、带人回宿舍、群聊阴阳怪气。冲突强度由前期状态决定。

| 状态组合 | 冲突形态 |
|---|---|
| 完美线倾向、宿舍信任高 | 弱冲突，玩家更像调停者，重点是规则重谈 |
| 纯摆烂倾向、宿舍互动低 | 弱冲突，玩家更自我，重点是逃避和低温共处 |
| 宿舍信任低、现实压力高、借物/作息问题累计 | 强冲突，可能出现摔门、公开站队、短期冷战 |

### 5A 节点序列

| 节点 | 场景与事件 | 玩家操作 | 输出 |
|---|---|---|---|
| `ACT5_5A01_DORM_COLD_START` | 夜里回到宿舍，公共桌面堆着外卖袋，空调温度被改，唐骁戴耳机但音乐漏音，周屿在床上不说话，陆沉把洗漱盆放回原位 | 收拾桌面、直接提醒唐骁、问周屿怎么了、先去走廊缓一缓 | 宿舍低温、个人压力、周屿信任或唐骁防御 |
| `ACT5_5A02_OLD_DEBTS_SURFACE` | 宿舍群弹出旧账：充电器、热水卡、衣架、夜归声响、谁把门禁卡放丢 | 翻聊天记录、线下摊开说、私聊其中一人、装作没看见 | 旧账清单、回避值、宿舍火药味 |
| `ACT5_5A03_LUCHEN_HALLWAY_LISTEN` | 走廊水房边，陆沉第一次把“不介意”说成“我只是懒得吵” | 追问、承认自己没注意、替唐骁解释、让陆沉自己说 | 陆沉表达度、林亦舟自省、后续班会/毕业回响 |
| `ACT5_5A04_RULE_REOPEN_MEETING` | 四个人在宿舍重谈规则，手机放在桌上计时，没人愿意先开口 | 站周屿、站唐骁、把自己问题放出来、要求白纸黑字、提前结束 | 弱修补、表面和平、站队裂痕或强冲突 |
| `ACT5_5A05_AFTER_ARGUMENT_MEAL` | 第二天食堂或宿舍门口，第一次决定要不要一起吃饭 | 主动喊人、只喊一个人、把饭放桌上不说话、各吃各的 | 修补温度、低温共处、短期冷战 |
| `ACT5_5A06_EXAM_WEEK_RELAPSE` | 期末前夜旧习惯复发：外放、灯光、资料散落、消息震动 | 按新规则提醒、忍到天亮、发群消息、当面爆发 | 规则可信度、压力爆点、第六幕宿舍状态 |
| `ACT5_5A99_DORM_OUTPUT` | 分线输出页，以物件和动作收束：门口拖鞋、公共桌面、空调遥控器、四人饭卡记录 | 玩家确认本线结果并写入记忆池 | `宿舍修补`、`低温共处`、`公开站队`、`短期决裂` |

## 强冲突分歧规则

强冲突不是固定剧情，而是概率与状态共同触发。触发条件包括：

- `宿舍互信 <= 35`
- `现实压力 >= 70`
- `回避值 >= 2`
- 前四幕触发过“借物不还、夜聊影响作息、宿舍公共区不收拾、替别人和稀泥失败”中的任意两项

满足两项时，5A04 进入高压对话；满足三项及以上时，5A04 增加强冲突选项与更重即时反馈。完美线和纯摆烂线默认不进入强冲突，除非玩家主动选择激化选项。

## 与其他分线的接口

5A 输出只影响宿舍状态和第六幕回响，不抢其他分线的主题位置：

| 输出标签 | 影响 |
|---|---|
| `宿舍修补` | 第六幕宿舍能自然出现共同复习、毕业互拍、收拾行李 |
| `低温共处` | 第六幕仍同住但对白减少，毕业活动存在距离感 |
| `公开站队` | 周屿或唐骁一侧温度上升，另一侧下降 |
| `短期决裂` | 第六幕需要通过毕业活动或最后一个新年做最低限度回收 |

5A 可以轻微接入其他线，但不让其他线打断单线。例如晚风只可通过夜间消息影响作息压力，父母线只可通过生活费延迟影响宿舍借钱，学业线只可通过资料散落和复习时间影响冲突。

## 验收标准

本设计落地后，应满足以下标准：

- 第五幕仍是多条清晰单线，不回到晚风核心。
- 5A 有完整“引爆、取证、对谈、复发、选择、输出”结构。
- 每个 5A 节点至少有 3 个可操作选项，关键节点有 4-5 个选项。
- 强冲突可以出现，但只在合理状态下出现。
- 完美线与纯摆烂线更偏弱冲突，符合“更关注自身、更自我”的玩家轨迹。
- 所有事件发生在宿舍楼、走廊、水房、食堂、教学楼等五神院沉浸版合理空间内。
- 5A 不改变主线事件，不挪动军训手机袋事件、汇演、班会、主持招新、毕业活动。
