# 第五卷第一百章《备注》设计记录

## 本轮目标

承接第九十九章结尾的活动复盘到场确认表，把第五卷收在“备注栏到底写什么”这一件具体小事上。章节要完成第五卷收束，不再开启新追查；台卡、说明、留存表、直播截图、4XX 宿舍分工和周屿临时提示规则都要回到一行可留存的事实里。

## 结构设想

1. 东区商业街后台晨间复盘：唐骁和林亦舟面对秦越、姜宁、许棠、梁思予，先核三份材料。
2. 备注栏争执：唐骁想写清周屿不是主持，秦越怕写得太长反而扩大争议；姜宁补主持事实，许棠补发布端说明，梁思予补留存编号。
3. 4XX 远程在场：唐骁把草稿发到 4XX 群，周屿只回“可以”，陆沉从图书馆发来排班间隙的核对意见。
4. 最终备注：用事实性短句确认“第二环节由姜宁主持；周屿临时提示规则；台卡原件与活动说明已留存；主持名单不补填”。
5. 回寝室：林亦舟和唐骁带复印件回青枫居，路过东区商业街早晨、图书馆、食堂和宿管窗口，把卷尾落回日常。
6. 第五卷尾声：公共桌空出一块位置，四个人没有正式和解，但一张新课表的位置出现，给第六卷入口留下状态。

## 文风要求

- 使用 `chinese-novelist` 原则：展示而非讲述，冲突由纸张、手机、笔迹、表格和动作推动。
- 不写“成长”“意识到”“后续影响”这类解释。
- 不出现玩家可见系统词：主线、副线、分线、节点、flag、触发、权重、判定、回主线、资源线。
- 不使用上帝视角句式，如“下一句话会”“命运开始分叉”。
- 不新增第五卷追查对象；这是收束，不是继续扩案。

## 需要保留的主轴

- 宿舍四人优先：周屿不在现场但通过签名照片和群消息进入；陆沉通过记录顺序进入；唐骁和林亦舟现场签字。
- 周屿不能被备注重新写成主持人，也不能被彻底洗成无关者。
- 唐骁的变化要体现在克制措辞，不写替人吵架。
- 林亦舟的变化要体现在少转发、少解释、把纸带回公共桌。
- 晚风只做现实消息回声，不替宿舍判断。

## 卷尾钩子

第五卷结束在一张空白课表或公共桌空位上。不是大团圆，而是四个人终于能把下一件事放在同一张桌上；第六卷可以从新的课程、实习、考证、毕业路径和第六幕硬结局入口继续。
