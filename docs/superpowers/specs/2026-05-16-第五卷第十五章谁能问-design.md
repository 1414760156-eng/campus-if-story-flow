# 第五卷第十五章《谁能问》设计说明

## 使用技能

- `superpower-brainstorming`：先确认本章只推进 5A 宿舍修补线，不把晚风、秦越或楼层同学抬成核心。
- `planning-with-files`：以本文件和执行计划承接断点，写完后回写目录与进度。
- `superpower-writing-plans`：先拆场景与风险，再改正文。
- `chinese-novelist`：正文展示而非讲述，避免系统口吻和总结式心理说明。
- `superpower-verification`：完成后做字数、地点和禁用表达扫描。

## 断点

第十四章结束于 4XX 公共桌贴上窄纸：

> 公共桌的事，可以问事，不先问人。

这一句已经从课程互看和宿管抽查转回寝室内部。第十五章要把它推进一步：当楼层同学、活动名单、课程组也开始“问进来”时，四人是否能守住问事不问人的边界。

## 本章目标

1. 让外部关系进入 4XX，但不夺走宿舍四人轴心。
2. 让“空表能不能借出去”成为具体事件，而不是抽象边界讨论。
3. 用罗泽、丁祺、何书南的寝室问题补强楼层生活，让支线人物有自己的困扰。
4. 用秦越的活动名单形成外部镜像：周屿需要给准话，但不能被活动吞掉。
5. 晚风线只作现实侧面回声，继续保持“不替别人说”的处理方式。
6. 章末留下下一步压力：空表被楼层群转发后，谁能改、谁能问、谁能看开始失控。

## 场景设计

1. 青枫居清晨：窄纸被隔壁路过同学看见，罗泽开玩笑问是不是寝室规章。唐骁不舒服，周屿用玩笑挡，陆沉注意到走廊目光。
2. 西园早饭：罗泽、丁祺、何书南讲自己寝室椅子、外卖架和晾衣绳的问题，想借“空表”。四人第一次讨论给不给。
3. C305：课程里助教要求互看便签整理为空白格式，强调只提交格式和例句，不提交寝室内容。沈嘉禾组侧面补充“空白也会被别人填成自己的事”。
4. 东区商业街小舞台消息：秦越要周屿确认临时替场，周屿学着先发时间地点和自己能不能到，不把“可能”说成“会去”。
5. 青枫居夜晚：罗泽拿着空表来敲门，丁祺和何书南在走廊等。4XX 只给空表，不给公共桌照片；何书南问能不能写“谁放坏椅子”，唐骁改成“椅背松动，暂未确认来源”。
6. 章末：楼层群里出现空表照片，另一个寝室同学开始把私人争执写进去。4XX 门口再次有人敲门，问“这格能不能写名字”。

## 风险控制

- 正文不出现“主线/节点/触发/变量/判定/系统提示”等游戏话。
- 正文不写“让林亦舟意识到”一类直白自我认知。
- 不让明德楼承担课程、班会、答疑或机房功能。
- 第五卷不出现 A302，计算机组成原理继续在教学楼 C305。
- 许澈不进入本章关系核心。
