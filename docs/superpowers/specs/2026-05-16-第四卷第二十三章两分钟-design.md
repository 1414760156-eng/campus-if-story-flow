# 第四卷第二十三章《两分钟》设计

## 目标

续写第四卷第二十三章，承接第二十二章末“课堂两分钟说明示例”“计算机二级今晚必刷”“周屿东区商业街小舞台走位”三条压力。章节要推进宿舍四人关系，不开启新线，不让支线抢主轴。

## 章节核心

“被当成示例”不是奖励，而是新的暴露。四个人前一晚刚补完站位图，第二天又要在课堂上把数据来源边界讲给全班听。唐骁因计算机二级题库被挤压而焦躁，陆沉用纸质材料托住追问，林亦舟负责把说明说得像人话，周屿嗓子未恢复，还要在晚上赶去东区商业街小舞台走位。

## 场景安排

1. 青枫居清晨：唐骁刷二级选择题出错，周屿含润喉糖，陆沉整理纸质本，林亦舟把两分钟说明压成三段。
2. 教学楼 A302 课前：助教要求先讲；同学对“为什么不拍视频”“为什么不补估”追问，四人各自接一部分。
3. 课堂后走廊：陈老师提醒边界说明可以作为课程材料，但不要写成表扬稿；唐骁错过二级群讲题，周屿收到秦越催到场。
4. 西园或便利店短场：四人用常温水、包子或饭团压一压疲劳，不写庆功。
5. 东区商业街小舞台前：周屿走位，秦越压缩主持词；林亦舟和唐骁/陆沉可短暂停留或远看，宿舍线仍在场。
6. 章末钩子：课堂材料被学习委员发到班级群作为参考，另一组同学私聊林亦舟索要“能不能直接套用”的版本；同时二级群继续催题，周屿小舞台临时缺人需要加一段。

## 约束

- 不使用“主线、副线、节点、触发、判定”等游戏系统词。
- 不写“让林亦舟意识到”“下一句话会”式上帝视角。
- 避开“觉得/发现/突然/忽然”等第四卷高风险词。
- 班会、课堂、示例说明只在教学楼，不进明德楼。
- 支线只提供压力和生活质感：秦越可出现，许澈不建立关系，晚风最多短消息。

## 预期效果

第二十三章把外卖架实验收束为“公共材料压力”，让四人短暂获得认可但付出新的时间代价。下一章可以继续写班级群套模板、二级错题、东区小舞台临时加词，或转入第四卷后段家庭/晚风/专业课交叠。
