# 第五卷第五十章《半截卡片》设计

## 目标

承接第四十九章结尾：旧预览半截卡片出现在朋友圈。把冲突写成小范围但真实的“私人转发怎么处理”，避免升级成校园舆情，同时让问题从线上滑向第二天活动现场的实体材料。

## 写作原则

- 不开启 5X，不引入许澈。
- 不把朋友圈截图写成全校围观，只写许棠看到、发圈人换图、四人处理边界。
- 宿舍四人优先：周屿参与处理但不被代言，唐骁压住质问，陆沉限制截图外传，林亦舟负责在几方之间传话。
- 支线人物做具体事：许棠联系发圈人，梁思予确认新卡片，秦越和姜宁负责活动现场材料。
- 场景从青枫居 4XX 过渡到次日东区商业街小舞台，遵循校园动线。

## 事件链

1. 林亦舟收到许棠发来的朋友圈截图，四人重新聚到公共桌。
2. 许棠说明这不是公开平台，只是朋友圈，但已经有人点赞评论。
3. 四人讨论是否要求删除、是否要追问谁保存，周屿明确不要逼别人道歉，只换图。
4. 发圈人私下回复“只是帮忙预热”，许棠把沟通压成换新图和改配文。
5. 梁思予补发正式卡片，秦越在活动大群固定使用新卡片。
6. 次日清晨四人赶去东区商业街小舞台核材料。
7. 现场线上卡片已换，但第一批打印出来的节目单/桌签仍残留旧导语，进入下一章。

## 禁区

- 不出现开发者口吻和系统提示。
- 不出现“让林亦舟意识到”等自我认知式旁白。
- 不使用明德楼、A302、公交、火车到武生院站等错误地点动线。
- 不写成全校舆情或大规模网暴，只保留大学生活里常见的“半公开转发”和材料残留。
