# 第四卷第四十一章《撤不回去》设计

## 写作目标

- 承接 `未完成项说明示例版_整理自22组.docx` 在课程群出现后的课堂现场，写清文档来源、撤回、责任和私人原因外传造成的现实压力。
- 把冲突继续压回青枫居四人关系：他们不是“出题人”，却因空目录被模仿而被推到说明位置；四人要在不替别人承担、不推开同学、不把梁嘉明继续暴露之间找做法。
- 保持第四卷的“暑假未停、学业和生活持续挤压”调性：唐骁二级入口、晚风票务投诉、周屿小舞台安排只做现实余音，不抢走课堂外传事件。

## 关键锁点

- 场景只使用教学楼 A302、阳光书屋、青枫居、图书馆西侧、东区商业街消息余音等合理动线。
- 明德楼不承载课堂、班会、咨询会、试训、机房或课程追问。
- 不启动 5X 唯一外流；第四卷只保留“想离开原关系”的情绪影子。
- 正文不出现玩家、系统、节点、触发、判定、主线、副线等开发口吻。
- 避免解释性句式，不写“让林亦舟意识到”“下一句话会”等上帝视角。

## 章节事件链

1. A302 群消息炸开：文件被撤回又重发截图，邓鸣作为误转发者露面，承认从罗泽组群里拿到文件后改名发到课程群。
2. 梁嘉明被私人理由公开刺到，先在门口或群里追问，再进教室要删源文件；他的情绪不是戏剧化崩溃，而是被同学看见后的尴尬和硬撑。
3. 陈老师与助教收束现场：要求相关组留五分钟，课堂继续；下课后只处理来源、撤回、授权和更正，不让任何人用“大家都是好意”盖过去。
4. 阳光书屋短会：罗泽、丁祺、邓鸣、梁嘉明和22组对齐更正文本。四人只提供“更正格式”，不替写具体句子。唐骁坚持留来源，周屿收住玩笑，陆沉要求本人确认，林亦舟负责把话说短。
5. 晚风票务线并行：她问能不能把室友朋友圈一起删掉；林亦舟只给做法，不代她们发声。
6. 青枫居夜里回收：四人回到公共桌，二级成绩入口卡顿、图书馆排班、东区小舞台录音并排。章节末，课程群出现助教通知：明早每组提交“更正说明与撤回确认”，且点名22组和罗泽组各派一人说明。

## 人物状态

- 林亦舟：要控制自己不把现场变成主持式圆场，更多做“删多余话、留事实”的动作。
- 周屿：被“公开场面”诱惑开玩笑，但这次能压住，说明他不是不会严肃，只是常用笑挡压力。
- 唐骁：对来源和文件名敏感，推动留证和更正；二级成绩入口只压在他肩上。
- 陆沉：更重视本人确认和边界，少说但落在关键处。
- 罗泽：好意与偷懒混杂，开始付出具体代价。
- 丁祺：比罗泽更早看见外传后果，帮助稳定沟通。
- 邓鸣：误转发者，不能写成恶人；他把“帮大家省事”当理由，正好照出模仿的真实危险。
- 梁嘉明：被公开私人理由后强撑，要求删掉并改成事实，不承担“故事素材”的位置。

## 结尾钩子

助教在课程群发布明早提交要求，四人以为只要更正就能收束，却看见群附件里多了一份“下载名单截图”，有人已经把文档转出课程群。
