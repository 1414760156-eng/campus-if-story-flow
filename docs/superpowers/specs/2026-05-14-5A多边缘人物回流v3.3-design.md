# 5A 多边缘人物回流 v3.3 设计说明

## 背景

v3.2 已经把湖边回流做成“先选晚风/夏知微，再选是否聊刚发生的宿舍事”的微心态聊天结构。用户进一步指出，同样的结构也应扩展到食堂侧门和后段外流：既然沈嘉禾、秦越等人物已经存在于其他主线，就可以在 5A 内作为边缘人物被拉回来，增加回流线的小说感和可操作性。

## 核心规则

- 第五幕的核心不是扩大边缘人物戏份，而是把玩家从混乱多线拉回几个强关系的人。宿舍四人关系仍是 5A 的主轴，边缘人物只承担补充世界厚度和照出主角心态的功能。
- 可以带入其他主线的命名边缘人物，但只能作为软回流或软擦肩。
- 这些人物不能解锁自己的主线，例如沈嘉禾不能开启 5D 学业线，秦越不能开启 5F 资源线。
- 每个边缘人物出现时都要给玩家至少两个话题方向：聊刚发生的宿舍/饭桌现实，或不聊主线只聊对方所在领域。
- 分支最终必须回到 5A 主轴，或在 5X 中继续外流定轨，不开放回流抉择。
- 影响范围限制为心态、人物好感、资源感、回避值、记忆标签和终章回响。

## 食堂侧门扩展

在 `ACT5_5A16_CANTEEN_SEAT` 中，除便利店买水和夏知微侧门路线外，新增两条食堂侧门软回流：

- 沈嘉禾：从复习资料和小组合作角度映照宿舍饭桌的沉默。
- 秦越：从活动名单、资源和临时机会角度映照林亦舟是否又想把注意力投向别处。

每条路线都应有“聊刚发生的饭桌/宿舍事”和“不聊宿舍，只聊对方领域”的差异，并最终回到 `ACT5_5A16E_SIDE_DOOR_BACK` 或 `ACT5_5A17_AFTER_MEAL_CHAT`。

## 5X 外流扩展

5X 外流线中，许澈仍是唯一新人关系核心。但可以让秦越作为已存在边缘人物擦肩出现：

- 秦越不把玩家带回 5F。
- 秦越只提供活动、名单、临时机会的软牵引。
- 这次擦肩强化外流线的主题：原来的各条线还会经过玩家身边，但都只是远处的影子，不再开放旧线抉择。

## 验收标准

- 标题包含 `v3.3`。
- 节点数不低于 73。
- 总选项数不低于 176。
- 心理级微选择不低于 50。
- 回环节点数不低于 41。
- 微心态聊天节点数不低于 20。
- `ACT5_5A16_CANTEEN_SEAT` 至少 4 种回流方式。
- 食堂侧门必须出现夏知微、沈嘉禾、秦越三名边缘人物。
- 5X 外流线必须保留许澈核心，并新增秦越软擦肩节点。
- 所有新增边缘人物节点不得解锁 5B-5F 或新增外流点。
