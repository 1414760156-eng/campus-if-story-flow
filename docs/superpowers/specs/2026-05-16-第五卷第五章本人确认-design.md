# 第五卷第五章《本人确认》设计记录

## 本章定位

承接第五卷第四章《能用的桌子》末尾的双重压力：课程群要求明天下午五点前提交课程协作时间表最终版，每人本人确认；晚风室友改口，不想交投诉材料，只想把被骗的钱追回来。

本章继续服务 5A 宿舍修补线。重点放在青枫居 4XX 公共桌、课程协作表、本人确认、晚风材料边界和家庭转账暗线。晚风线只作为现实压力，不抢走宿舍四人的中心位置。

## 叙事原则

- 展示而非讲述：用签名栏、打印纸、撤回的语音、聊天截图、桌面物件和空椅子呈现人物状态。
- 冲突驱动：冲突来自时间表最终版、本人确认、晚风室友改口、父亲未回复、宿舍公共桌能否持续被使用。
- 宿舍优先：周屿、唐骁、陆沉、林亦舟必须共同在场处理课程表；任何外部压力都要回到公共桌。
- 不替别人写：课程表、票务材料、活动证明、图书馆排班和家庭私事各自归位。
- 地点合规：班会/课程相关内容使用教学楼或青枫居；不把明德楼用于课堂或班会。

## 场景安排

1. 青枫居 4XX 夜间公共桌：四人拆糖纸，面对课程群与晚风消息；唐骁把最终版表格打印出来，要求各自改自己的栏。
2. 青枫居走廊与水房：周屿接秦越补录消息，顺手在走廊里听见隔壁寝室代签争执，强化本人确认的现实感。
3. 阳光书屋短停：林亦舟为晚风整理追回钱的流程，只给步骤，不替室友写原因；沈嘉禾短暂出现，提供“材料和愿不愿意交材料不是一件事”的实际经验。
4. 青枫居 4XX 次日中午：四人把最终版写到可提交状态，陆沉坚持图书馆排班只写本人能确认的时段，周屿不把小舞台证明当课程贡献，唐骁不替任何人润色。
5. 教学楼 C305 外走廊：提交前最后核对，学习委员催统一签名，22 组拒绝代签，留下下一章课堂抽查入口。

## 人物推进

- 林亦舟：把晚风票务追回钱的步骤和课程协作表分开；父亲的转账消息继续悬着，母亲的生活用品消息留在抽屉和快递箱里。
- 周屿：不再用玩笑把活动压力塞进课程表；补录表只作个人活动材料，签名由自己写。
- 唐骁：从“把表做完”转向“只把能确认的写进去”；不替别人把话改漂亮。
- 陆沉：坚持本人排班本人写，少说但把签退和换班截图分清。
- 晚风：通过微信、语音和室友改口进入现实压力；她不能替室友做决定，也不能让林亦舟替她们完成材料。
- 沈嘉禾：作为阳光书屋短暂交叉人物，提供资料分类经验，不替主角下结论。

## 章节钩子

结尾落到教学楼 C305 外的提交前：学习委员拿着一沓表格催签，22 组只交本人确认的版本；课程群随后通知明天课堂随机抽两组，问“空着的地方为什么空着”。这把压力推向下一章。
