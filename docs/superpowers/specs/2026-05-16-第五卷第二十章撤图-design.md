# 第五卷第二十章《撤图》设计记录

## 写作目标

- 承接第十九章末尾楼层群出现留底页局部截图。
- 把“公开页 / 留底页”从方法推进到风险：内部材料存在意义，但一旦被截走，也会成为新的伤害。
- 继续以青枫居 4XX 四人为中心，丁祺、罗泽、何书南、宿管阿姨、秦越、晚风只承担镜像压力。
- 不开启 5X，不让许澈进入关系核心，不改变第五卷 5A 宿舍修补方向。

## 章节结构

1. 青枫居 4XX 夜间：楼层群截图出现后，周屿带着麻辣烫回来，四人先压住互相指责，确认截图角度和纸夹位置。
2. 走廊与一楼宿管窗口：宿管阿姨要求先撤图、别堵门、别在群里审人；同时指出 4XX 把留底页放在公共桌边缘也有问题。
3. 丁祺当面解释：他不是单纯恶意，而是觉得 4XX 要求别人分公开和留底，自己却在背后记原话。唐骁和周屿被迫把“我们自己的话”和“拿别人作材料”说清。
4. 秦越和晚风镜像：社团内部备注截图、票务留底目录外传欲望分别照出同类问题。
5. 青枫居 4XX 收束：四人重新放置牛皮纸夹，公开发一条短说明，不公开留底内容；章末课程群通知周一 C305 讨论“材料被截走后的更正说明”。

## 人物推进

- 林亦舟：压住替所有人解释的冲动，只写能公开的话，并在晚风线里阻止把留底目录发成泄愤材料。
- 周屿：承认自己把夹子推得太靠外，也承认被截出的那句话确实伤人。
- 唐骁：从要求立刻抓人，转向承认留底页也需要保管边界。
- 陆沉：继续用动作拆分责任，提出留底页只留目录、内容另收。
- 丁祺：不是工具人。他的越界来自被公开规则压过后的反弹，要求 4XX 也解释自己的标准。
