# 第五卷第六十九章《六点四十》设计

## 目标

承接第六十八章“明早八点前到明德楼当面复核”，写完整一章清晨材料复核。章节必须把地点限定为青枫居到明德楼融媒体工作室的合理动线，不把明德楼误写为课程、班会或答疑空间。

## 叙事职责

- 青枫居 4XX 清晨：闹钟、源文件、U 盘、截图、白卡、没睡透的四人状态。
- 路线：从青枫居出发，经校内清晨道路到明德楼，体现时间紧、校园醒来、早餐摊和宿管阿姨的短促生活感。
- 明德楼用途：融媒体工作室/材料复核。老师查看源文件、取消预约记录、旧预览截图、群内说明，不承载教学或班会。
- 宿舍四人优先：周屿负责本人边界，唐骁负责文件时间戳和硬话收住，陆沉负责清单和留存，林亦舟负责在几方之间传话但不抢结论。
- 支线人物仅软参与：许棠、梁思予、秦越承担材料端和活动端工作；晚风只以短微信外部回声出现，不进入核心。

## 冲突与钩子

本章冲突不是“旧句还没删干净”，而是当面复核时发现旧预览的传播路径比群里表面追问更靠近权限端：失效页面截图不是普通围观者随手点开，而是有人拿过后台预览链接。章末把下一章压力留给“谁有预览权限、谁把链接带出了工作小群”的追问。

## 风格约束

正文不使用系统口吻、数值提示、上帝视角预告或抽象认知总结。心理状态通过动作、物件、语气和停顿呈现。禁用词继续按总控扫描。
