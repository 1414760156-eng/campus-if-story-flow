# 5A 回流小段扩写 v2.2 设计

## Summary

用户指出：5A v2.1 的回流结构方向正确，但回流里的剧情过少，玩家从主轴离开后只经历一个选择就回主线，显得刻意、机械。v2.2 的目标是保留“短回流”结构，但让每个回流都至少有两拍：离开现场后的环境/心理承接，以及一个具体的人物、消息或物件互动，再回到主轴。

## Design

- 不改变 5A 主轴事件顺序，不新增外流点，不跳入 5B-5F。
- 4 个现有回流点都扩为两节点小段：
  - 楼外长椅：`ACT5_5A01A_OUTSIDE_BENCH -> ACT5_5A01B_OUTSIDE_RETURN -> ACT5_5A02_DOORWAY_PAUSE`
  - 水房洗脸：`ACT5_5A10A_WASHROOM_RESET -> ACT5_5A10B_WASHROOM_MESSAGE -> ACT5_5A11_TABLE_TIMER`
  - 便利店买水：`ACT5_5A16A_STORE_WATER -> ACT5_5A16B_STORE_RETURN -> ACT5_5A17_AFTER_MEAL_CHAT`
  - 走廊两页：`ACT5_5A19A_HALLWAY_REVIEW -> ACT5_5A19B_HALLWAY_BACK -> ACT5_5A20_RELAPSE_CHOICE`
- 每个新增节点必须：
  - 标记 `回环节点: true`
  - 指定同一个 `回流节点`
  - 有至少 2 个选项
  - 每个选项有 80 字以上反馈
  - 不带 `外流点`
  - 不解锁 5B/5C/5D/5E/5F

## Acceptance Criteria

- 标题标记 `v2.2 回流小段扩写`。
- `chain` 节点数从 31 增至 35。
- 总选项数不少于 92。
- 每个回流小段从触发点进入后，需要至少两次选择才抵达回流主轴。
- 原唯一外流点仍只有 `ACT5_5A15_DOOR_SLAM_RISK`。
- 原 v2.1 微选择规则继续有效。

## Scope

本轮只扩写 5A 四个已存在的回流段，不新增新回流点，不重写主轴。
