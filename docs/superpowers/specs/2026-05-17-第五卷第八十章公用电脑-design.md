# 第五卷第八十章《公用电脑》设计记录

## 目标

承接第七十九章末尾 `host_temp_03` 公用电脑账号，把明德楼融媒体工作室内的材料核对带到东区商业街后台。第八十章不寻找单一替罪羊，而是把临时文件进入正式素材的原因写成更真实的大学活动漏洞：公用电脑自动登录、共享盘同步、散场后统一收包、多个同学轮流坐下操作，导致“谁拖进去”与“谁造成后果”无法简单重合。

## 本章设计

- 开场仍在明德楼融媒体工作室：老师要求马上去看公用电脑，避免在群里继续猜。
- 动线从明德楼出发，经自强路、健源体育场外侧，抵达东区商业街后台，写出校园空间移动感。
- 东区商业街后台要有具体生活质感：折叠桌、缠线带、半干奶茶印、贴纸、舞台灯箱、旧键盘、自动登录的公用账号。
- 宿舍四人继续在场：唐骁追具体操作，周屿拦住“谁坐过就谁背”的粗暴指认，陆沉把时间、文件、账号、动作拆开记录，林亦舟收住晚风外部问包消息。
- 秦越、袁承、许棠、梁思予和老师参与核对。叶青可作为已有边缘人物通过记录/电话进入，说明散场收包并非恶意，而是旧习惯。
- 章末钩子：公用电脑最近位置露出 `正式素材_给冯喻.zip`，修改时间 02:04，把下一章压力引向“冯喻拿到的到底是哪一包”。

## 写法约束

- 正文不写游戏开发口吻，不写解释性心理总结。
- 不使用禁用表达和上帝视角提示。
- 明德楼只用于融媒体核对；东区商业街只作为活动后台和商业街场景。
- 不引入许澈，不开启 5X。
- 晚风只保留外部回声，不抢占第五卷核心。

## 预期推进

第八十章应完成一轮完整小节：从明德楼提出核验，到东区商业街后台开机查记录，再以 `正式素材_给冯喻.zip` 结束。下一章可继续追冯喻收到包的来源、用途和二次传播。
