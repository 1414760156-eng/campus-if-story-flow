# 第五卷第九章《留着原样》设计

## 目标

承接第八章结尾的整改照片与设备原始记录要求，把 5A 宿舍修补线从“当晚摆好、当晚拍照”推进到“隔天、后天是否还能保持”。本章重点不是新增大事件，而是让课程记录、宿舍公共区和晚风票务三处都要求保留原样，压出四人的真实生活习惯和责任分配裂缝。

## 叙事策略

- 以青枫居 4XX 的物件持续状态开场：公共桌、话筒线、设备清单、洗衣液、外卖袋、陆沉旧包、四行纸都必须继续在场。
- 课程场景继续使用教学楼 C305，明确属于大二上《计算机组成原理》；不再出现 A302。
- 晚风线只做现实镜像：她那边也遇到“材料不能只发一次、要留得住原始记录”的问题，但不抢走 5A 的宿舍中心。
- 冲突落在具体动作：谁把东西临时挪回去、谁忘了补原始记录、谁想先拍照再恢复舒服状态、谁要赶另一个场。
- 结尾保留下一章钩子：课程群与楼层群同时要求“当前状态”，四人必须面对一次更公开、更难临时补救的检查。

## 场景安排

1. 青枫居清晨：昨晚拍照后桌面看似干净，但起床后的使用让东西重新溢出。
2. 西园或北园短饭：四人讨论“留原样”的实际成本，不写成开会说教。
3. 教学楼 C305：助教要求保留本人设备原始记录，指出有组拍完照后删掉原始记录。
4. 阳光书屋/图书馆短停：林亦舟处理晚风票务材料，强调不要替付款人补话。
5. 青枫居傍晚：楼层群抽查“当前公共区”，有人提议先拍再挪，四人分歧变尖。

## 风格约束

- 正文不出现系统、节点、变量、判定、回流等游戏设计词。
- 不写“让林亦舟意识到”“下一句话会”等上帝视角提示。
- 不用“周屿怕、唐骁怕、陆沉怕”式并列心理总结。
- 用物件位置、消息时间、手上动作和短句推进人物状态。
