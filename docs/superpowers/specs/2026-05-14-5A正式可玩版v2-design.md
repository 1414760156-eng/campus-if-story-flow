# 5A 正式可玩版 v2 设计

## Summary

5A v1 已经能跑通完整宿舍修补线、唯一外流线和一个回流点，但仍有部分节点偏直线。v2 的目标是提升可玩性：增加多个回流点，扩关键节点选项密度，同时保持主线完整、唯一外流点不变、外流线独立定轨。

## Chosen Approach

采用“小范围增密”的方案，不重写 5A。

- 新增 3 个回流点：
  - `ACT5_5A10A_WASHROOM_RESET`：规则会前去水房洗把脸，可能遇到陆沉或看到周屿消息，回到十分钟计时。
  - `ACT5_5A16A_STORE_WATER`：第一顿饭前不进食堂，去便利店买水，可能碰见唐骁或陆沉，回到饭后群聊。
  - `ACT5_5A19A_HALLWAY_REVIEW`：期末前夜去走廊/水房缓一下，处理学业压力与宿舍规则复发，回到复发选择。
- 扩关键节点选项：
  - `ACT5_5A11_TABLE_TIMER` 增加“先写底线再开口”。
  - `ACT5_5A20_RELAPSE_CHOICE` 增加“先问陆沉还能不能撑一下”。
  - `ACT5_5A10_BEFORE_MEETING`、`ACT5_5A16_CANTEEN_SEAT`、`ACT5_5A19_EXAM_NIGHT_RETURN` 增加路线级回流选项。
- 保持唯一外流点：
  - 外流点仍只有 `ACT5_5A15_DOOR_SLAM_RISK`。
  - 外流线仍为 `5X_离场外流线`，第六、第七幕只走外流单线。

## Node Standards

- 普通心理选择：只改状态、反馈、后续温度。
- 路线级选择：可以短暂进入回环节点，但必须回到指定主轴。
- 外流选择：全 5A 只有一个，进入独立外流线，不回旧线。
- 关键冲突节点：至少 3 个选项，且选择前铺垫不少于 250 字。
- v2 目标：回环节点不少于 4 个，总选项不少于 68 个。

## Acceptance Criteria

- 数据标题标记为 `正式可玩版 v2`。
- `回环节点` 不少于 4 个。
- 必须存在以下回环节点及回流点：
  - `ACT5_5A01A_OUTSIDE_BENCH -> ACT5_5A02_DOORWAY_PAUSE`
  - `ACT5_5A10A_WASHROOM_RESET -> ACT5_5A11_TABLE_TIMER`
  - `ACT5_5A16A_STORE_WATER -> ACT5_5A17_AFTER_MEAL_CHAT`
  - `ACT5_5A19A_HALLWAY_REVIEW -> ACT5_5A20_RELAPSE_CHOICE`
- `外流点` 数量仍为 1。
- `ACT5_5A11_TABLE_TIMER`、`ACT5_5A15_DOOR_SLAM_RISK`、`ACT5_5A20_RELAPSE_CHOICE` 均至少 3 个选项。
- 正常路线不会误入回环或外流节点。
- 外流线规则保持不变。

## Open Risks

- 回流点过多会稀释宿舍主轴，因此本轮只新增 3 个，且每个只处理一次人物动作或心理暗线。
- 食堂回流点不能变成资源/实践线；期末回流点不能变成学业线。它们只服务 5A 宿舍修补。
