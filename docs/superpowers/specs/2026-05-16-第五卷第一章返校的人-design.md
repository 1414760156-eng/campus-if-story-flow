# 第五卷第一章《返校的人》设计记录

## 目标

第四卷已经以上册终点收束。本轮进入第五卷《所有路都窄下来》，开章要完成三件事：

1. 承接第一至四卷的暑假签字、本人材料、家庭生活费、晚风现实化和青枫居公共桌旧账。
2. 把上册的多线生活压成几条更清楚的现实压力：宿舍修补、家庭转账、晚风见面、课程实验、主持活动、勤工排班和证书考试。
3. 只写 5X 的远处气味，不启动正式外流。许澈不进入稳定关系。

## 写法约束

- 使用 chinese-novelist 原则：展示而非讲述，以物件、动作、场景和对话推动人物。
- 正文不使用系统词、策划词和自我总结句。
- 不写“让林亦舟意识到”“下一句话会”“周屿怕/唐骁怕/陆沉怕”等上帝视角句式。
- 明德楼只用于学工、融媒体和行政材料；本章课程和班会通知放在教学楼。
- 武生院站按地铁阳逻线动线写，不写公交换乘中心或国铁终点。

## 章节功能

《返校的人》不是强冲突章，而是下册压力重排章：

- 武生院站和校门负责把暑假的外部世界带回学校。
- 快递站、青枫居 4XX 和公共桌负责让家庭物件、课程文件和宿舍旧账重新出现。
- 周屿、唐骁、陆沉和林亦舟各自带回一条新学期压力。
- 夏知微、沈嘉禾、秦越和晚风用短场景或消息进入，不抢宿舍四人主轴。
- 章末用同一张桌上的多条通知作为钩子，进入第五卷后续单线收束。

## 关键物件

- 林亦舟：行李箱、校园卡、常温水、母亲包裹、旧手机壳、晚风微信。
- 周屿：靠门桌、耳机线、主持稿、润喉糖、秋季小舞台排班。
- 唐骁：靠窗桌、线夹、新学期课表、实验分组记录、羽毛球场预约。
- 陆沉：阳台门边旧背包、勤工牌、饭卡、小票、图书馆排班。
- 公共桌：第四卷留下的“已签完”便利贴、空调遥控器、插排、新学期班会通知。

## 结尾钩子

四个人刚把东西放回自己的位置，班群、课程群、主持队、勤工群和晚风消息几乎同时压来。结尾不写命运分叉，只写桌面变窄、手机同时亮、门外查寝表喊声进来，为下一章进入具体寝室公约和课程协作压力做准备。
