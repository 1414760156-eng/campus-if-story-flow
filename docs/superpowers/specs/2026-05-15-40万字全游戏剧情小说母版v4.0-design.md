# 40万字全游戏剧情小说母版 v4.0 Design

## Summary

本轮目标不是继续局部修 5A，也不是把现有节点表简单扩写，而是为《武生院沉浸版》建立一套完整的长篇小说母版。母版覆盖序幕/第一幕至第六幕，以及终章《最后一个新年》，用连续小说的方式补足主线之间的大学生活、人物互动、现实矛盾、心理暗线和支线人物行动。

小说母版的文学性优先级高于游戏拆分。它首先必须作为一部长篇校园群像小说成立：人物要有吸引力，章节要有悬念，事件要有现实重量，关系要有曲折，文字要能让读者继续读下去。后续互动剧情拆分只是它的次级用途：支线对话、回流段、小事件、关系变化、资源变量、强冲突概率、结局回响都从母版里取材，避免每次单独改某一幕时出现剧情跳跃或人物漂移。

## Chosen Approach

采用“长篇母版分卷写作”方案，总量不少于 40 万字，目标区间为 42-48 万字。

它会被拆成多个 Markdown 文件，而不是一个巨型文档：

- 总控卷：固定世界观、主线锁点、地图动线、人物弧光和文风禁令。
- 正文卷：按时间顺序写成真正的校园群像小说。
- 支线插章卷：补充不一定进入主线正文、但可以被互动节点调用的人物事件。
- 拆分索引卷：把小说段落映射到未来可选节点、心理选项、关系变量、资源变量和结局回响。

推荐这种方式的原因：40 万字级别如果只写成剧情表，会缺少现场感；如果只写成小说，又难以拆成游戏。分卷母版可以同时保留小说连续性和互动系统可用性。

## Non-Drift Locks

母版不得改动以下锁点：

1. 宿舍四人关系优先：周屿、林亦舟、唐骁、陆沉是情感主轴。
2. 关键事件保持：军训手机袋事件、汇演、班会、主持招新、毕业活动。
3. 第五幕不是晚风核心，而是多线压力拆成多条清晰单线。
4. 5A 是宿舍修补线；5C 才是晚风现实化关系线。
5. 许澈只属于唯一 5X 硬外流线，外流后不回 5A-5F。
6. 第六幕形成 9 个主预结局；终章只调整抵达状态和关系温度，不推翻硬结局。
7. 地点遵循武生院地图和动线逻辑，优先使用青枫居、银杏路、晨光湖、西园餐厅、东区商业街、晨光体育场、健源体育场、明德楼、阳光书屋、图书馆等已确认锚点。

## Location Reality Rule

主线发生地必须固定在武生院校园内，主要现实事件发生在武汉。校外、家庭、其它城市、线上空间可以自由发挥，但必须有真实支撑：交通时间、学生消费水平、武汉城市语境、大学生行动能力、家庭经济状况和人物关系都要合理。

执行边界：

- 主线课堂、宿舍、军训、汇演、班会、招新、毕业活动以武生院为主场。
- 校外事件优先落在武汉可理解的生活半径内，例如光谷活动、武汉商业街、地铁/车站、校外餐馆、兼职或展会空间。
- 家庭段落可以发生在主角家乡，但必须和春节、暑假、返校、生活费、父母工作等现实节点绑定。
- 线上剧情可以跨地域，但必须通过游戏、微信、抖音、电话、朋友圈、网购、转账等真实媒介落地。
- 其它地点可以自由补充，但不能把主线带离武生院，也不能让玩家像瞬移一样从校园跳到不合逻辑的场景。

## Extra-Game Narrative Rule

允许在小说母版中大量增加“游戏之外的生活剧情”。这些剧情可以是未必直接进入正式试玩节点的校园日常、支线插章、人物之间的过场、边缘事件或长对话，用来支撑后续支线对话和世界厚度。

但这些新增内容属于母版参考层，必须遵守四条边界：

1. 不改变主线必经事件的发生顺序和结果。
2. 不改变宿舍四人关系的核心走向。
3. 不替代第五幕既定的多线收束结构。
4. 不把玩家没有选择过的支线强行写成正式经历。

可增加的内容包括：课间闲谈、便利店偶遇、宿舍旧账、社团边缘任务、被骗小事件、晚风聊天碎片、父母生活费暗线、夏知微拍摄插章、沈嘉禾资料协作、秦越活动名单、许澈外流线以外的遥远预埋感。但这些内容进入游戏时必须通过条件化记忆、可选回流、软互动、支线素材或后日谈回响承载。

## Target Length

目标总量：420,000-480,000 中文字。

建议分配：

| 卷 | 内容 | 目标字数 |
|---|---|---:|
| 00 | 总控、规则、人物弧光、支线索引 | 35,000 |
| 01 | 序幕与第一幕：入学、家庭、宿舍初识 | 45,000 |
| 02 | 第二幕：军训、手机袋、汇演 | 45,000 |
| 03 | 第三幕：早八、招新、小组作业、热闹分流 | 55,000 |
| 04 | 第四幕：暑假不是暂停键、家庭和支线过载 | 55,000 |
| 05 | 第五幕：多线收束，含 5A-5F 与 5X | 120,000 |
| 06 | 第六幕：寒假后到大四上冬季，准结局形成 | 70,000 |
| 07 | 终章：最后一个新年、毕业、离校 | 55,000 |
| 08 | 互动拆分索引与素材库 | 25,000 |

最低完成标准：总字数不得低于 400,000。

## Writing Rules

使用 `chinese-novelist` 的三条核心原则：

- 展示而非讲述：用动作、物件、消息、空间和对话表现心理变化。
- 冲突驱动：每一节至少有一个现实摩擦、误会、隐瞒、欲望差异或关系代价。
- 悬念承接：每一节末尾留下可进入下一节或可拆成互动节点的钩子。

文学性优先规则：

- 每一章先按小说阅读体验成立，再考虑可拆成哪些游戏节点。
- 不为了选项便利牺牲章节节奏。
- 不为了变量清晰把人物写成说明牌。
- 重要支线可以用伏笔、错过、误会、沉默、旁观和插章慢慢养出来，不急着立刻变成可点击选项。
- 章节结尾优先服务阅读钩子，其次才服务节点钩子。
- 游戏之外的生活剧情可以大幅增加，只要它让人物更真实、关系更复杂、后续支线更有依据。

玩家可见正文禁止：

- 上帝视角预告。
- 反复写“意识到、明白、发现”式成长总结。
- 用变量、线路、判定、触发、回主线等系统语言。
- 把人物写成过度正向的功能 NPC。
- 让支线人物替林亦舟讲道理或替宿舍作裁判。

## Reality Conflict Requirements

母版必须加入合理但有影响力的大学现实挑战：

- 寝室生活习惯差异：外放、作息、卫生、借物、空调、床帘、洗衣房、水卡、公共桌。
- 钱和家庭变化：生活费、红包备注、母亲是否继续买生活用品、父亲工作压力、家庭饭桌沉默。
- 被骗和小算计：音乐会票、活动名额、证明材料、网络聊天、兼职或资源承诺。
- 感情和暧昧：晚风从游戏、微信、抖音到现实见面的关系边界。
- 老师和规则：陈老师的误会、过程材料、推荐语、班会通知。
- 支线人物之间的交叉：夏知微、沈嘉禾、秦越、许澈等必须有自己的行动线，不只是等主角触发。

强冲突可以出现，但必须概率化和状态化。完美线与纯摆烂线更关注自我路径，通常弱冲突；中间摇摆线、低互信、高回避、高现实压力时更容易进入强冲突。

- `rg` 检查玩家可见母版正文不出现明显系统口吻。
- 用字符统计记录当前总量和各卷进度。
