# 第五卷第五十七章《只写一句》设计记录

## 断点

第五十六章章末，开场已经开始，侧屏正常亮出新字幕。秦越手机收到主持队临时群消息：胡嘉宁撤图后仍问能否开场后写一句“差点被误当成电影海报的一次调试”，不放图，只写一句。

## 本章目标

- 继续 5A 宿舍修补线，不开启 5X，不引入许澈。
- 把冲突从“图片外散”推进到“语言包装”：图撤了，但一句漂亮花絮也会把旧残留重新公开化。
- 现场活动照常进行，后台处理不能抢走舞台，也不能让人物用大段说教停住剧情。
- 宿舍四人分工明确：周屿指出这不是他能不能开玩笑，而是旧残留不能被做成素材；唐骁拆开公开花絮、内部记录和后台日志；陆沉记录提议、回复和审核端；林亦舟处理晚风侧面刷到的外部说法，并协助把文案压成可执行规则。
- 支线人物继续有行动：秦越作为群主回复，许棠把花絮审核收回融媒体流程，梁思予守字幕端，夏知微守拍摄素材，晚风从外部平台提醒“半边字”还在被转述。

## 冲突结构

1. 秦越看到“只写一句”，活动已开始，不能停场。
2. 周屿拒绝被包装成“可以开玩笑的人”，但不要求全场追责。
3. 唐骁要求把“公开文案”和“后台事故记录”分开，不允许用诗化说法替代处理记录。
4. 秦越在临时群发短句：不写调试残留，不写电影海报，不写彩排事故；花絮仅使用正式现场和授权素材。
5. 胡嘉宁回复“那幕后感怎么写”，说明她不是恶意，而是习惯把活动包装好看。
6. 许棠让花絮文案走融媒体审核，梁思予补字幕端不再提供截图，夏知微只交授权照片。
7. 晚风发来外部平台文字截图，不发图，只说有人已经把“半行白字”写成梗。
8. 第一段节目结束，掌声压过后台消息，现场继续。
9. 章末钩子：花絮共享文档自动生成标题《那半行白字》，创建者不是胡嘉宁，而是“模板副本”，把下一章压力推向模板和默认标题。

## 写法要求

- 展示而非讲述，用舞台声、掌声、手机光、白卡、后台群推进。
- 不写“让林亦舟意识到”“下一句话会”等上帝视角。
- 避免正文出现禁用游戏术语和 A302、许澈。
- 地点限定东区商业街小舞台、台侧、侧屏控制台、主持队临时群、活动材料群和微信。
