# 第四卷第三十七章《第十九题》设计稿

## 写作边界

- 承接第三十六章结尾：周六计算机二级考试、周屿主持词、陆沉图书馆排班、晚风票务诈骗后续同时压来。
- 第四卷仍只埋“引导式外流感”，不真正开启 5X 外流线；许澈如出现只能作为远处生活影子，不进入强关系。
- 宿舍四人关系优先：唐骁的考试焦虑、周屿的主持现场、陆沉的拒绝替班、林亦舟的消息处理都必须回到青枫居公共桌形成彼此照面。
- 不使用明德楼承载课堂、班会、机房、补考、咨询会等内容；机房和考试放在八号教学楼，图书馆勤工放在图书馆西侧，舞台相关放在东区商业街或健源体育场外场。
- 正文避免系统口吻、避免上帝视角预告、避免“让某人意识到/下一句话会/怕什么怕什么”的解释性写法。

## 本章事件骨架

1. 青枫居清晨：夜归登记后的次日疲惫，唐骁反复核对二级准考证和第 19 题，周屿背主持词，陆沉整理排班确认，林亦舟回复晚风票务截图。
2. 八号教学楼机房：唐骁进入二级考试/模拟正式考场，因过度在意第 19 题而在保存路径、命名或最后检查上吃亏；不是灾难，但让他承认“会做”和“做完”不是一回事。
3. 图书馆西侧：陆沉拒绝替别人顶班，但在自己的班内把书车和扫码工作收完，边界通过动作呈现。
4. 东区商业街小舞台：周屿用更短、更稳的串词处理现场延误，不再靠玩笑把所有空白塞满。
5. 晚风票务后续：晚风通过微信、语音、截图询问“怎么才算证据”，林亦舟把真实平台订单、付款链和转让规则拆开看，不替她下结论。
6. 夜里回到青枫居：四人把一天里各自没做完的东西摊开，唐骁等成绩，周屿改词，陆沉确认不替班，林亦舟把晚风“还想买下一场”的消息压在屏幕里；结尾钩子指向实验四后续和生活压力继续并行。

## 关键人物弧光

- 林亦舟：从“帮忙判断真假”转向“把证据拆清楚”，不代替晚风和她室友做决定。
- 周屿：现场不靠抢话和玩笑撑场，而是学会留出空白、提前确认。
- 唐骁：从刷题式把握转向承认考试现场的流程也需要被认真对待。
- 陆沉：拒绝替班不是冷淡，而是把自己的责任做完后按时离开。
- 晚风：不只是聊天对象，而是把现实诈骗、证据链和关系试探带进林亦舟生活的一条线。

## 场景与生活细节

- 青枫居一楼宿管窗口、4XX 公共桌、夜归登记本、准考证、旧电脑包、胶带缠线、润喉糖。
- 八号教学楼机房、准考证核验、身份证、保存路径、考试结束铃声。
- 图书馆西侧、书车、扫码枪、还书箱、阳光书屋或星辰书屋作为学习分流。
- 东区商业街小舞台、便利店、奶茶店、学生唱歌、秦越的现场确认。
- 票务诈骗使用微信语音、截图、平台订单、转账记录、抖音刷到演出切片等真实沟通方式。
