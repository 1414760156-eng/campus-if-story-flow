# 第四卷第四章《低空飞过》设计

## 目标

承接第三章末尾“数据结构入门”成绩加载未出的钩子，写出成绩查询、补考预警和暑假空寝室对林亦舟、周屿、唐骁、陆沉的具体影响。章节必须把 `05_人物外貌与穿着索引.md` 和 `06_青枫居四人寝室与桌面索引.md` 落到正文里，让四张桌子、插排、空调遥控器、旧耳机、手机消息和父母转账都参与叙事。

## 叙事边界

- 不改变主线锁点，不提前进入第五卷。
- 第四卷只写引导式外流感，不开启 5X。
- 许澈本章不正面出场；如需延续远影，只能通过便利店、工牌或小票痕迹。
- 明德楼不承载班会、课程、机房、试训或咨询会。
- 补考要写成现实压力，不写成笑料。

## 本章核心事件

1. 教务系统加载完成，林亦舟“数据结构入门”卡在不体面的分数线上。
2. 宿舍群里周屿高数低空飞过、唐骁克制地给出资料整理方式、陆沉在同一间空寝室里陪林亦舟把页面看完。
3. 家庭线同步压来：母亲问成绩，父亲转账备注仍挂在手机里，林亦舟不能再用“还没查”挡过去。
4. 第二天补考相关信息、资料包、经验帖和同学闲聊进入校园生活，补考开始影响吃饭、回校、图书馆和宿舍自尊。
5. 章末用补考安排提前发布形成下一章钩子，把暑假后半段压回返校、资料、补考和班会前压力。

## 人物弧光

- 林亦舟：从“先把成绩藏起来”到把分数说给陆沉听，但仍没有完全告诉母亲和晚风。回避不消失，只是第一次被现实堵在同一张桌前。
- 陆沉：不替林亦舟解释，也不替他决定，只用旧耳机、勤工表、食堂小票和一句实在话陪他把夜熬过去。
- 周屿：用玩笑和“低空飞过”护住面子，随后用远程插科打诨试图把林亦舟从沉默里拉出来，露出他热闹背后的不安。
- 唐骁：本能地整理补考资料和时间表，但要避免写成辅导员式说教；他的帮助要有边界，也会因为语气过于规整让人难受。
- 晚风：不做心理咨询师，只通过微信、游戏大厅、语音和现实化的短句参与。她提供陪伴入口，但不替宿舍关系裁判。

## 场景与物件

- 青枫居 4XX：四张桌子、空椅子、周屿耳机线、唐骁计划表、陆沉旧耳机、林亦舟手机和父亲转账。
- 公共区：插排、空调遥控器、公共桌、小票、外卖袋、垃圾桶，承接宿舍旧账。
- 图书馆 / 阳光书屋：暑期留校学生、成绩查询后的补考资料、座位纸条。
- 食堂二楼或便利店：补考相关闲聊和情绪出口，避免只在寝室内推进。

## 文风要求

- 使用 chinese-novelist 原则：展示而非讲述，冲突驱动，结尾留钩子。
- 正文禁止系统口吻、上帝视角预告、解释性心理总结。
- 不使用正文禁用词：节点、变量、触发、回流、系统提示、意识到、忽然、突然、发现、觉得、害怕、主线、副线、分线、flag、权重、判定、资源线、NPC 等。
