# 第五卷第八十六章《半句》设计

## 本章定位

承接第八十五章末尾“半句已经独自往前走了一夜”。本章继续在东区商业街小舞台后台核对这半句的去向：它不是凭空变成正式文本，而是在冯喻的转发、胡嘉宁的整理、陈景的打开和导语初稿的备注里，一次次被看成“可以先参考”。本章不新增大事故，不让某一个人突然背下全部责任，而是把半句怎样被不同的人各自省掉一点，写成可见的动作链。

## 场景与动线

- 场景仍固定在东区商业街小舞台后台。
- 时间继续接第八十五章深夜，商铺准备关灯，后台仍在数话筒、收线、拷文件。
- 具体物件：旧一体机、陈景电脑、白卡、素材交接群截图、胡嘉宁电脑、话筒箱、外卖页、导语候选文档。
- 不进入明德楼，不新增校外交通，不写 A302，不引入许澈。

## 人物功能

- 林亦舟：在群聊截图、晚风消息和现场催促之间压住节奏，只做必要转交。
- 周屿：继续守住“不能只怪最后写出旧句的人”，也不让“大家都忙”冲掉责任。
- 唐骁：追问“先看一下”为什么会变成“先用一下”，火气仍在，但追问要落到具体动作。
- 陆沉：继续把原话、转述、看见者、下一步动作和少掉的字写成可核对的表。
- 黎旻：确认自己语音被截短后的影响，仍不替别人解释，只补台侧真实压力。
- 袁承、冯喻、胡嘉宁、陈景：分别交出半句在不同群、文件夹和文档里的留痕。
- 老师：把讨论从情绪拉回第一处正式写入路径。

## 叙事目标

1. 冯喻翻出素材交接群，确认自己把袁承半句带进更大的群。
2. 胡嘉宁承认她按“台侧可以留”把口播包放进 `02_旁边_别发`，以为已经避开正式正文。
3. 陈景说明自己看到的是被转过的短句，不是黎旻原语音。
4. 陆沉把“原话”和“被看见的话”并排写，让半句的缺口肉眼可见。
5. 林亦舟收到晚风不催饭的消息，但不让外部关系接管现场。
6. 章末追到 `导语候选_第一版.docx` 顶端灰字：它像提醒，也像入口，把下一章压力留给第一版文档。

## 禁止写法提醒

- 不写系统、分支、判定、触发、变量等游戏口吻。
- 不写“让林亦舟意识到”“他突然发现”“下一句话会”等解释性旁白。
- 不写“你怕我怕”式心理并列。
- 不把晚风写成解决问题的人。
- 不让老师、黎旻或任何边缘人物替宿舍四人完成关系推进。
