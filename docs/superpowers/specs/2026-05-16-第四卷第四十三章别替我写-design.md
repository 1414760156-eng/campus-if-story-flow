# 第四卷第四十三章《别替我写》设计

## 目标

承接第四十二章“空表互换填写”的课堂压力，把第四卷后段从文档外传追责推进到更深一层：即使工具和规则被公开，学生仍会把听来的印象、个人判断和别人的空白写进材料里。

## 写作边界

- 场景限定在武生院内：青枫居、打印店、教学楼 A302、北园餐厅、阳光书屋和图书馆西侧。
- 明德楼不承载课程、班会、答疑、机房或上机。
- 仍以宿舍四人为主轴，晚风票务投诉只做镜像支线。
- 许澈不进入本章。
- 第四卷只继续埋外流气味，不开启 5X。

## 事件设计

1. 青枫居清晨：四人带空表去打印，唐骁坚持不留示例，周屿的小舞台排班和陆沉图书馆工牌并排出现，林亦舟继续收到家庭和晚风的现实消息。
2. 教学楼 A302：随机互换空表。22 组按事实给罗泽组填写“外班复习群删除无法确认”；另一个小组给 22 组写下“成员个人活动导致进度不稳”，把课堂听来的印象当成表内事实。
3. 课堂追问：陈老师追问依据，填表组说是“之前听过”。唐骁把“听过”和“空表可见”分开，周屿把玩笑压回去，陆沉指出勤工签退照片也不能被别人写成贡献不足。
4. 北园餐厅：饭桌不做道理总结，写出螺蛳粉气味、二级成绩入口、父亲分笔转账、母亲问包裹的细节，让压力落回大学生活。
5. 阳光书屋：四人整理新备注“本表可见 / 课堂听说 / 个人印象 / 需要本人确认”，同时晚风删掉维权模板里的夸张词，只提交能证明的材料。
6. 青枫居夜里：课程群要求明天说明被划掉的那一行，压力回到周屿和 22 组。

## 文风要求

- 使用动作、物件、对话推进心理，不写上帝视角结论。
- 不出现“让林亦舟意识到”“这一步会被记进”“下一句话会”等禁用表达。
- 结尾保留悬念：被划掉的一行明天要公开说明。
