# 第五卷第四十二章《旧资料》设计

## 目标

承接第四十一章章末“姜宁 更新了第 2 行 / 周屿资料已补齐”，把冲突推进到“曾经本人填过的旧资料，是否可以在新场景里自动沿用”。本章继续 5A 宿舍修补线，不改变周屿本人申请暂退、不参与主持人背景展示的事实，不开启 5X，不引入许澈。

## 章节核心

旧资料不是假资料，却也不是当前授权。周屿曾经主动写过自己想被看见的话，如今这些话被别人从旧表里搬出来，反而让他无法说“不”。本章要让读者看见：修补关系不是把人护成永远不被提及，而是能陪他把旧的、真的、但此刻不该被拿出来的东西收回去。

## 场景和人物

- 青枫居 4XX：四人第一时间发现协作记录，核对“补齐”的具体内容。
- 活动材料群：姜宁说明自己用了去年主持队旧资料，许棠要求撤下最终版。
- 青枫居一楼宿管窗口：宿管阿姨用旧借钥匙记录类比“旧登记不能替今天本人确认”。
- 东区商业街小舞台：秦越和姜宁短场对话，显出活动压力和人情误差。
- 阳光书屋 / 微信：夏知微处理摄影社旧授权表，晚风处理旧聊天截图被再次引用。
- 章末钩子：姜宁撤下资料后，活动群里有人转出一张旧主持队招新截图，截图上周屿当年写过一句“我不怕被看见”。

## 文风约束

- 展示而非讲述，不写系统口吻。
- 避免“让林亦舟意识到”“下一句话会”“周屿怕/唐骁怕/陆沉怕”等解释性句式。
- 不写“主线、节点、触发、变量、回流、判定、资源线、系统提示”等游戏开发词。
- 明德楼不承载班会、课程、机房、答疑。本章不需要使用明德楼。
- 不出现 A302、许澈、公交、火车到武生院站等误用。
