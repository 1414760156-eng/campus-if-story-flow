# 第五卷第八十四章《旧稿勿用》设计

## 本章定位

承接第八十三章结尾陈景本地草稿路径暴露 `主持口播_临时合并\旧稿勿用\主持词_旧版_灯光.docx`。本章不把陈景写成唯一责任人，也不把“旧稿勿用”当作一句简单的证据，而是把后台赶稿、压缩包解压、文件搜索、资源管理器预览窗和“我没点开过”的认知落差写清楚。

## 场景与动线

- 场景固定在东区商业街小舞台后台和后台旁旧打印台。
- 仍使用旧一体机、陈景笔记本、白卡、U 盘、活动证、水瓶、舞台外收场声。
- 不进入明德楼，不新增校外交通。

## 人物功能

- 林亦舟：不做主持式总结，只在关键处追问“看到”和“打开”的差别，并收住晚风外部回声。
- 周屿：阻止现场把“没点开过”直接变成狡辩，同时要求不能用“不记得”抹掉动作。
- 唐骁：追问“没点开也看见了”的事实差别，火气要锋利但不写成宣判。
- 陆沉：继续把表格拆细，新增“解压、搜索、预览、复制、残句”。
- 陈景：不是恶意抄袭者，而是急着显得会写，靠搜索和预览误把旧稿残句带进脑子。
- 胡嘉宁、冯喻、秦越、老师：保持在后台流程链中，提供文件、时间和现实压力，不抢宿舍四人中心。

## 叙事目标

1. 复现陈景操作：下载压缩包、解压到本地、搜索“灯”、资源管理器预览窗显示旧稿内容。
2. 让“没点开”变得复杂：没有双击打开文件，但通过搜索命中和预览窗看到了句子。
3. 把旧稿内容具体呈现：旧稿内含“所有灯光都会回到舞台”“同一束灯下”等句子，能解释第二版第一行来源。
4. 章末留下新钩子：旧稿内有黎旻早先批注，明确写过“这版别给文案看”，说明提醒曾经存在，但在压缩包流转里失效。

## 禁止写法提醒

- 不写系统口吻、分支口吻和变量口吻。
- 不写“让林亦舟意识到”“下一句话会”等上帝视角句。
- 不写“你怕我怕”式并列心理总结。
- 不写 A302、许澈。
- 不让晚风吞掉本章，只保留外部消息一两处。
