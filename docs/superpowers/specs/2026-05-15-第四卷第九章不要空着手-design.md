# 第四卷第九章《不要空着手》设计

## 目标

承接第八章晚风光谷邀约和生活费错位，把“周六现实见面前夜”写成一章具体的大学生准备：路线、钱、礼物、室友起哄、舞台热闹、支线人物擦肩和夜归门禁共同压住林亦舟。

## 叙事边界

- 不正式写光谷见面，只写前夜准备和换乘成本。
- 许澈仍只作为便利店/校道远影或短句出现，不建立新人关系。
- 明德楼不承载班会、课程、补考、答疑或咨询。
- 宿舍四人保持主轴：周屿通过健源体育场小舞台和起哄露出表达欲；唐骁通过考试计划和花销建议露出规则感；陆沉通过勤工排班、伞和常温水继续稳住现实；林亦舟则在母亲转账、父亲叮嘱和晚风消息之间准备第一次现实靠近。

## 场景安排

1. 青枫居 4XX：周六路线、生活费余额、礼物争论、四人桌面和各自周末任务。
2. 图书馆 / 阳光书屋：陆沉勤工、唐骁计算机二级模拟、补考成绩未出造成空窗。
3. 东区商业街 / 健源体育场边：周屿小舞台临时串场，夏知微和许棠作为拍摄/融媒体支线擦肩，情侣礼物和争执作为亲密关系对照。
4. 便利店 / 青枫居门口：许澈短句远影、宿管阿姨门禁、晚风发来光谷地铁口和穿着信息。
5. 章末：林亦舟把路线截图、钱、学生证、小礼物和常温水放进包里，留下明日现实见面的钩子。

## 文风要求

- 使用 `chinese-novelist` 原则：动作和物件承载心理，不写系统口吻。
- 保留真实大学生活：礼物笨拙、室友怂恿、考试报名、乐跑、舞台试音、奶茶店、地铁换乘成本。
- 禁用“让林亦舟意识到”“下一句话会”“你怕我怕”式表达，正文中也避免“突然/忽然/发现/觉得”等解释性口头词。
