# 第四卷第五章《不要卡点》设计

## 使用技能

- superpower-bootstrap
- superpower-brainstorming
- planning-with-files
- superpower-writing-plans
- chinese-novelist
- superpower-verification

## 本章目标

承接第四章末尾“补考同学务必提前返校，不要卡点”。本章要把暑假后半段从“成绩压在桌面上”推进到“返校压力有了真实路线”：教学楼 C201、教学楼 B204、图书馆、打印店、青枫居、银杏路和阳光书屋都要成为具体位置。

本章不提前开启第五卷 5X，只写第四卷的引导式外流感：林亦舟可以短暂想离开青枫居和家庭消息，但所有动作都被补考、班会、晚风白天聊天、父母信息错位和室友提前返校拉回原生活。

## 关键承接

- 补考：数据结构入门 59 分，补考地点教学楼 C201。
- 班会：开学第一周班会暂定教学楼 B204。
- 家庭：父亲曾在武生院站见面并给钱；母亲只知道“明天再说”，后续需要追问。
- 晚风：夜聊退到白天，不能像心理咨询师，要用微信、抖音、游戏状态和现实见面退让来推动关系。
- 宿舍四人：周屿、唐骁、陆沉都要以各自性格靠近林亦舟，但不能把林亦舟写成被拯救的人。
- 世界观：加入资料包骗局、打印店、四六级和计算机二级排队、生活费和生活用品变化、青枫居桌面细节。

## 本章场景

1. 图书馆二楼继续列复习表，教学楼 C201 和 B204 通知压实返校时间。
2. 林亦舟沿银杏路和自强路去教学楼试走路线，路过打印店和公共教室。
3. 打印店里有人打印四六级准考证、计算机二级资料，也有人问补考包过资料，老板用普通口吻提醒不要信。
4. 晚风白天发语音和抖音截图，谈室友被骗善后，也把光谷见面往后挪。
5. 周屿和唐骁用宿舍群的玩笑、车票截图、复习资料和提前返校信息修补上一章“不要替我”的硬处。
6. 母亲追问父亲是否到过学校，形成下一章家庭线钩子。

## 文风约束

- 展示而非讲述，用动作、物件、消息和地点推进。
- 不写系统口吻，不写变量解释。
- 禁用“发现/觉得/忽然/突然/意识到”等正文扫描词。
- 明德楼不承接班会、专业课、试训、上机或机房内容。
