# 第五卷第三十八章《没遮住》设计

## 使用技能

- superpower-brainstorming：确认本章只推进 5A 宿舍修补线，不开启 5X，不引入许澈。
- superpower-writing-plans：先确定事件链、地点和钩子，再落正文。
- planning-with-files：记录设计、执行和进度，保证自动续写可恢复。
- chinese-novelist：以场景、动作和对话写出压力，不写系统口吻和总结句。
- superpower-verification：写完后检查字数、禁用表达、地点逻辑和章末钩子。

## 承接断点

第三十七章末，拍照者在楼层群发所谓“只露顶部”的截图预览，却没有遮干净周屿头像和“主持队周屿”备注。新章从群聊里短暂沉默开始，压力从“完整私聊能不能发”转为“身份被带出房间后还能不能收住”。

## 本章目标

1. 让周屿的主持队身份被楼层群看见，产生跨圈层传播风险。
2. 保持宿舍四人优先：林亦舟稳住回复，唐骁压住截图扩散，陆沉提出登记和撤图，周屿面对自己也成了暴露者。
3. 支线镜像不抢戏：秦越小舞台、夏知微摄影社、晚风朋友圈和 C305 课堂只照出同类边界。
4. 地点沿用第五卷生活动线：青枫居 4XX、青枫居一楼宿管窗口、西园餐厅、教学楼 C305、东区商业街小舞台消息、阳光书屋、微信。
5. 章末留下主持队群里已经出现截图的钩子，下一章转向跨群传播和撤回后的二次收束。

## 事件链

1. 楼层群沉默后出现“这不是周屿吗”，4XX 立即要求撤图，不截图、不二次转发。
2. 青枫居一楼宿管窗口登记：撤回时间、图片暴露位置、谁已经保存不再公开追问；宿管阿姨用快递面单和夜归名单提醒“遮一半等于没遮”。
3. 西园餐厅中，罗泽、丁祺、何书南承认小群已经有人认出周屿，几个人把“别问本人、别转主持队”的话压成短句。
4. 教学楼 C305，陈老师用一张材料截图让学生找“还能认出谁”，把打码从形式拉回“能不能识别”。
5. 秦越发来东区商业街小舞台消息：有人把截图发到主持队小群问是不是周屿，周屿要求不要解释成活动风波。
6. 夏知微在阳光书屋处理招新照片，裁掉半张脸和工作牌，补足“边角信息也能认人”的镜像。
7. 晚风处理朋友朋友圈截图，林亦舟不替她写道理，只帮她把拒绝公开完整聊天的短句改清楚。
8. 章末秦越发来主持队群截图，截图上有人问“这个是你们那个主持周屿吗”，压力越过楼层群。

## 禁止写法

- 不写“让林亦舟意识到”“下一句话会”“这一步会被记”等总结句。
- 不写玩家、节点、触发、变量、判定、回流、主线、副线等开发词。
- 不写“周屿怕、唐骁怕、陆沉怕”式并列心理。
- 不使用明德楼承载课程或班会，不写 A302，不写许澈。
