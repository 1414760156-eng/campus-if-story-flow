# 5A 回流聊天微心态 v3.2 设计说明

## 背景

v3.1 已经把 5A 宿舍修补线的回流系统升级为命名人物、多页回流和唯一 5X 外流线。但当前试玩反馈指出：回流点仍然偏“功能性”，例如碰到夏知微、许澈之后，很快就被拉回主线，缺少真正能聊开的生活感。

v3.2 的目标不是继续增加新分线，而是把已经存在的相遇点扩写成“微心态聊天段”。玩家可以聊近况、未来、关系、被记录的感觉、短暂轻松的错觉；这些选择只影响心理状态、人物关系、记忆标签和终章回响，不改变 5A 主轴，不新增外流点。

## 设计目标

1. 湖边回流不再只是“走到湖边再被系统送回”，而是先选对话对象，再选话题方向：
   - 玩家先选择和晚风聊天，或和夏知微聊天。
   - 下一层再选择是否聊刚发生的宿舍事。
   - 聊宿舍进入更深的现实对话；不聊宿舍则聊游戏、抖音、照片、毕业影像或未来。
   - 所有分支最后都回到 5A 主轴，但会造成心态和好感差异。

2. 夏知微回流不再只是“拍照/边界提醒”，而是能打开话匣子：
   - 聊林亦舟最近是否变安静。
   - 聊宿舍气氛和“被记录”的不自在。
   - 聊毕业活动、未来、影像作品和空椅子的含义。
   - 最后由手机震动、楼灯或食堂动线把玩家拉回现实。

3. 许澈外流线不再只是“新人出现”，而是让外流更有宿命感：
   - 许澈先以轻松的近况聊天出现。
   - 玩家短暂得到“不用解释旧关系”的舒适感。
   - 聊到未来和活动邀约时，关系像是有了新入口。
   - 但聊天继续后显出逢场作戏的空感，最终进入 5X 来源特异结局。

4. 每个新增聊天节点至少提供两个可选互动：
   - 选项必须标记为 `心理级选项: true`。
   - 选项不得标记为 `外流点: true`。
   - 选项不得解锁 5B、5C、5D、5E、5F 等旧分线。
   - 影响范围限制在人物关系、心态、自我清晰、回避值、记忆标签等软变量。

## 节点调整

### 湖边夏知微软回流

原结构：

`ACT5_5A01C_LAKESIDE_SOFT_RETURN -> ACT5_5A01C2_LAKESIDE_XIA -> ACT5_5A01D_LAKESIDE_BACK`

v3.2 结构：

晚风分支：

`ACT5_5A01C_LAKESIDE_SOFT_RETURN -> ACT5_5A01W_WANFENG_CHAT_CHOICE -> ACT5_5A01W2_WANFENG_DORM -> ACT5_5A01W3_WANFENG_DORM_REPLY -> ACT5_5A01D_LAKESIDE_BACK`

不聊宿舍的晚风分支：

`ACT5_5A01C_LAKESIDE_SOFT_RETURN -> ACT5_5A01W_WANFENG_CHAT_CHOICE -> ACT5_5A01W4_WANFENG_LIGHT -> ACT5_5A01W5_WANFENG_LIGHT_BACK -> ACT5_5A01D_LAKESIDE_BACK`

夏知微分支：

`ACT5_5A01C_LAKESIDE_SOFT_RETURN -> ACT5_5A01C2_LAKESIDE_XIA -> ACT5_5A01C3_LAKESIDE_RECENT -> ACT5_5A01C4_LAKESIDE_FUTURE -> ACT5_5A01D_LAKESIDE_BACK`

不聊宿舍的夏知微分支：

`ACT5_5A01C_LAKESIDE_SOFT_RETURN -> ACT5_5A01C2_LAKESIDE_XIA -> ACT5_5A01C4_LAKESIDE_FUTURE -> ACT5_5A01D_LAKESIDE_BACK`

新增节点职责：

- `ACT5_5A01W_WANFENG_CHAT_CHOICE`：晚风微信聊天框，玩家选择聊刚发生的宿舍事或不聊宿舍。
- `ACT5_5A01W2_WANFENG_DORM`：聊宿舍时，晚风听见主角从门口退出来，但不替他判谁对谁错。
- `ACT5_5A01W3_WANFENG_DORM_REPLY`：晚风把夜聊边界推回现实，提醒“可以跟我说，但不能只跟我说”。
- `ACT5_5A01W4_WANFENG_LIGHT`：不聊宿舍时，晚风通过游戏和抖音给出轻松出口。
- `ACT5_5A01W5_WANFENG_LIGHT_BACK`：轻松之后仍回到宿舍主线，避免晚风变成 5C 入口。
- `ACT5_5A01C3_LAKESIDE_RECENT`：夏知微聊近况和宿舍状态，玩家可选择承认疲惫或把话题推回她的拍摄。
- `ACT5_5A01C4_LAKESIDE_FUTURE`：夏知微聊未来、毕业影像和“人被记录以后会不会真的留下”，玩家可选择追问她的未来或承认自己还不敢想以后。

### 食堂侧门夏知微软回流

原结构：

`ACT5_5A16C_SIDE_DOOR_XIA -> ACT5_5A16D_SIDE_DOOR_CAMERA -> ACT5_5A16E_SIDE_DOOR_BACK`

v3.2 结构：

`ACT5_5A16C_SIDE_DOOR_XIA -> ACT5_5A16D_SIDE_DOOR_CAMERA -> ACT5_5A16D2_SIDE_DOOR_FUTURE -> ACT5_5A16E_SIDE_DOOR_BACK`

新增节点职责：

- `ACT5_5A16D2_SIDE_DOOR_FUTURE`：夏知微围绕空椅子、毕业影像和以后还会不会有人坐回这里展开，玩家选择保留希望或承认散场感。

### 5X 许澈离场外流线

原结构：

`ACT5_5X01_STAIRWELL_MUTE -> ACT5_5X02_SOUTH_GATE_EDGE -> ACT5_5X03_STRANGE_SEAT -> ACT5_5X99_OUTFLOW_OUTPUT`

v3.2 结构：

`ACT5_5X01_STAIRWELL_MUTE -> ACT5_5X02_SOUTH_GATE_EDGE -> ACT5_5X02A_XUCHE_RECENT -> ACT5_5X02B_XUCHE_FUTURE -> ACT5_5X03_STRANGE_SEAT -> ACT5_5X03A_XUCHE_EMPTY_ECHO -> ACT5_5X99_OUTFLOW_OUTPUT`

新增节点职责：

- `ACT5_5X02A_XUCHE_RECENT`：许澈聊近况、课、活动和“今晚怎么一个人出来”，玩家可选择顺着轻松聊或稍微提到宿舍烦躁。
- `ACT5_5X02B_XUCHE_FUTURE`：许澈聊以后、社团活动和毕业前认识新人的轻松，玩家可选择追问或接受邀约式错觉。
- `ACT5_5X03A_XUCHE_EMPTY_ECHO`：聊天延长后出现空感，许澈手机里还有别的局，玩家发现这段关系没有根，进入 5X 输出。

## 验收标准

- 标题必须包含 `v3.2`。
- 完整链路节点数不低于 66。
- 总选项数不低于 160。
- 心理级微选项不低于 34。
- 湖边晚风路径至少 5 页。
- 湖边夏知微路径至少 5 页。
- 食堂侧门夏知微路径至少 4 页。
- 5X 许澈外流路径至少 7 页，并仍然只有一个硬外流入口。
- 所有新增聊天节点必须包含 `微心态聊天节点: true` 和 `聊天主题`。
- 所有新增聊天选项都必须是心理级选项，不得改变主线或解锁旧分线。
