# 第四卷第四十八章《联系人》设计

## 使用的创作技能

- superpower-bootstrap：确认本轮续写必须先看规则、再动笔。
- superpower-brainstorming：先确定第四卷收束前的动作重心，而不是直接堆煽情。
- planning-with-files：读取总控、进度表、正文断点与根目录规划文件，保持断点一致。
- superpower-writing-plans：把本章拆成可执行写作步骤。
- chinese-novelist：正文采用中文小说式叙事，靠动作、物件、对话和空间调度表现压力。
- superpower-verification：写完后用字数、禁用表达、地点和交通规则扫描验收。

## 本章定位

第四十八章承接第四十七章结尾父亲来电，处理“紧急联系人顺序”和暑假去向登记。它不是家庭线爆发章，而是第四卷上册收束前的一个现实性收紧：林亦舟不再替父母任何一方补解释，只按学校表格要求和自己能确认的事实填写。宿舍四人围绕“本人确认”的课业训练被推回生活材料、家庭电话、暑假安排和桌面收拾。

## 必须保留

- 主线锁点不改，不引出正式第五卷外流线。
- 宿舍四人优先，周屿、唐骁、陆沉必须有具体动作。
- 明德楼不承载班会、课程、答疑、机房或课堂确认。
- 暑假去向登记、紧急联系人、生活费暗线、母亲袜子包裹必须自然并排。
- 晚风票务线只作现实支线余压，不抢宿舍四人关系核心。

## 叙事策略

- 避免“让林亦舟意识到”“下一句话会”“怕什么怕什么”的上帝视角句式。
- 父亲电话里只给具体要求：别写太细、别让老师找他、别告诉母亲钱晚点。
- 林亦舟用短句和保存动作回应，情绪藏在阳台、外卖架、联系人顺序和未喝的奶茶里。
- 宿舍桌面整理分为四堆：课程、主持、勤工、家庭/去向，各自拿回本人能确认的材料。
- 章末把下章压力推到教学楼的暑假安全确认或纸质签字，为第四卷终章/上册总结留入口。

## 场景安排

1. 青枫居阳台：父亲来电，紧急联系人顺序和生活费迟到浮出水面。
2. 青枫居公共桌：四人整理暑假去向、个人材料和期末周尾声。
3. 校园快递站/西园餐厅：母亲包裹、父亲小额转账、同学离校碎片补出大学生活真实感。
4. 阳光书屋/图书馆西侧/东区商业街余音：晚风、陆沉、周屿各自支线只作为同一天的现实回声。
5. 青枫居夜里：四人把去向登记保存，学习委员通知教学楼签字，章节收束。
