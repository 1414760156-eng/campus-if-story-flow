# 考试时间线与生活血肉世界观设计

## 目标

补强《武生院沉浸版剧情小说母版 v4.0》的世界观底盘：把四六级、计算机二级、普通话、教师资格证、竞赛、创新创业大赛、运动会等长期压力写成大学时间流逝的背景；同时建立无名生活人物与情绪出口规则，让食堂、商业街、后街、校医院、奶茶店、修车摊、KTV、台球厅和水果摊成为有血肉的校园记忆。

## 设计原则

- 考试证书不必都成为主事件，但要成为消息、报名、复习、缺考、二战、证书截图、朋友圈和聊天背景。
- 四六级尤其适合作为时间流逝标尺：一次报名、一次没过、一次二战、一次查分，大学就往前推一段。
- 竞赛和运动会可以成为强剧情点，但只在人物能力、欲望和关系压力足够时放大。
- 无名生活人物不需要姓名，也不需要改变主线，但可以反复出现、说话、问候、记住口味，成为大学生情绪出口。
- 正文可以写这些人，但不要用“NPC”这种游戏词；母版规则文档可以解释其功能。
