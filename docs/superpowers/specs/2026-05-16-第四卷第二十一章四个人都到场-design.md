# 第四卷第二十一章《四个人都到场》设计说明

## 目标

承接第二十章末尾助教通知“四名成员都到场”，写出撤下 400 米栏以后压力没有消失：周屿仍要去 B206 复排，也要回到外卖架材料说明现场；唐骁需要把问答清单做得能被追问；陆沉负责原始记录保管；林亦舟补站位与场地叙述。章节重点是“共同承担”不是口号，而是四个人在不同地点和时间表中把话讲清。

## 必须承接

- 文件已从“待周屿确认”改成提交版，但助教追加要求四名成员到场。
- 周屿已撤下 400 米栏，运动会名单会更新，形成可见失落。
- B206 明晚六点半复排仍在，秦越要求他不要用撤项换偷懒。
- 外卖架材料要说明缺测处理、原始记录保管和现场站位。

## 场景安排

1. 次日早上青枫居 4XX：唐骁做追问清单，林亦舟画站位草图，陆沉把原始记录装入透明文件袋，周屿看到运动会群更新替补名字。
2. 西园餐厅/打印店短场：四人吃饭或打印纸质材料，生活人物短句增加真实感。
3. 阳光书屋：三人预演问答，周屿去 B206 前补一轮；晚风只通过消息短暂出现，不替代宿舍关系。
4. 教学楼 B206：秦越调周屿停顿和气息，给出“别把退赛讲成梗”的活动边界。
5. 教学楼 A302：外卖架说明现场，助教随机追问四人。每人回答自己能负责的部分，周屿被问中途离开，不能讲漂亮话。
6. 章末钩子：材料通过口头说明，但助教要求补交一张清晰站位图；同时 B206 发来小舞台名单或嗓音复查，下一章继续压缩时间。

## 写法约束

- 不使用系统/游戏口吻，不出现“主线、节点、触发、判定”等词。
- 不使用“让林亦舟意识到”“周屿怕/唐骁怕/陆沉怕”“下一句话会”等总结式句子。
- 心理暗线通过动作和物件呈现：替补名单、透明文件袋、打印纸、站位草图、润喉水、B206 话筒。
- 明德楼不出现；班会/说明/复排地点都在教学楼或阳光书屋。

## 本章预期效果

周屿撤项后的空落被运动会名单和 B206 台词压住；宿舍四人第一次把“缺测、不补写、中途离开”在公开追问里讲出来。章节不写成胜利，结尾用站位图补交和 B206 后续名单继续留下压力。
