# 第五卷第四十六章《第一帧》设计

## 目标

承接第四十五章末尾“叶青没有用照片，却用了旧视频《开场回忆素材_可用.mp4》”。本章要把压力从文件包误发推进到剪辑工程和导出预览：旧视频已经在剪辑轨道里配过音乐、切过节奏，不能只靠一句“删掉”解决。

## 事件链

1. 青枫居 4XX 夜里收到叶青截图，四人先不让更多人点开旧视频，只要求叶青提供第一帧、文件属性、已导出版本和发送范围。
2. 周屿坚持只看第一帧和轨道位置，不让旧视频在宿舍重复播放；唐骁担心不看完整内容会漏掉问题，陆沉提醒越多人播放越难收住。
3. 叶青说明她已导出一个 28 秒预览版发给秦越和姜宁，音乐卡点用到了这段旧视频的三秒。
4. 姜宁崩一下，但不写成被审判；她主动列出自己发过的三个对象、时间和文件名。
5. 东区商业街小舞台现场，秦越拿到预览版，原本想赶紧让叶青改，许棠要求先停转、补更正说明、再替换导出。
6. 夏知微用“原片试色小样被当成正式照片”的旧事做镜像；晚风用抖音草稿被别人提前发的经历做镜像。
7. 章末叶青发来第二个问题：旧视频不止在剪辑轨道里，还已经被配进活动开场词稿，稿子里写着“看见当年的自己”，署名是秦越。

## 场景与动线

- 青枫居 4XX：宿舍四人公共桌、手机截图、剪辑截图、常温水、盖饭盒和电脑屏幕。
- 东区商业街小舞台：活动预览、秦越、姜宁、许棠和叶青的线上线下协作。
- 微信聊天：夏知微、晚风作为镜像，不抢宿舍四人中心。

## 文风约束

- 不写解释性成长总结。
- 不写上帝视角分叉句。
- 不使用玩家可见禁词。
- 不开启 5X，不引入许澈。
- 明德楼不进入本章。

## 自检

- 主轴仍是 5A 宿舍修补。
- 周屿不是被保护对象，而是参与制定处理方式的人。
- 姜宁不是反派，而是误打包后需要具体补救的人。
- 章末保留“口播稿也引用旧视频”的钩子。
