# 第五卷第十六章《谁来改》设计

## 目标

承接第十五章楼层群转发空表后的压力，写出空表被更多人拿去使用、改写、署名和误用后，青枫居 4XX 必须重新划清边界：表可以借，名字不能借；可以帮人把事情写清楚，不能替别人背书。

## 叙事位置

本章仍属于第五卷 5A 宿舍修补线。宿舍四人关系优先，楼层同学、C305 课堂、秦越替场和晚风票务只作为镜像与外部压力进入，不抢走中心。

## 核心冲突

1. 楼层群有人把 4XX 的空表改成“楼层公共版”，加上“责任人”等更像贴标签的栏位。
2. 罗泽、丁祺、何书南想继续借表，但也想借 4XX 的名字压寝室里不回消息的人。
3. C305 课堂要求提交空白格式和改动说明，陈老师只接受可复用的空格式，不接受把别人寝室问题搬上讲台。
4. 秦越把小舞台设备交接也套进表格，周屿必须学会给设备时间与归还状态，不把玩笑写进证明。
5. 晚风票务线也有人想用表格写“骗钱”，林亦舟只能保留付款、账号、平台处理这些具体事实。

## 场景与动线

- 青枫居 4XX：清晨看到未贴出的窄纸，楼层同学来敲门，公共桌成为边界谈判点。
- 西园早餐：罗泽、丁祺、何书南带着改坏的表来问，人物可坐在湿伞、热粥和 A4 纸之间说话。
- 教学楼 C305：计算机组成原理课堂，只讲空白格式和改动理由，不出现 A302。
- 东区商业街小舞台消息：只通过周屿手机进入，保持秦越为活动镜像。
- 北园粉面窗口或寝室走廊：林亦舟处理晚风票务信息，现实压力贴近但不夺走 5A。
- 夜间 4XX：有人想署 4XX 名，四人写出新的窄纸。

## 人物推进

- 林亦舟：从“帮着改字”走向“只给可执行路径”，不替晚风或楼层同学定性。
- 周屿：从会用玩笑缓和场面，转向给明确时间、设备归还和边界短句。
- 唐骁：意识到规则一旦被外借会变形，因此坚持去掉指向人的栏位。
- 陆沉：仍少说，但在关键处把“借表”和“借名字”分开。

## 文风要求

使用 chinese-novelist 原则：动作和对话承载心理，不写总结式觉悟；结尾留钩子。正文不得出现系统、节点、变量、判定、回流等策划词，也不使用“让林亦舟意识到”“下一句话会”等上帝视角句。
