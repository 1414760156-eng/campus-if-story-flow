# 第五卷第四十九章《还没发》设计

## 目标

承接第四十八章结尾的旧预览截图流入周五活动大群。写出“未定稿材料在群聊里被当成定稿”的现实压力，让宿舍四人在不扩大冲突的前提下完成一次更成熟的协作。

## 写作原则

- 不新增大事件，不开启 5X。
- 场景限定在青枫居 4XX、周五活动大群、私聊和活动材料协作。
- 周屿不再只是被保护的人，他要参与措辞并要求别人不要替他美化。
- 唐骁负责处理群聊秩序，但克制“冲上去吵”的冲动。
- 陆沉负责留档和事实核对，不做道德总结。
- 林亦舟作为协调者，只写行动和观察，不写自我顿悟句。

## 事件链

1. 活动大群有人发旧预览截图，称标题“有感觉”。
2. 唐骁想马上要求撤回，周屿提醒别把话写成替他洗白。
3. 陆沉提出先确认旧图从哪里流出、是否已经转发到别处。
4. 许棠、梁思予从推送端确认旧预览不是定稿，转发卡片需要重新生成。
5. 秦越在大群中用具体责任说明，不再用空泛道歉压过别人。
6. 截图发送者撤回，但群里出现“没恶意”“至于吗”的轻飘反应。
7. 姜宁、夏知微、晚风分别提供外部视角：别追着删每部手机里的图，要截断它成为公开材料的机会。
8. 四人确认新说明、新封面、新转发卡片。
9. 结尾钩子：旧图虽在活动群撤回，却已被人截到朋友圈或外部平台。

## 禁区

- 不使用开发者视角词。
- 不使用“让林亦舟意识到”“下一句话会”等上帝视角句式。
- 不出现明德楼、A302、许澈。
- 不把群聊冲突写成校园舆情，只保留局部失控感。
