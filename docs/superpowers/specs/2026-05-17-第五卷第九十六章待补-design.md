# 第五卷第九十六章《待补》设计记录

## 目标

承接第九十五章末尾“互动抽奖 / 主持人：待补”，把周屿是否临时回到麦前写成现场压力，而不是热血复归。重点不是让周屿重新当主持，而是让 4XX 在一个很小、很具体的环节里学会把空栏分开承担：谁念规则，谁看奖品，谁核台卡，谁拦截图。

## 场景

- 主场：东区商业街小舞台后台、幕布边和第二环节台侧。
- 现场元素：抽奖箱、礼品袋、侧屏名单、直播手机、台卡空栏、观众起哄。
- 远端：晚风只通过微信提醒“有人截到周屿”，不拉成新的解释支线。

## 人物职责

- 周屿：拒绝被直接填回主持人栏，但愿意临时念抽奖规则，且要求写清“临时协助”。
- 唐骁：防止“待补”又被人顺手填成周屿，负责核抽奖规则和奖品数量。
- 陆沉：把主持、协助、规则提示、奖品核对分成不同栏，避免一个名字承担全部。
- 林亦舟：搬礼品袋、看晚风消息、拦住外部直播截图扩散。
- 秦越：把活动端执行说明改成能让台侧照做的短句。
- 蒋策、姜宁：带出舞台端的急迫和对周屿回归的惯性期待。

## 情绪推进

本章要让“修补”落成动作：不是四个人抱团和好，而是每个人在一张台卡前承担一点，且不替别人填整栏。周屿的选择不应是“回归主持”，而是“只念规则、写清身份”；这既保留他的边界，也让宿舍四人共同把活动推进下去。

## 章末钩子

第二环节顺利走完，但台下和小群里已经有人截到周屿拿麦的瞬间。章末留下“临时协助被看成回归”的后续压力。
