# 第五卷第八十一章《先用》设计记录

## 目标

承接第八十章末尾 `正式素材_给冯喻.zip`，追查冯喻到底拿到哪一包，以及他如何把“给我复盘用”的材料继续发进交接群。本章重点不是把冯喻写成反派，而是写清楚大学活动里“先用”“先收齐”“别等”这些省事话如何让待核材料进入下一层传播。

## 本章设计

- 场景继续在东区商业街后台，先复制留存 `正式素材_给冯喻.zip`，再查看包内目录。
- 包内目录包含照片、侧屏、口播、花絮等混杂材料，其中口播文件夹仍含 `主持口播_临时合并.zip`。
- 冯喻通过电话/赶到后台说明自己收包是为了复盘素材，不是为了确认口播；但他确实把整包转发给“周五复盘素材交接（18）”。
- 宿舍四人继续担任场内压力结构：唐骁追“为什么整包转发”，周屿压住单人定罪，陆沉拆分“收到、下载、转发、看过、采用”，林亦舟收住晚风那边的外部包截图。
- 章末钩子落在冯喻发送时配的一句“素材都在，文案先用，别等我筛”，把下一章推向“先用”如何进入图文初稿。

## 写法约束

- 正文不写游戏口吻、变量口吻、裁判口吻。
- 不写解释性心理结论，不用上帝视角提示。
- 明德楼只作为上一段来源，不在本章出现教学用途。
- 不引入许澈，不打开 5X。
- 冯喻有责任，但不是单一恶人；流程旧习惯和模糊指令同样重要。

## 预期推进

本章完成“公用电脑包 -> 冯喻 -> 素材交接群”的完整小节，并把下一章压力推进到文案初稿和图文引用材料。
