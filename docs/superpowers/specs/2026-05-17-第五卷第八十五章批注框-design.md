# 第五卷第八十五章《批注框》设计

## 本章定位

承接第八十四章结尾，旧稿里露出黎旻 23:58 的批注：“这版别给文案看，太像以前汇演稿。台侧对词可以留，发稿不要碰。”本章不把黎旻写成来解释一切的人，也不让批注成为万能证据，而是把“提醒存在过”继续往下拆：批注写在 Word 右侧，语音发在候场小群，真正转述出去时只剩半句。

## 场景与动线

- 场景仍固定在东区商业街小舞台后台。
- 具体物件：旧一体机、陈景电脑、白卡、手卡夹、话筒箱、胶带、活动证、手机语音条、后台门缝里的收场声。
- 不进入明德楼，不新增校外交通，不写 A302，不引入许澈。

## 人物功能

- 林亦舟：只做现场核对中的压缩与传递，不下结论；晚风消息最多作为未吃饭的外部压力。
- 周屿：守住旧句不能被继续包装，也不让黎旻一句“我写过了”变成脱责。
- 唐骁：追问批注“给谁看”，语音“发在哪”，以及为什么“台侧可以留”能被传成可用。
- 陆沉：把批注位置、可见对象、语音位置、转述句、失效位置落表。
- 黎旻：疲惫、实际、带着台侧忙乱赶来。她确实提醒过，但提醒只留在自己以为有效的位置。
- 秦越、袁承、胡嘉宁、陈景、冯喻、老师：共同提供流转链上的动作，不抢宿舍四人中心。

## 叙事目标

1. 黎旻从台侧被叫来，带着话筒箱和胶带痕迹，呈现真实后台工作状态。
2. 复核 Word 批注：批注存在，但只在打开文档正文后可见，搜索预览与压缩包目录都看不见。
3. 复核 01:23 语音：黎旻曾说旧稿只给台侧对词，不要往图文那边传。
4. 追到转述失真：袁承把“台侧对词可以留，发稿不要碰”转成“台侧可以留，补字的人先看一下”。
5. 章末留下下一章压力：语音无法被截成半句，但文字转述已经只剩半句。

## 禁止写法提醒

- 不写系统、分支、判定、触发、变量等游戏口吻。
- 不写“让林亦舟意识到”“他突然发现”“下一句话会”等解释性旁白。
- 不写“你怕我怕”式心理并列。
- 不让晚风、黎旻或老师替宿舍四人完成本章的关系推进。
