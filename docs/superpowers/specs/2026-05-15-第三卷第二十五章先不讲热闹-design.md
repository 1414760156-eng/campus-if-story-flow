# 第三卷第二十五章《先不讲热闹》设计记录

## 目标

承接第二十四章末尾“各位晚上好，我今天先不讲热闹”和 B206 二轮/上机提交双重压力，把周屿主持招新线从“练习表达”推到“能否被放进候选名单”的现实门槛，同时让宿舍四人继续面对计科学业和互相催促的生活摩擦。

## 必守约束

- 主持队二轮发生在教学楼 B206，不写进明德楼。
- 程序设计基础提交压力回到青枫居和机房/电脑端，不把课程逻辑挪到行政空间。
- 宿舍四人优先：周屿承担 B206 二轮与代码截止冲突；唐骁守截止、守文件命名和个人提交；陆沉只讲思路与样例，不代写；林亦舟在宿舍和晚风之间处理“帮忙但不替人说话”的边界。
- 晚风室友音乐节票被骗线只做现实支线推进：保留记录、找辅导员、停止私下转账，不变成夸张案件。
- 支线人物自然出现：秦越在 B206 给二轮标准和试岗机会；罗泽可在课程群继续索要或反馈；许棠只以模板归档消息轻触，不喧宾夺主。

## 文学处理

- 使用 chinese-novelist 原则：用两个窗口、两条名单、B206 灯光、青枫居键盘声、课程群提醒推进。
- 不写抽象心理总结，不写游戏口吻，不写“节点/触发/判定/回流”等正文禁用词。
- 冲突落在动作和对话里：唐骁催提交、周屿嫌被盯、陆沉把样例纸推回来、林亦舟删改给晚风的消息。

## 章节结构

1. 周六白天，青枫居补代码与改主持稿并行，周屿想先去 B206，唐骁逼他先把能提交的部分整理清楚。
2. 晚风发来被骗室友后续：辅导员登记、室友不愿告诉家里，林亦舟只给步骤，不抢话。
3. 傍晚去 B206，秦越要求一分钟开场不要卖惨，也不要把水房变成道德课。
4. 周屿用“先不讲热闹”开场，被指出有力但太重；秦越给他东区商业街小舞台试岗机会，同时强调迟到和不交作业都不是梗。
5. 周屿回青枫居赶上机提交，宿舍四人短暂争执后各自守住边界。章末课程群与主持队群分别出现“未提交名单预警”和“候选名单明早出”。

## 本章钩子

两个名单同屏出现：一个来自课程群，一个来自主持队群。第三卷下一章可继续用“名单”收束热闹、责任和分流。
