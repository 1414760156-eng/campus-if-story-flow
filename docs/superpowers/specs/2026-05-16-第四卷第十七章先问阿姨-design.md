# 第四卷第十七章《先问阿姨》设计记录

## 使用技能

- superpower-bootstrap：延续技能检查和项目约束。
- superpower-brainstorming：先把“外卖架统计”拆成真实校园冲突，避免写成顺滑课堂作业。
- planning-with-files：沿用现有 specs/plans/progress 结构恢复断点。
- superpower-writing-plans：写清本章验收项后再动正文。
- chinese-novelist：用动作、物件、对话和场景推进心理，不写系统口吻。
- superpower-verification：完成后跑字数和禁用表达扫描。

## 本章承接

第十六章结尾停在陈老师要求“外卖取餐统计需先说明数据来源，必要时请征得宿管同意”。周屿问阿姨会不会同意。第十七章需要把这个问题写成一次真实接触校园规则的事件：作业要用校园生活，但校园生活里有人的隐私、时间和不耐烦。

## 本章职责

1. 主场固定在青枫居 4XX、青枫居一楼宿管窗口和外卖架，可短暂经过打印店或便利店。
2. 宿管阿姨必须有现实经验和稳定权力：门禁、外卖架、晚归登记、丢餐、架子拥堵。
3. 四人分工要可见：
   - 林亦舟负责把作业意图说成人话。
   - 周屿负责开口和缓和现场，但不能用玩笑糊弄。
   - 唐骁负责表格和说明纸，学会删掉过度记录项。
   - 陆沉负责隐私边界和现场站位，提出不拍脸、不记寝室号、不记取件码。
4. 现场必须有轻冲突：有人找不到外卖，有人误会他们在记录取餐码或查寝，楼层群出现质疑。
5. 晚风只通过消息给林亦舟一个外部回声，不替宿舍给结论。
6. 结尾留下下一章钩子：说明纸贴出后，楼层群追问“谁在统计”，把课堂作业压力推回公共沟通。

## 文风禁线

- 不写“让林亦舟意识到”“他突然发现”“下一句话会”等解释性句子。
- 不写“你怕我怕他怕”式并列心理总结。
- 正文避开：主线、副线、分线、节点、flag、触发、权重、判定、回主线、资源线。
- 第四卷继续避开：觉得、发现、突然、忽然、让林亦舟、下一句话、怕被、怕规则、怕自己。
