# 第五卷第八章《照着放》设计说明

## 目标

续写第五卷第八章《照着放》，承接第七章末尾的 C305 本人设备复核和宿管阿姨公共区抽查，把“写在纸上的边界”推进为“现实里能不能照着放”。

## 核心原则

- 宿舍四人优先。晚风票务只做镜像压力，不抢走 5A 宿舍修补中心。
- 继续使用 `07_学期课程表与教室索引.md`：计算机组成原理相关核验仍在教学楼 C305，不回到 A302。
- 生活习惯摩擦必须具体：公共桌、插排、外卖袋、话筒线、洗衣液、旧包、床帘、键盘声，都要以动作和物件推进。
- 不写系统提示、不写变量解释、不写上帝视角预测。心理暗线靠行为呈现。

## 章节事件设计

1. 青枫居清晨：四行纸还在公共桌上，但真实寝室已经开始偏离。周屿的话筒线没完全收好，唐骁的核验清单占了公共桌，陆沉旧包被挤到门后，林亦舟母亲寄来的袜子和洗衣液仍需要安置。
2. C305 核验：助教要求本人设备和提交时间对应。别组出现代登、借设备和“只是顺手”的小事故。22 组四人各自通过，但过程让他们意识到“本人”不是口头承诺。
3. 晚风镜像：晚风按林亦舟建议要求付款人自己出面，对方开始撤退或威胁不管。林亦舟不接私发截图，只给晚风可执行的现实路径：付款人、收款方、聊天约定、平台记录和辅导员/相关人。
4. 公共区抽查：宿管阿姨下午检查青枫居 4XX，不看四行纸写得好不好，只看放得对不对。她指出公共桌和照片之间的差异，让四人把纸上规矩改成更具体可执行的摆放。
5. 章末钩子：楼层群要求今晚十点前发整改照片，且不许拍完再挪回去；同一时间课程群通知下次要提交设备异常说明。宿舍修补从“解释清楚”进入“持续维持”。
