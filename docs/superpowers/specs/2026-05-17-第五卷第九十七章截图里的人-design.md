# 第五卷第九十七章《截图里的人》设计记录

## 写作目标

- 承接第九十六章末尾“周屿临时拿麦被截图误读为回归”。
- 把外部误读压到散场后的具体处理里，不再开启新的无限追查链。
- 保持宿舍四人为核心：周屿守住边界，唐骁想解释但改成事实短句，陆沉拆清“拿麦”和“主持名单”，林亦舟处理晚风那边的转发压力。
- 加入东区商业街散场、奶茶店收摊、台卡留存、晚归登记和宿管阿姨等生活细节，让事件回到青枫居。

## 冲突焦点

- 截图只拍到周屿拿麦，没拍到台卡旁边的“临时提示规则”。
- 活动端需要公开说清事实，但不能把周屿重新推到台前，也不能把姜宁的主持身份冲掉。
- 晚风那边已经看到二次转发，林亦舟要给出可说的一句话，同时不把私人截图继续扩散。
- 周屿不想被当作“回来了”，也不想让别人替他辩白成新的表演。

## 章节收束

- 活动端发布短说明：第二环节为姜宁主持，周屿临时提示规则，主持名单不作调整。
- 周屿只签“临时提示规则”，不签主持栏。
- 散场后四人回青枫居，宿管阿姨让四个人一一补晚归登记。
- 章末以晚归登记本把外部截图的喧闹压回同一扇门，为第九十八章回寝室处理旧账做入口。
