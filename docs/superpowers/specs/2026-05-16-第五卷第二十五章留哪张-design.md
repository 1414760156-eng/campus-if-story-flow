# 第五卷第二十五章《留哪张》设计

## Summary

承接第二十四章末楼层群追问“原截图留不留”。本章继续推进第五卷 5A 宿舍修补线，重点把“保留原截图作为私下查证材料”和“继续在群里传播旧截图”拆开，让 4XX 在留证、撤错、公开更正之间找到窄办法。

## Chosen approach

采用“手按手机 - 宿管窗口定边界 - 西园餐桌补现实疑问 - C305 三类材料 - 支线镜像 - 夜里封存期限”的结构。

这样能守住宿舍四人主轴：周屿想把旧截图留住，免得以后又被改写；唐骁想立刻标注来源、时间和不可外发；陆沉提醒旧图留着也会变成新的压力；林亦舟把楼层群回复压成可执行句子。罗泽、丁祺、何书南、沈嘉禾、秦越、夏知微、晚风只用于照出同一问题的不同边界，不抢走 4XX。

## Scene beats

1. 青枫居 4XX 夜里，唐骁按住屏幕，四人围绕“留原图、删旧图、换更正图、只留记录”分歧。
2. 青枫居一楼宿管窗口，宿管阿姨要求旧图别再往群里挂，若要留只能放到楼长可查的记录里，先让群里停转旧截图。
3. 西园餐厅中，罗泽、丁祺、何书南带来椅背材料的同类问题：错误旧图要不要留下，谁能看，什么时候删。
4. 教学楼 C305 中，陈老师要求把材料分成“公开更正、私下查证、应撤旧图”三类，不把证据当成继续传播的理由。
5. 秦越小舞台设备视频、夏知微摄影社登记表、晚风票务聊天记录分别作为镜像，处理旧视频、旧误注和骂人截图的保存边界。
6. 夜里 4XX 把原截图封进旧文件袋附页，只留编号和保存期限；楼层群有人继续问“留到什么时候”，章末把压力推向保存期限和删除权。

## Constraints

- 不开启 5X，不让许澈出现。
- 不写“让林亦舟意识到”等解释性句子。
- 不写“下一句话会”式预告。
- 正文不出现游戏系统词。
- 课堂地点继续使用教学楼 C305；明德楼不承载课程、班会、答疑、机房或上机。
- 交通不新增校外动线。
- 不把更正写成公开审人，不把支线人物写成判官。
