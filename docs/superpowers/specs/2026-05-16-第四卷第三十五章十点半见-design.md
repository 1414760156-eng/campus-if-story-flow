# 第四卷第三十五章《十点半见》设计说明

## 写作目标

承接第三十四章结尾的周五冲突处理记录要求，把“冲突双方都要到场说明取舍”从课程通知压进当天晚上的时间表。重点写现实挤压：唐骁的计算机二级直播、周屿的东区商业街小舞台收尾、陆沉的图书馆西侧顶班、林亦舟与晚风关于“表格会骗人吗”的消息，都必须回到青枫居 4XX 的同一张公共桌。

## 场景边界

- 可用场景：西园餐厅、教学楼外、青枫居 4XX、东区商业街小舞台、图书馆西侧、星辰书屋或楼下外卖架。
- 不使用明德楼承载课程、班会、机房、答疑或上机。
- 不新增校外交通动线。
- 许澈不出现。

## 人物推进

- 林亦舟：记录自己造成的 README 冲突，不再把组长位置写成“整理者无错”。晚风消息只提供旁照，不替他处理宿舍。
- 周屿：收尾不能加玩笑，和写冲突取舍一样，需要把话压住；他必须在小舞台之后带着被压住的话回寝。
- 唐骁：在二级直播和冲突记录之间被挤压，容易把记录写成规范答案；本章让他把“为什么保留说一声”写进记录。
- 陆沉：只顶一小时勤工，九点前回寝，不让别人替他写自己到场确认。

## 事件设计

1. 西园餐厅中午：四人把冲突记录拆成四栏，约定十点半在青枫居碰头；唐骁不愿拖，周屿想先去小舞台，陆沉报出自己图书馆时间。
2. 下午空档：林亦舟在青枫居起草记录，晚风追问“表格会不会把人写轻”；他删掉太漂亮的组长说明。
3. 傍晚分头：周屿去东区商业街小舞台，秦越压他收尾；唐骁补二级直播，老师讲第 18/19 题合并；陆沉去图书馆西侧顶班，星辰书屋/图书馆里的同学也在备考。
4. 九点后回流：三个人陆续回到青枫居，中途用楼下外卖架、常温水、润喉糖和旧电脑作为生活锚点。
5. 十点半公共桌：四人打开冲突记录，唐骁想写“规则更清晰”，周屿要求写“说一声”的人味，陆沉要求写“到场确认”不是替答，林亦舟把自己造成冲突写进第一栏。
6. 章末钩子：记录初稿完成，但课程群又发明天课堂要抽“被审核人反问审核人”的提醒，压力从冲突记录继续转向互审关系。

## 文风约束

- 展示而非讲述，用动作、表格删改、消息、舞台收尾和旧电脑等待表现心理。
- 避免第四卷高风险词：觉得、发现、突然、忽然、让林亦舟、下一句话、怕被、怕规则、怕自己。
- 不写玩家/系统口吻，不出现主线、副线、节点、触发等正文禁词。
