# 第五卷第九十章《没回到群里》设计

## 使用技能

- planning-with-files：承接第八十九章断点和进度统计。
- brainstorming：本章追“看出来了为何没有形成退回动作”，不另起新事故。
- writing-plans：明确场景、人物功能、证据推进和章末钩子。
- chinese-novelist：用手机记录、私聊、截图和现场反应表现冲突。
- verification：写后统计并扫描禁用表达、地点误用和不宜词。

## 本章目标

承接秦越在 `周五发布前确认` 小群里发出“这句开头是不是太像以前那版？”。本章追查这句话之后，许棠、梁思予、老师和秦越各自做了什么，解释为什么“看出来了”没有变成“退回并停用”。

## 场景与人物

- 主场仍为东区商业街后台，所有人围着手机、白卡、旧一体机和截图。
- 明德楼只作为许棠所在的融媒体工作室背景，通过电话和截图接入。
- 宿舍四人优先：
  - 周屿：指出问题不只是有没有发，而是异议有没有被看见。
  - 唐骁：追问“等老师看”和“先别发”的差别。
  - 陆沉：记录群聊、私聊、时间、动作和谁没看见。
  - 林亦舟：短暂收住晚风消息，不让外部线吞掉场面。

## 章节推进

1. 秦越拿出完整聊天，确认他在 02:57 提出像旧版。
2. 许棠翻出自己与老师的私聊：老师回过“先别发，明早重看”，但这句话没有发回确认群。
3. 梁思予说明她只看见群里“等老师看”，没有看见“先别发”，于是继续保留草稿和改后截图。
4. 唐骁追问：谁说了停、停在哪里、谁没有收到。
5. 老师承认当时在收设备和看借用表，私聊回复后没有回群；她不推责，但也要求所有人把动作说全。
6. 陆沉把表格拆成“异议出现、私聊回应、群内未同步、后续草稿保留”。
7. 章末许棠私聊里还有一张 `改后预览_0306.png`，图上标题已经改了，分享摘要仍残留旧句，为下一章追“改了哪里，没改哪里”留钩子。

## 风格提醒

- 不写作者总结，不写上帝视角预告。
- 不使用系统、节点、触发、判定等游戏口吻。
- 不让老师成为夸张反派；冲突来自真实工作流的分散和沉默。
