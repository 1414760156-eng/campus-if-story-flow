# 第五卷第十七章《改回谁的》设计记录

## 写作目标

- 承接第十六章末尾“被别人改过的空表”和“表可以借，名字不借”。
- 继续推进 5A 宿舍修补线，不开启 5X，不改变第五卷收束方向。
- 把冲突从“能不能借表”推进到“别人把表改成好听的话以后，事实和当事人的声音是否被抹掉”。
- 外部人物只作为镜像和压力来源：罗泽、丁祺、何书南、沈嘉禾、秦越、晚风均出现，但不抢占 4XX 宿舍四人的中心。

## 场景与动线

1. 青枫居 4XX：窄纸仍贴在公共桌边，楼层群里出现把事实改成“友好沟通”的版本。
2. 西园餐厅：罗泽、丁祺、何书南带来二次改表，暴露“写得好听”与“写清楚事”的冲突。
3. 教学楼 C305：计算机组成原理课堂继续检查“被别人改过的空表”和修改说明。
4. 教学楼走廊、东区商业街小舞台消息、北园二楼消息：秦越和晚风分别用设备交接、票务投诉提供现实镜像。
5. 青枫居 4XX 夜间：四人把自己的窄纸补成第二行“改字不改事”，形成下一章压力。

## 人物推进

- 林亦舟：从“递空表”转向“保留事实不替人润色”，继续在晚风票务线中克制介入。
- 周屿：不再用玩笑遮掉尴尬，学会把小舞台交接里的评价词删掉，只留时间和动作。
- 唐骁：对扩散失控保持警惕，但不再一味拒绝，开始承认有些原话需要保留位置。
- 陆沉：继续把可公开内容和寝室内部内容分开，给出“能改字，别改掉事”的底层判断。
- 罗泽等楼层人物：从借表者变成二次修改者，带来“好听版”的具体后果。
- 沈嘉禾：用“敏感内容/不展示内容”的措辞差异做侧面参照。
- 秦越、晚风：分别从活动和票务两个现实场景补强“改字不改事”的必要性。

## 风格约束

- 正文不写系统口吻，不写游戏开发术语。
- 不用“让林亦舟意识到”“下一句话会”等解释式句子。
- 不使用明德楼承载课程、班会、答疑、机房或上机。
- 第五卷课堂继续锁定教学楼 C305，避免 A302 残留。
