# 第五卷第五十五章《半行白字》设计

## 使用的技能

- `chinese-novelist`：正文采用展示而非讲述、冲突驱动、章末留钩。
- `planning-with-files`：继续使用根目录 `task_plan.md`、`progress.md` 和母版 progress 记录断点与执行结果。
- `superpower-brainstorming`：在既有母版框架内做短设计，不改大纲。
- `superpower-writing-plans`：写入执行计划后再动正文。

## 断点

第五十四章结尾，侧屏黑底预览时旧句“同一束灯下，看见当年的自己。”只闪出半行。周屿放下水瓶，旧材料并没有真正离开现场。

## 本章目标

第五十五章不升级为公开事故，而是把风险压进“预览缓存、拍屏、群内误传、倒计时”四个具体动作里。宿舍四人继续围绕同一件事分工：周屿拒绝再被旧叙述套住，唐骁追查文件路径和预览状态，陆沉留证并限制扩散，林亦舟在现场人员之间传话。

## 场景与动线

仍在东区商业街小舞台后台与侧屏控制台，所有动作围绕控制台、侧屏、字幕志愿者群、融媒体老师和台口倒计时发生。不离开武生院，不使用明德楼，不涉及课程教室。

## 冲突推进

1. 旧句短暂闪出后，现场几个人都看见了，但不能立刻追究。
2. 叶青区分“正式播放”和“预览残留”，打开工程缓存与历史缩略图。
3. 唐骁要求查清旧句来源，不接受“已经删了”的口头说法。
4. 陆沉记录黑屏预览、缓存清理和再次测试，避免事后各说各话。
5. 夏知微提醒不要让控制台本身成为新的素材。
6. 字幕志愿者已经拍了一张侧屏闪字，发到小群里，章末留下撤不撤的压力。

## 禁用写法提醒

正文不写系统口吻，不写“让林亦舟意识到”，不写“下一句话会”，不写并列心理总结，不出现 `A302` 和 `许澈`。
