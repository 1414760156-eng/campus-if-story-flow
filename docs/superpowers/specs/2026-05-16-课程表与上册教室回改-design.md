# 课程表与上册教室回改设计记录

## 背景

用户指出“大学不是一直在 A302 上课”，并进一步确认正确逻辑应为：同一门课同一学期可以固定教室，不同课程、不同学期、考试、补考和班会应有不同地点。剧情本来随时间推进，因此每个阶段都应有固定课程表，课程地点的远近也可以成为日常剧情来源。

## 设计决定

1. 新增课程表索引文件，作为正文和互动拆分的底层调用表。
2. 总控规则从“教室必须分散使用”改为“同课固定、跨课分散”，避免把同一门课的连续课堂误判成地点错误。
3. 上册 1-4 卷不做大范围重写，只在课程表入口、第一次课堂出现、第四卷 A302 首次集中出现处补明原因。
4. 第五卷当前课程地点从 A302 纠偏为 C305，并同步设计、计划、进度记录。
5. 上册开发测试版重新生成，让开发侧读取到最新源正文。

## 课程地点逻辑

- 大一上：专业导论 A101；高等数学 C201；大学英语 B306；程序设计基础理论课 B307；程序设计实验在八号教学楼三楼机房；班会 B204。
- 大一下至第四卷：数据结构实验固定 A302；线性代数/离散数学导论使用 B307/C305；计算机二级和补测进八号教学楼机房；补考和答疑使用 C201；暑假安全签字 B204。
- 大二上第五卷：新学期班会 B206；计算机组成原理 C305；后续操作系统导论可用 B307，数据库可用 C201 或机房。
