# 第四卷第十六章《换组理由》设计记录

## 使用技能

- superpower-bootstrap：继续遵守技能检查和追踪。
- superpower-brainstorming：先审本章事件功能，避免只写表格填空。
- planning-with-files：用现有计划、发现记录和进度保持连续。
- superpower-writing-plans：把本章写作拆成可验收检查项。
- chinese-novelist：展示而非讲述，用物件、动作、消息和场景承载心理。
- superpower-verification：完成后运行新鲜字数统计和禁用表达扫描。

## 本章承接

第十五章结尾停在西园餐厅饭桌：互评表下载、运动会报名、周五主持、计算机二级准考证、图书馆排班和晚风消息同时挤进手机。第十六章要把“继续同组还是申请换组”从文件选项压成四个人的现实犹豫。

## 本章职责

1. 主场从西园餐厅转回青枫居，再可短暂经过阳光书屋、校园快递站或图书馆排班处。
2. 互评表不能写成顺滑填写，要写每个人怎么面对“贡献”“沟通”“是否继续同组”这些词。
3. 四人都要有压力：
   - 周屿：周五主持队临时主持、运动会报名、想被看见又怕误事。
   - 唐骁：计算机二级准考证和规则感，担心实验二继续混乱。
   - 陆沉：图书馆排班、经济/勤工压力和“申请换组”一栏带来的低声犹豫。
   - 林亦舟：组长责任、晚风消息、家庭/生活费暗线和“写人话”的难度。
4. 实验二选题要落到具体校园数据：运动会报名数据、图书馆借阅记录、外卖取餐统计；不能变成抽象项目讨论。
5. 第四卷引导式外流可轻轻出现：林亦舟短暂想绕到便利店或后街，但必须回到青枫居和表格。
6. 结尾留钩子：互评表提交后，陈老师或助教发来“实验二分组确认/换组申请名单”，或周五三重时间冲突进一步逼近。

## 文风禁线

- 不写“让林亦舟意识到”“他突然发现”“下一句话会”等上帝视角。
- 不写“怕被/怕规则/怕自己”并列心理句。
- 不写主线、副线、节点、触发、判定、资源线等游戏口吻。
- 避免第四卷扫描词：觉得、发现、突然、忽然。
