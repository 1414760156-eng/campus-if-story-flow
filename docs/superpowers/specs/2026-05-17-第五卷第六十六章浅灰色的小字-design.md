# 第五卷第六十六章《浅灰色的小字》写作设计

## 目标

承接第六十五章末尾的公众号分享摘要残留旧句。标题已经改掉，但真正流到群卡片、朋友圈预览和手机分享里的，仍有一行浅灰色小字。第六十六章要把问题从“改标题”推进到“每个会被看见的位置都要核对”，继续压实 5A 宿舍修补线，不开启 5X。

## 场景与路线

- 主场景：青枫居 4XX 公共桌。
- 远程支援：许棠、梁思予、秦越通过工作小群、截图、验证码、语音短句参与。
- 明德楼只作为刚才融媒体工作室的来源，不在本章承担课堂、班会、考试或训练功能。
- 晚风只作为外部转发环境的短消息提醒，不抢占宿舍四人中心。

## 人物推进

- 周屿：拒绝“好看化”和“把旧话改成更漂亮的结尾”，要求把看得见的位置逐个清掉。
- 唐骁：第一反应是立刻冲回明德楼找人，随后在公共桌前把火气压进核对动作。
- 陆沉：把标题、摘要、正文首行、封面、群卡片、朋友圈预览、预约发送拆成不同位置，不替任何人下结论。
- 林亦舟：处理晚风外部提醒，只回必要信息，把注意力拉回宿舍公共桌。
- 许棠、梁思予、秦越：不只是远程 NPC，而是补出后台字段、审核口、验证码和预约发送信息的具体参与者。

## 冲突钩子

看似改完标题和摘要后，截图底部又露出“预约发送时间：明早八点三十分”。章末停在这个时间点，逼下一章处理定时发布、审核人、发布端和宿舍四人能否在明早前拦住旧卡片。

## 避免事项

- 不出现许澈。
- 不写 A302。
- 不用系统口吻，不写“触发、判定、主线、节点”等开发表达。
- 不写“让林亦舟意识到”“下一句话会”等上帝视角句式。
- 不把晚风写成核心，不让她替宿舍线做决定。
