# 第四卷第四十九章《签字》与上册总结设计

## 使用的创作技能

- superpower-bootstrap：本轮继续按技能流程执行。
- superpower-brainstorming：先确定第四卷收束方式，不把 B204 签字写成机械手续。
- planning-with-files：读取总控、进度、断点、task_plan、findings 和根 progress。
- superpower-writing-plans：把正文收束和上册总结拆成可执行任务。
- chinese-novelist：正文用动作、对话、物件、空间和沉默推进，不写解释性心理总结。
- superpower-verification：完成后做字数、禁用表达、地点交通和章节顺序检查。

## 本轮目标

续写第四十九章，完成第四卷作为“上册终点”前的正式收束：周一教学楼 B204 纸质签字、暑假去向确认、联系人顺序、留校/返家安排和青枫居公共桌最后一次整理。随后生成一份上册阶段总结，供后续把 1-4 卷作为上册、5-7 卷作为下册时参考玩法和剧情调性。

## 第四十九章定位

第四十九章不是新冲突爆炸，而是把第四卷积累的几条现实压力压到同一张纸上：本人签字、联系人、留校申请、返家截图、父亲转账、母亲包裹、宿舍公共桌。章节需要有明确事件推进：B204 现场出现“能不能替签”的小摩擦，林亦舟和宿舍四人用前几章练出来的方式处理它；同时每个人都被暑假推向不同方向。

## 必须遵守

- B204 是教学楼场景，符合班级/安全确认逻辑；不使用明德楼承载班会或课程。
- 周屿、唐骁、陆沉、林亦舟四人都要在签字和离校/留校安排中有具体动作。
- 晚风票务线只作余波，不抢第四卷收束核心。
- 许澈不出现或只保留前文远影，不进入关系核心。
- 不写“让林亦舟意识到”“下一句话会”等禁用表达。

## 章节场景设计

1. 青枫居清晨：拖箱子、绿豆糕、四栏文件夹和校园卡；四人确认 B204 签字材料。
2. 西园餐厅短饭：离校同学、四六级耳机、行李箱和宿舍互相提醒。
3. 教学楼 B204：学习委员、辅导员/老师、纸质签字表；有人试图替已离校室友签字，被要求本人补说明。
4. B204 走廊：四人签完后短暂停留，讨论周三谁送地铁、谁留校、谁回家；不要大段告别。
5. 青枫居夜里：公共桌清空一部分，四栏文件夹收好；四人把各自东西放回各自位置，第四卷以暑假真正开始的空位收束。

## 上册总结设计

新增 `docs/剧情小说母版_v4.0/上册阶段总结_第一至四卷.md`。总结内容包括：

- 上册的文学定位：从入学到暑假，日常现实主义群像和关系基础。
- 四卷各自完成的叙事功能。
- 宿舍四人弧光状态。
- 关键支线状态：晚风、父母、夏知微、沈嘉禾、秦越、许棠、许澈远影。
- 对后续玩法的提示：上册偏探索积累、多线并行和微选择；下册偏收束、强选择、唯一外流和结局回响。
