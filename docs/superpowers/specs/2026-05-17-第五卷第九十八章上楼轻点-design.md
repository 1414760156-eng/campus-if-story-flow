# 第五卷第九十八章《上楼轻点》设计记录

## 写作目标

- 承接第九十七章末尾“谁也没法替谁补”的晚归登记，把外部截图余波带回青枫居 4XX。
- 让宿舍修补落到更具体的生活动作：轻声上楼、开灯、洗澡顺序、椅子声、公共桌、台卡放置、群消息回复。
- 不继续扩大战线，不再追截图来源，而是处理“别人问到寝室门口时，谁来说”。
- 让周屿短句回应班级群，不让唐骁和林亦舟替他说；唐骁压住替人出头，陆沉留存台卡，林亦舟收住晚风和室友群的回声。

## 冲突焦点

- 青枫居走廊有人睡了，四个人必须把情绪压进小动作里。
- 台卡是事实留存，但摆在公共桌上会再次提醒周屿；收起来又像把问题藏掉。
- 班级群有人把截图带进宿舍关系里，唐骁想替周屿回，周屿选择用一句短事实自己说。
- 晚风的消息也到寝室门口，林亦舟需要回应，但不能让晚风成为宿舍事件的外部裁判。

## 章节收束

- 周屿在群里只发事实短句：临时念规则，主持不是我，别转图。
- 四人把台卡夹进陆沉记录本，不贴上墙，也不扔掉。
- 熄灯后每个人手机仍有各自压力：活动复盘、证书提醒、排班、晚风晚安。
- 章末用公共桌上四杯少冰柠檬水的水圈和台卡边角，留出第五卷最后收束入口。
