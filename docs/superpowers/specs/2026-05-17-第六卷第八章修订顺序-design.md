# 第六卷第八章《修订顺序》设计记录

## 写作目标

承接第七章结尾的修订顺序表，把“第一版能打开但问题不少”推进到“谁先改、谁复测、为什么先改”的责任拆分。章节重点不是扩大新事件，而是让四人围绕表格发生一次有摩擦但可收住的分工重排，为后续第二版、复测记录和准结局材料留下可调用依据。

## 锁点

- 宿舍四人仍为叙事中心：唐骁负责页面改动，林亦舟负责复测记录，周屿检查说明文字，陆沉核对公开规则和记录来源。
- 地点限制在青枫居 4XX、教学楼旁打印店、教学楼 C407、西园餐厅短场，不使用明德楼承载课堂或班会。
- 第六卷继续控制篇幅，不复刻第五卷逐材料追查的长链条。
- 不引入新强支线，不让晚风替代宿舍关系，只作为微信短句和现实感回声。
- 本章结尾必须给出下一章入口：周五前第二版页面、最新版复测记录、说明页修订痕迹和现场复测。

## 情节骨架

1. 青枫居 4XX 夜里继续写修订顺序表，第二行“旧时间”引发“改动人”和“复测人”混写争执。
2. 陆沉把签字拆成改动、复测、核对三类，周屿压住唐骁想把“好改”写成优先理由的冲动。
3. 周三打印前，四人在教学楼旁打印店核对最新页码和复测记录，林亦舟把自己的名字从“修”栏挪到“复测”栏。
4. C407 检查中，冯助教和陈老师追问排序理由：为什么先清除、再旧时间、再说明页。
5. 唐骁现场演示第一项已改，第二项仍未完成，林亦舟读复测记录，四人承认未完成项并保留到周五。
6. 章末陈老师要求周五交第二版和现场复测，林亦舟名字被写进“复测到场”格。

## 表达要求

- 展示而非讲述，不写“成长”“意识到”“关系变化”等总结句。
- 具体物件推动情绪：修订顺序表、红笔、打印纸、文件名、复测记录、C407 桌面、手机消息。
- 对话保留大学生口吻，避免“你怕我怕他怕”式心理归纳。
- 章末留钩子，但不使用上帝视角提示。
