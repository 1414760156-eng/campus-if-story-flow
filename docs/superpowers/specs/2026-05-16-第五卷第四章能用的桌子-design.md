# 第五卷第四章《能用的桌子》设计稿

## 目标

承接第三章《晚归登记》结尾，让“公共桌照片”接受真实检查。第四章不扩大路线，不启动 5X，只继续 5A 宿舍修补线：四个人必须在第二天早晨把桌面、本人事项和各自时间冲突处理清楚。

## 章节功能

- 宿舍四人优先：周屿面对八点二十东区商业街补录和楼长检查的冲突；唐骁从“整理出一张好看的桌子”转向承认桌子要真的可用；陆沉保留图书馆排班和盖章表，不被拉去补照片；林亦舟把晚风票务材料、父亲转账消息和寝室检查分开。
- 事件驱动：楼长和宿管阿姨上楼检查公共桌、周屿带回的话筒线需要处理、唐骁课程协作表需要重新放置、陆沉图书馆表不能被挪走、晚风压缩包和父亲消息继续在手机里挤压。
- 支线只加压强：秦越通过补录消息进入；晚风通过票务材料进入；父亲通过半额转账和短消息进入；不让她们替宿舍矛盾给结论。
- 生活血肉：西园早餐、青枫居楼下外卖架、图书馆西侧和东区商业街小舞台作为同一天的动线，不写明德楼，不写校外公交式交通。

## 场景路径

1. 青枫居 4XX 清晨：四人被楼长检查公共桌前的声音叫醒，桌面照片和真实桌面不一致。
2. 青枫居公共桌检查：宿管阿姨和楼长看见话筒线、胶带、公共药盒、盖章表、课程协作表，各自追问物品归属。
3. 西园餐厅短饭桌：四人用早餐分出“谁今天能处理什么”，避免把周屿补录、陆沉排班、唐骁协作表和林亦舟手机材料揉成一件事。
4. 分流并回到同一张桌：周屿去东区商业街补录，陆沉去图书馆西侧，林亦舟处理晚风和父亲消息，唐骁留在 4XX 重拍真实桌面；傍晚再次把更新后的桌面照片、物品位置和本人说明放到一起。

## 人物弧光

- 林亦舟：从拿手机处理所有消息，转成把每条消息放回自己的位置；他不替晚风交材料，也不替父亲遮掩家庭变化。
- 周屿：从想先赶补录，转成留下本人说明和物品归属，再去东区商业街；他用行动补一块桌面空缺。
- 唐骁：从拍一张整齐照片，转成接受桌面有真实使用痕迹；规则不再只为看起来漂亮。
- 陆沉：不让盖章表和旧本子被当作公共证明；他可以把东西摆到桌上，但话仍然由自己说。

## 章节钩子

傍晚公共桌重新拍照后，课程群又发出“明天交课程协作时间表最终版”的通知；同一时间，晚风说她室友不想交投诉材料了。第五章可以继续把“本人事项”推到课程、晚风和家庭三条压力里。

## 禁用表达提醒

正文不使用系统口吻和解释性套话；不写“让某人意识到”“下一句话会”“这一步会被记”等上帝视角；不使用“忽然、突然、发现、觉得、害怕、主线、副线、分线、节点、触发、回流、变量”等高风险词。
