# 第四卷第十八章《不写满》设计记录

## 使用技能

- superpower-bootstrap：继续遵守技能检查和项目边界。
- superpower-brainstorming：把丁祺的追问拆成真实公共空间难题，而不是辩论题。
- planning-with-files：按现有 specs/plans/progress 结构记录本次续写。
- superpower-writing-plans：先写可验收计划，再进入正文。
- chinese-novelist：用群消息、纸张、桌面、现场站位和课堂材料承载冲突。
- superpower-verification：完成后做字数和禁用表达扫描。

## 本章承接

第十七章结尾：丁祺在楼层群问“那如果有人不说介意，但其实介意呢？”，晚风发来宿舍群外卖拿错截图，说“你们这题，还真不小。”第十八章要把这个追问落成行动：四人不能用“没人反对就是同意”糊弄，也不能因为麻烦直接把作业扔掉。

## 本章职责

1. 主场在青枫居 4XX、青枫居一楼外卖架、宿管窗口和教学楼 A302。
2. 丁祺的问题必须推动规则修改：外卖架观察从“尽量完整”改成“允许缺测、少写、写局限”。
3. 宿管阿姨给出生活经验：不说的人也可能介意，所以站远、少记、别挡路、别追问。
4. 四人关系推进：林亦舟承认限制，周屿控制急躁，唐骁接受缺测，陆沉把“没人说”改成“不能替人说”。
5. 晚风只作为外部回声，不替他们给答案。
6. 结尾用助教或陈老师的课堂通知，把他们推到下一章的数据来源公开说明。

## 文风禁线

- 正文不写主线、副线、分线、节点、flag、触发、权重、判定、回主线、资源线。
- 不写“让林亦舟意识到”“让他明白”“他突然发现”“下一句话会”。
- 第四卷继续避开：觉得、发现、突然、忽然、让林亦舟、下一句话、怕被、怕规则、怕自己。
- 不把陈老师、阿姨或晚风写成说教工具。
