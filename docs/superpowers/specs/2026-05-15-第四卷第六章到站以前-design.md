# 第四卷第六章《到站以前》设计

## 目标

承接第五章结尾母亲追问“父亲为什么知道 C201”，把家庭信息错位、补考压力、室友提前返校和晚风现实陪伴压到同一条暑假后半段里。第四卷仍只写引导式外流感，不开启第五卷 5X。

## 叙事约束

- 班会、补考、答疑和机房相关地点继续放在教学楼 / 机房，不使用明德楼。
- 明德楼只保留学工、融媒体、行政材料、说明会属性，本章不写明德楼。
- 许澈只允许以便利店排班表、店员远影、活动群名字等方式出现，不建立稳定关系。
- 周屿、唐骁回校必须服务宿舍修补，不写成单纯搞笑返场。
- 晚风通过微信、语音、抖音、现实善后和光谷延期存在，不替家庭或宿舍下判断。
- 不写系统口吻，不写“让林亦舟意识到”“下一句话会”“这一步会被记进”等解释性文本。

## 本章事件骨架

1. 林亦舟听完母亲语音后，先处理两张票：一张原本回家，一张周二返校。退票和保留票形成家庭线的可见物证。
2. 父亲发来短信，要求不要让母亲多想；母亲要求把父亲在武生院站说过的话讲清。林亦舟被夹在两种说法之间。
3. 林亦舟和陆沉去图书馆或阳光书屋复习，路上经过乐跑、晨光体育馆、东区商业街或便利店，让校园生活继续有血肉。
4. 周屿提前到武生院站，唐骁也发来车次；他们并不是来“救”林亦舟，而是各自带着自己的暑假狼狈回到青枫居。
5. 四人重新在青枫居同一间寝室落位，桌面、床位、公共桌、外卖架、打印资料和空调遥控器承担关系修补。
6. 章末用教学楼 C201 答疑 / B204 返校班会通知加压，留下下一章进入补考前夜的钩子。

## 人物弧光重点

- 林亦舟：不再只做消息中转，开始把“哪些话不能替父母互传”落到具体行动里。
- 周屿：用玩笑遮掩回校的别扭，但这次愿意先到站、先占座、先把自己的椅子坐热。
- 唐骁：继续带资料和规则，但把“你必须按我来”改成“我把纸放这，你自己选”。
- 陆沉：仍然少话，通过水、座位、路线、图书馆时间陪伴，不抢解释权。
- 晚风：不治疗林亦舟，只把被骗善后、白天排队和自己的延迟回复摆在他面前。

## 结尾钩子

辅导员在班级群通知：周二返校后，补考同学到教学楼 C201 参加答疑，周三上午按准考安排考试；同晚 B204 返校班会照常。家庭消息与班级通知同时亮起，林亦舟必须在同一晚回到两张桌子前。
