# 第五卷第五十九章《谁能撤》设计记录

## 目标

承接第五十八章末尾融媒体老师追问“这条卡片谁能撤？”本章继续在东区商业街小舞台现场推进，不新开支线，不开启 5X，不引入许澈。核心冲突从“原卡片撤回”推进到“引用框归属和撤回窗口”：不是谁肯负责就能删掉所有痕迹，群聊里的每一条引用都有自己的发送者、时间和可撤范围。

## 场景

- 东区商业街小舞台台侧、观众区前排、控制台旁。
- 材料群、主持队临时群、摄影社小群和微信私聊作为手机内场景。
- 舞台节目继续推进，第三个节目准备上场，后台没有暂停处理的余裕。

## 人物动作

- 秦越：作为群主先想直接点名要求撤回，随后改为私聊和找人，承担“群主也不能替别人撤消息”的现实挫败。
- 唐骁：判断引用框只能由发送者撤，盯住撤回时间窗口，但需要压住公开追责冲动。
- 陆沉：记录原卡片撤回、引用消息发送者、联系时间、是否超过撤回时间，不写“已完全清除”。
- 周屿：继续阻止把胡嘉宁或引用发送者推出去公开道歉；他关心的是旧标题停止流动，不是让谁当众难堪。
- 许棠：和融媒体老师协调材料群更正口径，避免更正消息再次重复旧标题。
- 夏知微：帮助确认引用发送者在观众区前排，但不拍人、不转截图。
- 林亦舟：在晚风外部提醒和现场寻找之间传话，守住“不让晚风当入口”的边界。

## 冲突推进

1. 融媒体老师问谁能撤，唐骁指出只能由引用发送者撤。
2. 材料群里引用旧卡片的人是冯喻，她此刻在前排拍摄或协助节目，手机静音。
3. 秦越想在群里点名，周屿和许棠拦下，改为私聊和线下找人。
4. 台上节目继续，现场人流、麦克风、手机静音和倒计时形成现实阻力。
5. 冯喻终于看到消息，但撤回窗口已经过去，留下下一章“撤不掉以后怎么办”的压力。

## 文风要求

- 不写抽象总结，不写游戏术语，不写上帝视角。
- 用倒计时、手机未读、舞台掌声、前排人群和引用框小预览展示压力。
- 结尾落在可见失败上：冯喻说超过时间了。
