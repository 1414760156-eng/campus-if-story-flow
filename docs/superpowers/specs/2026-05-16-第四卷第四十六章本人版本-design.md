# 第四卷第四十六章《本人版本》设计记录

## 目标

承接第四十五章期末周安排，继续把第四卷后段收向“个人材料由本人核对、本人补交”。本章重点不是再制造新事故，而是让上册后段的几条现实压力各归各位：课程个人说明、二级成绩、勤工签退、主持排班、家庭转账、晚风票务补证和期末周复习。

## 写作约束

- 宿舍四人关系优先，晚风只作为票务审核支线余压，不抢主轴。
- 第四卷只埋引导式外流感，不开启 5X；许澈不出现。
- 课堂与材料核对发生在教学楼 A302、教学楼旁打印店或阳光书屋；明德楼不承担课程语义。
- 使用桃园餐厅或东园餐厅补充餐饮空间，避免长期只去西园。
- 不写系统/变量/节点/判定等游戏口吻，不写“让林亦舟意识到”和上帝视角预告。

## 章节设计

1. 青枫居清晨：期末周安排表、二级成绩截图、勤工工牌、主持排班表和袜子包裹分散在四张桌面。四人各自核对个人材料，林亦舟收到父亲“这周晚点”的短消息。
2. 教学楼旁打印店：学生集中打印期末复习资料、四六级耳机提醒、普通话报名表和课程个人说明。唐骁不让周屿拿自己的二级截图当“压力说明”。
3. 教学楼 A302：陈老师强调个人说明只收本人账号、本人文字。学习委员想让林亦舟统一润色，被拒；罗泽组有人想借 22 组格式，课堂冲突压回“格式可以看，句子不能代写”。
4. 桃园餐厅短饭桌：四人换一个饭桌吃饭，唐骁的二级庆祝热度退掉，陆沉排班冲突出现，周屿收到秦越活动总结催促，林亦舟家庭线继续以转账缺口存在。
5. 阳光书屋与图书馆西侧：林亦舟帮晚风压短补证说明，但不替她发泄；陆沉用本人账号上传签退说明，周屿在活动总结里删掉“大家都很配合”。
6. 青枫居夜间：四人各自上传本人版本。章末课程群通知周五最后一次课堂确认，未确认项不得由组长统一说明，为第四卷上册收尾前最后确认铺垫。

## 预期钩子

章末以周五最后一次课堂确认收束，下一章可写第四卷结束前的“最后确认”：谁交上去了，谁还卡着，谁仍想让别人替自己把话说漂亮。
