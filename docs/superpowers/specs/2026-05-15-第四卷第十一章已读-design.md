# 第四卷第十一章《已读》设计

## 目标

承接光谷现实见面后的次日回声，把“见过一次以后”的关系变化写具体：消息不再只是夜聊，变成已读、照片、小票、路线、补考成绩空窗、室友追问边界和家庭钱款错位。章节不写大爆点，用多个现实小压力把第四卷继续往第五卷前的收束感推。

## 叙事边界

- 晚风线保持现实化，但不替宿舍或家庭解决问题。
- 许澈不进入本章。
- 宿舍四人优先：周屿不能只负责起哄，必须有主持队复盘压力；唐骁不能只负责管人，必须有计算机二级/四级压力；陆沉不能只负责递水，必须有图书馆勤工与排班压力。
- 补考成绩不必立刻出分，可用“待录入/复核/通知”制造空窗和钩子。

## 场景安排

1. 青枫居雨后早晨：贴纸、黑伞、钥匙扣照片、室友追问与边界。
2. 西园餐厅 / 乐跑路线：周屿被主持队复盘消息牵走，唐骁看二级错题，陆沉排班。
3. 图书馆 / 阳光书屋：补考成绩待录入，晚风消息变少但更具体；家庭转账和生活费继续错位。
4. 晨光体育馆边 / 银杏路：乐跑补公里数、运动和课程压力成为大学时间流逝。
5. 章末：教务系统状态变化或班级群通知，留下补考结果即将到来的钩子。

## 写法要求

- 使用 `chinese-novelist` 原则：展示而非讲述，冲突驱动，结尾留钩。
- 用物件写关系：手机壳贴纸、湿伞、奶茶小票、钥匙扣照片、成绩页面、错题本、排班表、红包备注。
- 避免“意识到/明白/下一句话会/你怕我怕”式解释，正文中尽量不出现“觉得/发现/突然/忽然”。
