# 第五卷第三十九章《名单里的人》设计

## 使用技能

- chinese-novelist：用场景、动作、对话写出“被保护式移除”的现实压力。
- superpower-brainstorming：确认本章承接第三十八章章末，不改 5A 方向，不开启 5X。
- superpower-writing-plans：先落事件链和验收口径，再改正文。
- planning-with-files：留存设计、计划、进度与恢复信息。
- superpower-verification：写完后跑字数、禁用表达、地点逻辑检查。

## 承接断点

第三十八章末，主持队小群出现楼层群残留截图，有人提议先把周屿从周五名单里撤下，理由是“免得影响活动”。新章从东区商业街小舞台现场开始，把私人信息外泄后的后果推进到活动名单和公共角色。

## 本章目标

1. 把“身份外泄”转成“是否从名单中撤人”的现实冲突。
2. 让周屿面对自己在主持队里的位置，不把他写成纯受害者或纯逞强者。
3. 让宿舍四人继续作为支点：林亦舟稳住外扩，唐骁拆理由，陆沉盯住名单变更，周屿自己开口。
4. 支线只做镜像：秦越、主持队学姐姜宁、夏知微、晚风进入，但不抢走宿舍中心。
5. 章末留下“名单被静悄悄改掉”的钩子，为下一章处理活动名单和本人确认继续蓄压。

## 事件链

1. 东区商业街小舞台：秦越给周屿看主持队小群，有人建议先撤周屿；周屿不让秦越在群里吵，要求先问变更理由。
2. 青枫居 4XX：四人围着“名单变更理由”讨论；唐骁把“影响活动”拆成空话，陆沉指出如果因为截图撤人，就把泄露后果转嫁给被露出的人。
3. 青枫居一楼宿管窗口：楼长登记截图外扩到活动群；宿管阿姨只管楼内登记，但提醒别让楼里的事带着姓名到外面绕一圈。
4. 教学楼 C305：陈老师用活动名单变更样例训练“能写什么理由、不能拿什么做依据”，让课堂压力照到周屿。
5. 阳光书屋：夏知微处理摄影社名单，把“出镜授权”和“活动工作人员名单”拆开；林亦舟收到晚风消息，晚风朋友想把她从共同群里移出去“省得尴尬”。
6. 东区商业街小舞台：姜宁出现，承认自己想用“暂缓名单”保护活动；周屿要求如果要撤，必须写“本人申请”或“活动安排”，不能写“避免影响”。
7. 章末：群里发出最新版流程表，主持人一栏从“周屿”变成“待定”，文件名写着“周五活动主持名单_无争议版”。

## 禁止写法

- 正文不出现开发词、系统词、解释性总结句。
- 不使用明德楼承载课程、班会或答疑。
- 不写 A302，不写许澈。
- 不写“周屿怕、唐骁怕、陆沉怕”式并列心理。
