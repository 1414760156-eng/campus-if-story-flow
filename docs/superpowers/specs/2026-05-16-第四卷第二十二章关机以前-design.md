# 第四卷第二十二章《关机以前》设计说明

## 目标

承接第二十一章结尾打印店卷帘门半落，写出四个人在“先吃饭还是先补正式站位图”之间做现实取舍。章节重点不是把材料问题漂亮解决，而是让读者看见：补一张图也会挤占晚饭、嗓子恢复、B206 复排和运动会失落，大学里的责任常常不是大事，而是一个打印店即将关机的晚上。

## 必须承接

- 助教要求明早八点前补交正式站位图。
- 打印店老板提醒再晚就关机。
- 周屿同时收到运动会替补训练照和主持队小舞台名单初版，名字仍在主持前半段但嗓音不稳可能调整。
- 北园二楼螺蛳粉作为陆沉偏好与晚饭出口可以出现，但不能让它直接化解冲突。

## 场景安排

1. 教学楼一楼打印店门口：四人短暂停住，决定先补图，不去吃完整晚饭。
2. 打印店内：老板等关机，唐骁用电脑重画站位图，林亦舟补空间方位，陆沉核对记录和遮挡区域，周屿用手机看运动会照片和主持名单，仍要参与确认。
3. 北园二楼或便利店短场：打印后四人匆匆吃东西，饭不是庆祝，而是补充体力；周屿嗓子不能吃辣，陆沉替他换清淡一点。
4. 青枫居 4XX 夜间：正式图发送助教，群里短暂安静；B206/主持队继续压来；运动会群更新替补训练安排。
5. 章末钩子：助教回“收到”，但陈老师或学习委员发来实验二课堂展示优秀/边界说明示例征集，可能要求他们把这次说明整理成班级参考；同时主持队要求周屿明晚东区商业街走位，第四卷继续多线挤压。

## 写法约束

- 正文不出现系统口吻或游戏词。
- 不用抽象心理总结；周屿的空落用手机照片、辣椒油、润喉水、被换掉的晚饭呈现。
- 空间只用教学楼打印店、北园二楼或便利店、青枫居 4XX，不跳明德楼。
- 晚风只可短消息出现，不能替代四人关系。

## 本章预期效果

四人完成正式站位图并发给助教，但付出的是晚饭被压缩、周屿嗓子继续被管、唐骁计算机二级复习继续往后挪。材料表面收束后，新的“示例征集/主持走位”把下一章压力续上。
