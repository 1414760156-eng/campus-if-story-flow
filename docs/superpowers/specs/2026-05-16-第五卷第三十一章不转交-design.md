# 第五卷第三十一章《不转交》设计

## 目标

承接第三十章末尾“给那位同学”的手写信，把 5A 宿舍修补线继续推进到“私下歉意能不能由旁人保存、等待、转交”的边界。核心不是阻止梁嘉明有愧疚，而是让 4XX 发现：不公开并不等于可以绕到私下，不发群也不等于对方必须在未来接收。

## 必守

- 保持 5A 宿舍修补线，不开启 5X。
- 不写许澈，不使用 A302。
- 课堂场景使用教学楼 C305。
- 明德楼不承载课堂、班会、机房、答疑或咨询。
- 正文不出现系统化游戏口吻和被用户指出的上帝视角句式。

## 场景与动作

1. 青枫居 4XX 门口：梁嘉明递来未封口的信，信封写“给那位同学”。四人不接、不拆、不代看，先把“能留在谁手里”说清。
2. 青枫居一楼宿管窗口：宿管阿姨用夜归登记、公告栏、代收东西的生活经验提醒，不要把“等他愿意看”变成“替他放在门口”。
3. 校园快递站：无收件码、无本人确认的包裹不能随便代领，映照不替别人接收私人内容。
4. 西园餐厅：罗泽、丁祺、何书南处理小群里“帮忙带句话”的旧尾巴，让边缘人物继续参与。
5. 教学楼 C305：陈老师把“可转交/不可转交”压成可见条件：本人请求、对方同意、转交人是否知道内容、转交后谁承担。
6. 支线镜像：秦越设备组、夏知微摄影社、晚风朋友圈/微信都出现“别人托我说一句”的压力，但只作为镜像，不抢四人中心。

## 结尾钩子

梁嘉明最终把信改成“本人留存，未递交”，请求 4XX 只帮他见证日期而非签字转交。夜里楼层群出现一张走廊照片，拍到白信封边角和梁嘉明的袖口，有人问“4XX 又在帮谁递信？”下一章转入“没递出去的东西也可能被围观”。
