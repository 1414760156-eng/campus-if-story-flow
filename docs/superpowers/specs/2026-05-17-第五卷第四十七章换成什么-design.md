# 第五卷第四十七章《换成什么》设计

## 目标

承接第四十六章结尾“这段也换掉”，把压力从旧视频素材转到口播文字、活动导语和推送标题。冲突核心不是“能不能夸周屿”，而是旧叙事被撤下后，现场必须用今天的事实重新开场。

## 事件链

1. 东区商业街小舞台现场，秦越承认口播稿用了旧主持队回忆句。
2. 周屿要求换掉，但不接受被重新包装成“更体面”的个人故事。
3. 姜宁担心开场缺一段情绪铺垫，秦越担心现场断气，许棠要求把活动导语写成可核查的今天。
4. 唐骁把口播稿、视频工程、推送导语三处版本分开命名，陆沉记录停用、替换、待确认三类材料。
5. 夏知微和晚风分别从照片说明、抖音草稿里补出镜像：不要替别人把旧话写成今天的话。
6. 四人夜里回青枫居，宿管阿姨补登记；公共桌上重新写出短口播。
7. 章末许棠发来推送预览，标题/导语仍然用了“当年的自己”，压力转向发布端。

## 场景

- 东区商业街小舞台：灯、话筒、音箱、奶茶店收摊、活动人群散去后的空场。
- 银杏路回程：不把场景传送回寝室，保留步行和夜归登记。
- 青枫居一楼宿管窗口：补登记，确认“出去几个人，回来几个人”。
- 青枫居 4XX 公共桌：口播稿改写、群消息、版本文件名。
- 微信/摄影社消息：夏知微、晚风作为镜像补充，不抢宿舍四人。
