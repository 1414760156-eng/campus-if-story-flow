# 小说阅读版生成 Design

## 目标

把当前已完成的小说正文整理成给外部读者试读的干净阅读版，便于他人从人物真实感、节奏、冲突强度、校园生活可信度和阅读吸引力上提出意见。

## 范围

本次只合并已经完成的正文卷：

- 第一卷《行李落在青枫居》
- 第二卷《把手机交出去》
- 第三卷《热闹把人带走》截至第十五章《不是花絮》

不纳入总控规则、人物弧光索引、互动系统、进度表、开发说明和任何游戏术语。不改写既有正文，只做卷名清理、封面说明、目录生成和阅读格式包装。

## 产物

在 `docs/剧情小说母版_v4.0/阅读版/` 下生成：

- `武生院沉浸版_小说阅读版_截至第三卷第十五章.md`
- `武生院沉浸版_小说阅读版_截至第三卷第十五章.html`
- `武生院沉浸版_小说阅读版_截至第三卷第十五章.txt`

Markdown 用于后续编辑，HTML 用于直接阅读，TXT 用于复制分享。

## 校验

- 阅读版必须包含三卷标题和 38 个章节标题。
- 阅读版必须包含第三卷第十五章和“武生院最近好多故事”结尾钩子。
- 阅读版不得包含总控、进度、互动系统、节点、变量、回流等开发视角内容。
- 正文禁用表达扫描沿用当前正文质量门槛。
