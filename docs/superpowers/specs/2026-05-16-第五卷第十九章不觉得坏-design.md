# 第五卷第十九章《不觉得坏》设计记录

## 写作目标

- 承接第十八章末尾周五 C305 抽“本人不认为被改坏”的说明。
- 把“改坏”从单方面错误推进到同一件事在不同位置上的两种需求：公开说法要让人还能共处，留底事实要让问题不被擦掉。
- 继续服务 5A 宿舍修补线，让公共桌和外部寝室争执都变成四人关系的镜面。
- 不开启 5X，不让许澈进入关系核心，不改变第五卷多线收束方向。

## 章节结构

1. 青枫居 4XX 清晨：四人围绕“本人不认为被改坏”拆分对外版本和留底版本。唐骁担心变成和稀泥，周屿指出硬话贴错地方也会伤人。
2. 西园餐厅：罗泽、丁祺、何书南说明为什么有人会想把文字写好听；椅子、外卖架和洗衣机旧账继续作为具体压力。
3. 教学楼 C305：陈老师让不同学生说明“这版是给谁看的”。课堂不替寝室判案，只要求区分公开材料和内部留底。
4. 东区商业街小舞台与晚风微信：秦越设备交接表、晚风票务材料各自出现“公开简写”和“内部不能删”的同类问题。
5. 青枫居夜间：4XX 把自己的公共桌记录夹分成“公开页”和“留底页”。章末楼层群出现一张留底页局部截图，把下一章压力引向内部材料外泄。

## 人物推进

- 林亦舟：继续只给现实路径，不替晚风或室友把争议写成误会。
- 周屿：学会公开措辞不等于掩盖，但也不把所有难堪都推到公开处。
- 唐骁：从追求事实完整，推进到承认材料也有使用场景。
- 陆沉：提出“公开页 / 留底页”的实际做法，守住内部事实不丢。
- 罗泽、丁祺、何书南：让边缘寝室争执不再只是工具，而是普通学生想保全面子又想解决问题的矛盾。
- 秦越、晚风：用活动交接和票务材料映照同一主题。

## 风格约束

- 正文不用系统、规则表、游戏术语。
- 不写解释式认知总结，不写上帝视角预告。
- 第五卷课堂继续使用教学楼 C305。
- 不让明德楼承接课程、班会、答疑、机房或上机。
