# 第五卷第七十五章《谁方便》设计

## 目标

承接第七十四章结尾“手卡缺字，谁方便先补”，继续拆解“方便”如何被误读成开放权限。章节要把秦越的含混原话、冯喻的扩张转发、陈景的省事转述、邵闻的扫码进入串成一条可见链条，但不把任何一人写成单纯反派。

## 场景位置

- 场景仍在明德楼融媒体工作室，属于融媒体复核和材料核对，符合地点规则。
- 本章继续第五卷 5A 宿舍修补线，不开启 5X，不让晚风、夏知微或其他支线抢走宿舍四人的核心位置。
- 周屿、唐骁、陆沉、林亦舟仍在桌边：周屿拦住替罪羊式追责，唐骁要求把“方便”和“允许”拆开，陆沉继续扩表，林亦舟只收紧外部消息的使用范围。

## 冲突推进

1. 秦越承认“谁方便先补”是自己发的，但说明对象是主持队临时群里的手卡同学，不是所有现场帮忙的人。
2. 冯喻承认自己为了加快台侧补字，把素材交接群二维码转到摄影社小群，并配了“手卡补字的人先扫”。
3. 陈景的上一句“秦越那边催手卡”来自冯喻转图和秦越催字的重叠，属于省事转述。
4. 老师要求把“方便”替换成具体范围：谁方便、方便做什么、能看什么材料、谁批准。
5. 章末发现二维码不是只在摄影社小群转过，还出现在“手卡补字临时备份”小群里，并有一个陌生备注“宣传口小邵”曾短暂入群又退群，把下一章压力推向临时备份群。

## 文风要求

- 用动作、对话、手机截图、表格空栏和人群沉默推进，不写抽象总结。
- 不出现系统口吻、游戏节点语言、上帝视角预告。
- 不出现“让林亦舟意识到”等解释性句式。
- 不写许澈，不让 5X 提前进入。

## 验证

写完后运行字数统计，扫描第五卷禁用表达、明德楼误用、交通误写和新章新增文本。
