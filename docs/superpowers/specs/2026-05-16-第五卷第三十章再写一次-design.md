# 第五卷第三十章《再写一次》设计记录

## 目标

承接第二十九章《给谁看》的断点：梁嘉明想公开道歉，但草稿里的“被我打扰的同学”“你不愿意接收我的道歉”再次把无关人写出来。本章不扩大成新的大事件，而是把压力落在一次又一次删改里，让 4XX、梁嘉明、罗泽/丁祺/何书南、沈嘉禾、秦越、夏知微和晚风分别面对同一个问题：公开文字到底写给谁看，写到什么程度才算收住。

## 核心冲突

梁嘉明认为“不写对不起”像逃避，但写得越真诚，越容易把对方重新推到人群里。4XX 要帮他把公开说明从“对某个人表达歉意”改成“停止自己的行为并请别人停止追问”，同时不能替无关人说原谅、拒绝、接收或不接收。

## 场景安排

- 青枫居 4XX：深夜围绕梁嘉明草稿进行第一轮删改，展示周屿的直觉、唐骁的文本边界、陆沉的停联原则和林亦舟从晚风处带来的现实回声。
- 青枫居一楼宿管窗口：宿管阿姨用公告栏和夜归登记提醒他们，歉意不能像通知一样贴给路过的人看，也不能堵人门口。
- 西园餐厅：罗泽、丁祺、何书南处理自己小群里“别问了”的善后，把外部传播继续压回个人能收住的地方。
- 教学楼 C305：陈老师让同一件事改成三版：公开说明、私下留存、自己记录；沈嘉禾补充校医院材料经验，22 组现场改梁嘉明草稿。
- 阳光书屋/东区商业街小舞台消息：夏知微、秦越、晚风分别作为镜像出现，照出摄影社撤展说明、设备替补说明和朋友圈边界。

## 写法约束

- 保持 5A 宿舍修补线，不开启 5X，不让许澈出场。
- 第五卷不使用 A302；课堂使用 C305；明德楼不承载班会、课程、答疑或机房。
- 不写系统化提示词，不出现“主线/副线/节点/触发/回流/判定/权重”等玩家可见词。
- 不写“让林亦舟意识到”“下一句话会”“这一步会被记”等解释句；心理变化通过删字、停顿、纸张、手机、桌面和对话呈现。
- 章末留钩子：梁嘉明带来一张手写纸条，封面仍写出会指向人的称呼，迫使 4XX 发现连信封也会把人写出来。

## 章末钩子

梁嘉明没有再发公开草稿，却带来一张折好的纸，想让 4XX 帮忙转交。信封封面写着“给那位同学”。陆沉看着封面，说出“封面也得重写”，把下一章压力推向私下转交与不转交的边界。
