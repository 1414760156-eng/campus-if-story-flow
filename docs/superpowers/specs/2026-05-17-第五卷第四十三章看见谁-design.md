# 第五卷第四十三章《看见谁》设计

## 目标

承接第四十二章章末旧主持队招新截图和“我不怕被看见”。本章要把冲突从“旧资料能不能沿用”推进到“当年愿意被看见，是否等于今天可以被任何人、在任何场合重新展示”。继续 5A 宿舍修补线，不开启 5X，不引入许澈，不改变周屿暂退与不参与主持人展示的事实。

## 章节核心

周屿当年那句“我不怕被看见”是真的，甚至很珍贵。本章不能把它写成笑柄或黑历史。真正的问题是：当年的勇气被拿到今天的活动群里，就从自我介绍变成别人证明他的材料。宿舍修补要落在 4XX 陪他把这句话从“别人可以拿来用”里拿回来，而不是替他否认曾经的自己。

## 场景和人物

- 青枫居 4XX：深夜看到旧截图外传，四人围绕“看见谁”拆出场合、对象和本人意愿。
- 活动材料群：有人用“他自己说不怕被看见”解释转发；许棠要求停止扩散。
- 青枫居一楼宿管窗口：旧照片和公告栏照片类比，宿管阿姨提醒旧照片可以留在相册里，但不能贴到新公告上。
- 东区商业街小舞台：次日早晨秦越找不到话筒，周屿忍住不递，姜宁当面道歉但不被写成反派。
- 夏知微、晚风镜像：夏知微处理旧摄影展海报被借用；晚风处理“以前说过想出去玩”的旧话被拿来证明今天该出现。
- 章末钩子：许棠发来公共盘路径，旧截图来自“主持队招新素材”共享文件夹，链接仍可查看，创建者显示秦越。

## 文风约束

- 展示而非讲述，动作和对话推进。
- 不写“让林亦舟意识到”“下一句话会”“周屿怕/唐骁怕/陆沉怕”等解释性句式。
- 不写“主线、节点、触发、变量、回流、判定、资源线、系统提示”等游戏开发词。
- 本章不使用明德楼；不出现 A302、许澈、公交、火车到武生院站等误用。
