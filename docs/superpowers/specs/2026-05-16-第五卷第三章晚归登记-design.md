# 第五卷第三章《晚归登记》设计稿

## 目标

承接第二章结尾的十点四十归寝钩子，把 5A 宿舍修补入口落成一个具体夜晚：周屿因东区商业街录制事故晚归，宿管阿姨要求本人登记，青枫居公共桌“照片里的整齐”和真实桌面重新撞在一起。

## 章节功能

- 宿舍四人优先：周屿晚归、唐骁按表登记、陆沉赶图书馆交班表、林亦舟同时被晚风票务材料和家庭消息拉扯。
- 事件驱动：背景板倒塌、录制重来、晚归签字、公共桌照片复核、图书馆交班表，全部用动作和对话推进。
- 支线只补压强：秦越出现于东区商业街录制现场；晚风通过微信打码材料继续推进票务补证；父亲“月底再看”继续压家庭线。
- 外部空气只作铺垫：周屿在宿舍楼外短暂停留，写出“不想马上进门”的气味，但不启动 5X，也不引入许澈关系核心。

## 场景路径

1. 青枫居 4XX：宿管阿姨门外提醒，唐骁捡胶带，陆沉准备去图书馆，林亦舟处理晚风消息。
2. 东区商业街小舞台：秦越和周屿处理倒下的背景板、话筒线、重录安排。
3. 青枫居楼下外卖架与宿管桌：周屿短暂停留，宿管阿姨要求本人填晚归时间。
4. 青枫居 4XX 公共桌：周屿回来后，照片、登记、自查表和真实桌面重新对上。

## 人物弧光

- 林亦舟：不替任何人解释，只把能分开的材料分开，把该回的微信回到边界内。
- 周屿：不再用玩笑盖过晚归，但仍对“被登记”有抵触；他的修补从填真实时间开始。
- 唐骁：从只想按表推进，转向把登记留给本人填写，不替周屿补句子。
- 陆沉：坚持自己的图书馆交班表和旧物边界，不为了宿舍现场无限让步。

## 禁用表达提醒

正文不使用系统口吻和解释性套话；不写“让某人意识到”“下一句话会”“这一步会被记”等上帝视角；不使用“忽然、突然、发现、觉得、害怕、主线、副线、分线、节点、触发、回流、变量”等高风险词。
