# 第五卷第七十七章《不要外传》设计

## 目标

承接第七十六章结尾 `发布端预览_不要外传.png` 来源待查，继续在明德楼融媒体工作室追问“不要外传”为什么仍被发进临时备份群。章节要把提醒、重命名、转发、可见范围四件事拆开，让冲突落在具体动作上，而不是落成空泛道理。

## 场景位置

- 场景仍在明德楼融媒体工作室，属于融媒体复核和材料核对，符合地点规则。
- 本章继续第五卷 5A 宿舍修补线，不开启 5X，不写许澈，不让晚风或夏知微抢走本章核心。
- 宿舍四人仍在桌边：周屿挡住把“不要外传”当笑话，唐骁追问谁发给谁，陆沉扩表记录原始来源、重命名者、实际发送范围，林亦舟只收紧外部截图和群内材料的使用方式。

## 冲突推进

1. 梁思予翻导出记录，确认 `发布端预览_不要外传.png` 是陈景在 01:05 发进手卡补字临时备份群的。
2. 陈景说明自己不是原始截图者，原图来自姜宁私聊，她让他判断这是不是旧预览，顺手写了“别外传”。
3. 陈景把图片另存并改名为 `发布端预览_不要外传.png`，再发进备份群，原意是提醒手卡同学不要按旧预览补字。
4. 唐骁追问“写了不要外传为什么还发九个人群”，周屿压住把陈景当唯一责任的冲动，指出提醒不等于边界。
5. 陆沉把表格加为：原图来源、私聊对象、重命名者、发送群、可见成员、是否有人下载、后续引用。
6. 章末袁承的消息露出：“那我按这张顺序改手卡”，说明“不要外传”的图不只被看见，还被当作参考使用，把下一章压力推向“手卡顺序”如何把旧预览带回正式稿。

## 文风要求

- 用导出记录、电话停顿、文件名、缩略图、手机滑动和表格空栏推进。
- 不写系统口吻、游戏术语、上帝视角预告和解释性自我认知。
- 不出现“让林亦舟意识到”“下一句话会”等禁用句式。
- 不写许澈，不把 5X 提前带入。

## 验证

写完后运行字数统计，扫描第五卷禁用表达、明德楼误用、交通误写和新章新增文本。
