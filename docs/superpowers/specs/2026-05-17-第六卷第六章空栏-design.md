# 第六卷第六章《空栏》设计记录

## 写作目标

- 承接第五章结尾：团队名单确认表中“负责人”和“测试记录负责人”仍空着。
- 把空栏落成具体责任：谁能写名字、谁只提供公开规则、谁只做文字口径、谁负责可点击页面和测试记录。
- 宿舍四人优先：冲突发生在 4XX 公共桌、晚饭、群消息、签名和周一机房要求里，不另开新核心人物。
- 第六卷控长：本章只推进团队确认与第一版交付压力，不展开完整演示。

## 场景与动线

- 从教学楼 C407 外回青枫居，路过篮球场，带入东区商业街鸡排饭的饭味和健源方向音响试麦声。
- 青枫居 4XX 公共桌作为核心场景，桌面物件包括团队名单确认表、图书馆公开规则截图、周屿带回的薄荷糖、唐骁电脑、陆沉文件袋、林亦舟打印纸。
- 次日短转到八号教学楼机房门口和机房内，形成第一版可点击页面的外部压力。

## 人物动作

- 林亦舟：从“等回去填”变成写下测试记录负责人，但会划清自己不替全组兜底。
- 唐骁：从默认自己扛全部转向只做负责人和页面统筹，不再把“负责人”当成全能承诺。
- 周屿：拒绝成为展示或说明负责人，只答应把说明页里的虚词删掉，让语言能被现场听懂。
- 陆沉：提供公开规则和记录格式，不提供内部便利，不让图书馆勤工身份被滥用。
- 晚风：只以微信短句推动林亦舟回答“你签的是你能做的，还是他们希望你做的”。

## 文风约束

- 正文不写系统口吻、变量口吻、上帝视角提示。
- 不使用“让林亦舟意识到”“下一句话会”“你怕我怕”等总结式表达。
- 避免将明德楼、公交、A302、许澈写入本章。
- 用饭盒、纸张、签字、文件名、机房门口排队和消息提示推动心理变化。
