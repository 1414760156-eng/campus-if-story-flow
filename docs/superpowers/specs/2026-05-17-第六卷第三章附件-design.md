# 第六卷第三章《附件》设计记录

## 调用技能

- planning-with-files：继续用文件记录本轮计划和进度。
- superpower-brainstorming：先确定第三章事件功能与篇幅边界。
- superpower-writing-plans：形成可执行写作步骤。
- chinese-novelist：按展示而非讲述、冲突驱动、章尾钩子写正文。
- superpower-verification：完成后跑统计和禁用表达、地点逻辑扫描。

## 本章定位

第三章进入大三下春季开学第一周。上一章结尾“附件栏空着”，本章让它变成现实：返校、寝室重新住满、方向表修订版提交、八号教学楼机房环境配置和第一轮材料核对。

## 本章目标

1. 推进时间段：春节后返校到大三下开学第一周。
2. 使用第六卷节制写法：只写一个代表性事件，不拆成多章。
3. 展示四人去向材料差异：
   - 唐骁有竞赛和项目草案，但“并行”太满，家里考研压力仍在。
   - 周屿有普通话报名、活动经历本人确认版和表达能力，但拒绝被默认当展示人。
   - 陆沉材料少，却有排班、课程提交记录和可协调时间的真实限制。
   - 林亦舟附件不足，商业街活动留存不能直接支撑项目实训。
4. 晚风只做现实回声：她问开学前见面，但林亦舟因附件和机房配置迟迟没有答应。

## 场景

- 青枫居 4XX：返校后床铺、桌面、家里带来的东西、公共桌重新住满。
- 西区快递站：家里寄来的生活用品和开学快递潮。
- 八号教学楼机房：项目实训环境配置、提交页、助教核对附件。
- 教学楼 C407：只作为下一次导师意向面谈通知，不在本章展开。

## 冲突

冲突集中在“材料能不能支撑你写下的方向”。不是老师训话，而是提交页、助教询问、同学材料和四人彼此的表格互相照出空洞。

## 结尾钩子

林亦舟提交后，页面在他的名字后留下红字：材料需补充。与此同时，C407 下一周导师意向面谈名单通知弹出，唐骁已经进第一批，林亦舟没有。
