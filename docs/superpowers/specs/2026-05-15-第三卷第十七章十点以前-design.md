# 第三卷第十七章《十点以前》设计

## Summary

本轮继续《武生院沉浸版剧情小说母版 v4.0》第三卷《热闹把人带走》，承接第十六章结尾“明早 A101 还有完整版吗？想听热水机后续。”写一个完整章节。

目标不是继续抽象讨论传播边界，而是把匿名围观压回真实课堂、最终版材料、宿舍四人站位和支线人物各自的生活渠道里。章节需保持宿舍四人优先，明德楼只用于融媒体/行政渠道，班级课堂仍在教学楼 A101。

## Chosen Approach

采用“课堂提交 + 课后分流 + 官方渠道诱惑”的推进方式。

1. 早晨青枫居到教学楼 A101：四人带着最终版和修改记录去交材料，手机评论区仍在发酵。
2. A101 课堂现场：有同学用匿名图文起哄，罗泽/丁祺/何书南在后排形成现实压力；周屿不能把自己继续写成段子，唐骁守住材料边界，陆沉补充证据留存，林亦舟负责把话从“后续表演”拉回“最终版报告”。
3. 陈老师处理方式：不公开审判谁，也不帮他们表演委屈，而是让他们补交一页“材料使用说明”和“删改记录”，把事情落到课程要求。
4. 课后分流：周屿仍去 B206，夏知微从摄影社回话，许棠从明德楼融媒体渠道传来新情况。
5. 章末钩子：匿名热闹刚压下去，学院融媒体反而想把“水房观察报告”改成一篇正式推文，形成更现实的新选择压力。

## Style Constraints

- 用动作、物件、对话推进，不写上帝视角预告。
- 不在正文中出现策划词、系统词、玩家提示词。
- 不写“让林亦舟意识到”式自我总结。
- 不写“周屿怕、唐骁怕、陆沉怕”式并列作文句。
- 场景必须符合武生院动线：青枫居 -> 教学楼 A101 -> 西园/银杏路/明德楼融媒体消息 -> B206。
