# 第五卷第五十三章《给镜头的稿》设计记录

## 目标

承接第五十二章末尾“融媒体老师提前到现场，问开场给镜头的是哪一份稿”。本章把现场白卡、主持口播、融媒体镜头稿和后期字幕四种材料分开，让“用哪一份稿”成为公开传播层面的新压力。

## 约束

- 不开启 5X，不引入许澈。
- 宿舍四人继续作为处理方式的核心：林亦舟负责协调问话，唐骁拆文本版本，陆沉留档流向，周屿确认本人边界。
- 融媒体老师只作为现实现场的专业压力，不替人物下结论，不写成长篇训话。
- 夏知微保留摄影边界，许棠、梁思予、秦越、姜宁继续处理活动材料。
- 场景限制在东区商业街小舞台、活动总群、材料群、拍摄位和台侧，不误用明德楼。

## 事件链

1. 融媒体老师提前到场，要求开场给镜头的稿、主持手卡、字幕稿保持一致。
2. 秦越下意识想把白卡递给老师，但唐骁发现字幕组还拿着昨晚口播摘要。
3. 梁思予承认她把“临时开场口播”转给字幕志愿者时没有同步白卡。
4. 陆沉将材料拆成四份：主持白卡、镜头稿、字幕稿、推送导语。
5. 周屿明确镜头稿也不能写“他回来了”“当年”等旧包装。
6. 融媒体老师要求给她一份纸面确认稿，并说明谁来确认画面字幕、谁确认主持口播。
7. 夏知微提醒拍摄会录到台侧白卡，不能让镜头里出现另一份旧稿。
8. 许棠把四份材料压成同源短稿，梁思予联系字幕志愿者停用昨晚摘要。
9. 章末字幕志愿者回复说，旧字幕条已经提前导入播放软件。

## 写法

重点写“同源”不是抽象原则，而是四份纸、四个手机、一个播放软件、一个机位和一支话筒之间的错位。用舞台灯、字幕机、摄像机红点、纸页边角和手卡夹子推进紧张感。
