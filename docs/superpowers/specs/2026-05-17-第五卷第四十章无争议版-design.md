# 第五卷第四十章《无争议版》设计

## 使用技能

- chinese-novelist：以具体名单、文件名、群消息和场景动作推动冲突。
- superpower-brainstorming：确认本章承接“待定”，仍走 5A 宿舍修补线，不进入 5X。
- superpower-writing-plans：先落事件链、地点和章末钩子，再写正文。
- planning-with-files：记录设计、执行和进度，便于自动续写恢复。
- superpower-verification：写完后检查字数、禁用表达、地点逻辑。

## 承接断点

第三十九章末，姜宁发出的最新版文件名为“周五活动主持名单_无争议版.xlsx”，开场主持栏从“周屿”变成“待定”。第四十章从这个文件名开始，把“无争议”的温和措辞写成新的排除。

## 本章目标

1. 把“待定”从临时写法推进到正式材料和传播后果。
2. 让周屿不是简单硬扛，而是要求把自己是否出场和截图风波拆开。
3. 让宿舍四人继续在公共桌上分工：林亦舟写短回复，唐骁拆文件名，陆沉盯确认流程，周屿自己决定是否申请复核。
4. 用 C305、阳光书屋、东区商业街小舞台和晚风共同群形成镜像，避免支线抢主轴。
5. 章末钩子：融媒体材料群收到“无争议版”后要求补一份主持人背景表，周屿发现自己的那一栏被空着，文件流向更正式的校内材料。

## 事件链

1. 银杏路/青枫居回程：周屿看见文件名和“待定”，情绪没有爆开，而是反复看文件名。
2. 青枫居 4XX：四人拆“无争议版”文件名，唐骁指出文件名已经给人定性；陆沉要求追问谁确认过；林亦舟帮周屿把回复压短。
3. 青枫居一楼宿管窗口：楼长提醒外扩登记只能说明截图流向，不能处理主持队名单；阿姨用“夜归名单不能因为别人起哄把人删掉”给生活映照。
4. C305：陈老师讲文件名和版本名也是公开材料的一部分，不能用“最终版/无争议版/内部版”偷塞判断。
5. 阳光书屋：夏知微将摄影社文件名从“无纠纷版”改成“姓名隐藏版”，晚风收到共同群“先冷处理”的消息。
6. 东区商业街小舞台：姜宁和周屿当面改名单；姜宁承认自己怕活动出问题，周屿提出要么按彩排结果，要么写本人申请，不接受“无争议版”。
7. 章末：姜宁改了群文件名和名单，却有融媒体材料群催主持人背景表；周屿打开表格，自己的那一栏空着。

## 禁止写法

- 正文不出现开发词、系统词、解释性总结句。
- 不写明德楼承担课程、班会或教学。
- 不写 A302，不写许澈。
- 不写“周屿怕、唐骁怕、陆沉怕”的并列心理。
