# 第四卷第二十四章《别直接套》设计

## 目标

续写第四卷第二十四章，承接第二十三章末的三重压力：班级群索要可直接套用的 Word 版、唐骁二级错题加讲、周屿东区商业街小舞台继续加一分钟。章节重点不是开新事故，而是让“帮忙”开始消耗四人，并制造一次合理的宿舍小冲突。

## 章节核心

四个人前面靠互相补位撑过了课堂示例和小舞台走位，但同一套互助方式开始过载。林亦舟不想把说明框架变成别人照抄的模板，唐骁急着回到二级错题，周屿嗓子被小舞台继续消耗，陆沉试图用格式和记录边界把事情压住。冲突落在具体动作：谁拿手机、谁写回复、谁错过讲解、谁还要开口。

## 场景安排

1. 东区商业街小舞台：歌手没到，秦越要求再加一分钟；周屿不能讲笑话，唐骁给出流程型串词，林亦舟同时收到 Word 版索要。
2. 小舞台后台/奶茶店旁：周屿试新加段，唐骁错过二级加讲开头，手机声音被音响盖住。
3. 返寝路上：林亦舟想回同学私聊，唐骁急躁，陆沉提出只发不可直接套用的空框架或 PDF。
4. 青枫居 4XX 夜里：四人围绕“要不要给 Word 版”发生小冲突；唐骁说出自己也有考试，不是所有空缺都该他补；周屿嗓子受限，玩笑收不住。
5. 章末钩子：林亦舟发出班级群回复后，助教/学习委员又要求明早前整理一页“空白边界说明表”；唐骁的二级群点名缺席；周屿收到小舞台录音。

## 约束

- 不使用游戏系统词和上帝视角预告句。
- 第四卷正文避开“觉得/发现/突然/忽然/让林亦舟/下一句话/怕被/怕规则/怕自己”。
- 明德楼不出现。
- 许澈不进入关系核心；晚风可不出现或只做短消息。
- 冲突要小而真实，不让唐骁崩坏；他可以烦躁，但仍会给出可执行边界。

## 预期效果

完成外卖架实验后续从课堂扩散到班级材料的压力，把“互相帮忙”的成本推到四人面前，为后续第五卷 5A 宿舍修补线埋下公共桌、手机、文件格式和谁替谁补位的旧账。
