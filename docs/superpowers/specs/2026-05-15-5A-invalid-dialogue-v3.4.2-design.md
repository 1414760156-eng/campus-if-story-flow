# 5A 无效对话与上帝视角纠偏 v3.4.2 Design

## Goal

把第五章 5A 当前试玩文本继续小说化，删除玩家可见文本中的无效提示、上帝视角预告、自我认知总结句和“小学生作文式”的排比解释。

## Scope

- 修改 `playtest/5a-full-data.js` 的玩家可见剧情、选项反馈和自动补足反馈。
- 不新增节点，不改变 5A 回流/外流结构，不移动唯一异质外流点。
- 不把 5A 改写成晚风线、家庭线或第一章文本。
- 保留现有可玩性指标、节点数量、选项数量和测试约束。

## Writing Rules

- 使用 `chinese-novelist` 的“展示而非讲述”原则：用门把、拖鞋、杯子、规则纸、玻璃、照片、耳机声等物件承载心理。
- 禁止玩家可见文本出现“让林亦舟意识到 / 林亦舟发现 / 忽然明白 / 下一句话会把……”等总结式句子。
- 禁止“周屿怕、唐骁怕、陆沉怕”这类并列心理解释，改成动作和反应。
- 地点按武生院地图纠偏：使用青枫居、晨光湖、西园餐厅、银杏路、东区商业街等具体地点，不再使用旧误称。

## Verification

- `rg` 扫描禁用短语，结果必须无命中。
- `node playtest/5a-full-data.test.js` 必须通过。
- `node --check playtest/5a-full-data.js` 必须通过。
- `node playtest/app.smoke.test.js` 必须通过。
