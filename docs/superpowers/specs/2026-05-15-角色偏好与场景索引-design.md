# 角色偏好与场景索引设计

## 目标

把“生活碎片”从模糊氛围升级为可追踪的索引系统。后续正文写某个角色去哪家店、吃什么、考什么、参加什么运动或为什么受伤，都要能回到前文暗示、角色性格、年级阶段和现实动线，避免人物随机行动。

## 设计原则

- 生活碎片必须明确：吃什么、去哪、和谁、为什么今天去、花多少钱、这次与以往有什么不同。
- 采用“先暗示后使用”：先在对话、外卖袋、饭桌玩笑、查分截图、运动器材、店员问候里露出偏好，再让角色在关键场景去对应地点。
- 偏好不是硬判定，而是概率逻辑。爱学习的人大概率四六级稳过，不爱学或拖延的人更容易反复考；爱吃的人更可能到处找好吃的；爱运动的人更可能受伤或临时替补；社交广的人更容易被活动和饭局牵走。
- 年级节奏必须合理：大一忙学业社团，暑假像学校剧情的喘气，大二开始学会生活，大三开始把生活偏好和去向压力混在一起，大四让旧店、旧证书、旧号码布成为遗憾回响。
- 期末周和补考必须纳入节奏：期末周改变校园质感、时间分配和宿舍作息；补考影响自尊、返校、聚餐和新学期心态，不能写成笑料。
- 个人偏好必须形成交点：独特喜欢负责人物辨识度，宿舍/社团/班级共同记忆负责群像厚度和终章遗憾。
