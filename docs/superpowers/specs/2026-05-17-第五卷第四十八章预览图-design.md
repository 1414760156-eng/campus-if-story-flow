# 第五卷第四十八章《预览图》设计

## 目标

承接第四十七章末尾推送预览图仍写“看见当年的自己”的钩子，把冲突从口播扩展到发布端的标题、导语、封面小字、摘要和转发语。重点不是再开新事件，而是让读者看到：视频和口播改掉以后，旧话仍可能藏在看似不起眼的发布位置里。

## 事件链

1. 青枫居 4XX 夜里看到许棠发来的推送预览图，旧导语仍在。
2. 唐骁追问预览图已发给谁，陆沉记录“标题、导语、封面小字、摘要、转发语”五处。
3. 许棠说明发布端有模板草稿，排版同学已经套过一版，旧句从口播稿被复制到导语和分享摘要。
4. 周屿想直接删光情绪句，但秦越担心推文失去现场感，姜宁不敢再替周屿写漂亮话。
5. 夏知微和晚风用照片说明、抖音分享标题作镜像，提醒改正文不等于改转发画面。
6. 青枫居公共桌重新拟定推文导语：只写今天现场和材料更正，不写“当年”“终于”“底气”等结论。
7. 章末新预览未正式发布，但旧预览截图已被活动群同学转发，下一章转向“未发布的截图也会流动”。

## 场景

- 青枫居 4XX 公共桌：手机长图、电脑文档、夜里灯光、外卖盒和登记表。
- 微信/活动材料群/主持组小群：发布端反馈和旧预览截图扩散。
- 摄影社消息、晚风抖音草稿消息：支线镜像，不抢主轴。
- 青枫居楼道/水房短场：用夜里泡面味、洗衣液味和楼道声保留寝室生活质感。
