# 第五卷第六十一章《两年前的模板》设计

## 本章职责

- 承接第六十章旧模板文件露出，继续查源而不变成抓个人认错。
- 将问题从“谁发错了”推进到“旧表达如何通过公共盘、复制文件夹和默认模板一路留到今天”。
- 保持 5A 宿舍修补线：四人通过核对文件、分工、留档和克制语言继续修补，不新开 5X。
- 晚风仅做外部回声，提示外部文字已淡下去，不抢现场核心。

## 人物动作

- 周屿：面对旧模板中“人物故事可替换姓名”的逻辑，明确拒绝自己被套进回忆模板，但不要求现场追人。
- 唐骁：想一次性删除模板库，转为建立“停用、留档、可用”三类清理清单。
- 陆沉：记录文件名、创建时间、修改时间、复制路径和实际使用者，不写心理总结。
- 林亦舟：在晚风消息与现场文件之间收住注意力，帮忙比对文件路径，不让外部回声吞掉现场。
- 秦越、梁思予、胡嘉宁、叶青、许棠：各自说明自己拿到、复制、套用或停用文件的环节。

## 场景与钩子

- 场景仍在东区商业街小舞台散场后的折叠桌、电脑屏幕、共享盘和关闭一半的舞台灯下。
- 章末钩子：旧模板文件夹内的使用说明写着“人物故事可直接替换姓名”，把问题推向主持队长期“套话模板”的惯性。
