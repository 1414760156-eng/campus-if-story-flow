# 第五卷第五十六章《原图》设计记录

## 断点

第五十五章章末，字幕志愿者私聊梁思予：侧屏拍屏已经被人存图，对方索要原图，并评价“半行白字挺有感觉”。现场距离开场只剩五分钟。

## 本章目标

- 继续推进 5A 宿舍修补线，不开启 5X，不引入许澈。
- 把冲突落在“原图是否继续外传、现场如何开场、旧字如何停止变成素材”三个具体动作上。
- 让周屿、唐骁、陆沉、林亦舟分别承担不同位置：周屿坚持不被包装，唐骁压住当众质问并改为可执行回复，陆沉留档但不扩散，林亦舟在梁思予、秦越、许棠、晚风之间传话。
- 支线人物不只做背景：梁思予、秦越、许棠、夏知微、晚风都要有动作，但不抢过宿舍四人。

## 冲突结构

1. 梁思予展示私聊，现场短暂失声。
2. 唐骁要求不要争论“好不好看”，只回复不能要原图、以同源稿为准。
3. 陆沉只记录必要来源和时间，不把原图继续转发给更多人。
4. 周屿拒绝让旧字变成“电影感素材”，也拒绝扩大到全群训话。
5. 志愿者回复：对方已经把半行白字转到主持队/活动小群，并有人想配朋友圈。
6. 许棠、秦越、梁思予分头截断发布端、主持端、字幕端。
7. 开场倒计时推进，现场不能停。
8. 章末钩子：主持队小群有人建议把“彩排事故”当开场梗，逼出下一章对公开玩笑化的处理。

## 写法要求

- 使用中文小说式场景推进，展示动作和对话，不写开发者口吻。
- 避免“怕我怕”“让林亦舟意识到”“下一句话会”等上帝视角和总结句。
- 避免新正文中出现 A302、许澈、明德楼误用、公交等地点/交通错误。
- 场景限定在东区商业街小舞台、侧屏控制台、台侧、活动群和微信聊天。
