# 第五卷第六十五章《挺有感觉》设计记录

## 目标

- 承接第六十四章末尾旧卡片被再次转发并配文“挺有感觉”，把技术残留推进到审美化、玩笑化和二次传播的现实冲突。
- 保持 5A 宿舍修补线：宿舍四人继续围绕同一件事学习如何“处理可见材料”，而不是开新线或进入 5X。
- 把场景从明德楼融媒体工作室自然带回青枫居，补足夜归、宿管窗口、路上消息和公共桌这类宿舍修补线的生活质感。

## 场景与动线

- 明德楼一楼融媒体工作室收尾：处理旧卡片二次转发，形成群内短说明。
- 明德楼外与回青枫居路上：保安催关灯、夜风、手机消息持续震动，形成“事情离开屏幕仍跟着人走”的压迫感。
- 青枫居一楼宿管窗口与 4XX：宿管阿姨用夜归登记和楼内告示提醒“过夜的东西不好收”，四人回到公共桌整理留存和停用清单。

## 人物职责

- 周屿：不把“挺有感觉”当成夸奖，要求别把旧标题继续当好看的句子使用。
- 唐骁：追究转发来源和群权限，但需要克制点名冲动。
- 陆沉：把旧卡片、转发者、处理方式和留存范围分开，避免证据本身继续外传。
- 林亦舟：处理晚风外部提醒，同时不让晚风线成为核心。
- 秦越、许棠、梁思予、胡嘉宁、叶青：继续承担后台、群说明、预览和素材清理的可见工作。

## 结尾钩子

- 群转发被收住后，公众号后台“分享摘要/封面摘要”仍残留旧句。下一章可以处理“不是标题也会被看见”的更隐蔽字段。

## 文风约束

- 不写“意识到/明白/发现/下一句话会”等总结预告。
- 不写系统口吻、策划词、玩家可见变量词。
- 明德楼只用于融媒体和材料处理；回宿舍走线要有空间感。
