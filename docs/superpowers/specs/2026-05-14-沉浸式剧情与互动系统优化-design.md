# 1-6幕沉浸式剧情与互动系统优化设计

## 目标

在不重写主线、不改变宿舍四人关系和关键事件顺序的前提下，把现有 1-6 幕剧情优化为更适合沉浸式剧情游戏接入的中文内容，并为支线人物补足可触发事件、关系回响、心理暗线和资源奖励。

## 不可改动的主线锁点

- 宿舍四人关系保持为周屿、主角林亦舟、唐骁、陆沉。
- 关键事件保持：报到入住、宿舍边界、军训手机袋事件、军训汇演、晨光大道招新、迎新晚会/周屿串场、暑假现实化、班会通知、晚风现实化、第六幕准结局、毕业公共收束。
- 玩家互动逻辑不推翻原选项，只做扩展：新增支线可见条件、事件回响、心理状态和资源奖励。
- 剧情只在武生院沉浸版场景体系内发生：武生院站、学校主入口、图书馆西门、图书馆仓库、中南公寓、北园、晨光湖、晨光大道、阳光书屋、明德楼、九龙壁舞台、健源体育场、后街、光谷校外实践点、家中等。

## 人物弧光设计

- 林亦舟：从被流程推着走，到学会问清、说明、取舍、修补。心理暗线是“不知道自己算不算真的在生活里出现过”。
- 周屿：从玩笑型室友，到用表达接住紧张和普通关系的人。弧光不写成主持天才，只写成敢站到灯下说两句。
- 唐骁：从规则提醒者，到帮助主角理解边界、责任、钱、记录的人。弧光是不把规则当冷漠，而当保护关系的方式。
- 陆沉：从安静、少说、现实压力隐现，到能被看见一点疲惫，也能接受一点陪伴。弧光不做苦情反转，只做低声打开。
- 沈嘉禾：资料、学业、方向感支线。她不是工具人，而是让主角看见“慢慢做深”的人。
- 夏知微：记录、影像、作品支线。她让主角看见地点、时间和记忆如何变成可留下的东西。
- 秦越：活动资源与人情网络支线。不能只奖励资源，要同时带来边界风险。
- 晚风：线上关系现实化支线。她不是情绪垃圾桶或恋爱奖励，而是会被主角作息、逃避和边界影响的具体人物。
- 父母/家庭：现实重量支线。重点不是施压，而是父亲工作、母亲复查、饭桌问题让主角学会把家也算进选择。

## 互动系统设计

现有 auto/choice 结构保留，新增七层扩展：

1. 关系温度：记录人物信任、靠近、低温、修补状态。
2. 心理暗线：记录回避、过载、边界、自我照顾、现实面对。
3. 支线记忆池：只回放玩家真正经历过的人和事。
4. 时间回响：关键选择必须在几周后、暑假后、冬天前产生后果文本。
5. 场景动线锁：每个节点限制可用地点，避免泛校园漂移。
6. 资源/奖励：资料、作品素材、活动证明、边界意识、稳定节奏等只影响后续文本和软判定。
7. 终章回收：终章只读取第六幕准结局和软变量，不推翻主预结局。

## 交付结构

- `1-6幕剧情优化稿.md`：面向阅读和策划校对，包含主线锁定、角色状态、支线扩展、关键节点小说化文本和互动节点说明。
- `沉浸式剧情互动系统_v2.json`：面向 Codex/前端接入，包含角色、状态、场景、主线锁点、节点扩展、选项效果、支线触发和终章回收规则。

## 验收标准

- 中文内容完整，不出现玩家可见的开发层词汇。
- JSON 可解析。
- 四人宿舍关系和关键事件不改变。
- 支线人物至少拥有可触发事件、互动选项、关系/心理/资源影响。
- 选择反馈不只写即时反应，包含延迟回响。
- 不出现超出武生院沉浸版场景体系的随机地点。
