# 第四卷第十三章《同一个文件夹》设计

## 写作目标

承接第十二章《六十一》的数据结构实验周通知。第十三章不再继续庆祝低分通过，而是把“刚松一口气”的林亦舟重新放回计科专业协作压力里：四个人看似天然成组，真正开始分工后，作息、能力、自尊、活动排班和文件规则全部浮上来。

## 必须完成的剧情职责

- 青枫居 4XX 内确定四人组，写清组长、代码、流程图、报告、测试截图和最终提交的分工。
- 唐骁的规则感要有用，也要产生压力：他能把任务拆开，但容易把别人逼到只剩执行。
- 周屿的主持队复盘和运动会预报名继续挤压实验时间，不把他写成纯粹不负责。
- 陆沉的图书馆勤工、旧电脑和流程图能力进入现场，不能让他只做沉默背景。
- 林亦舟刚 61 分通过，想补上亏欠，但代码能力和家庭消息仍然拉扯他。
- 晚风只通过消息和短语音存在，不替实验小组或宿舍关系下结论。
- 结尾留钩子：最终提交前，文件夹里出现版本混乱或重复文件，压到下一章。

## 场景与动线

- 主场：青枫居 4XX、教学楼/八号教学楼机房、图书馆讨论区或星辰书屋。
- 可短停：西区外卖架或便利店，用来补生活感和外流感，但不能开启 5X。
- 不使用明德楼承载课程、班会、补考、答疑、机房或上机。
- 不新增校外交通动线。

## 文风约束

- 使用 `chinese-novelist` 原则：展示而非讲述，冲突驱动，结尾留钩子。
- 不写系统口吻、变量口吻、上帝视角总结。
- 正文避开：`这一步会被记进`、`后续会影响`、`系统记录`、`输出结果`、`让林亦舟意识到`、`让他明白`、`他突然发现`、`下一句话会`、`此刻决定了`、`命运开始分岔`、`主线`、`副线`、`节点`、`触发`、`判定`、`回主线`、`资源线`。
- 第四卷继续避开解释性高风险口头禅：`觉得`、`发现`、`突然`、`忽然`、`让林亦舟`、`下一句话`、`怕被`、`怕规则`、`怕自己`。
