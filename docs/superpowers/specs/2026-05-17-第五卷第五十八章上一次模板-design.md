# 第五卷第五十八章《上一次模板》设计记录

## 目标

承接第五十七章末尾胡嘉宁两秒语音：“我没改标题，它自己套了上一次模板。”本章不扩大为新支线，不开启 5X，也不引入许澈，只把冲突从“某个人一句花絮写法不合适”推进到“活动材料默认模板会把旧表达重新带回公开场”的流程惯性。

## 场景

- 东区商业街小舞台台侧。
- 活动已经开场，第一组节目结束，第二组准备上台。
- 材料群、主持队临时群、共享文档权限页、微信聊天作为手机内场景。
- 晚风只作为外部刷到文字残留的镜像，不参与解决。

## 人物动作

- 秦越：作为主持队临时群群主，先想追问“谁建的模板”，后被拉回当下可处理的权限、改名、说明和撤回。
- 唐骁：抓住标题、权限、共享卡片和材料群流向，不让“自动套模板”成为免责句。
- 陆沉：记录文档来源、权限、改名时间、材料群卡片残留和更正说明，不替所有传播端作证。
- 周屿：不让胡嘉宁成为替罪羊，但也不让“模板”被说成无辜。他只要求别把旧表达换壳继续流动。
- 许棠：从融媒体审核端接管花絮稿，把文档改成普通标题，要求材料群以重命名文档为准。
- 夏知微：继续控制拍摄边界，提醒截图和文档卡片也会变成材料。
- 林亦舟：在晚风提醒和台侧处理之间传递必要信息，克制索图冲动。

## 冲突推进

1. 胡嘉宁语音说明标题来自上一次模板。
2. 众人发现问题不只是标题本身，而是共享文档卡片已经进入材料群。
3. 唐骁、陆沉、许棠拆出四件事：改名、改权限、停止使用旧卡片、发更正说明。
4. 周屿把责任从“谁尴尬”拉回“不要让漂亮标题继续替现实遮羞”。
5. 文档改名后，材料群里的旧卡片预览仍未更新，留下下一章钩子。

## 文风要求

- 用舞台、手机屏、文档卡片、灯光和后台动作展示压力。
- 不写抽象自我认识，不写上帝视角分叉，不写“谁怕谁怕”式并列心理。
- 结尾要落在可见物上：灰白色文档卡片仍挂着旧标题。
