# 第五卷第六十七章《八点三十分》写作设计

## 目标

承接第六十六章章末“预约发送时间：明早八点三十分”。本章要把压力从“看见残留字段”推进到“是否连夜取消预约发送”，并让取消预约后预览链接失效，制造下一章的现实误会。

## 场景与路线

- 主场景仍在青枫居 4XX 公共桌，时间为深夜。
- 远程参与者为许棠、梁思予、秦越、胡嘉宁，均通过小群截图、电话、语音和后台预览参与，不抢宿舍四人的核心位置。
- 明德楼只作为刚才融媒体后台来源，不在本章承载课程、班会或普通活动。
- 晚风只通过短消息提示外部有人问链接，不成为决策者。

## 人物推进

- 周屿：明确宁可明早晚一点发，也不要让旧话按点自动出去。
- 唐骁：想立刻按取消，火气仍在，但这次愿意等秦越和许棠确认后再点。
- 陆沉：把取消预约、保存草稿、预览链接失效、给老师说明分成不同动作。
- 林亦舟：负责把晚风外部消息收住，并把对外说明压成不煽情的短句。
- 许棠、梁思予、秦越：完成后台取消、截图留存、给融媒体老师说明的具体动作。

## 章节钩子

预约取消后，旧预览链接失效。活动临时群里有人问“链接怎么打不开了，是不是把刚才那版删了”。下一章可以处理“停用旧稿”和“删除痕迹”之间的误会。

## 避免事项

- 不出现许澈、A302。
- 不使用系统口吻和开发表达。
- 不写“让某人意识到”“下一句话会”等上帝视角。
- 不把晚风提升为核心。
