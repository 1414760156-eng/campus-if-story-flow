# 第五卷第四十五章《发出去的包》设计

## 目标

承接第四十四章结尾的压缩包断点，写清“旧入口已关，但待剪包已经流出”的下一层责任。章节不把姜宁写成单薄犯错者，而是让她在活动压力、志愿者协作和材料混乱里露出真实失误：为了赶进度把“主持人背景备用”一起打包发出。冲突重点落在已经发出去的包如何追回、替换和说明。

## 冲突链

1. 许棠在东区商业街小舞台旁指出压缩包已经发给剪辑志愿者，打包人显示姜宁。
2. 姜宁承认打包过快，但不能用删包假装没发；秦越想立刻电话兜底，周屿要求先问清下载、打开、是否进工程文件。
3. 青枫居 4XX 夜里把处理步骤压成可执行清单：停用旧包、发替换包、询问下载和使用状态、留下撤换说明。
4. 校园快递站作为生活镜像：发错取件码不能只删短信，得找拿走包裹的人，照出“已发出去”的现实重量。
5. 夏知微和晚风作为边缘镜像：旧相册被改语境、旧群邀请被转发，补足“公开过不等于可再次改用”的同类压力。
6. 结尾由剪辑志愿者叶青回复：照片没用，但用了压缩包里的旧试麦视频，引出下一章“视频比照片更难收回”的新压力。

## 场景与人物

- 东区商业街小舞台、奶茶店外小圆桌：许棠、姜宁、秦越、周屿、林亦舟参与材料确认。
- 青枫居 4XX：宿舍四人把现场混乱拆成清单，宿舍关系仍优先。
- 校园快递站：用取件码、错拿包裹和工作人员短句压实生活逻辑。
- 微信聊天：夏知微、晚风以支线镜像出现，不抢走宿舍修补线。
