# 第五卷第九十四章《先停在这里》设计记录

## 目标

承接第九十三章末尾“03:16 以后按旧图对过的内容还算不算”，把此前连续追查的链条转入收束：不是继续追每一个看过旧图的人，而是在东区商业街后台把已经发生的材料动作分成“停用、重做、留痕”三类，给第五卷后段的活动开场、关系后果和第六卷余波留下入口。

## 场景

- 主场：东区商业街小舞台后台。
- 关联地点只通过电话和群消息出现：明德楼融媒体工作室、活动端小群、确认群。
- 不新增校外地点，不触发硬外流，不让支线人物抢走宿舍四人核心。

## 人物职责

- 林亦舟：把外部扩散压住，短回晚风，不把现场话变成对外解释。
- 周屿：把讨论从“谁该背锅”压回“谁能看见、谁已经做过什么”。
- 唐骁：追问“还算不算”的现实后果，但在章内接受不能无限追人。
- 陆沉：把背面白卡重新分成停用、重做、留痕三栏，承担收口动作。
- 秦越：交代活动端按旧图对过的顺序、站位、口播和台侧提醒，并负责回到活动端逐项停用。
- 许棠：负责发布端和预览图停用说明，承认私聊没有回群带来的后果。
- 梁思予、陈景：分别处理文案端和导出记录边界，不再补漂亮解释。
- 老师：补发事实，要求公开发布停、内部材料重做、已经发生的沟通留痕。

## 情绪推进

本章不写彻底和解，也不写单人认罪。重点是“事情不能算没发生，但也不能继续用旧图往下滚”。人物要通过具体动作表达疲惫和克制：停笔、删半句、把手机翻过来、把错名文件移出桌面、把群里一句话改短。

## 章末钩子

活动没有因为材料问题完全停下。外面开始试音或灯光，后台有人问今晚还开不开场。第五卷由追查链条转向“带着不完整的处理结果继续上场”。
