# 第四卷第三十九章《不放进去》设计

## 使用的技能

- superpower-bootstrap
- superpower-brainstorming
- planning-with-files
- superpower-writing-plans
- chinese-novelist
- superpower-verification

## 章节目标

承接第三十八章末尾“周一 A302 课堂展示未完成项如何记录”的压力，把“哪些材料不该交”写成公开场合里的具体冲突。四个人必须在同一间教室里各自开口，说明自己没有把某些现实压力、情绪材料或私人材料塞进实验四证明里。

## 锁定边界

- 场景主轴在青枫居 4XX、西园餐厅、教学楼 A302、银杏路和青枫居夜间公共桌。
- 明德楼不承载班会、课程、答疑、机房、试训或课堂展示。
- 晚风票务投诉只作为现实支线余音，不替代宿舍四人主轴。
- 唐骁二级成绩未定只作为个人压力，不改实验四提交逻辑。
- 第四卷只埋外流感，不开启 5X；许澈不进入关系核心。

## 事件设计

1. 青枫居清晨，四人把“不放进去”的材料写在白纸上：票务材料、小舞台证明、二级考试结果、图书馆替班细节、私人情绪。
2. 西园早餐，旁人听说 22 组要展示，半开玩笑地问能不能借模板，提前引出“能不能发 Word 版”的后续压力。
3. 教学楼 A302，22 组投影四类提交材料，林亦舟说明提交包只有课程证据，私人材料不入包。
4. 唐骁、周屿、陆沉分别被追问：未确认的考试、活动证明、签退照片为什么不能当作实验四证据。
5. 陈老师把问题压回证据边界：不是把生活交给课程评判，而是材料能否支撑一句话。
6. 课后学习委员和同学追问 Word 版，章末形成下一章“模板被借用/套用”的压力钩子。

## 文风执行

- 展示而非讲述：用白纸、投影、U 盘、手机屏幕、椅子和文件夹来承载压力。
- 对话要有校园真实感，避免上帝视角判断和自我认知总结。
- 不写“让他明白”“下一句话会”“命运分岔”等解释性句式。
- 章节结尾留下可续写问题，而不是把课堂展示完全解决。
