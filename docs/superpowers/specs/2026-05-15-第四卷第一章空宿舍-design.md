# 第四卷第一章《空宿舍》设计记录

## 目标

开启第四卷《夏天没有停下来》。第三卷以课程群名单、主持队试岗观察、生活费备注和晚风反诈空表收束，第四卷需要把热闹后的余波转入暑假：宿舍变空、家庭生活费变化、晚风从线上进入现实邀约、陆沉留校勤工、周屿小舞台后的被注视、唐骁把暑假也排成表。

## 本章承接

- 青枫居逐渐变空，离校和留校形成对照。
- 林亦舟收到父亲转账延迟和母亲“别乱花”的新语气。
- 晚风通过微信、抖音和光谷见面邀约进入现实。
- 周屿小舞台试岗后被人送水和起哄，承接亲密关系试错线。
- 陆沉暑假留校勤工，现实压力变得更可见。
- 唐骁把暑假学习、乐跑、实验和返校安排写进表格。

## 写作要求

- 用空宿舍、床铺、行李箱、空调遥控器、外卖袋、转账备注、微信语音、抖音预告等物件承载心理。
- 不写系统口吻，不出现正文禁用词。
- 明德楼本章不承载课程、班会、试训或机房内容。
- 结尾留钩子：晚风提出现实见面，家庭电话同时压来。
