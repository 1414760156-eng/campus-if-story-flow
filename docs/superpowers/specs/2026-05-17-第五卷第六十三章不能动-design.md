# 第五卷第六十三章《不能动》设计记录

## 目标
- 承接第六十二章结尾“这个模板不能动，明天复盘还要用”。
- 保持第五卷 5A 宿舍修补线，不新开 5X，不改变第六卷和终章硬结构。
- 将冲突从“谁误用了模板”推进到“旧模板是否应保留、如何停用、谁承担交接成本”。
- 保持宿舍四人优先：周屿、林亦舟、唐骁、陆沉分别以态度、传话、追问和记录推动现场。

## 场景与动线
- 夜晚：东区商业街小舞台散场后的折叠桌，收尾、装袋、处理群消息。
- 回寝：从东区商业街回青枫居，经过宿管窗口，宿管阿姨只作为生活秩序与夜归细节出现。
- 次日上午：明德楼融媒体工作室。此处只承担融媒体复盘、公共盘清理、材料归档，不承担班会、课程、考试或教学任务。

## 冲突设计
- 上一届“不能动”的意思不是护短，而是担心删掉旧模板后下一轮活动无人能写，呈现大学组织交接的现实惯性。
- 秦越不能再用“我负责”概括所有问题，需要把公共盘、旧文件、使用中的模板分开。
- 唐骁追问“不能动是不能删，还是不能停用”，推动语言精确化。
- 周屿坚持“不再替换姓名”，但不要求销毁文件，避免强行胜利。
- 陆沉把“原件留档、工作区停止使用、新稿另建”写成可执行路径。

## 叙事规则
- 使用 chinese-novelist 原则：以动作、物件、对话推进，不写系统口吻和上帝视角总结。
- 不使用禁用表达：主线、副线、节点、触发、变量、判定、系统记录、让林亦舟意识到、下一句话会等。
- 不出现 A302，不使用“许澈”。
- 不让晚风成为核心，仅通过微信短句作为外部压力余波。

## 章末钩子
- 在明德楼融媒体工作室打开“复盘推送_今晚先发版”或同类预备稿。
- 预备稿没有再用旧照片，却沿用旧标题。
- 章末落在：“标题第一行，还是那束灯。”
