# 第五卷第十八章《谁算改坏》设计记录

## 写作目标

- 承接第十七章末尾周三 C305 要求说明“谁改坏，谁改回，为什么改回”。
- 把上一章的“改字不改事”推进到更尖锐的问题：谁有资格说一版表被改坏了，谁又愿意承认自己参与过改坏。
- 宿舍四人继续为中心，外部人物罗泽、丁祺、何书南、沈嘉禾、秦越、晚风只提供镜像压力。
- 不开启 5X，不让许澈进入关系核心，不改变第五卷 5A 宿舍修补方向。

## 章节结构

1. 青枫居 4XX 清晨：四人挑选 C305 要讲的“改坏后又改回”案例，唐骁想写罗泽表，周屿指出不能替罗泽承认。
2. 西园餐厅：罗泽、丁祺、何书南对“谁算改坏”产生分歧；建议者、下笔者、同意发出的人都不想成为唯一名字。
3. 教学楼 C305：陈老师要求说明“参与过哪些改动”，不是把改坏扣给某一个人。沈嘉禾用“敏感内容”的改动经历作参照。
4. 东区商业街小舞台与晚风微信：秦越和晚风分别遇到“评价词是谁写的”“误会是谁建议的”。
5. 青枫居夜间：4XX 把本组说明改成“改坏不是罪名，是一段需要说清的过程”，章末转向下一次“本人不承认”的课堂压力。

## 人物推进

- 林亦舟：继续克制帮忙冲动，只给结构，不替人认错。
- 周屿：把玩笑收住，学会区分“建议过”和“写下去”的差别。
- 唐骁：从急着划掉错误，推进到承认自己也可能把圈出来的动作变成压力。
- 陆沉：保持事实边界，把“谁改坏”拆成看见、建议、下笔、同意发出的过程。
- 罗泽三人：呈现普通学生对责任署名的真实回避。
- 沈嘉禾、秦越、晚风：补出措辞、活动交接、票务材料中的同类问题。
