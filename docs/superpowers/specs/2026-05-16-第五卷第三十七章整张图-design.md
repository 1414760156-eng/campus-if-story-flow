# 第五卷第三十七章《整张图》设计记录

## 目标

承接第三十六章末拍照者询问“能不能发完整私聊”，继续推进第五卷 5A 宿舍修补线。第三十七章不改变主线锁点，不开启 5X，不引入许澈，只把当前冲突从“只剩三句是否摘句”推进到“完整材料是否等于可以公开”。

## 叙事设计

本章采用现实主义校园群像写法，核心冲突落在周屿作为私聊另一方被拖入公开场。拍照者想用完整截图自证没有被断章取义，4XX 则必须分清完整材料给谁看、公开材料能写什么。宿舍四人仍为中心：周屿面对主持式圆场失效，唐骁守住材料边界，陆沉提出“聊天另一方也要同意”，林亦舟在晚风线里照见同类问题。

## 场景安排

- 青枫居 4XX：楼长转来拍照者请求，四人讨论完整私聊的归属。
- 青枫居一楼宿管窗口：宿管阿姨用快递面单、取件码和姓名地址类比，强调本人能提交材料，不等于能公开对话另一方信息。
- 西园餐厅：罗泽、丁祺、何书南处理小群里“发完整就清白”的说法。
- 教学楼 C305：陈老师讲“完整材料”和“公开摘录”的差别，仍使用大二上计算机组成原理场景。
- 东区商业街小舞台、阳光书屋、晚风微信：分别以录音原文件、原片、聊天截图形成镜像。

## 文风约束

正文中避免系统口吻、抽象说教和上帝视角，不使用“主线/节点/触发/判定/变量/回流”等玩家可见元词，不出现“让林亦舟意识到”“下一句话会”“你怕我怕”式表达。心理暗线通过手机、截图、外卖架、快递面单、桌面动作和沉默来呈现。

## 章末钩子

拍照者最终没有等到完整同意，先在楼层群发出一张“只露最上方”的截图预览。内容未完全展开，但周屿的头像和备注没有遮干净，把下一章压力推向“对话另一方是否同意被公开”。
