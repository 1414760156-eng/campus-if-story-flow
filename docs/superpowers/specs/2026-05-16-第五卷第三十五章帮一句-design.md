# 第五卷第三十五章《帮一句》设计

## 断点

第三十四章末，公开更正已经贴出，拍照者完成私下登记，却继续私聊周屿：“你们能不能在群里帮我说一句，我真的不是故意的？”

## 章节目标

- 延续 5A 宿舍修补线，不开启 5X，不引入许澈。
- 把冲突从“公开说明最后一行”推进到“谁有资格替别人说不是故意”。
- 周屿的表达能力受到考验：他想用一句圆场话把气氛压下去，但这一次“圆场”本身会变成背书。
- 唐骁守住规则边界，陆沉补出“替他说”会让被拍者再次被迫回应，林亦舟把晚风线里“不替朋友出面”的经验压回现实。
- 支线镜像继续使用秦越小舞台、夏知微摄影社、晚风微信和 C305 课堂，不抢宿舍四人中心。

## 场景安排

1. 青枫居 4XX 夜里：拍照者私聊周屿，四人围绕“帮一句”和“背书”起小争执。
2. 青枫居一楼窗口：宿管阿姨把“本人补充”和“别人证明”拆开，要求不要堵门、不要让 4XX 代发。
3. 西园餐厅：罗泽、丁祺、何书南处理小群里“4XX 都不帮他说，是不是太狠”的风向。
4. 教学楼 C305：陈老师用匿名样例训练“代为解释”和“转述本人陈述”的差别。
5. 东区商业街小舞台、阳光书屋和晚风微信：分别出现“帮我说一句”的现实镜像。
6. 章末钩子：拍照者自己在楼层群发言，却写成“我已经和 4XX 沟通过，他们也知道我不是故意的”，把下一章压力推向“沟通过”被拿来当背书。

## 文风约束

- 展示而非讲述，用私聊截图、登记本、餐桌短句、课堂改写和微信停顿推动。
- 不写系统口吻、不写变量、节点、回流、判定等游戏词。
- 不使用“让林亦舟意识到”“下一句话会”“周屿怕/唐骁怕/陆沉怕”等解释性模板。
- 明德楼不承载课程、班会、机房或答疑。
