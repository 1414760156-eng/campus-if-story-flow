# 第四卷第二十五章《空白表》设计说明

## 目标

承接第二十四章末尾的三块屏幕：空白边界说明表、唐骁二级错题截图、周屿小舞台录音。章节重点不是再扩大外卖架事件，而是让“帮忙带来的新活”压回青枫居 4XX，使四个人在同一间屋里重新分配各自的时间和责任。

## 叙事策略

- 主场放在青枫居 4XX 夜里，保持宿舍四人优先。
- 用桌面、耳机、截图、录音进度条、空白表格和公共桌动作呈现人物状态。
- 林亦舟负责把群里的需求压成可公开的空白栏。
- 唐骁必须把二级第 6 套错题截图补交，不能继续被宿舍事务吞掉。
- 周屿必须面对小舞台录音，但不在本章完成漂亮成长，只给出低声听完和不立刻辩解。
- 陆沉负责把“可以写什么”和“不该写什么”分清，但不替别人解决所有问题。

## 场景安排

1. 青枫居夜里：四人围着公共桌定顺序，先把每个人必须在今晚完成的事摊开。
2. 空白表格：只留栏目，不留句子，栏目包括观察地点、记录范围、不记录内容、原始形式、缺测时段、不补写原因、可公开材料。
3. 小冲突：唐骁不愿再被临时拉走，周屿想开玩笑缓冲但嗓子压不住，林亦舟犹豫是否继续接同学私聊，陆沉用纸面分工止住互相抢话。
4. 个人事务回收：唐骁补错题截图，周屿点开录音，林亦舟把空白表发给助教，陆沉把最终文件名写清。
5. 下一章钩子：助教收到空白表，但课程群又发布数据结构实验三，要求周日前提交并随机抽查运行过程；公开材料压力暂时收束，专业课压力接上。

## 文风约束

- 不写系统口吻，不写“这一步会被记进”类说明。
- 不写上帝视角预告，不用“让林亦舟意识到”“下一句话会”等解释句。
- 避免“觉得、发现、突然、忽然”等惰性表达。
- 不使用“怕被/怕规则/怕自己”三连式句法。
- 章节结尾保留具体钩子，不抽象总结。

## 自检

无 TODO；不改变第五卷和第六卷结构；第四卷只保留引导式外流气味，本章不写许澈关系推进。
