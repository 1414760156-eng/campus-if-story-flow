# 第三卷第二十二章《求别曝光》设计

## 目标

承接第二十一章章末匿名校园号发布柳园洗衣房照片和“求别曝光”评论，把公共表达的代价从“能不能发”推进到“发出后谁承担”。本章要让事件继续有现实阻力：删除慢、截图快、评论区只认热闹、柳园同学内部也会互相埋怨。

## 叙事边界

- 主场仍在武生院。
- 宿舍四人留在青枫居处理信息和边界，不越权代写、代发、代吵。
- 沈嘉禾在柳园内部承担现场压力，让支线人物不只是传话。
- 许棠在明德楼融媒体工作室给出平台和学院流程边界，明德楼不承接班会、课程或试训。
- 周屿仍在教学楼 B206 承接主持队压力，和匿名事件形成“被临时插活”和“被公共叙事拖走”的呼应。

## 本章事件设计

1. 匿名号帖子扩散，第一条“求别曝光”被更多评论淹没。
2. 柳园室友方苒出场，确认照片里的粉色洗衣篮属于她，个人物品被公开导致真实伤害。
3. 沈嘉禾在柳园楼层群劝撤图，被质疑“是不是帮计科那几个说话”，支线人物承担现实冲突。
4. 青枫居四人决定不下场吵，只做留证、私信撤图、模板说明和流程建议。
5. 许棠提醒：融媒体不能命令匿名号，只能协助留证、平台投诉、避免继续投喂材料。
6. B206 插入周屿一场主持队临时任务压力，秦越要求他别用“会说话”替所有人接活。
7. 匿名号最终编辑或删除原帖，但截图已进更多群，事件没有被干净收束。
8. 章末由陈老师的课堂消息开启下一章：带材料使用说明来教学楼 A101，讲删掉了什么，也讲没能删掉什么。

## 文风要求

- 使用 chinese-novelist 原则：展示而非讲述，冲突驱动，结尾留钩子。
- 不写游戏系统口吻，不写“这一步会被记”“下一句话会”等上帝视角句式。
- 避免反复心理总结，用屏幕、评论、物品、门口停顿、洗衣篮、折叠评论、手写备注承载人物变化。
