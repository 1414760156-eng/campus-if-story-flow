# 5A 回流/外流分型系统 v3 设计

## Summary

本轮把 `playtest/5a-full-data.js` 从 v2.2 的“延长版单线回流”升级为 v3 的“分型回流/分型离场”。目标不是增加失控的多线，而是让玩家在关键空间动作上有真实可操作感：暂时离开现场可以遇到室友，也可以遇到本线外的边缘人物；最后的离场根据前面积累，分成 5A 内部收束或真正进入 5X 新线。

## Chosen Approach

采用“唯一硬离场点 + 多个软回流点 + 判定型终端”的方案。

- `同质回流`：仍围绕 5A 宿舍关系展开。玩家离开主现场后遇到周屿、唐骁或陆沉，发生一段微心态对话，只影响宿舍互信、单人信任、回避值、修补温度等状态，然后回到 5A 主轴。
- `异质软回流`：玩家暂时走向湖边、操场、便利店外沿等空间，遇到晚风消息或其他边缘人物。它只提供心理牵引、资源提示或关系温度，不解锁 5B-5F，不改变主线归属，之后回到 5A 主轴。
- `同质收束`：唯一硬离场点上，如果宿舍关系值足够，玩家选择不继续正面修补时不会进入新人线，而是得到 5A 自己的小结局，写入第六幕入口判定。
- `异质外流`：唯一硬离场点上，如果宿舍关系不足、回避值过高、短期决裂风险过高，玩家选择离开时进入 `5X_离场外流线`。该线在第六、第七幕只走自己的单线，同场景角色只能软互动。

推荐把“同质外流”命名为“同质收束”，避免和真正外流混淆。

## Data Model

新增或规范以下字段：

- `流向类型`: `"同质回流" | "异质软回流" | "同质收束" | "异质外流"`
- `回环节点`: `true` 用于所有回流节点。
- `回流节点`: 指向回到 5A 主轴的节点。
- `软串场角色`: 记录异质软回流里出现但不拉线的角色或关系，如 `"晚风"`、`"影像社边缘同学"`。
- `禁止解锁分线`: `true` 表示该节点或选项不得跳入 5B-5F。
- `离场判定`: 用于唯一硬离场点的阈值说明。
- `第六幕入口标签`: 用于同质收束或异质外流写入后续幕。

## Node Changes

### Corridor Vertical Slice

`ACT5_5A01_CORRIDOR_RETURN` 不再只有一个“不进门”回流：

- `不进门，去楼下坐一会儿`：进入同质回流，楼下遇到周屿或唐骁，发生微心态对话，再回门口。
- `去湖边绕一圈`：进入异质软回流，收到晚风消息或碰见边缘人物，短暂意识到“离开寝室也不是答案”，再回门口。

两条都不改变 5A 主轴，只改变心理和关系状态。

### Later Loop Points

后续三个回流点保留两拍以上结构，但要标明流向类型：

- 水房洗脸：偏同质回流，可出现陆沉或唐骁的低声对话。
- 便利店买水：偏同质回流，也可加入异质软回流的一次短暂外部人物擦肩。
- 期末走廊两页：偏同质回流，强调学业压力和宿舍规则复发。

### Unique Terminal Departure

`ACT5_5A15_DOOR_SLAM_RISK` 仍是唯一硬离场点。该节点新增两类结果：

- `同质收束`：关系阈值足够时，进入 5A 小结局节点，宿舍没有完全修好，但形成可读的低温稳定或有限修补，并进入第六幕判定。
- `异质外流`：关系阈值不足时，进入现有 `ACT5_5X01_STAIRWELL_MUTE`，之后走新人新线。

## Threshold Rule

阈值只作为数据说明和测试约束，不在当前静态试玩器里做复杂 UI 判定；但节点必须写清楚判定含义。

建议判定：

- 同质收束：`宿舍互信 >= 48` 且 `回避值 <= 5` 且 `短期决裂风险 <= 0`
- 异质外流：`宿舍互信 < 48` 或 `回避值 >= 6` 或 `短期决裂风险 >= 1`

试玩器当前仍通过玩家选项显式进入对应节点；后续若升级引擎，可让 `engine.chooseOption` 根据 `判定条件` 自动分流。

## Non-Goals

- 不把 5A 回流重新做成第五幕多线混杂。
- 不让晚风在 5A 成为核心，只作为消息、状态或软串场。
- 不新增第二个硬外流点。
- 不改变军训手机事件、汇演、班会、主持招新、毕业活动等主线锚点。
- 不重写第六幕或终章，只写入可被后续读取的标签。

## Testing

测试需要覆盖：

- 标题包含 v3。
- 至少存在 1 个 `同质回流` 节点与 1 个 `异质软回流` 节点。
- 所有 `异质软回流` 都设置 `禁止解锁分线: true`，且不解锁 5B-5F 或 5X。
- `ACT5_5A01_CORRIDOR_RETURN` 同时提供楼下同质回流和湖边异质软回流。
- 唯一硬外流点仍只有 `ACT5_5A15_DOOR_SLAM_RISK`。
- 存在 `同质收束` 输出节点，且写入 `第六幕入口标签`。
- `5X_离场外流线` 仍保持定轨，不回旧线。
