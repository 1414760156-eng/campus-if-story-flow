# 第四卷第四十五章《自己写自己的》设计记录

## 目标

在第四十四章“三类不能替写的句子”之后，继续把第四卷后段收向“上册结束前的材料边界”。本章不另开新关系线，仍以青枫居四人为主轴，把课堂互改、计算机二级结果、家庭转账与晚风票务审核压到同一张公共桌上。

## 写作约束

- 地点只使用青枫居、教学楼 A302、西园餐厅、校园快递站、阳光书屋、图书馆西侧和东区商业街小舞台余音。
- 明德楼不承载课程、班会、答疑、试训、机房或上机。
- 不写系统口吻，不写变量、节点、判定、回主线等玩家可见词。
- 不写“让林亦舟意识到”“下一句话会”“你怕我怕”式总结。
- 第四卷只埋外流感，不开启 5X；许澈不进入本章。

## 章节设计

1. 青枫居早晨：唐骁计算机二级成绩从“待发布”变成“合格”，但课程群马上压来三类句子改写；林亦舟收到父亲短转账和母亲袜子取件提醒，家庭线以小物件进入本章。
2. 教学楼 A302：各组从三类越界句里各选一句改写。22 组不再做全班示例，只写自己的版本。课堂重点放在“事实、听说、本人说明”三种栏目边界。
3. 课堂冲突：有同学想把“家里原因”“活动原因”“关系问题”写成可交句，唐骁、周屿、陆沉分别用自己的方式拦住越界，陈老师把任务压回每组亲自确认。
4. 校园快递站与西园餐厅：袜子包裹、二级成绩截图、父亲短备注、餐桌小庆祝共同显示生活仍在继续。
5. 阳光书屋：晚风票务审核被要求继续补证，林亦舟只帮她压短可确认事实，不替她把愤怒写成定论。
6. 青枫居夜里：四人把三类句子写成自己的提交版本；课程群发来期末周与个人材料提醒，进入第四卷上册收束前的新压力。

## 预期钩子

章末以“期末周、个人材料、未确认事项不得代交”收束。下一章可继续写第四卷上册收尾：公共桌上的材料各归各位，班级进入期末前最后一次课堂确认。
