# 第四卷第八章《座位表》设计

## 目标

承接第七章 C201 答疑和 B204 返校班会，把补考当天写成第四卷的现实压力场：学业失败感、宿舍轻修补、家庭生活费错位、晚风现实见面压力和第四卷引导式外流同时出现，但不提前开启 5X。

## 叙事边界

- 补考、答疑、座位表和考后走廊均放在教学楼，不借用明德楼。
- 许澈只作为便利店远影或一句短话出现，不和林亦舟形成稳定关系。
- 章节不写系统语言，不写玩家可见变量，不直接讲心理成长。
- 宿舍四人仍是章内主轴：周屿用玩笑刹车，唐骁用规则和饭局修补，陆沉用水、路和沉默陪伴，林亦舟在补考与家庭消息之间被迫接住自己的位置。

## 场景安排

1. 青枫居清晨：座位表、学生证、早餐、宿管阿姨提醒，把考试感落到物件上。
2. 教学楼 C201：补考现场、手机放置、名单边界、试卷中的卡壳和收卷。
3. 教学楼走廊：有人议论补考名单，四人用很小的动作挡住公开羞耻。
4. 东区商业街东北饺子馆：唐骁的饮食偏好变成集体饭局，四人关系在饭桌上轻微回暖。
5. 便利店和青枫居夜晚：许澈远影、晚风光谷消息、父母生活费错位，章末留下第五卷前的现实压力。

## 写法要求

- 使用 `chinese-novelist` 原则：展示而非讲述，冲突驱动，章末保留钩子。
- 具体化校园细节：青枫居、银杏路、教学楼 C201、东区商业街、便利店、宿管阿姨、东北饺子馆、学生证、座位表、取件/小票/转账。
- 不使用“让林亦舟意识到”“下一句话会”“你怕我怕”等解释性或上帝视角表达。
