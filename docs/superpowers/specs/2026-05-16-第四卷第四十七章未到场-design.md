# 第四卷第四十七章《未到场》设计

## 写作目标

承接第四十六章结尾的两条压力：课程群要求“本人未到场需本人提前提交文字说明”，父亲要求林亦舟“别跟你妈说晚点”。本章不把两者写成道理，而把它们放进同一天的动作里：课堂点名、同学代说、缺席说明、母亲电话、父亲转账、宿舍公共桌。

## 必须守住

- 主场仍在武生院，课堂确认放在教学楼 A302，不使用明德楼承载课程、班会、机房或答疑。
- 宿舍四人优先：周屿、唐骁、陆沉、林亦舟都要在课堂确认和夜里公共桌里有动作。
- 晚风线只作现实支线镜像：票务材料继续推进，但不抢走宿舍四人关系。
- 许澈不进入本章关系核心。
- 不写系统口吻，不出现“节点、变量、触发、回主线、系统提示”等玩家可见词。
- 避免“让林亦舟意识到”“下一句话会”“你怕我怕他怕”等总结式句子。

## 章节结构

1. 青枫居清晨：父亲未回复的消息、母亲袜子与生活费问候、四人带材料去 A302。
2. 教学楼 A302：最后一次课堂确认，出现同学想替未到场成员解释的现场；陈老师要求只读本人留下的文字，不补原因。
3. 走廊与打印店：学习委员、罗泽等人处理缺席说明和纸质版，形成现实余波。
4. 北园/阳光书屋/图书馆西侧/东区商业街：四人各自处理二级、主持总结、勤工排班、票务补证。
5. 青枫居夜里：母亲电话和父亲消息继续夹击；四人围绕暑假去向登记和本人确认收束本章，给第四卷上册总结前再留一个家庭钩子。

## 风格约束

以物件和动作推进：打印纸、签名表、手机未读、袜子包裹、工牌、主持队文件、课程确认表。少写抽象心理，多写停顿、删字、换文件名、放下手机、谁没有接谁的话。
