# 第四卷第十九章《十二点二十》设计记录

## 使用技能

- superpower-bootstrap：继续遵守技能检查和项目边界。
- superpower-brainstorming：把“中午还得下楼”拆成时间挤压事件，而不是单纯补记录。
- planning-with-files：按现有 specs/plans/progress 结构记录本次续写。
- superpower-writing-plans：先写可验收计划，再进入正文。
- chinese-novelist：用身体动作、路线、群消息、外卖架和体育场现场推动人物关系。
- superpower-verification：完成后做字数和禁用表达扫描。

## 本章承接

第十八章结尾：助教要求外卖架观察组提交场地说明、缺测规则和原始记录样例；周屿中午还要去晨光体育场参加 400 米栏试跑。他问“所以，我们中午还得下楼？”第十九章要把“少写一点”放进真实时间压力里：他们知道不能为了作业强行补满，但截止和活动会逼他们想抢时间。

## 本章职责

1. 主场从教学楼 A302 下课后，经过西园/青枫居一楼外卖架，再到晨光体育场和图书馆排班边缘。
2. 四人分工不能顺滑：
   - 周屿要在外卖架记录和 400 米栏试跑之间选择先后。
   - 林亦舟被迫临时补位，但不能把他写成万能协调者。
   - 唐骁要接受中午样例不完整，不能用规则把现实压平。
   - 陆沉因为图书馆排班不能无限兜底。
3. 运动会必须具体：晨光体育场、跑道、栏架、体育委员、试跑确认、周屿喉咙和身体状态。
4. 外卖架观察继续保留宿管阿姨、取餐同学、缺测和场地说明。
5. 晚风用微信短消息做外部回声，不替四人解决。
6. 结尾留下明晚 A302 提交材料的压力，并让周屿身体/主持双压继续抬高。

## 文风禁线

- 正文不写主线、副线、分线、节点、flag、触发、权重、判定、回主线、资源线。
- 不写“让林亦舟意识到”“让他明白”“他突然发现”“下一句话会”。
- 第四卷继续避开：觉得、发现、突然、忽然、让林亦舟、下一句话、怕被、怕规则、怕自己。
- 不把运动会写成热血爽点，要保留赶场、喘、失误和现实拖拽。
