# 第六卷后半晚风核心关系写作计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use $superpower-subagents (recommended) or $superpower-executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking via update_plan.

**Goal:** 将第六卷后半从“多线均摊”改为“林亦舟与晚风靠近，以及晚风生活圈牵引主角的新关系卷”。

**Architecture:** 第五卷已经完成站队和爆发，第六卷后半不重复爆炸，而是写关系窄化后的新选择。主轴为林亦舟和晚风，宿舍四人作为旧生活圈持续回响，夏知微/沈嘉禾/许棠/秦越等只作为偶然插入或对照，不再平均展开。

**Tech Stack:** Markdown 小说正文；`chinese-novelist` 写作原则；`planning-with-files` 进度管理；`count_v4.py` 字数统计与关键词扫描。

---

## 总体定位

第六卷后半的根本问题不是“项目还能不能交”，也不是提前完成所有关系收尾，而是：

林亦舟在宿舍关系变窄以后，是否真的选择了新的亲密关系，还是只是借晚风逃离 4XX 的旧账。

晚风不是单一安慰角色，也不是传统固定女主。她应承担三个功能：

- 亲密关系核心：林亦舟最靠近的人。
- 新生活圈入口：她的朋友、室友、光谷/武生院外部活动、线上状态和现实见面带出另一套关系规则。
- 镜子：她让林亦舟看见自己不是“中立调解者”，而是也会缺席、逃避、误读和被别人等待的人。

## 体量控制

第五卷体量已经达到约 40 万字符，是“矛盾爆发 + 支线资料库 + 关系站队”的超长卷。第六卷不应只是第五卷后的附录，而要成为下册里独立成立的“关系靠近与生活圈重组卷”。因此第六卷目标不再按 14-16 万字符处理，调整为约 30 万字符。

第六卷建议目标：

- 最终总量：约 28-32 万字符，目标中位数约 30 万字符。
- 当前前十一章：约 5.3 万字符。
- 后半建议扩为第十二章至第三十章：新增约 23-25 万字符。
- 单章体量：每章约 10,000-14,000 字符，采用 2-4 个小节。
- 章节数量：建议最终 30 章左右，不用追第五卷 100 章的爆发式密度，但必须有足够篇幅让晚风关系圈、宿舍退位、现实好感对照和外部失败暧昧自然长出来。

扩容方式不是堆支线，而是让每章有稳定三层：

- 晚风主关系推进：聊天、见面、误读、甜、等待或错过。
- 晚风生活圈展开：室友、朋友、旧暧昧、共同活动、光谷/武生院外生活场景。
- 宿舍旧生活圈回响：4XX 群、项目、课程、缺席、不等人、不叫人、公共桌空位。

这样第六卷会比第五卷短一截，但不会轻到失衡；它不是第五卷的尾声，也不是全书关系收尾，而是下册里独立的第二座重量：第五卷写“关系炸开”，第六卷写“新关系长出来，旧关系退位并留下后果”。真正的“所有关系慢慢走远、大家忙毕业工作，只靠学校固定必到时间点重聚”，放到第七卷。

## 关系圈配置

### 核心关系：林亦舟与晚风

- 关系方式：微信聊天、抖音状态、语音、线下短见面、光谷/武生院周边路线、错过消息。
- 推进节奏：从互相理解到现实靠近，再到因为项目、寝室、考试和晚风生活圈产生误读。
- 必须有甜：散步、一起等地铁、一起吃东西、互相保留状态、短暂牵手或接近牵手。
- 必须有代价：林亦舟错过寝室小节点、少回 4XX 群、钱和时间消耗、学习安排被挤压、晚风也被自己的生活圈牵走。

### 现实好感对照：夏知微 / 沈嘉禾

- 不写成狗血三角恋。
- 功能是让林亦舟意识到“现实中有人刚好在身边”也会产生动摇。
- 出现方式：图书馆、摄影、活动后勤、阳光书屋、星辰书屋、晨光湖或雄关短谈。
- 作用：和晚风形成线上靠近/现实靠近的对照。

### 晚风生活圈

可引入 2-3 个具名人物，不能全是无名 NPC：

- 晚风室友：知道晚风和林亦舟聊天，偶尔调侃或提醒她别太上头。
- 晚风旧暧昧或共同活动男生：不是反派，而是现实存在的比较对象，让林亦舟感到不安。
- 晚风同圈朋友：一起拍视频、去光谷、看演出、吃饭或玩桌游/K 歌，构成林亦舟进入新圈子的门槛。

### 宿舍旧生活圈

四人不再默认同行。

- 唐骁：项目、二级、竞赛/实习压力；对林亦舟缺席不一定质问，但会不再默认等他。
- 周屿：普通话、活动圈、运动/主持余波；可能用玩笑掩盖“你现在也有你的人了”。
- 陆沉：图书馆、勤工、实习与钱；更少出声，但他的退出最明显。
- 4XX 群：从热闹群变成事项群，消息越来越短。

## 章节级计划第一段：第十二章至第二十章

这一段先完成关系核心的建立：林亦舟靠近晚风，进入晚风生活圈边缘，同时宿舍旧生活圈开始不等他。

### 第十二章《接收人》

**目标：** 承接第十一章“毕业后接收位置 / 接收人”空着，同时让晚风成为林亦舟下一阶段最靠近的人。

**体量：** 约 9,000-10,000 字符，三小节：C407 接收人追问、4XX 回声、晚风线下邀约。

- [ ] C407 或机房里，冯助教追问毕业后接收位置是否继续空着。
- [ ] 唐骁、周屿、陆沉分别只承认自己的维护范围，没有人愿意接整组以后。
- [ ] 林亦舟没有替他们圆场，只说“暂不指定接收人”。
- [ ] 晚风发来消息，不是安慰，而是约他晚上去武生院站外或光谷附近见一面。
- [ ] 章末钩子：林亦舟第一次没有在 4XX 群里解释自己晚上去哪里。

### 第十三章《站外的风》

**目标：** 写林亦舟和晚风第一次真正进入“线下日常”的亲密推进。

**体量：** 约 10,000-12,000 字符，三小节：武生院站外见面、地铁/面馆/光谷短行、回寝缺席后果。

- [ ] 场景建议：武生院站外、地铁站附近面馆、光谷返程、东区商业街夜晚。
- [ ] 晚风不只听林亦舟讲宿舍，也讲自己的室友、朋友圈和旧关系。
- [ ] 林亦舟发现晚风并不是只围着他转，她也有自己的生活圈。
- [ ] 两人有甜：一起等车、分一份热食、短暂靠近或差点牵手。
- [ ] 代价：林亦舟错过 4XX 群里项目演示前的小确认。
- [ ] 章末钩子：唐骁在群里只发一句“算了，我先写”。

### 第十四章《她的朋友》

**目标：** 晚风生活圈正式出现，林亦舟进入新圈子的边缘。

**体量：** 约 10,000-12,000 字符，三小节：晚风生活圈聚会、旧暧昧/共同朋友出现、林亦舟回到宿舍后的错位。

- [ ] 引入晚风室友和一位共同圈男生/旧暧昧对象。
- [ ] 场景建议：光谷商场、K 歌、桌游、奶茶店、短视频拍摄点或学校周边聚餐。
- [ ] 林亦舟不是被排挤，而是明显不熟悉他们的玩笑和节奏。
- [ ] 晚风对他照顾，但照顾本身让别人看出暧昧。
- [ ] 林亦舟感到开心，也感到自己像临时加入别人已经很熟的桌子。
- [ ] 章末钩子：晚风旧暧昧对象随口问“你们现在算什么”。

### 第十五章《没叫他》

**目标：** 宿舍旧生活圈反向施压，不靠大吵，而靠缺席和小动作。

**体量：** 约 9,000-10,000 字符，三小节：上课/机房没叫人、项目事项群变短、晚风邀约与补说明冲突。

- [ ] 早课/机房/项目复测，唐骁没有像以前一样叫林亦舟。
- [ ] 周屿也没有在群里打圆场，只发课程地点或截图。
- [ ] 陆沉把材料发群，格式很清楚，但没有多余话。
- [ ] 林亦舟赶到后发现事情没耽误，但他已经不是默认在场的人。
- [ ] 晚风消息同时进来，问他晚上要不要出来。
- [ ] 章末钩子：林亦舟第一次在“去见晚风”和“回寝室补说明”之间犹豫。

### 第十六章《别人也会等》

**目标：** 强化晚风不是单向等待者，她也会被林亦舟耽误，也有自己的委屈。

**体量：** 约 9,000-11,000 字符，三小节：林亦舟迟到或取消、晚风生活圈替她发声、宿舍那边也不再等他。

- [ ] 林亦舟因为项目/宿舍临时事迟到或取消见面。
- [ ] 晚风没有大闹，只把状态改掉或很久不回。
- [ ] 晚风室友或朋友替她说一句不重但刺的话。
- [ ] 林亦舟意识到自己也在把“看情况”留给别人。
- [ ] 宿舍这边唐骁/周屿也因为他的迟回，把事情各自处理掉。
- [ ] 章末钩子：两边都没有等他。

### 第十七章《好感对照》

**目标：** 加入现实好感对照，但不写狗血争夺。

**体量：** 约 9,000-10,000 字符，三小节：现实同行、微妙动摇、未告知晚风的隐瞒。

- [ ] 夏知微或沈嘉禾在图书馆/书屋/摄影现场与林亦舟有一段自然同行。
- [ ] 对方能帮他解决一个现实问题，比如素材、照片、说明页、活动记录或学习资料。
- [ ] 林亦舟感到轻松，因为现实在身边的人不用解释太多时间差。
- [ ] 晚风同时发来消息，语气正常，却让林亦舟产生微妙心虚。
- [ ] 这章的重点是“大学里好感不只有一种来源”，不是让林亦舟马上摇摆。
- [ ] 章末钩子：林亦舟没有把这次同行告诉晚风。

### 第十八章《难看的喜欢》

**目标：** 写一条外部失败暧昧/短恋样本，让晚风线有现实参照。

**体量：** 约 10,000-12,000 字符，三小节：外部短恋翻脸、晚风态度、林亦舟意识到自己与晚风也不天然安全。

- [ ] 失败样本可来自晚风生活圈或活动圈：送礼、节日告白、朋友圈公开、聊天截图、冷处理后翻脸。
- [ ] 事件不压主线，但要让林亦舟和晚风都看见。
- [ ] 重点不是渣男渣女标签，而是大学生“享受在一起，却不懂责任”的幼稚和疼。
- [ ] 晚风对这件事有自己的态度，可能比林亦舟更清醒。
- [ ] 林亦舟发现自己和晚风也不是天然安全。
- [ ] 章末钩子：晚风问林亦舟，“你以后也会这样冷掉吗？”

### 第十九章《演示当天》

**目标：** 项目主线收束，同时让主角两种生活圈同日挤压。

**体量：** 约 10,000-12,000 字符，三小节：项目演示、晚风生活圈同日事件、演示成功后的错过。

- [ ] 演示当天，四人只负责各自部分，配合专业但不亲密。
- [ ] 林亦舟负责复测和页面点击，唐骁负责页面结构，周屿负责说明页口径，陆沉负责来源。
- [ ] 晚风或她生活圈当天也有事件，需要林亦舟回应或到场。
- [ ] 林亦舟必须选择先完成项目，晚风不是不理解，但等待会留下痕迹。
- [ ] 演示成功，不写成大团圆。成功后四人各自去不同地方。
- [ ] 章末钩子：晚风那边的聚会已经散了，只给他留下一条语音。

### 第二十章《不接所有人的以后》

**目标：** 第六卷中段阶段性收束，确认“谁也不接所有人的以后”，但不提前写毕业走远。

**体量：** 约 10,000-12,000 字符，三小节：项目归档、林亦舟和晚风关系谈话、4XX 公共桌与第七卷入口。

- [ ] 项目归档范围阶段性写清，但毕业后的真正流向不在本章处理完。
- [ ] 毕业后接收人仍不写具体人，只先写文件位置和停止条件。
- [ ] 林亦舟和晚风进行一次关系谈话：不是确定甜蜜终点，而是承认他们正在靠近，也都可能因为现实缺席。
- [ ] 4XX 公共桌还在，但四个人已经不再围着同一件事。
- [ ] 夏知微/沈嘉禾/晚风生活圈只做短短回声，不展开新线。
- [ ] 章末钩子：第六卷后半进入“关系现实考验”段，而不是第七卷毕业收尾。

## 章节级计划第二段：第二十一章至第三十章

这一段用于把第六卷扩成完整的 30 万字关系卷。它不再开很多新线，而是围绕“晚风核心关系已经形成后，现实怎样考验它”继续深入。

### 第二十一章《她也有旧账》

**目标：** 晚风不是纯白纸，她也有过去的暧昧、误会或没有说清的人。

**体量：** 约 11,000-13,000 字符，三小节：晚风旧关系露面、林亦舟不安、宿舍那边项目归档催促。

- [ ] 晚风生活圈中旧暧昧或曾经走近的人出现。
- [ ] 此人不写成反派，而是写成“他们确实有过一段模糊靠近”。
- [ ] 林亦舟第一次感到自己不是晚风生活里的唯一新鲜事。
- [ ] 4XX 群同时催项目归档材料，林亦舟回得慢。
- [ ] 章末钩子：晚风问他是不是介意。

### 第二十二章《没有资格介意》

**目标：** 林亦舟意识到自己也没有完全坦白自己的现实好感对照。

**体量：** 约 11,000-13,000 字符，三小节：晚风旧暧昧余波、夏知微/沈嘉禾现实同行回刺、林亦舟说不出口。

- [ ] 林亦舟想问晚风旧关系，却想到自己没有告诉她夏知微/沈嘉禾那次同行。
- [ ] 晚风不逼他，但语气开始变短。
- [ ] 宿舍里唐骁或周屿一句玩笑刺中他：你现在也学会不说了。
- [ ] 章末钩子：林亦舟删掉一条已经打好的解释。

### 第二十三章《两个群》

**目标：** 用 4XX 群和晚风生活圈群形成强对照。

**体量：** 约 10,000-12,000 字符，三小节：4XX 事项群、晚风生活圈热闹群、林亦舟在两个群之间错位。

- [ ] 4XX 群只剩文件、时间、确认、收到。
- [ ] 晚风生活圈群热闹、表情包多、临时约饭、开玩笑。
- [ ] 林亦舟在晚风群里显得拘谨，在 4XX 群里又显得缺席。
- [ ] 宿舍有人不再等他确认，直接推进。
- [ ] 章末钩子：两个群同时弹消息，他先点开晚风那边。

### 第二十四章《礼物》

**目标：** 写大学暧昧里笨拙但真实的送礼。

**体量：** 约 12,000-14,000 字符，四小节：挑礼物、被室友看见、晚风反应、钱和时间代价。

- [ ] 林亦舟想给晚风准备一个不贵但用心的礼物。
- [ ] 可选：围巾、耳机套、相机小配件、书、演出票、奶茶卡、手写卡片。
- [ ] 宿舍室友看见后有不同反应：唐骁嘴硬，周屿调侃，陆沉提醒别超预算。
- [ ] 礼物送出时甜，但也暴露林亦舟对晚风生活圈了解不够。
- [ ] 章末钩子：晚风收下，却没有马上发朋友圈。

### 第二十五章《朋友圈》

**目标：** 处理“发不发、怎么发、谁看见”的暧昧现实。

**体量：** 约 11,000-13,000 字符，三小节：晚风朋友圈、室友看见、旧暧昧点赞。

- [ ] 晚风可能发了半张礼物照片，也可能只发风景不发人。
- [ ] 林亦舟反复刷新，表现年轻人的在意。
- [ ] 旧暧昧对象点赞或评论。
- [ ] 4XX 有人看见但没人直接问。
- [ ] 章末钩子：周屿只说一句“你别把点赞当判决书”。

### 第二十六章《第一次牵手》

**目标：** 写真正亲密推进，但必须带着现实压力。

**体量：** 约 12,000-14,000 字符，四小节：约见、散步、牵手、错过寝室关键消息。

- [ ] 场景建议：晨光湖、雄关、凌空栈道、光谷返程、武生院站外。
- [ ] 牵手不能写得浮夸，要笨拙、短暂、带一点不确定。
- [ ] 晚风主动或半主动都可以，但要符合她性格。
- [ ] 同时 4XX 项目/课程/毕业活动有一个需要他确认的小节点。
- [ ] 章末钩子：他回寝时，公共桌上的纸已经被别人签完。

### 第二十七章《签完了》

**目标：** 宿舍旧生活圈对亲密推进进行反作用。

**体量：** 约 10,000-12,000 字符，三小节：回寝、签完、没人责备但位置变化明显。

- [ ] 唐骁不吵，只说“我们先签了”。
- [ ] 周屿调侃变浅，陆沉补一句“你看一下有没有问题，不改就算”。
- [ ] 林亦舟发现不是被排挤，而是别人学会不等他。
- [ ] 他想解释晚风，但没有人问。
- [ ] 章末钩子：晚风发来“你到了吗”，他看着公共桌没有立刻回。

### 第二十八章《难听的话》

**目标：** 安排一次真正难看的外部暧昧翻脸，映照林亦舟和晚风。

**体量：** 约 12,000-14,000 字符，四小节：翻脸发生、聊天截图、众人站队、晚风和林亦舟分歧。

- [ ] 外部失败样本可以是晚风生活圈情侣/暧昧对象。
- [ ] 翻脸原因要现实：冷暴力、截图、礼物钱、朋友圈公开/删除、节日没陪。
- [ ] 不写怀孕误传。
- [ ] 晚风看得更清醒，林亦舟看得更迟钝。
- [ ] 章末钩子：晚风说“我们别变成这样”，但两人都没有接下一句。

### 第二十九章《最后一次演示》

**目标：** 项目/学业线最终收束，与晚风关系同时受压。

**体量：** 约 11,000-13,000 字符，三小节：最后演示、成果归档、晚风约定被挤压。

- [ ] 四人各自完成自己的部分，配合成熟但不亲密。
- [ ] 老师认可他们“边界清楚”，但这句话听起来不全是夸奖。
- [ ] 晚风那边有一个需要林亦舟到场的节点。
- [ ] 林亦舟完成演示后赶过去，但已经晚了。
- [ ] 章末钩子：晚风没有生气，只说“我知道你会先把那边做完”。

### 第三十章《后来有了形状》

**目标：** 第六卷终章，完成“新关系有了形状，旧关系退到必要节点”的卷名回收；只把第七卷毕业走远的入口露出来，不提前完成收尾。

**体量：** 约 12,000-14,000 字符，四小节：项目归档、晚风关系谈话、4XX 各自去向、第七卷毕业入口。

- [ ] 项目文件归档，毕业后接收人仍不指定具体人，只写位置和停止条件。
- [ ] 林亦舟和晚风谈清楚：他们不是完全错过，也不是简单确定关系，而是都看见彼此的生活不是只围着对方。
- [ ] 可写一次明确但克制的亲密确认，比如牵手更久、拥抱、或一句不躲的话。
- [ ] 4XX 公共桌仍在，但四人已经各自有新安排；他们还能因为课程、项目、活动节点重聚，但不再每天自然聚在一起。
- [ ] 第七卷入口出现：毕业活动、去向表、实习/工作安排、论文/答辩、学校固定必到节点；不写最终走散，只把走散机制建立起来。

## 第七卷边界

第七卷才处理真正收尾：

- 毕业、实习、工作、论文、答辩、去向表让所有关系自然变远。
- 宿舍四人不再靠日常黏在一起，只靠班会、毕业活动、答辩、拍照、离校手续、最后一个新年等学校固定必到节点重聚。
- 晚风线也不能在第六卷彻底定死。第七卷处理它是否继续、淡化、错过或留下不完全确定的余味。
- 第七卷的情绪重点是：不是大吵后散，而是每个人都有事要忙，关系被现实慢慢挪远。

## 必须避免

- 不要把第六卷写成“晚风拯救林亦舟”。
- 不要让晚风生活圈抢走宿舍主线。
- 不要把夏知微/沈嘉禾写成强行三角恋。
- 不要把唐骁、周屿、陆沉写成只会吃醋或只会阻止主角恋爱。
- 不要再用表格说明规则代替人物冲突。
- 不要让每章都四人围桌开会。

## 验收标准

- 第六卷后半每章都有林亦舟和晚风关系的推进或反作用。
- 每章至少有一次宿舍旧生活圈对这段关系的回响。
- 至少出现一个晚风生活圈具名人物，并形成现实压力。
- 至少出现一次现实好感对照，但不扩写成独立主线。
- 至少出现一次外部失败暧昧/恋爱样本。
- 第六卷收束时，读者能明确感到：林亦舟不是多线乱跑，而是在一种新亲密关系里重新认识自己的缺席和选择。
