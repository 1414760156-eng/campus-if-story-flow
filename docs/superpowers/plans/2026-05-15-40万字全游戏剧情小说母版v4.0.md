# 40万字全游戏剧情小说母版 v4.0 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use $superpower-subagents (recommended) or $superpower-executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking via update_plan.

**Goal:** 建立并持续写作不少于 40 万字的《武生院沉浸版》全游戏剧情小说母版，为后续所有支线、选项、回流和互动节点提供连续叙事依据。

**Architecture:** 采用分卷 Markdown 母版。先写总控与进度表，再按序幕/第一幕、第二幕、第三幕、第四幕、第五幕、第六幕、终章、支线插章、互动拆分索引逐卷推进。每卷既是小说正文，又保留可被后续游戏化拆分的场景锚点。

**Tech Stack:** Markdown 文档、PowerShell 文件检查、`rg` 关键字验收、`chinese-novelist` 小说化原则、`planning-with-files` 进度记录。

**Writing Priority:** 小说母版高于互动系统。先写成能独立成立、具有悬念和文学吸引力的长篇小说，再从正文中拆分互动节点；不得为了方便拆节点而牺牲章节节奏、人物复杂度和现实细节。
