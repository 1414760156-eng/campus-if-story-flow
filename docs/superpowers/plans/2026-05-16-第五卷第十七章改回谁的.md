# 第五卷第十七章《改回谁的》执行计划

## 技能流程

- 使用 `superpower-brainstorming` 明确本章冲突：空表被二次润色后，事实变轻。
- 使用 `planning-with-files` 读取母版目录、进度和第五卷尾章，保持断点连续。
- 使用 `superpower-writing-plans` 形成本章执行顺序。
- 使用 `chinese-novelist` 原则写正文：展示而非讲述，人物通过动作、物件和对话推动。
- 使用 `superpower-verification` 在完成后运行统计和文本扫描。
