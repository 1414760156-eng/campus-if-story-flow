# 全卷游戏剧情连续性二轮优化计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use $superpower-subagents (recommended) or $superpower-executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking via update_plan.

**Goal:** 在不推翻现有 1-7 卷小说母版的前提下，把全卷整理成可供互动剧情游戏开发使用的时期外流点母版，确保主线锁点不漂移，同时让玩家在少数关键时期选择林亦舟靠近的人或生活方向，产生更多事件、代价和毕业结局差异。

**Architecture:** 保留母版正文作为剧情源，不按独立成稿逻辑压缩，也不优先开发独立支线外传。二轮优化重点不是把每个事件都拆成分支树，而是提取第三幕、第四幕、第五幕的少数时期外流点，并为每条路线补核心事件链、代价节点和毕业结算入口。未被选择的人仍按母版主线生活，只改变与林亦舟的交集频率、关系温度和结算文本。

**Tech Stack:** Markdown 正文、`rg` 定向扫描、`count_v4.py` 字数统计、人工 `chinese-novelist` 审校、互动节点结构化表。
