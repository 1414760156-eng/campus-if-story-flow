# 第四卷第四十八章《联系人》执行计划

## 目标

续写第四十八章，承接父亲电话和暑假去向登记，把第四卷后段的“本人版本/本人确认”从课堂材料推到家庭联系人、生活费、暑假安排和宿舍桌面收拾，为第四卷最终收束做准备。

## 步骤

1. 读取断点和进度文件，确认第四十七章结尾停在林亦舟接通父亲电话。
2. 写青枫居阳台父亲电话：只用具体话语和动作呈现家庭压力。
3. 写青枫居公共桌整理：四人各自确认暑假、课程、主持、勤工和个人材料。
4. 写白天生活碎片：快递站、西园餐厅、阳光书屋、图书馆西侧、东区商业街余音，补生活感但不抢主线。
5. 写夜里回收：去向登记完成，联系人顺序不改，学习委员通知教学楼纸质签字。
6. 更新进度表、v4.0 progress 和根目录 progress。
7. 运行字数统计、禁用表达扫描、明德楼误用扫描、交通误写扫描和章节顺序扫描。
