# 5A 回流小段扩写 v2.2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use $superpower-executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking via update_plan.

**Goal:** 将 5A 的 4 个回流点从“一次点击回主轴”扩写成至少两拍的小段剧情，降低刻意感。

**Architecture:** 继续使用 `playtest/5a-full-data.js` 的静态 chain。每个回流段在原回环节点后插入一个延展节点，主轴正常选项继续显式解锁回流后的主轴节点，从而跳过回流小段。

**Tech Stack:** Plain JavaScript data module, Node.js tests, static HTML playtest.
