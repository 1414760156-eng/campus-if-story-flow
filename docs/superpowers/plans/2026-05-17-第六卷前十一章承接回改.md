# 第六卷前十一章承接回改计划

## 目标

暂停第六卷后续，不写第十二章。只回改《第六卷 后来有了形状》前十一章，让它明确承接第五卷“站队、缺席、私下吐槽、示威小动作、外部补位”的后果。

## 回改原则

- 不新增独立大事件，不推翻既有项目实训、确认页、维护到毕业主线。
- 四人不再默认天天一起行动；同寝但不同场，公共桌只在必要时重新把人拉回来。
- 第五卷留下的词要继续疼：`本人确认`、`四人确认`、`最终确认`、`先用`、`备注`、`可用`、`空着`。
- 每一次看似正确的分工都要有代价：有人觉得被甩开，有人觉得被继续拖住，有人把沉默理解成站队。
- 晚风和支线人物只能补上林亦舟生活里的空缺，不能替代宿舍主线；亲密推进必须带来时间、精力、寝室缺席和后续误读。
