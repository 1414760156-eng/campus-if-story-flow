# 第五卷第四十八章《预览图》执行计划

## 写作步骤

1. 从青枫居 4XX 公共桌接上第四十七章末尾，四人盯着许棠发来的旧导语预览图。
2. 让唐骁追问预览图流向，陆沉拆出需要检查的五个位置。
3. 通过许棠消息说明排版模板和分享摘要的现实问题，不把矛盾写成抽象宣传风险。
4. 写秦越、姜宁、周屿之间的二次拉扯：现场感、漂亮话、本人边界。
5. 插入夏知微和晚风的短消息镜像，落到照片说明和抖音转发标题。
6. 让 4XX 在公共桌上改出新版导语和转发语，保持事实口径和画面感。
7. 章末让旧预览截图先于正式发布进入活动群，形成下一章钩子。
