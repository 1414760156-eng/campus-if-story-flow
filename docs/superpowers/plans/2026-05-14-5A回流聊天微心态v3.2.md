# 5A 回流聊天微心态 v3.2 执行计划

## 目标

把 v3.1 的湖边晚风/夏知微回流、食堂侧门夏知微回流和许澈外流相遇点扩写为可读、可选、可回响的微心态聊天段。新增内容只增加沉浸感和可操作心态，不改变 5A 主线、不新增外流点、不把玩家拉回旧多线混乱。

## 执行步骤

1. 更新测试
   - 标题断言升级到 `v3.2`。
   - 节点数、选项数、心理级选项数阈值升级。
   - 更新湖边晚风、湖边夏知微、食堂侧门夏知微、5X 许澈路径。
   - 新增聊天节点断言：必须有命名角色、聊天主题、至少两个心理级选项，且不得解锁旧线或新增外流。

2. 更新 `playtest/5a-full-data.js`
   - 把湖边回流改成“先选晚风/夏知微，再选是否聊刚发生的宿舍事”的两层微分支。
   - 在湖边晚风路径插入 `ACT5_5A01W_WANFENG_CHAT_CHOICE`、`ACT5_5A01W2_WANFENG_DORM`、`ACT5_5A01W3_WANFENG_DORM_REPLY`、`ACT5_5A01W4_WANFENG_LIGHT`、`ACT5_5A01W5_WANFENG_LIGHT_BACK`。
   - 在湖边夏知微路径插入 `ACT5_5A01C3_LAKESIDE_RECENT` 和 `ACT5_5A01C4_LAKESIDE_FUTURE`。
   - 在食堂侧门夏知微路径插入 `ACT5_5A16D2_SIDE_DOOR_FUTURE`。
   - 在 5X 许澈路径插入 `ACT5_5X02A_XUCHE_RECENT`、`ACT5_5X02B_XUCHE_FUTURE`、`ACT5_5X03A_XUCHE_EMPTY_ECHO`。
   - 更新数据标题到 v3.2。

3. 同步试玩入口和文档
   - 更新 `playtest/5a-full-playtest.html` 标题。
   - 更新 `docs/5A节点验收表_v2.md` 的版本、路径和统计。
   - 更新 `1-6幕剧情优化稿.md` 的 v3.2 规则说明。
   - 更新 `沉浸式剧情互动系统_v2.json` 的 v3.2 结构记录。

4. 运行验证
   - `node playtest/5a-full-data.test.js`
   - `node playtest/app.smoke.test.js`
   - `node playtest/engine.test.js`
   - `node --check playtest/5a-full-data.js`
   - `node --check playtest/app.js`
   - `node --check playtest/5a-data.js`
   - JSON 解析检查。
   - 统计节点、选项、心理级选项、回流节点和外流节点。

5. 更新 planning 文件
   - 在 `findings.md` 写入 v3.2 叙事发现。
   - 在 `progress.md` 写入本轮 skill 检查、实现结果和验证证据。
   - 在 `task_plan.md` 写入 v3.2 落地状态。
