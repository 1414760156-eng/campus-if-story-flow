# 全卷环境描写功能化回改方案

> **For agentic workers:** REQUIRED SUB-SKILL: Use $superpower-subagents (recommended) or $superpower-executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking via update_plan.

**Goal:** 将《武生院沉浸版剧情小说母版 v4.0》第一至第六卷中重复、无功能的环境描写改成服务冲突、动作、关系和后果的小说式叙事。

**Architecture:** 先不大规模重写主线，只做“环境功能化”专项回改。按问题严重程度分层推进：第六卷后半优先，第六卷前半和第五卷次之，第四卷重点回改，第三卷中等回改，第一二卷轻修。每一轮都先改章首和高频无功能转场，再运行扫描验证。

**Tech Stack:** Markdown 正文文件、`rg` 扫描、`count_v4.py` 字数统计、`chinese-novelist` 叙事原则、现有 `progress.md` / `findings.md` / `task_plan.md` 记录体系。

---

## 一、总判断

这次回改不是“把门缝换成窗户”“把手机震换成微信响”。问题根源是叙事习惯：正文常用环境声作为开场、转场和沉默填补，导致人物还没行动，文本先铺了早晨、走廊、风、阿姨、水声、门缝、手机震动。

按 `chinese-novelist` 标准，环境描写只有在承担以下功能时才保留：

- 制造阻碍：排队、门禁、机房不开门、地铁换乘、打印店拥堵。
- 暴露关系：谁不再叫谁、谁不再等谁、谁不再留座、谁把消息发群里而不私聊。
- 承载后果：饭凉了不是气氛，而是有人没吃饭被看见；文件湿了不是天气，而是材料作废。
- 提供选择：地点不是风景，而是生活方式入口，决定人物去向和关系圈。

其余环境描写默认删掉，或改成动作/对话/具体消息直接进入事件。

## 二、统一回改规则

### 规则 1：章首优先事件直入

禁止把大多数章节写成：

> 第二天早上，青枫居四楼……

优先改为：

> 唐骁把演示顺序发进群里时，林亦舟还没回 4XX。

章首必须尽快回答一个问题：今天谁被迫做选择，谁没被等，谁的责任被推到台面上。

### 规则 2：环境必须被人物使用

弱写法：

> 四张桌子在清晨里各有各的乱法。

强写法：

> 周屿翻主持稿时把唐骁的题库纸压住，唐骁抽了两次没抽出来，只说：“你这张嘴能不能别压我二级？”

桌面、衣服、外设、小票、便签、寝室座位只能通过动作、借用、遮掩、碰撞、争执、迟到代价自然露出。

### 规则 3：沉默不能总靠声响补

手机震、门响、风吹、脚步声不能反复替代关系压力。

关系紧张时优先写“取消的动作”：

- 本来要叫醒他，最后只在群里发一句。
- 本来顺手递饭，最后把袋子放到公共桌另一头。
- 本来等他一起走，最后把“已到”发给别人。
- 本来解释，最后把手机扣下。

### 规则 4：NPC 不能只做广播

宿管阿姨、食堂阿姨、打印店老板、快递站工作人员可以保留，但必须有功能：

- 截住人，迫使选择。
- 看见四人是否同进同出，反衬关系变化。
- 一句生活化话刺中人物处境。
- 让规则落地，比如夜归登记、门禁名单、外卖拿错、快递取件。

如果只是“阿姨喊别堵门”“阿姨喊上楼轻点”，删。

### 规则 5：时间流逝不能靠日期堆叠

“第二天早上 / 周一上午 / 晚上九点”要被课程、考试、项目、社团和关系后果替代：

- 四级报名表推到群顶。
- 程序设计实验换到八号教学楼机房。
- 普通话测试训练册被周屿压在主持稿下。
- 期末周座位表、补考名单、留校表和回家票互相挤压。

## 三、回改优先级

### 第一优先级：第六卷第十七至第十九章

原因：刚写完，问题最明显，且第六卷后半是晚风核心关系成型期，不能再用环境模板削弱张力。

重点：

- 第十七章《好感对照》开场必须从沈嘉禾的现实约定切入，不再从门缝、拖把水声、早晨醒来切入。
- 第十八章《难看的喜欢》如果保留西园餐厅，必须让排队、剩菜、加蛋、付款、没吃饭成为情绪和时间代价，不是生活背景。
- 第十九章《演示当天》必须从演示提前、唐骁发顺序、4XX 群短句、晚风生活圈同日事件切入，不写“第二天最早响的是走廊里的门”。

目标效果：

- 晚风线的甜与代价同场出现。
- 4XX 退位通过“不等、不叫、不追问、只发事项”呈现。
- 环境只作为选择现场，不作为气氛铺垫。

### 第二优先级：第六卷第一至第十六章

原因：第六卷整体要承接第五卷站队后果，当前仍有大量风、走廊、手机震、早晨模板，会把关系窄化写软。

重点：

- 第一章去除“门缝被风吹响”式开场，改成方向表、第一格、谁先签、谁没说话。
- 第三章返校不靠箱轮声堆气氛，改成开学后旧关系没有恢复原样。
- 第十五章《没叫他》保留“不叫”的动作，删掉无功能醒来铺垫。
- 第十六章《别人也会等》保留晚风等待代价，删掉过多站外风和手机震转场。

目标效果：

- 第六卷不再像“项目材料日常”，而是每一章都让某段关系更窄、更清楚或更疼。

### 第三优先级：第五卷全卷

原因：第五卷是矛盾爆发卷，最不能被早晨、走廊、门缝、阿姨喊稀释。它已经有 100 章，越长越需要强节奏。

分批：

- 第1-20章：本人确认、空栏、账号、照片、改稿。章首改成表格、截图、谁没来、谁不签。
- 第21-40章：旧袋、转交、删图、名单。章首改成责任动作和外部误读。
- 第41-60章：旧资料、模板、节目继续。章首改成旧句被放回、谁先用、谁沉默。
- 第61-80章：二维码、退群、公用电脑。章首改成截图、撤回、小群消息和“谁方便”。
- 第81-100章：先用、可用、摘要、发布前确认。章首改成词义滑移和四人分流。

目标效果：

- 第五卷从“材料追查流水”变成“每一步都有人站队、误读、蛐蛐、示威、拆伙”。
- 宿舍关系裂痕必须在动作里出现：不帮、不叫、不等、不转交、不解释。

### 第四优先级：第四卷

原因：第四卷问题仅次于第五六卷。暑假、补考、期末、家庭线本来应该有很强压力，但现在大量时间、阿姨、走廊、手机震把压力写软。

重点：

- 暑假空宿舍不能只写空，要写谁先离开、谁留下、谁的东西没带走、谁没被告知。
- 补考和期末周不能只写早起和走廊，要写教室分散、课程表压力、复习资源、谁帮谁占座、谁没叫谁。
- 家庭线不能只靠手机震和母亲消息，要让生活费、快递、生活用品、父亲转账、母亲忘买东西这些变量落地。
- 宿管阿姨只保留能改变行动或刺中关系的句子。

目标效果：

- 第四卷成为第五卷爆发前的压力蓄水层，而不是暑假日常集。

### 第五优先级：第三卷

原因：第三卷是问题开始出现的地方。社团、主持、招新、早八、课表展开以后，日常写法开始顺手。

重点：

- 第一章“四张桌子在清晨里各有各的乱法”必须动作化。
- 早八和水房不能反复作为开场，要改成课程表、谁迟到、谁帮谁带早餐、谁没等谁。
- 招新和主持线要更早把周屿的外部关系、秦越、夏知微等支线人物带入动作。

目标效果：

- 第三卷保留热闹感，但不让热闹变成环境堆叠。

### 第六优先级：第一、二卷轻修

原因：第一卷承担入学空间建立，第二卷承担军训纪律压力，环境本身有功能，不适合过度删除。

第一卷只修：

- 个别醒来开场。
- 静态床位/桌面介绍。
- 抽象总结句，比如“生活开始落灰”一类，如不通过动作承接则改弱。

第二卷只修：

- 章首天气/集合背景，优先改成纪律动作。
- 手机袋事件保留环境压力，因为它本身是主线事件。
- 汇演、总结会保留集体场面，但减少无功能走廊声和天色铺垫。

## 四、每章回改工作流

每章按以下顺序处理：

- [ ] 读章首前三段，判断是否在事件前铺了无功能环境。
- [ ] 标出所有“早上、清晨、门缝、走廊、阿姨喊、手机震、风、水声、睁眼”。
- [ ] 对每处环境问四个问题：是否制造阻碍、暴露关系、承载后果、提供选择。
- [ ] 四个问题都否，删除。
- [ ] 有一个是，改成动作化表达。
- [ ] 检查章尾是否只靠风声、手机震、门响收束；如果是，改成人物选择或未完成动作。
- [ ] 检查是否出现静态介绍式段落；如果是，改成碰撞、借用、遮掩、争执、迟到代价。

### 质检标准

- 命中不要求归零，因为有些环境有功能；但每个保留命中都必须能解释用途。
- 每章章首必须尽量在前三句内出现人物动作、任务压力、关系后果或具体消息。
- 第五、六卷不能再出现“先铺环境，再慢慢进事”的章节。
- 宿管阿姨、食堂阿姨、快递站、打印店、走廊、水房等生活血肉保留，但必须参与事件。
- 修改后不得破坏主线锁点、宿舍四人优先、第五幕多线收束、5X 唯一外流、终章不推翻第六幕硬结局。

## 六、建议执行方式

先做一轮小样，不直接全卷批量改：

1. 先回改第六卷第十七至第十九章。
2. 给用户看回改前后差异。
3. 用户确认口味后，再按同标准推第六卷前十六章。
4. 第六卷完成后，再进第五卷。
5. 第五、六卷稳定后，再处理第四、三、二、一卷。

这样最稳，因为第六卷问题最贴近当前争议，回改样本能最快确定“删多少、留多少、怎么保留大学生活血肉”。
