```markdown
### 5A 宿舍修补线样板

5A 不是为了制造狗血，而是把“住在一起”这件事变得具体。大学宿舍里真正磨人的是小事：空调温度、外放声音、桌面垃圾、借走的充电器、夜里回来的门声、公共区到底谁收。玩家前四幕如果一直和稀泥，这里会看见和稀泥的代价；如果一直只顾自己，这里会看见低温关系的后果。

#### 5A01 返寝冷场

场景：宿舍楼夜间，林亦舟从走廊推门进来。公共桌面上有外卖袋，空调遥控器被压在资料下面，唐骁戴着耳机但音乐漏出来，周屿床帘拉着，陆沉把洗漱盆从门边挪开。

玩家选项：

| 选项 | 即时反馈 | 状态变化 |
|---|---|---|
| 先把桌面收掉 | 陆沉看一眼，没有说谢谢，但把另一个空袋也拿起来 | `宿舍互信 +3`、`自我压抑 +1` |
| 直接提醒唐骁声音太大 | 唐骁摘下一边耳机，说“你早说啊”，语气不重但脸冷 | `唐骁防御 +4`、`宿舍火药味 +3` |
| 问周屿是不是还没睡 | 床帘里停了两秒，周屿说“没有，你们继续” | `周屿委屈 +3`、解锁 `5A02_周屿旧账` |
| 先去走廊缓一缓 | 门合上后，里面短暂安静，随后群聊弹出新消息 | `回避值 +1`、进入 `5A02_群聊旧账` |

#### 5A02 旧账浮出

场景：宿舍群聊与门口走廊交替。旧账不是一次爆炸，而是一条接一条消息把人逼到台面上。

玩家选项：

| 选项 | 即时反馈 | 状态变化 |
|---|---|---|
| 翻聊天记录确认谁借了什么 | 找到唐骁借充电器、自己忘还热水卡、周屿替大家交过一次水费 | `事实清单 +1`、`自责 +2` |
| 线下把话摊开 | 唐骁说“那就现在说”，周屿终于拉开床帘 | `宿舍火药味 +5`、解锁 `5A04_规则重谈` |
| 私聊陆沉问真实情况 | 陆沉回得很慢：“不是一天两天。” | `陆沉表达度 +3`、解锁 `5A03_走廊倾听` |
| 装作没看见 | 群聊停了，但第二天没人一起去食堂 | `回避值 +1`、`宿舍互信 -4` |

#### 5A03 陆沉走廊倾听

场景：水房边，灯管有轻微电流声。陆沉洗杯子，林亦舟站在门口。

玩家选项：

| 选项 | 即时反馈 | 状态变化 |
|---|---|---|
| 追问“你到底介意什么” | 陆沉说“我不是不介意，我只是懒得吵。” | `陆沉表达度 +5`、`林亦舟自省 +3` |
| 承认自己以前没注意 | 陆沉擦干杯子，说“你不是没注意，你是以为会过去。” | `宿舍互信 +4`、`回避值 -1` |
| 替唐骁解释他压力大 | 陆沉笑了一下：“你又开始替别人说话。” | `和稀泥标签 +1`、`陆沉距离 +3` |
| 让陆沉自己在会上说 | 陆沉沉默很久，说“那你别替我圆。” | 解锁 `5A04_陆沉直说` |

#### 5A04 规则重谈

场景：宿舍四人围着公共桌面，手机倒扣在桌上计时十分钟。规则不是班规，是能不能继续住下去。

玩家选项：

| 选项 | 即时反馈 | 状态变化 |
|---|---|---|
| 先把自己的问题说出来 | 周屿表情松一点，唐骁没有立刻顶回来 | `宿舍互信 +6`、`自尊防御 -2` |
| 站周屿，要求唐骁改作息 | 唐骁笑了一声：“行，就我一个人有问题。” | `周屿信任 +5`、`唐骁防御 +8` |
| 站唐骁，说大家都别太敏感 | 周屿把床帘重新拉上 | `唐骁亲近 +4`、`周屿信任 -7` |
| 要求白纸黑字写规则 | 陆沉拿笔，周屿补水费，唐骁写外放时间 | `规则可信度 +6`、`宿舍火药味 -3` |
| 提前结束，说别为了小事闹 | 群聊当晚改名，没人再说话 | `回避值 +2`、可能进入 `强冲突` |

#### 5A05 争吵后的第一顿饭

场景：第二天食堂窗口或宿舍门口。关系修不修，不看谁说得漂亮，看还会不会一起端着饭坐下。

玩家选项：

| 选项 | 即时反馈 | 状态变化 |
|---|---|---|
| 主动喊四个人一起吃 | 唐骁犹豫，周屿看陆沉，陆沉先拿饭卡 | `修补温度 +5` |
| 只喊昨晚被你站的一方 | 另一方在朋友圈发了一张空桌照片 | `公开站队 +1` |
| 买一份饭放在桌上不说话 | 周屿或唐骁会根据关系值决定是否收下 | `低温修补 +3` 或 `低温共处 +1` |
| 各吃各的 | 本线进入低温共处，冲突不爆但关系变薄 | `宿舍互信 -3`、`低温共处 +1` |

#### 5A06 期末前夜复发

场景：期末前夜，资料散在桌上，手机震动不断，外放声又漏出来。此前写下的规则第一次被现实压力检验。

玩家选项：

| 选项 | 即时反馈 | 状态变化 |
|---|---|---|
| 按新规则提醒 | 对方关声音或收桌面，规则真正生效 | `规则可信度 +5`、`第六幕宿舍稳定` |
| 忍到天亮 | 第二天复习效率下降，没人知道你为什么烦 | `自我压抑 +3`、`学业压力 +2` |
| 发群消息提醒 | 文字很冷，关系温度取决于之前修补结果 | `宿舍火药味 +2` 或 `规则可信度 +2` |
| 当面爆发 | 若强冲突条件满足，出现摔门、站队、短期冷战 | `短期决裂 +1` 或 `公开站队 +1` |

#### 5A99 宿舍线输出页

输出标签：

| 条件 | 标签 | 第六幕回响 |
|---|---|---|
| `宿舍互信 >= 60` 且 `规则可信度 >= 5` | `宿舍修补` | 共同复习、毕业互拍、行李整理时有自然互动 |
| `宿舍互信 35-59` | `低温共处` | 仍同住，但饭桌和毕业活动对白减少 |
| `公开站队 >= 1` | `公开站队` | 周屿或唐骁一侧升温，另一侧降温 |
| `短期决裂 >= 1` | `短期决裂` | 第六幕毕业活动需要最低限度回收，否则终章只保留淡淡一笔 |
```
