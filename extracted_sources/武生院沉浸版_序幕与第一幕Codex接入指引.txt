# 武生院沉浸版｜序幕与第一幕 Codex 接入指引（回改版）

## 1. 接入目标

序幕与第一幕负责完成三件事：

1. 建立玩家从“收到录取通知书”到“真正住进武生院”的进入感。
2. 提前埋下家庭、宿舍、室友、晚风、军训、社团招新的轻量入口。
3. 为第二幕军训和第三幕社团分流提供稳定状态字段。

玩家可见文本禁止出现“主线 / 副线 / 权重 / flag / 触发 / Codex / route”等开发层语言。

## 5. 进入第二幕条件

第一幕结束后进入第二幕：军训服发放与军训开始。

第二幕可读取：

```js
const act2Input = {
  dorm_bond,
  zhou_yu_bond,
  tang_xiao_trust,
  lu_chen_trust,
  self_planning,
  boundary_awareness,
  family_departure_seen,
  first_night_wanfeng_contact,
  knows_military_uniform_at_library_warehouse
}
```

重点承接：

- 军训服统一到图书馆仓库领取。
- 日常军训地点优先晨光体育场。
- 军训汇演地点为健源体育场。
- 宿舍关系不突然熟透，只从第一晚和宿舍整理自然推进。

## 6. 去剧透与过滤规则

序幕与第一幕不显示后续人物命运，不提示后续分支。

正确写法：

> “这些名字正在从屏幕里走出来。”

错误写法：

> “如果你后面走陆沉线……”

前端只展示当前节点文本。所有状态只在内部记录，不进入玩家文案。
