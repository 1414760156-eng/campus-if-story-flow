# 武生院沉浸版大学成长 H5｜第三幕 Codex 接入指引

## 1. 第三幕定位
第三幕为“大一第一学年，热闹开始分流”。它负责把第一幕入校、第二幕军训之后的大学生活铺开，并为第四幕暑假入口提供状态。

运行原则：玩家文案只写生活化剧情，不显示“主线 / 副线 / 权重 / flag / 触发”等开发词；分支回响只显示玩家已经发生过的内容。

## 4. 选择写入建议

- `media`：明德楼三楼、夏知微、记录意识、素材整理、守机位、备份素材
- `academic_expression`：阳光书屋、沈嘉禾、资料整理、小组作业、复习框架
- `performance_social`：周屿、九龙壁舞台、新生串场、展示表达、普通成长
- `activity_department`：秦越、明德楼一楼学工、物料、活动资源、责任边界
- `work_study`：唐骁、陆沉、勤工公告、规则、光谷现实入口
- `gaming_social`：周屿、后街夜市、宿舍群、晚风夜聊、普通生活

3-08 允许试两个方向；3-09/3-10 决定哪个方向更常出现、哪个方向保持联系、是否过载、是否没有固定组织。

## 6. 去剧透显示规则
- 玩家文本不列举所有可能经历。
- 条件回响只显示当前状态命中的一条或数条。
- 不在玩家文案中出现“如果你走过、主线、副线、权重、flag、触发、Codex”等词。
- 3-15 至 3-18 所有新生参与都必须是“边缘参与 / 一小段 / 跟着学长学姐 / 完成小责任”，不能显示为主导晚会或解决复杂问题。
